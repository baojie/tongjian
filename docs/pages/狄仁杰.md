# [[狄仁杰]]

[[狄仁杰]]（630—700年），字怀英，并州太原人，[[唐高宗]]及[[武后]]时期著名贤臣。以正直不阿、敢于直谏著称，历任大理丞、宁州刺史、地官侍郎等职，在[[武则天]]掌权期间多次维护法律公正，被后人称为"梁公"。

## 执法公正

高宗时期，"大理丞太原[[狄仁杰]]奏：'二人罪不当死。'上曰：'善才等斫陵柏，我不杀则为不孝。'仁杰固执不已"（202-079），力争依法处置，拒绝皇帝以孝道名义枉杀，体现其司法精神。

## 弹劾权贵

武后时期，"侍御史[[狄仁杰]]劾奏弘机导上为奢泰，弘机坐免官。左司郎中王本立恃恩用事，朝廷畏之。仁杰奏其奸，请加显戮"（202-119），连续弹劾权贵，不畏强权。

## 宁州德政

狄仁杰为宁州刺史时，"耆老歌刺史德美"（203-144），百姓自发歌颂其政绩，被御史发现后上报朝廷。

## 历史评价

[[司马光]]通过狄仁杰的记录，展示了唐代铁面御史的精神风貌。狄仁杰在[[武则天]]专权时期能够坚持法制、为民请命，是盛唐时代官僚制度中法治精神的代表。
