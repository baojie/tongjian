# [[赵国]]

[[赵国]]，嬴姓赵氏，战国七雄之一，由[[三家分晋]]而来，都邯郸（今河北邯郸）。赵国在战国中期一度强大，以[[赵武灵王]]的军事改革著称，后因[[长平之战]]惨败而衰落，最终被[[秦国]]所灭。

## 立国根基

赵国之祖[[赵简子]]、[[赵襄子]]在春秋末年积累力量，[[智伯]]水灌晋阳，邯郸仓库虽实，"浚民之膏泽以实之，又因而杀之，其谁与我！其晋阳乎"（001-010），[[赵襄子]]坚守晋阳，联合魏、韩反灭智伯，建立赵国。

## 胡服骑射

[[赵武灵王]]是赵国最杰出的君主，"赵武灵王北略中山之地……与肥义谋胡服骑射以教百官"（003-054），打破礼制束缚，引入北方骑射战法，使赵国军事实力大幅提升。赵武灵王甚至在诸侯争相称王时独持清醒："无其实，敢处其名乎？"（002-123）。

## 长平之战与衰亡

[[长平之战]]（前260年）中，赵王中反间计，以[[赵括]]代替老将[[廉颇]]，白起坑杀赵军四十余万，赵国从此一蹶不振。此后虽有[[李牧]]等名将苦撑，终难挽回，前228年都城邯郸被秦攻破，赵国灭亡。
