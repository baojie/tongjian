# [[前316年]]

本年事见于《[[资治通鉴概述|资治通鉴]]》**[[第003卷]]**（周纪三）。

公元前316年，巴蜀两国相互攻伐，俱向秦国求援。秦惠文王本欲伐韩，廷臣张仪与司马错展开"伐韩"与"伐蜀"之争。司马错以富国强兵之论说服惠文王，秦军当年即出兵灭蜀，将巴蜀纳入秦国版图。蜀地地广物博，秦国由此获得稳固的大后方，实力急剧增强。

## 司马错伐蜀之议

张仪主张伐韩，司马错力主先取巴蜀：

> 巴、蜀相攻击，俱告急于秦。秦惠王欲伐蜀，以为道险狭难至，而韩又来侵，犹豫未能决。司马错请伐蜀。张仪曰："不如伐韩。"……司马错曰："……得其地足以广国，取其财足以富民，缮兵不伤众而彼已服焉。拔一国而天下不以为暴，利尽西海而天下不以为贪，是我一举而名实附也。"王从错计，起兵伐蜀。十月取之，贬蜀王，更号为侯，而使陈庄相蜀。
> （003-016）

## 历史意义

秦灭巴蜀是秦国扩张史上的重要转折点。蜀地成为秦国东出中原的粮草基地和战略后方，日后秦始皇统一六国时，巴蜀的粮食供应发挥了关键作用。司马错关于"先取蜀以充国力"的战略主张，也成为中国历史上著名的军事战略思想。
