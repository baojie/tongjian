# [[项燕]]

[[项燕]]（？—前224年），楚国名将，是楚国最后抵抗[[秦国]]灭国的主将，也是[[项梁]]之父、[[项羽]]之祖父。

## 抵御秦军

秦王政二十三年（前224年），[[王翦]]率六十万大军伐楚，"追之，令壮士击，大破楚师，至蕲南，杀其将军项燕，楚师遂败走"（007-013）。

## 陈胜的旗号

[[陈胜]]起义时，以项燕为旗号之一，"乃诈称公子扶苏、项燕，为坛而盟，称大楚"（007-084），可见项燕在楚人心中的号召力。

## 历史影响

项燕虽败，其家族将复仇理念传承下去。儿子[[项梁]]和孙子[[项羽]]后来领导了秦末反秦运动，"楚虽三户，亡秦必楚"的谶语通过项氏家族得到了实现（007-102）。
