# [[彭越]]

[[彭越]]（？—前196年），昌邑人，楚汉之际著名将领，善于游击扰敌、断绝粮道，封梁王。他是帮助[[汉高祖刘邦|刘邦]]击败[[项羽]]的关键力量之一，汉朝建立后以谋反罪被诛，成为刘邦诛杀异姓诸王的重要一例。

## 起于渔间

彭越出身草莽，"【彭越】，昌邑人，常渔巨野泽中，为群盗"（008-054），与[[汉高祖刘邦|刘邦]]初遇时便以部众相从，成为汉军早期盟友。

## 游击断粮

彭越的核心战略价值在于以游击战扰乱楚军后方，断绝粮道。当[[汉高祖刘邦|汉王]]在彭城大败后，"【彭越】皆亡其所下城，独将其兵北居河上，常往来为汉游兵击楚，绝其后粮"（010-021），以灵活机动的游击战术持续消耗项羽。[[张良]]在分析汉军可以依靠的力量时，特别提出"【彭越】与齐反梁地；此两人可急使"（009-042），将彭越列为与英布并列的关键盟友。

## 封王据梁

彭越攻下大量梁地城池，"【彭越】将其兵三万馀人归汉。[[汉高祖刘邦|汉王]]曰：'彭将军收魏地得十馀城……'"（009-039），论功封梁王。楚汉战事关键阶段，"【彭越】攻徇梁地，下睢阳、外黄等十七城"（010-026），不断扩大在梁地的控制。

## 被杀与影响

汉朝建立后，彭越被以谋反罪诛杀，其被杀直接引发了英布的叛乱——"往年杀彭越，前年杀韩信；此三人者，同功一体之人也，自疑祸及身，故反耳"，揭示了汉初异姓诸王覆灭的连锁效应。

## 历史评价

[[司马光]]通过彭越的经历，展示了楚汉相争中游击战术的重要意义，以及汉初功臣难逃诛杀命运的历史悲剧。
