# [[颜真卿]]

[[颜真卿]]（709—785年），字清臣，琅琊临沂人，唐朝著名书法家与忠义名臣。[[安史之乱]]爆发前，"平原太守[[颜真卿]]知禄山且反，因霖雨，完城浚壕，料丁壮，实仓廪。禄山以其书生，易之"（217-044），以敏锐洞察与周密准备应对叛乱，是河北诸郡中最早做好防御的官员。

## 最早起兵勤王

[[安禄山]]叛乱后，"[[颜真卿]]召募勇士，旬日至万馀人，谕以举兵讨[[安禄山]]，继以涕泣，士皆感愤"（217-050），迅速集结力量起兵讨贼，感召力强大。

## 联络义军

颜真卿"自平原遣杲卿甥卢逖潜告杲卿，欲连兵断禄山归路，以缓其西入之谋"（217-054），积极协调各地忠义力量，策划夹击叛军。

## 历史评价

[[司马光]]通过颜真卿的事迹，记录了唐朝忠义之士在[[安史之乱]]中的壮举。颜真卿以文臣之身，凭借先见之明与感召力，成为平叛的关键人物。其书法名传千古，忠义精神更被后世奉为典范，最终以殉节方式证明了士大夫的最高操守。
