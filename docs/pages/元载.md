# [[元载]]

[[元载]]（？—777年），唐代宗时期宰相。他以财政才干获宠，主持度支、盐铁等财政要务，却逐步走向结党专权之路，勾结宦官[[李辅国]]，排挤[[刘晏]]、[[颜真卿]]等正直官员，广收贿赂。代宗忍无可忍，最终将其赐死，抄家时发现胡椒八百石，其贪腐之甚令人咋舌。

## 理财获用

[[安史之乱]]后，元载以财政才能受到重用，"御史中丞【元载】为户部侍郎，充句当度支、铸钱、盐铁兼江淮转运等使。载初为度支郎中，敏悟善奏对"（222-039），迅速进入权力核心。

## 勾结宦官

元载善于攀附权贵，"【元载】希[[李辅国]]意，以罪诬之也"（222-073），借宦官之力打击政敌，"以货结内侍董秀，使主书卓英倩潜与往来，上意所属，载必先知之，承意探微，言无不中"（223-024），内外勾连，权势日盛。

## 排斥正直

[[颜真卿]]请求先谒皇陵，"【元载】不从，真卿怒曰：'朝廷岂堪相公再坏邪！'载由是衔之"（223-022），对直言之臣怀恨在心，伺机报复。又诬陷[[刘晏]]，构陷忠良，"诸军俱发；……以宗正卿李岘为黄门侍郎、同平章事。遵庆既去，【元载】权益盛"（223-024），专权日深。

## 历史评价

[[司马光]]通过元载的兴衰，揭示[[安史之乱]]后唐朝政治腐败的深层机制：宦官与奸相相互勾连，排斥忠良，蚕食国政，是中唐政局持续动荡的重要根源。
