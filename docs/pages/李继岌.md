# [[李继岌]]

[[李继岌]]（？—926年），五代后唐庄宗[[李存勖]]之子，封魏王。他以西川四面行营都统身份统帅伐蜀大军；以[[郭崇韬]]为实际军事决策者的配合关系完成灭蜀；以询问蜀将"何不降"而彰显威名；最终在班师归途中遭遇庄宗已死的政变，以兵败自杀收场。

## 以魏王充西川都统，崇韬为副

李继岌以皇子身份统帅大军，"以【魏王继岌】充西川四面行营都统，崇韬充东北面行营都招讨制置等使，军事悉以委之"（273-106），名为都统，实际军事权力在[[郭崇韬]]手中，体现了后唐庄宗以皇子居名、以大臣掌实的军事体制。

## 连克蜀地诸州，前蜀节节败退

李继岌在军事上取得了辉煌胜利，"【魏王继岌】至剑州，蜀武信节度使兼中书令王宗寿以遂、合、渝、泸、昌五州降"（274-004），随后"【魏王继岌】至绵州，蜀主命翰林学士李昊草降表"（274-008），前蜀在后唐大军压境下迅速土崩瓦解。

## 询问降将"何不拒战"

李继岌以简洁对话彰显威名，"【魏王继岌】诘之曰：'居大镇，拥强兵，何以不拒战？'对曰：'畏大王神武'"（274-017），以言简意赅的问答方式展示了后唐军威对蜀将的震慑，成为《资治通鉴》中关于伐蜀的经典细节。

## 宦官矛盾激化，郭崇韬被杀

军中暗流最终爆发，"初，帝遣宦者李从袭等从【魏王继岌】伐蜀；继岌虽为都统，军中制置补署一出郭崇韬，崇韬终日决事，将吏宾客趋走盈庭"（274-014），宦官与郭崇韬的矛盾在胜利后达到顶点，皇后教令使继岌杀崇韬，成为后唐政治崩溃的关键转折。

## 历史评价

[[司马光]]通过李继岌，展示了五代"皇子挂帅"制度的内在矛盾：以无实际权力的皇子为名义统帅，以能臣为实际决策者，看似平衡了皇权与臣权，实则在胜利后制造了权力真空和猜忌——郭崇韬被杀、继岌班师途中庄宗已死，灭蜀的功业在内部政治清洗中化为了王朝崩溃的前奏。
