# [[豆卢革]]

[[豆卢革]]（？—928年），字大用，昌黎人，五代后唐开国宰相。他以卢质推荐为由进入晋王幕府；以行台左丞相身份参与后唐建国；以正式任相巩固了新政权的文治基础；是后唐庄宗在灭梁建国过程中倚重的首席文臣之一。

## 以卢质推荐进入晋王幕府

豆卢革以举荐入仕的方式开始其在后唐的仕途，"义武节度判官【豆卢革】、河东观察判官卢程为之；王即召革、程拜行台左、右丞相，以质为礼部尚书"（272-003），在卢质固辞出相后，以义武节度判官的幕僚身份被推荐，意外获得了行台丞相的高位，完成了从藩镇幕僚到最高文职的跨越。

## 以门下侍郎入新朝正式为相

后唐建国后，豆卢革获得正式宰相职务，"以【豆卢革】为门下侍郎，卢程为中书侍郎，并同平章事；郭崇韬、张居翰为枢密使，卢质、[[冯道]]为翰林学士"（272-009），以门下侍郎同平章事成为后唐开国内阁的核心成员，是新朝文治体系建立的重要奠基者。

## 留守东京，稳定后方

豆卢革在军事决战时期承担后方稳定责任，"仍命【豆卢革】、李绍宏、张宪、王正言同守东京"（272-031），在庄宗亲征的关键时刻坐镇后方，为前线作战提供行政保障。

## 历史评价

[[司马光]]通过豆卢革，展示了五代政权建立时对名门文臣的政治需求：[[李存勖]]以沙陀军事力量打天下，但在建立正统政权时需要以汉族士大夫的参与来增强合法性。豆卢革出身昌黎豆卢氏这一唐代名门，其入相既是对个人才能的认可，更是后唐以唐朝正统继承者自居、以名门士族为装点的政治象征。
