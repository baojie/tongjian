# [[高欢]]

[[高欢]]（496—547年），字贺六浑，渤海郡蓨县（今河北景县）人，南北朝时期东魏的实际统治者，北齐的奠基人，庙号太祖，谥号神武帝。他出身贫寒，历经多次投附，最终挟天子以令诸侯，主导东魏政局近二十年。

## 草莽起家

高欢早年随杜洛周起义，后转投葛荣，再归附尔朱荣。"[[高欢]]、段荣、尉景、蔡俊先在杜洛周党中，欲图洛周，不果，逃奔葛荣，又亡归尔朱荣"（152-014），辗转多方，审时度势。

在尔朱荣麾下，高欢逐渐展露才干，"荣以[[高欢]]为前锋"（152-017），得到重用。

## 控制北魏

尔朱荣死后，高欢趁乱坐大。尔朱兆曾以"代我者，唯贺六浑耳"评价他的能力，"乃以[[高欢]]为晋州刺史"（154-050），却不料此举正是养虎为患。

高欢后扶植元善见为孝静帝，迁都邺城，建立东魏，"屯胡关大王山六旬，乃引兵东出"（155-013），以武力为后盾掌控朝政。

## 与宇文泰争雄

高欢与[[宇文泰]]长期东西对峙，两人均为北朝一代枭雄，互相牵制，形成东西魏的分裂格局。高欢死后，其子[[高洋]]废东魏建北齐，完成了高氏代魏的最后一步。

## 历史评价

[[司马光]]通过高欢的崛起，深刻分析了权臣借势篡位的历史模式。高欢一生长于用兵，善于笼络人心，但其统治的东魏实为北齐的前奏，终究未能亲手称帝，留有遗憾。
