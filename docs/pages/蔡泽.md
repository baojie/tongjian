# [[蔡泽]]

[[蔡泽]]（？—？），[[战国]]末期[[秦国]]丞相，封[[纲成君]]，以能言善辩著称。

## 劝退范雎

[[范雎]]的亲信[[郑安平]]、[[王稽]]等人犯罪，[[范雎]]处境尴尬，有心退让。蔡泽"闻之，西入秦，先使人宣言于[[范雎|应侯]]"，声称自己将取代[[范雎]]（006-003）。

蔡泽会见[[范雎]]后，以[[商鞅]]、[[白起]]、[[孙吴|吴起]]等功成身退的典故，劝[[范雎]]主动让贤；又以"进不失廉，退不失行"的道理说服[[范雎]]推荐自己。

## 继任为相

[[范雎]]将蔡泽推荐给秦王，蔡泽继任为[[秦国]]丞相，封纲成君，由此继续辅佐[[秦国]]走向统一的进程（006-003）。

## 历史评价

蔡泽是[[战国]]晚期[[秦国]]的重要政治人物，他成功地以语言艺术劝动[[范雎]]主动退位，展示了[[战国]]时期[[纵横家]]的说服之术。《[[资治通鉴概述|资治通鉴]]》以此记录了权力的有序更迭与谋士的智慧（006-003）。
