# [[韩非]]

[[韩非]]（约前280—前233），[[韩国]]诸公子，战国末期法家思想的集大成者，与[[李斯]]同师于荀子。著《韩非子》五十五篇，系统阐述"法、术、势"三位一体的统治理论，对中国政治思想影响深远。

## 思想建树

韩非继承[[商鞅]]之法、申不害之术、慎到之势，融合为完整的法家体系："见韩之削弱，数以书干韩王，王不能用"（006-104），韩非的理论本为拯救弱韩而生，却未能被韩王采纳。

[[司马光]]引扬子之评："师申、商之法，行韩非之说，憎帝王之道，以贪狼为俗"（017-005），儒家对法家"刻薄伤化"的批评也因此留存史册。

## 入秦与悲剧

秦王[[秦始皇|嬴政]]读《韩非子》，大为赞赏，发兵攻韩，迫使韩王遣韩非使秦。然而[[李斯]]嫉其才，进谗言："韩非，韩之诸公子也。今欲并诸侯，非终为韩不为秦，此人情也"（006-105），秦王未及重用，韩非被投入狱中，不久被迫自尽。

## 历史评价

扬雄曰："韩非作《说难》之书而卒死乎说难，敢问何反也？"（006-106）——韩非一生研究说服之道，却终究未能说服秦王信任自己，成为历史的深刻讽刺。[[司马光]]借此警示：才学再高，若无政治信任，终究无用武之地。
