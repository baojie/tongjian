# [[李存孝]]

[[李存孝]]（？—894年），本名安敬思，代州飞狐人，五代后唐第一猛将，[[李克用]]义子。他以三百骑伏击擒获孙揆；以五百骑独保晋阳不被汴军攻破；以功高被谗致嫌；最终以叛罪被车裂；是五代乱世中"功高不赏"政治逻辑与义父子情义纠葛的悲剧缩影。

## 以三百骑伏击擒获孙揆

李存孝以奇袭闻名，"揆发晋州，【李存孝】闻之，以三百骑伏于长子西谷中。揆建牙杖节，褒衣大盖，拥众而行。存孝突出，擒揆及赐剑"（058-054），以三百骑突袭建节出行的朝廷大将，充分体现了其迅猛的战术突破能力。

## 以五百精骑保晋阳

李存孝在晋阳危急时刻发挥关键作用，"遣李存孝将五千骑救之"（258-059），以精骑"绕汴寨呼曰：'我，沙陀之求穴者也，欲得尔肉以饱士卒，可令肥者出斗'"（258-060），以心理震慑与实际战力令汴军不敢轻进，是晋阳能够守住的关键战力。

## 功高遭谗，与李存信不睦

李存孝因功高和内部矛盾走向悲剧，"【李存孝】自谓擒孙揆功大，当镇昭义，而君立得之，愤恚不食者数日"（258-061），与李存信长期不睦，"存信有宠于克用，存孝在邢州，欲立大功以自结于克用，而存信从中谗毁之"（259-041）。

## 城陷被执，车裂而死

李存孝在邢州城破后归降，"【李存孝】登城谓[[李克用]]曰：'儿蒙王恩得富贵，苟非困于谗慝，安肯舍父子而从仇雠乎！'"（059-096），虽然表达了悔恨，仍难逃被车裂处死的命运。

## 历史评价

[[司马光]]通过李存孝，展示了五代义父子体制的根本性张力：李存孝是[[李克用]]义子集团中战功最高者，但正是这种超凡战功在义子体制下产生了难以驾驭的威胁感。谗毁可以利用，猜忌早已存在——李存孝的悲剧，是五代武将以力取天下却无法以力自保的政治困境写照。"克用语及存孝，流涕不已"（259-111），义父之哀，并不能逆转义子之死。
