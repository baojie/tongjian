# [[孙坚]]

[[孙坚]]（155—191年），字文台，吴郡富春人，东汉末年将领，[[孙策]]、[[孙权]]之父，是[[孙吴]]政权的奠基人。

## 早年功绩

[[孙坚]]以勇武著称，早年镇压山贼：吴郡司马孙坚召募精勇，得千余人，助州郡讨许生（057-035）。

## 讨伐黄巾与平定叛乱

黄巾之乱时，孙坚随左中郎将朱儁南征，"司马孙坚率众先登"，克宛城，大破黄巾（058-060）。后又参与西北平叛，建议主将拦截粮道以困贼（058-079）。

灵帝末年，长沙贼区星作乱，"以议郎孙坚为长沙太守，讨击平之，封坚乌程侯"（058-099）。

## 讨伐董卓

初平元年（190年），天下州郡起兵讨[[董卓]]，孙坚亦起兵响应。荆州刺史王睿素来轻视孙坚，孙坚以其阻碍为名杀之（059-075）。孙坚率军入洛阳，是最先进入洛阳的诸侯之一。

## 历史评价

孙坚骁勇，是东汉末年军阀割据时期最有军事才能的将领之一，《[[资治通鉴概述|资治通鉴]]》多处记载其征战经历，视其为[[孙吴]]立国的武力根基。
