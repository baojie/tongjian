# [[公孙衍]]

[[公孙衍]]，字犀首，战国时期著名纵横家，合纵运动的重要推手。与[[苏秦]]、[[张仪]]并列为战国纵横家代表，主张六国联合抵抗强秦。

## 纵横地位

"又有魏人公孙衍者，号曰犀首，亦以谈说显名。其馀苏代、苏厉、周最、楼缓之徒，纷纭遍于天下，务以辩说取悦诸侯"（003-040），公孙衍在纵横家阵营中地位仅次于苏秦、张仪。

## 孟子的批评

孟子对纵横家有精彩评论，有人问："张仪、公孙衍，岂不大丈夫哉！一怒而诸侯惧，安居而天下熄。"孟子答曰："是恶足以为大丈夫哉！君子……"（003-041），认为纵横家以权谋取悦诸侯，不过是"妾妇之道"，非真正的大丈夫。

## 五国伐秦

公孙衍推动合纵，曾联合魏、赵、韩、楚、燕五国攻秦，是[[合纵连横|合纵运动]]在苏秦之前的重要实践者，虽成效有限，但对制约秦国扩张起到一定作用。

## 历史地位

[[司马光]]记录公孙衍，意在展示战国时期"游说之风"的全貌——纵横家以一己之口影响天下格局，折射出战国政治的本质：力量与言辞的博弈。
