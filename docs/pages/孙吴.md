# [[孙吴]]

"孙吴"，是春秋时代兵家孙武（[[孙子]]）与[[战国|战国时代]]名将[[吴起]]的并称，两人分别著有《孙子兵法》与《[[吴起|吴子]]》，代表中国古代军事思想的双峰。

## 在《资治通鉴》中的运用

[[司马光]]记载[[战国|战国时代]]的军事辩论，引用"孙吴"为兵家最高权威：

> 孙吴用之，无敌于天下，岂必待附民哉！（006-004）

这是临武君与[[荀子|荀况]]辩论时所引，认为孙武、吴起的兵法追求"势利"与"变诈"，不必仁义也能称雄，而[[荀子]]则以"仁人之兵"反驳，揭示了兵家与儒家政治观的根本分歧。

## 两位宗师

- **孙武**（孙子）：春秋末期吴国军事家，《孙子兵法》传世
- **[[吴起]]**：[[战国|战国时期]]名将兼政治改革家，历仕鲁、魏、楚三国

## 历史意义

"孙吴"并称的使用，显示了两位军事家在后世评价中的并列地位。《[[资治通鉴概述|资治通鉴]]》通过辩论将其与"王道"对立，体现了[[司马光]]对纯粹功利军事思想的批判性审视（006-004）。
