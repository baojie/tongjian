# [[隋文帝]]

[[隋文帝]]（541—604年），名杨坚，隋朝开国皇帝，在位23年（581—604年）。结束了南北朝三百余年的分裂局面，统一中国，奠定了唐代盛世的制度基础。

## 建立隋朝与统一

杨坚以北周外戚身份夺权，建立隋朝，改元开皇。开皇九年（589年）灭陈，完成南北统一，是中国历史上继秦汉之后第三次大一统。

## 开皇之治

隋文帝勤于政务："文帝勤于为治，每临朝，或至日昃，五品已上，引坐论事"（193-074）。他推行均田制、府兵制，改革官制，建立三省六部，国家粮仓充盈。唐太宗后来对比道："开皇十四年大旱，隋文帝不许赈给，而令百姓就食山东，比至末年，天下储积可供五十年"（192-082）。

## 晚年与误国

隋文帝晚年性情多疑，废太子杨勇，立次子[[隋炀帝|杨广]]，为隋朝灭亡埋下祸根。604年，杨广弑父（或说文帝病死），隋文帝结束了其统治。

## 历史评价

[[唐太宗]]曾以隋文帝为镜，称其有开创之功，却乏包容大度（191-009，193-074）。[[司马光]]在《[[资治通鉴概述|资治通鉴]]》中记载了唐初君臣多次以隋文帝为鉴的对话，视其为"勤政有成却家规不修"的复杂帝王形象。
