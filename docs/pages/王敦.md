# [[王敦]]

[[王敦]]（266—324年），字处仲，琅邪临沂人，[[王导]]的堂兄，东晋初年最强大的权臣。他长期掌握荆州军事实力，两度举兵东进攻建康，控制朝廷，是东晋"王与马共天下"政治格局的主要缔造者。

## 早年仕途

王敦早年任职于西晋，曾随行朝廷：太傅越征讨期间，以王敦为"扬州刺史"（087-007），使其掌握东南军政要地。此前他任中书监时，已洞察朝局走向，"谓所亲曰：太傅专执威权，而选用表请，尚书犹以旧制裁之，今日之来，必有所诛"（087-005），可见其政治嗅觉敏锐。

东渡后，晋元帝司马睿以"王敦为扬州刺史，寻加都督征讨诸军事"（087-065），由此掌握长江上游重兵。

## 两度起兵

322年，王敦以诛宦官刘隗为名举兵，攻陷建康，杀戮重臣，晋元帝忧愤而死。王敦把持朝政，荣宠无双，却以疾病缠身无力坐实帝位。

324年，王敦二度起兵，此时晋明帝已联络各方准备反击。王敦死于军中，起兵随之瓦解，乱事平息。

## 历史评价

王敦的叛乱深刻影响了东晋政局，使皇权长期屈服于门阀士族的军事力量之下。[[司马光]]通过记载此事，揭示了士族专权与皇权式微之间不可调和的张力。
