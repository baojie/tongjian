# [[田子方]]

[[田子方]]，[[战国]]初期魏国著名学者，[[魏文侯]]的老师之一，据传是[[子夏]]的弟子，兼具儒道两家思想。

## 魏文侯尊师

[[魏文侯]]以[[卜子夏]]、田子方为师，"每过段干木之庐必式"（001-015），礼贤下士之名传遍天下，吸引四方贤士归附魏国。

## 历史意义

田子方是[[魏文侯]]礼贤下士的重要对象，其存在使魏国在战国初期成为文化重心，间接奠定了魏国称霸的基础。《[[资治通鉴概述|资治通鉴]]》通过魏文侯尊师的记载，说明"得贤者昌"的治国原则（001-015）。
