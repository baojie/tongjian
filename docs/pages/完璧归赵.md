# [[完璧归赵]]

**完璧归赵**，[[战国]]时期著名外交事件，[[蔺相如]]奉命出使[[秦国]]，以智勇保住[[和氏璧]]完好归赵，留下千古成语。

## 事件经过

[[赵惠文王]]得楚国[[和氏璧]]，[[秦昭王]]欲以十五城换取，赵国君臣两难："赵王得楚和氏璧，秦昭王欲之，请易以十五城。赵王欲勿与，畏秦强，欲与之，恐见欺"（004-057）。

[[蔺相如]]自荐使秦。入秦后，见秦王无意给城，便以"璧有瑕"为由取回和氏璧，斥责秦王无诚信，并以"人璧俱碎"相威胁，迫使秦王不得不按礼相待，最终派人先将和氏璧送回赵国，完璧归赵。

## 历史意义

此事使[[蔺相如]]声名大噪，[[赵惠文王]]拜其为上大夫。"完璧归赵"成为中国历史上以智谋战胜强权的外交典范，与[[渑池之会]]、[[将相和]]并称蔺相如三大功绩。
