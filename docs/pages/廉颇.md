# [[廉颇]]

[[廉颇]]，[[战国|战国时期]][[赵国]]著名将领，[[战国]]四大名将之一，以骁勇善战著称。与文臣[[蔺相如]]"[[将相和]]"的故事，以及晚年"廉颇老矣，尚能饭否"的典故，成为中国文化中的经典。

## 守卫国家

在[[渑池之会]]前，廉颇与[[蔺相如]]共同商议，力劝赵王出行："王不行，示赵弱且怯也。"廉颇亲自送赵王至国境，并留下著名的应急部署：若王三十日不返，则立太子为王，使秦无所图谋（004-070）。

## 长平之战前的谋划

在秦攻上党之初，廉颇率军在长平据守，坚壁不出。秦军久攻不下，以反间计使赵王换将，以[[赵括]]代廉颇，最终导致长平惨败（005-041）。

廉颇被调离前，赵王曾问廉颇和[[乐乘]]能否救援被围困之军，廉颇回答"道远险狭，难救"（005-012），体现其务实的军事判断。

## 廉颇老矣

廉颇晚年流亡楚国，赵王曾考虑重新启用他抗击[[秦国]]，遣使问"廉颇老矣，尚能饭否"，廉颇一顿饭吃了一斗米、十斤肉，上马弓马娴熟以示强健，然使者受人贿赂，回报赵王"廉将军虽老，尚善饭，然与臣坐，顷之三遗矢"，赵王因此不再召廉颇（006-078）。

## 历史评价

廉颇是[[战国]]名将的典型，以其刚直勇猛和晚年壮心不已著称。[[司马光]]通过记录其生平，展示了[[战国|战国时期]][[赵国]]从强盛到衰落的历史轨迹。
