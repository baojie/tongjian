# [[刘琨]]

[[刘琨]]（271—318年），字越石，中山魏昌人，西晋末年忠臣名将，封广武侯。他年轻时与[[祖逖]]一同"闻鸡起舞"，立志报国；五胡乱华时期坚守并州十余年，屡次以少御多，最终兵败被杀，成为西晋末年忠义精神的代表人物。

## 年轻结谊

刘琨年轻时与[[祖逖]]情谊深厚，共为司州主簿，"范阳[[祖逖]]，少有大志，与[[刘琨]]俱为司州主簿。同寝，中夜闻鸡鸣，蹴琨觉曰：'此非恶声也！'"（088-077），二人于鸡鸣时共同起舞练武，砥砺报国之志，是中国历史上著名的励志典故。

## 纵横八王之乱

八王之乱中，刘琨效力于范阳王虓，"虓以【刘琨】为司马"（086-011），参与各王势力的争夺。多次奔走斡旋，"【刘琨】说冀州刺史[[晋阳|太原]]温羡，使让位于范阳王虓"（086-021），以口才和智谋游走于各方势力之间。

## 坚守并州

西晋覆亡前后，刘琨以并州刺史坚守[[晋阳]]，"【刘琨】至上党……时[[晋阳|并州]]饥馑，数为胡寇所掠，郡县莫能自"（086-051），在极端艰难的条件下守护一方汉家土地。"【刘琨】使上党太守刘惇帅鲜卑攻壶关"（086-104），以借胡兵御胡的策略周旋于各方势力。

## 历史评价

[[司马光]]高度赞扬刘琨的忠义之志，认为他在西晋灭亡后仍坚持抗争，是乱世中守护华夏文明的精神支柱。刘琨与[[祖逖]]共同代表了东晋之前那一代人以"闻鸡起舞"精神砥砺前行的意志。
