# [[曹参]]

[[曹参]]（？—前190年），[[西汉]]开国功臣、将领，封[[平阳侯]]，后继[[萧何]]为丞相，以"[[萧规曹随]]"著称。与[[萧何]]、[[韩信]]齐名，是[[汉初三杰]]以外最重要的建国功臣之一。

## 从军与封侯

[[曹参]]与[[萧何]]同为[[汉高祖刘邦|沛公]]的掾吏。陈胜起义后，沛令欲响应，萧何、曹参劝曰："君为秦吏，今欲背之，率沛子弟，恐不听。愿君召诸亡在外者，可得数百人，因劫众……"（007-101）。

楚汉之争中，曹参随汉王转战，屡立战功。汉四年，曹参与灌婴协助[[韩信]]击魏（009-052）。天下平定后，"平阳侯[[曹参]]，身被七十创，攻城略地，功最多，宜第一"，位列列侯功第一（011-040）。

## 萧规曹随

[[萧何]]临终荐曹参继任丞相。曹参任相后，一切遵循萧何旧章，史称"萧规曹随"，以无为而治稳固了汉初的休养生息局面。

## 历史评价

[[司马光]]及后世史家均视曹参为[[文景之治]]的重要铺垫者，其"萧规曹随"的政治智慧成为儒家黄老政治的典范。
