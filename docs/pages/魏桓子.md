# [[魏桓子]]

[[魏桓子]]（？—约前446），名驹，春秋晚期[[晋国]]魏氏的领袖，[[三家分晋]]历史进程中的核心人物之一。在[[智伯]]专权、威逼三家割地时，魏桓子暗中与赵、韩谋划反击。

## 智伯索地事件

[[智伯]]向魏桓子索地，"又求地于魏桓子，桓子欲弗与"（001-010）。谋士任章劝谏：不如先给，以使[[智伯]]骄横，等他向其他诸侯索地，其他人必然反抗，"魏子从之，复与之万家之邑一"（001-010），魏桓子采取韬晦之策。

## 三家反击智伯

[[智伯]]联合韩、魏攻赵，水灌[[晋阳]]城，危急时刻，魏桓子在关键时刻以肘信号[[韩康子]]，暗示反水之意：

> 智伯行水，魏桓子御，韩康子骖乘。智伯曰："吾乃今知水可以亡人国也。"桓子肘康子，康子履桓子之跗，以相感知（001-011）。

三家随后倒戈，反引水灌[[智伯]]军营，擒杀[[智伯]]，三分其地，魏国由此奠基。
