# [[魏征]]

[[魏征]]（580—643年），字玄成，钜鹿下曲阳人，初仕[[太子建成]]，"太子中允王珪、洗马[[魏征]]说太子曰：'秦王功盖天下，中外归心……'"（190-065），后被[[唐太宗]]重用，成为贞观朝最重要的谏臣。

## 转仕太宗

[[玄武门之变]]后，[[唐太宗]]召问魏征，"汝何为离间我兄弟！"魏征从容对答，"[[魏征]]、王珪表请陪送至墓所"以礼葬太子、齐王（192-010），太宗由此认识其忠直之才，委以重用。

## 直言极谏

魏征多次上疏言事，其中"[[魏征]]上疏，以为：'陛下欲善之志不及于昔时，闻过必改少亏于曩日……'"（195-003），直指太宗懈怠之处。又论君臣信任："臣为王虑……虽竭力尽诚，犹未免有败"（195-008），力陈用人之道。面对太宗疑问，魏征直言：贞观之初"恐人不谏，常导之使言"，而今"虽勉从之，犹有难色"（195-027）。

## 君臣典范

[[唐太宗]]曾赞："贞观以来，绳愆纠缪，魏征之功也"（195-027），并赐刀以示褒奖。魏征死后，太宗叹曰"以铜为镜可以正衣冠，以史为镜可以知兴替，以人为镜可以知得失……朕失一镜矣"。

## 历史评价

[[司马光]]高度肯定魏征的谏臣精神，将其视为"贞观之治"不可或缺的要素。魏征敢于在皇帝面前讲真话，以隋亡为鉴反复劝谏，代表了儒家士大夫对"致君尧舜"理想的最高实践。
