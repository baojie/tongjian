# [[张角]]

[[张角]]（？—184年），巨鹿人，东汉末年黄巾起义领袖。他以黄老之术创立"太平道"广收信众；以预谋举事被奸细告发而提前起兵；以黄巾乱六州震动东汉朝廷；被卢植连战击破，最终病死于军中，是东汉王朝崩溃的直接导火索。

## 以妖术教授，号"太平道"

张角以宗教手段聚集力量，"初，巨鹿【张角】奉事黄、老，以妖术教授，号'太平道'"（058-036），以黄老思想为旗帜、以治病传教为手段，广泛吸收底层民众，在东汉末年社会危机中迅速积累起庞大的信众基础。

## 举事消息泄露，朝廷犹豫

张角起事前的预谋险遭扼杀，"若不赦宥，轻与【张角】合谋，为变滋大"（058-040），面对唐周告密，朝廷内部在镇压与赦宥之间犹豫，最终错失提前剿灭的机会。

## 黄巾乱起，震动六州

张角的起义撼动了整个东汉，"今【张角】起于赵、魏，黄巾乱于六州"（058-049），黄巾之乱规模空前，波及六州，直接暴露了东汉地方军事体系的空洞，迫使朝廷启用地方武装，为后来的军阀割据埋下伏笔。

## 被卢植连战击破

张角在战场上遭到有力反击，"北中郎将卢植连战破【张角】，斩获万馀人"（058-053），朝廷正规军在卢植指挥下对张角部取得阶段性胜利，但随着董卓等人的介入，平叛战局变得更加复杂。

## 历史评价

[[司马光]]通过张角，展示了东汉末年政治崩溃的深层逻辑：一个以宗教聚众的叛乱领袖，之所以能够动摇拥有四百年根基的王朝，根本原因在于吏治腐败与社会矛盾已积重难返。张角以"太平道"许诺的"太平"，恰恰是东汉末年民众最渴求却最缺乏的东西——他的起义不是原因，而是症状。黄巾之乱真正的遗产，是迫使东汉朝廷授权地方自募兵马，从而开启了军阀割据的时代。
