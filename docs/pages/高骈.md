# [[高骈]]

[[高骈]]（821—887年），字千里，唐朝晚期著名将领，高崇文之孙。他因平定安南、击败南诏而功勋卓著，声名大振；后受命坐镇淮南，却在黄巢之乱中拥兵自保，坐失勤王良机，致使长安沦陷，留下晚节不保的历史评价。

## 平定安南

南诏侵扰安南多年，高骈奉命讨伐，"乃以骈为安南都护、本管经略招讨使。骈，崇文之孙也，世在禁军"（250-091），出身将门世家，领兵南下。监军李维周屡屡使绊，"监军李维周恶骈，欲去之，屡趣骈使进军"（250-105），高骈仍克服阻碍，坚持用兵。

## 大破南诏

高骈与援军协力出击，"【高骈】得以益其军，进击南诏，屡破之"（250-117），连战连捷。在围困交趾时，"【高骈】围交趾十馀日，蛮困蹙甚，城且下"（250-121），终将南诏逐出安南，"置静海军于安南，以【高骈】为节度使"（250-123），此后又凿通航路，"静海节度使【高骈】募工凿之，漕运无滞"（250-130）。

## 历史评价

[[司马光]]笔下的高骈兼具将才与局限：平安南一役展现了唐朝将帅的军事能力，而黄巢之乱中养寇自重的选择，则折射出晚唐藩镇割据、各自为政的深层困境，是唐朝覆亡的重要推手之一。
