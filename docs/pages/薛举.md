# [[薛举]]

[[薛举]]（？—618年），隋末陇右军事强人，起兵金城，自称秦帝，是李唐建立初期西北方向最强劲的割据势力。他多次击败唐军，令[[唐太宗|李世民]]遭受挫败，死后其子薛仁果继位，终被[[唐太宗|李世民]]所灭。

## 起兵陇右

隋末大乱之际，薛举乘机起事，"汾阴【薛举】，侨居金城，骁勇绝伦，家赀巨万，交结豪杰，雄于西边，为金城府校尉"（183-050），以豪富之身纠合豪杰，掌握陇右军事力量。

"【薛举】自称秦帝，立其妻鞠氏为皇后，子仁果为皇太子。遣仁果将兵围天水，克之，举自金城徙都"（184-015），正式称帝，建立割据政权。

## 威胁李唐

唐朝建立后，薛举被视为西北最大威胁，"以[[唐太宗|秦王世民]]为元帅，将八总管兵以拒之"（185-050），[[李渊]]多次派重兵与之对抗。薛举与突厥联合，扩大威胁，"郝瑗说【薛举】，与[[梁师都]]及突厥连兵以取长安"（185-029），战略图谋直指关中。

## 兵败与灭亡

薛举在与唐军的交战中曾多次得手，但因其突然病逝，政权转入其子薛仁果手中，最终被[[唐太宗|李世民]]彻底击灭。[[司马光]]记录了薛举在穷途末路时询问臣下"自古天子有降事乎"（184-054），折射出枭雄末路的困境与悲凉。
