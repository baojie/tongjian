# [[夫差]]

[[夫差]]，春秋末期吴国君主，[[杨行密|吴王]]阖闾之子，以复仇起家，以骄败亡，是中国历史上"成也骄傲、败也骄傲"的典型君主。

## 《资治通鉴》中的夫差形象

《[[资治通鉴概述|资治通鉴]]》多处以夫差为鉴戒。伍子胥在阖闾时代深受信任，到夫差时代却被赐死，"昔伍子胥说听于阖闾而吴远迹至郢；夫差弗是也，赐之鸱夷而浮之江"（004-073），揭示了君主用人之道的差异。

夫差又被列入"以极武而亡"的君主之列："昔者吴王夫差、[[智伯]]、[[秦始皇]]，皆以极武而亡"（012-038），是穷兵黩武招致覆灭的历史典型。

前秦[[苻坚]]南伐时，部下也引夫差为鉴："夫差、孙皓皆保据江湖，不免于亡"（104-100），以说明地势之险不足以保国。

## 历史意义

夫差的失败印证了[[司马光]]反复强调的主题：骄傲自满、不听忠谏是君王覆亡的根本原因。其与越王勾践的对比，成为后世"卧薪尝胆"故事的历史背景（004-073，012-038）。
