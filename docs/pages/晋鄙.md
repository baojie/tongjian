# [[晋鄙]]

[[晋鄙]]，[[战国|战国时期]]魏国将领，[[信陵君]]窃符救赵事件的历史主角之一。晋鄙手握魏国十万援军，却处于政治两难之中，最终以生命为代价成为了一段历史教训。

## 救赵受命却停兵观望

[[长平之战]]后，[[秦国]]围攻[[邯郸]]，楚、魏各遣援军。魏王派晋鄙将兵十万救赵，"魏王亦使将军晋鄙将兵十万救赵"（005-052）。然而[[秦国]]威胁魏王，魏王恐惧，"遣人止晋鄙，留兵壁鄴，名为救赵，实挟两端"（005-053），晋鄙奉命停兵观望，不敢前进。

## 被信陵君夺兵

[[信陵君|魏公子]][[长孙无忌|无忌]]多次请求魏王命令晋鄙出兵，"数请魏王敕晋鄙令救赵"（005-055），均告失败。最终[[信陵君]]盗取兵符，"持节至晋鄙军，合符欲夺其军"（005-057），晋鄙怀疑，被朱亥椎杀，[[信陵君]]夺取兵权，率兵救赵成功。

## 被秦用来中伤信陵君

晋鄙死后，其旧部也成为政治工具。秦王"求得晋鄙客，令说魏王曰：'公子亡在外十年矣，今复为将，诸侯皆属，天下徒闻信陵君而不闻魏王……'"（006-042），以晋鄙的旧关系网离间魏王与信陵君，导致信陵君被夺兵权。

## 历史评价

晋鄙处于君命与战机之间，其悲剧折射出[[战国]]末年大国权贵在君主意志与战略利益之间的两难困境（005-052，005-057）。
