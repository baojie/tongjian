# [[甘露之变]]

甘露之变，发生于唐文宗太和九年（835年）十一月，是唐朝最惨烈的宦官政变之一。文宗与宰相[[李训]]、御史[[郑注]]秘密谋划，以含元殿左仗有"甘露"降下为诱饵，企图将宦官首领[[仇士良]]等人诱入伏兵圈中一网打尽。然因事机不密、计划败露，[[仇士良]]等宦官反扑，当场捕杀宰辅大臣，事后株连数千人，史称"甘露之变"。

## 事变背景

文宗在位，深苦宦官专权，与[[李训]]、[[郑注]]密谋夺权，"上外虽包容，内不能堪。【[[李训]]】、[[郑注]]既得幸，揣知上意，训因进讲，数以微言动上"（245-036），君臣密谋铲除宦官势力。

## 诱敌计划

事变当日，以左仗有甘露为名诱引宦官前往查验，"【甘露之变】……顾左、右中尉[[仇士良]]、鱼志弘帅诸宦者往视之。宦者既去，训遽召郭行馀、王璠曰：'来受敕旨！'"（246-010），伏兵已在帷幕后待命。

## 事败反扑

[[仇士良]]察觉异常，"【[[仇士良]]】等至左仗视甘露，韩约变色流汗。士良怪之曰：'将军何为如是？'俄风吹幕起，见执兵者"（245-061），宦官随即反扑，"自是天下事皆决于北司，宰相行文书而已。宦官气益盛，迫胁天子"（245-071），朝廷大权彻底落入北司之手。

## 历史影响

文宗此后"自【甘露之变】，意忽忽不乐，两军球鞠之会什减六七，虽宴享音伎杂遝盈庭，未尝解颜"（245-096），郁郁而终。后世为诸冤臣昭雪，"上以【甘露之变】，惟[[李训]]、[[郑注]]当死，自馀[[王涯]]、贾餗等无罪，诏皆雪其冤"（249-059）。

## 历史评价

[[司马光]]将甘露之变视为唐朝宦官专权登峰造极的标志：皇帝不得不以诈术谋诛宦官，事败后反遭挟持，天子沦为傀儡，标志着唐朝皇权的彻底式微。
