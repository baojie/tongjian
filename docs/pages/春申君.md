# [[春申君]]

[[春申君]]（黄歇，?—前238年），楚国令尹（丞相），封"[[春申君]]"，[[战国四公子]]之一，以能言善辩和外交才能著称，后专擅楚政，最终被[[李园]]所杀。

## 救楚太子立功

春申君早年随楚太子入质于秦，楚顷襄王病重，设计帮助太子逃回楚国。顷襄王薨后，太子即位为楚考烈王，"以黄歇为相，封以淮北地，号曰春申君"（005-036）。

## 救赵

[[邯郸之战]]中，"楚王使春申君将兵救赵"，与[[信陵君]]、[[平原君]]共同解[[邯郸]]之围（005-052）。

## 历史评价

春申君是[[战国四公子]]中政治生涯最长、最具实权的一位，长期执掌楚国国政。[[司马光]]记其事迹，展示了战国晚期楚国的政治格局，以及贵族专权如何影响国家命运。
