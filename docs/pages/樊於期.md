# [[樊於期]]

[[樊於期]]，[[秦国]]将军，因罪逃亡投奔燕太子丹，成为[[荆轲]]刺秦计划中的关键人物。他以自杀、献出头颅的方式，为[[荆轲]]提供了接近秦王[[秦始皇|嬴政]]的信物，以生命践行了对[[燕国]]的最后忠诚。

## 亡命燕国

秦王暴怒，"将军樊於期得罪，亡之燕；太子受而舍之"（006-128）。太子丹收留了这位[[秦国]]通缉要犯，谋士鞠武警告此举太过危险，太子丹却以"不忍"为由坚持。

## 自献头颅

[[荆轲]]制定刺秦计划，需要秦王最想得到的两样东西：[[督亢]]地图和樊於期头颅。[[荆轲]]私下拜见樊於期，"[[荆轲]]乃私见樊於期曰：'秦之遇将军，可谓深矣，父母宗族皆为戮没！今闻购将军首，金千斤，邑万家，将奈何？'"（006-129）。樊於期慨然，立刻自刎，以一死换取[[荆轲]]入秦的机会。

## 历史评价

樊於期以生命为[[荆轲]]之行开路，是"士为知己者死"精神的极致体现。[[司马光]]记载此事，展示了战国末年[[秦国]]横暴下义士们的抗争精神，以及"以死明志"这一传统价值观的力量与局限。
