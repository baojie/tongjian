# [[西汉]]

**西汉**（前206—公元8年），中国历史上第一个长期稳定的大一统王朝，[[汉高祖刘邦]]于楚汉战争胜利后建立，定都长安，历[[汉高祖刘邦|高祖]]、惠帝、[[隋文帝|文帝]]、景帝、武帝至平帝共12位皇帝，直至[[王莽]]篡位建立[[王莽|新朝]]而终结。

## 建立与巩固

西汉建立后，经[[汉文帝]]、[[汉景帝]]的"[[文景之治]]"休养生息，国力大增，为[[汉武帝]]的鼎盛奠定基础。《[[资治通鉴概述|资治通鉴]]》第一部分（卷一至卷六十八）详细记载了西汉从建立到[[王莽]]篡汉的全过程。

## 汉武帝时代

[[汉武帝]]在位期间（前141—前87年），汉朝达到鼎盛：北击[[匈奴]]（[[卫青]]、[[霍去病]]），西通西域（[[张骞]]），南平百越，设立太学推行儒学，确立了中国文化的主体框架。

## 历史意义

西汉是《[[资治通鉴概述|资治通鉴]]》记载时间最长的单一王朝段落。[[司马光]]通过西汉历史，探讨了君臣关系、外戚干政、藩王问题等核心政治议题，"[[文景之治]]"与"武帝盛世"更成为后世君王效仿的典范。
