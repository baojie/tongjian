# [[乐乘]]

[[乐乘]]，字武襄，战国末期赵国将领，封武襄君，是[[廉颇]]晚年政治失意的直接相关人物。

## 早期参与马服之战

赵奢对阵秦军时，赵王问计廉颇与乐乘，"赵王召廉颇、乐乘而问之曰：'可救否？'皆曰：'道远险狭，难救'"（005-012），而赵奢则主张出击并最终获胜，乐乘此时判断保守，与廉颇相同。

## 击败燕国栗腹

燕国趁赵国长平之战后出兵，廉颇为将逆击，"败栗腹于鄗，败卿秦、乐乘于代"（006-021）——此处乐乘为燕国将领，战败。注意此处乐乘是燕将，并非赵将。

## 代廉颇引发出奔

赵悼襄王即位后，"使武襄君乐乘代廉颇。廉颇怒，攻武襄君，武襄君走，廉颇出奔魏"（006-051）。赵王以乐乘取代廉颇，廉颇大怒攻打乐乘，乐乘奔走，廉颇也因此出走魏国，赵国由此失去了最重要的老将。

## 历史意义

乐乘的任命是赵国人才流失的导火索之一，[[司马光]]记此事，以证君主用人失当造成的连锁损失（006-051）。
