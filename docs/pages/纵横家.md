# [[纵横家]]

**纵横家**，战国时期重要政治流派，以外交辞令与战略谋划为手段，游说各国君主，影响国际格局。主要代表为[[苏秦]]（主张"合纵"抗秦）和[[张仪]]（主张"连横"事秦），两人均出自[[鬼谷子]]门下。

## 合纵与连横

- **合纵**：六国联合以抗[[秦国]]，[[苏秦]]为主要推手，曾身挂六国相印，使秦国十五年不敢出函谷关。
- **连横**：各国分别与秦结盟，[[张仪]]为代表，以割地事秦瓦解合纵联盟。

## 鬼谷师承

"张仪者，魏人，与苏秦俱事鬼谷先生，学纵横之术，苏秦自以为不及也"（002-092），纵横家宗师为隐士鬼谷子，培养了众多游说之才。

## 历史评价

[[司马光]]引扬子（扬雄）之评："仪、秦学乎鬼谷术而习乎纵横言，安中国者各十馀年，是夫？曰：诈人也。圣人恶诸"（003-042）。儒家视纵横家为"诈术"，然其客观上延缓了秦国统一的步伐，亦对战国外交产生了深远影响。
