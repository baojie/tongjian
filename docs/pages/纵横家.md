# [[纵横家]]

**纵横家**，[[战国|战国时期]]重要政治流派，以外交辞令与战略谋划为手段，游说各国君主，影响国际格局。主要代表为[[苏秦]]（主张"[[合纵连横|合纵]]"抗秦）和[[张仪]]（主张"[[合纵连横|连横]]"事秦），两人均出自[[鬼谷子]]门下。

## 合纵与连横

- **[[合纵连横|合纵]]**：六国联合以抗[[秦国]]，[[苏秦]]为主要推手，曾身挂六国相印，使[[秦国]]十五年不敢出函谷关。
- **[[合纵连横|连横]]**：各国分别与秦结盟，[[张仪]]为代表，以割地事秦瓦解[[合纵连横|合纵]]联盟。

## 鬼谷师承

"[[张仪]]者，魏人，与[[苏秦]]俱事[[鬼谷子|鬼谷先生]]，学纵横之术，[[苏秦]]自以为不及也"（002-092），纵横家宗师为隐士[[鬼谷子]]，培养了众多游说之才。

## 历史评价

[[司马光]]引扬子（扬雄）之评："仪、秦学乎鬼谷术而习乎[[合纵连横|纵横]]言，安中国者各十馀年，是夫？曰：诈人也。圣人恶诸"（003-042）。儒家视纵横家为"诈术"，然其客观上延缓了[[秦国]]统一的步伐，亦对[[战国]]外交产生了深远影响。
