# [[燕国]]

[[燕国]]，姬姓，战国七雄之一，都蓟（今北京）。燕昭王时代一度强盛，以[[乐毅]]领五国联军伐齐，攻克七十余城，达到鼎盛。太子丹之死与[[荆轲]]刺秦的悲剧，是燕国灭亡的直接导火索。

## 燕昭王与乐毅

燕昭王礼贤下士，广纳贤才，"燕昭王日夜抚循其人，益为富实，乃与[[乐毅]]谋伐齐"（004-045）。[[乐毅]]统率五国联军伐齐，连下七十余城，差点灭亡齐国——是战国历史上最壮观的军事奇迹之一。

## 合纵中的燕国

[[苏秦]]以燕国为出发点推动[[合纵连横|合纵]]，"今乐君亲为寡人破齐……燕国固乐君所有，非燕之所得也"（004-071），体现燕国在战国外交中扮演的关键角色。

## 荆轲刺秦与灭亡

太子丹不能忍受[[秦国]]压迫，派[[荆轲]]持[[樊於期]]头颅与[[督亢]]地图赴秦，刺秦王未成，反而彻底激怒[[秦国]]。秦军随即大举伐燕，前222年燕国灭亡，成为战国七雄中倒数第二个被灭的国家。

## 历史意义

燕国的历史是战国"强国不义、弱国图存"主题的缩影，[[司马光]]记录[[荆轲]]刺秦一事，着重探讨了"义气"与"成败"之间的永恒张力。
