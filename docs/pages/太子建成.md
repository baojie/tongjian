# [[太子建成]]

[[太子建成]]（589—626），名李建成，唐高祖李渊长子，唐朝初立后被立为太子，但在与弟弟[[唐太宗|李世民]]的权力角逐中失败，[[玄武门之变]]中被杀。

## 太子之位

唐高祖建唐，以嫡长子李建成为太子，建成也确实参与了唐朝的建立，镇守后方，处理政务。然而李世民在军事上屡建奇功，声望日隆，兄弟之间矛盾逐渐激化。

## 与世民的角力

"礼部尚书李纲领太子詹事，太子建成始甚礼之。久之，太子渐昵近小人，疾秦王世民功高，颇相猜忌"（187-096），建成开始对世民采取防范措施，双方都在拉拢朝中势力。

建成受命出征稽胡，"太子建成获稽胡千馀人，释其酋帅数十人，授以官爵，使还，招其馀党"（189-004），也展示了一定的军事能力。

## 玄武门之变

[[玄武门之变]]中，"世民帅长孙无忌等入，伏兵于玄武门"，建成与元吉入朝途中遭到伏击，"世民弓无不中"，建成当场身亡，死后追谥"隐太子"。

## 历史评价

后世对太子建成的评价多有争议。[[司马光]]记录玄武门之变时，较为客观地呈现了双方的争斗，并未一面倒地为李世民辩护，承认建成之死是一场残酷的权力清算。
