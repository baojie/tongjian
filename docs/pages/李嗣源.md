# [[李嗣源]]

[[李嗣源]]（867—933年），本名邈吉烈，字若虚，沙陀人，[[李克用]]养子，后唐明宗，五代后唐第二位皇帝。他起自军旅，以勇猛谨慎著称，924年登基后推行节俭、整顿吏治，是五代十国时期少有的明君，在位十年政治相对清明。

## 军中英武

李嗣源早年在[[李克用]]麾下征战，以沉稳著称。"[[李嗣源]]性谨重廉俭，诸将相会，各自诧勇略，嗣源独默然，徐曰：'诸君喜以口击贼，嗣源但以手击贼耳'"（258-062），以行动代替言语的风格令人折服。

在对抗后梁的战争中，李嗣源屡建军功，"会横冲都将[[李嗣源]]以所部兵至，谓嗣昭曰：'吾辈亦去，则势不可支矣，我试为公击之'"（261-087），关键时刻主动请战，是晋军的主要支柱之一。

[[晋阳]]被围时，李嗣源与李嗣昭"数将敢死士夜入氏叔琮营，斩首捕虏，汴军惊扰"（263-012），守城克敌，忠勇双全。

## 即位与仁政

[[唐庄宗|后唐庄宗]][[唐庄宗|李存勖]]末年政治腐败，926年李嗣源被拥立为帝，是为后唐明宗。他即位后减轻赋税，整顿吏治，注重农业生产，使五代乱世中出现了一段相对安定的局面。

## 历史评价

[[司马光]]对李嗣源多有正面评价，视其为五代难得的实践仁政的君主。其"以手击贼"的务实作风，是那个时代武将出身帝王的难得品格。
