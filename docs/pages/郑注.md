# [[郑注]]

[[郑注]]（？—835年），唐文宗时期大臣，以医术游历四方，以善揣人意闻名。他先依附宦官王守澄，借医术获得宪宗、文宗的信任，入主朝廷后与[[李训]]合谋诛杀宦官，秘密策划[[甘露之变]]，然事败于凤翔，被监军张仲清所杀。

## 以医入幕

郑注早年游历，"翼城人【郑注】，眇小，目下视，而巧谲倾谄，善揣人意，以医游四方，羁贫甚"（243-007），以才智和医术逐步打入权贵圈子。

## 进医获宠

宪宗病风之际，王守澄推荐郑注，"上始得风疾，不能言。于是王守澄荐昭义行军司马【郑注】善医。上征注至京师，饮其药，颇有验，遂有宠"（244-089），郑注由此跻身权力核心。

## 陷害宋申锡

郑注权势初显，便参与了对政敌的政治陷害。[[宋申锡]]曾密谋诛除宦官，被郑注察觉，"[[宋申锡]]与御史中丞宇文鼎受密诏诛【郑注】，使京兆尹王璠掩捕之。璠密以堂帖示王守澄，注由是得免，深德璠"（245-024）。郑注不仅逃过一劫，还借机将[[宋申锡]]构陷定罪，展示了其危险的政治手腕。

## 合谋诛宦

文宗时期，郑注与[[李训]]被重用，共同策划诛宦，"【[[李训]]】、郑注为上画太平之策，以为当先除宦官，次复河、湟，次清河北"（245-041），胸怀宏图。又合力排挤宦官势力，"左神策中尉韦元素、枢密使杨承和、王践言……【[[李训]]】、郑注因之出承和于西川，元素于淮南，践言于河东"（245-039）。

## 事败被杀

[[甘露之变]]中，"注知训已败，复还凤翔。【[[仇士良]]】等使人赍密敕授凤翔监军张仲清令取注，仲清惶惑……押牙李叔说仲清……遂杀注及其子弟、将校约数百人"（245-070），郑注就此覆灭。

## 历史评价

[[司马光]]通过郑注，展示了晚唐政治中投机者的典型路径：以小术入大局，既善揣帝意图，又深知宦官之患，却终因谋划失当而殒命，与[[李训]]共同酿成了[[甘露之变]]的惨剧。
