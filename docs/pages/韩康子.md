# [[韩康子]]

[[韩康子]]（？—前424），名虎，春秋晚期晋国韩氏领袖，[[三家分晋]]的参与者之一，[[韩国]]的实际奠基人，其子孙最终建立战国七雄之一的韩国。

## 智伯索地事件

[[智伯]]专权，向韩康子索地，"智伯请地于韩康子，康子欲弗与。段规曰：'智伯好利而愎，不与，将伐我；不如与之。彼狃于得地，必又他索，他索必不与，不与必伐之，则我得所以与之矣'"（001-010）。谋士段规的建议是以割地养虎之计，等待智伯骄横失败。

## 宴会上的侮辱

智伯不仅索地，还公开羞辱韩康子，"与韩康子、魏桓子宴于蓝台。智伯戏康子而侮段规"（001-009），韩康子的家臣智国当场警告，认为这会招来祸端。韩康子则隐忍不发，等待时机。

## 三家灭智

智伯水灌晋阳，决定性关头，[[魏桓子]]以肘示意，韩康子以足回应，三家默契配合，"三家以国人围而灌之"（001-011），反戈一击，擒杀智伯，三分其地，韩国由此奠基。
