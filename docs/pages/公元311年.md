# [[公元311年]]

本年事见于《[[资治通鉴概述|资治通鉴]]》**[[第087卷]]**（晋纪九）。

公元311年，汉赵前军大将军呼延晏率兵二万七千南下攻洛阳，汉军前后十二战皆捷，六月攻克宣阳门，入南宫；始安王刘曜入西明门，杀太子及诸王公，士民死者三万余人，遂发掘诸陵，焚宫庙官府。晋怀帝出华林园欲奔长安，被汉兵追执，幽于端门，后被迁往平阳。

## 汉兵攻入洛阳，怀帝被虏

汉军连败晋兵，怀帝被追执：

> 汉太后单氏卒，汉主聪尊母张氏为皇太后……汉主聪使前军大将军呼延晏将兵二万七千寇洛阳，比及河南，晋兵前后十二败，死者三万馀人。始安王曜、王弥、石勒皆引兵会之……王弥、呼延晏克宣阳门，入南宫，升太极前殿，纵兵大掠，悉收宫人、珍宝。帝出华林园门，欲奔长安，汉兵追执之，幽于端门。
> （087-079）

## 历史意义

永嘉五年洛阳之陷、怀帝被俘是西晋正式灭亡的象征。五年后（316年）长安亦陷，西晋彻底终结。数十万士族百姓"衣冠南渡"，形成了东晋偏安江南的局面；北方则进入了五胡十六国的长期乱局，这是中华文明历史上空前的民族大动荡。
