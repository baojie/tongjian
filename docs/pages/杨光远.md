# [[杨光远]]

[[杨光远]]（？—？），五代后唐、后晋将领，在[[石敬瑭]][[晋阳]]之战中发挥关键作用。他先以副部署身份参与围攻[[晋阳]]，后被契丹援军反包围于晋安寨；粮尽困极之际，劝[[张敬达]]降契丹不成，遂自行将其杀死，率众降于新立的后晋，因功获封高位，是五代武将中以"反戈"立功的典型人物。

## 参与征晋

[[石敬瑭]]起兵[[晋阳]]，[[李从珂|后唐末帝]]遣军征讨，杨光远以副帅身份出战，"以义武节度使【杨光远】为副部署"（280-015），与主将[[张敬达]]共同主持围攻事宜，"以[[张敬达]]充[[晋阳]]四面招讨使，以【杨光远】为副使"（280-019）。

## 晋安被围

契丹援军抵达，唐军转攻为守，杨光远被困晋安寨，"[[张敬达]]、【杨光远】、[[安审琦]]以步兵陈于城西北山下，契丹遣轻骑三千，不被甲，直犯其陈"（280-030），在极端困境中坚守。

## 谏降议和

被围日久，粮草告竭，杨光远力主投降以保全兵众，"【杨光远】、[[安审琦]]，劝敬达降于契丹，敬达曰：'吾受明宗及今上厚恩，为元帅而败军，其罪已大，况降敌乎'"（280-044），[[张敬达]]慷慨拒绝。

## 弑帅降晋

[[张敬达]]拒降后，杨光远遂将其杀死，率众降于[[石敬瑭]]所建的后晋，"以【杨光远】为侍卫马步军都指挥使"（280-045），以弑主之功跻身后晋军方高位。此后[[石敬瑭]]又加封其为节度使，"杨光远降"（280-046），彻底完成立场转换。

## 历史评价

[[司马光]]通过杨光远，展示了五代武将的道德困境：在极端绝境中，以功利主义抉择取代忠义节操，弑杀主将以换取荣华，是五代"以力为本"军事文化的缩影，与[[张敬达]]的以死殉义形成鲜明对照。
