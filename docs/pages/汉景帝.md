# [[汉景帝]]

[[汉景帝]]（前188—前141年），名刘启，[[汉文帝]]长子，在位16年（前157—前141年）。继承[[隋文帝|文帝]]宽仁之政，但在位期间采纳[[晁错]]削藩建议，引发[[吴楚七国之乱]]；以[[周亚夫]]为将平乱，三月即平，此后国力持续积累，至武帝时达到顶峰。

## 七国之乱与周亚夫

"孝景皇帝下前三年"吴楚等七国反（016-002）。景帝拜[[周亚夫]]为太尉，"将三十六将军往击吴、楚"（016-013），凭借坚守昌邑、断绝楚军粮道之策，三个月内平定[[吴楚七国之乱|七国之乱]]。

## 治国理念

汉景帝继续轻徭薄赋，"孝景皇帝上"年间进一步削减赋税（015-074）。景帝晚年因废太子刘荣、改立武帝之母王夫人之子为储，造成宫廷纷争，但国家大政稳定。

## 历史评价

[[司马光]]以"孝景皇帝"为卷名标识，显示对景帝传统谥号的认同。[[文景之治]]期间，[[西汉|汉朝]]府库充实，"都内之钱贯朽而不可校，太仓之粟陈陈相因"，是秦汉时期难得的治世。景帝平定[[吴楚七国之乱|七国之乱]]后，诸侯王势力大为削弱，中央集权得到强化，为[[汉武帝]]的大一统奠定基础。
