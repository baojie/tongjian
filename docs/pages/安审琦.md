# [[安审琦]]

[[安审琦]]（？—？），五代后晋名将，金全之子，历仕后唐、后晋多朝，以骑战见长。他参与了[[石敬瑭]][[晋阳]]被围的守卫战，此后在后晋多次对辽战役中担任马军要职，是五代[[晋国]]对辽防御体系的重要组成部分。

## 出任节度

安审琦以军功晋升，"以彰圣都指挥使【安审琦】领顺化节度使。审琦，金全之子也"（279-091），以将门子弟身份步入高位。

## 参与晋安守卫

[[石敬瑭]][[晋阳]]被围，安审琦出任马军要职，"以安国节度使【安审琦】为马军都指挥使"（280-015），协同[[高行周]]、[[符彦卿]]守卫要地。在晋安寨被围时，"[[张敬达]]、[[杨光远]]、【安审琦】以步兵陈于城西北山下，契丹遣轻骑三千，不被甲，直犯其陈"（280-030），在极端困苦中坚守。

## 劝降不获

[[张敬达]]宁死不降，"[[杨光远]]、【安审琦】劝敬达降于契丹，敬达曰：'吾受明宗及今上厚恩，为元帅而败军，其罪已大，况降敌乎'"（280-044），安审琦的劝降被[[张敬达]]慷慨拒绝。

## 屯兵御辽

后晋出帝时期，安审琦继续出任对辽方向重要职务，"护国节度使【安审琦】会诸道兵屯邢州"（284-046），担任御辽前线的要帅，"[[张从恩]]、马全节、【安审琦】悉以行营兵数万，陈于相州安阳水之南"（284-049），参与多路抗辽部署。

## 历史评价

[[司马光]]通过安审琦的生涯，展示了后晋军事将领的整体面貌：从将门子弟起步，历经多次对辽苦战，忠于职守而缺乏足以改变战局的能力，是那个时代大多数将帅的真实写照。
