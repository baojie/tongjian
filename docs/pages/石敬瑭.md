# [[石敬瑭]]

[[石敬瑭]]（892—942年），沙陀族，[[李嗣源|后唐明宗]]女婿，后晋开国皇帝。早年随后唐征战，"左射军使石敬瑭与梁人战于河壖……横冲兵马使[[刘知远]]以所乘马授之，自乘断甲者徐行为殿"（271-004），英勇善战。后与后唐末帝反目，以割让燕云十六州为条件借契丹兵灭后唐，建立后晋。

## 灭后唐

石敬瑭叛后唐，"石敬瑭至[[晋阳]]，以部将[[刘知远]]、周瑰为都押衙，委以心腹；军事委知远"（278-028），以沙陀族将领为心腹，向契丹求援。"[[石敬瑭]]使裨将李琼以劲兵突入封丘门……自西门入，遂据其城"（274-076），最终攻入[[洛阳]]。

## 割让燕云

石敬瑭以割让幽云十六州、称辽太宗为父为代价取得契丹支持，史称"儿皇帝"。燕云十六州的丧失使中原门户洞开，此后数百年北方政权面临契丹/辽的南下威胁。

## 历史评价

[[司马光]]详细记录石敬瑭割地称臣的经过，视之为五代乱世中最严重的政治道德堕落，其"儿皇帝"之名成为后世批评软弱外交的最极端案例。燕云十六州直到宋太宗时代仍是未能收复的心头之患。
