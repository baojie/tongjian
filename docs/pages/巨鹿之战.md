# [[巨鹿之战]]

[[巨鹿之战]]（前207年），秦末[[项羽]]率楚军大破秦军主力的决定性战役，以"破釜沉舟"的英雄气概著称，是中国历史上以少胜多的经典战例。

## 战前形势

章邯击败[[项梁]]后，率秦军主力北上，与王离军合围赵国巨鹿城。"张耳与赵王歇走入巨鹿城，王离围之。陈馀北收常山兵，得数万人，军巨鹿北。章邯军巨鹿南棘原"（008-040）。

赵国危急，楚怀王命[[宋义]]、[[项羽]]率兵救赵。

## 项羽的决断

[[项羽]]杀宋义，夺取兵权。随后令全军渡过漳河后，"皆沈船，破釜甑，烧庐舍，持三日粮，以示士卒必死，无一还心"（008-052），此即"破釜沉舟"典故之由来。

楚军九战皆胜，"遂虏王离。章邯军棘原，兵少食乏"，秦军主力被歼灭。

## 历史意义

巨鹿之战彻底打垮了秦朝最后的主力，各诸侯军皆"跪行至辕门，莫敢仰视"，项羽威震天下。此后章邯投降，秦朝覆灭已成定局（008-052至008-053）。
