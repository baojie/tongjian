# [[平原君]]

[[平原君]]（赵胜，?—前251年），[[赵武灵王]]之子，封平原君，[[战国四公子]]之一，以礼贤下士、食客众多著称，在[[邯郸之战]]中积极推动合纵抗秦。

## 养士

> 赵王封其弟胜为平原君。平原君好士，食客常数千人。
> （003-089）

## 赵奢收税

赵国田部吏[[赵奢]]向平原君家收租税，平原君家不肯缴纳，赵奢依法处置，"杀平原君用事者九人"。平原君怒欲杀赵奢，赵奢以大义劝谏："君于赵为贵公子，今纵君家而不奉公则法削，法削则国弱，国弱则诸侯加兵，是无赵也。"平原君折服，荐赵奢于赵王（005-009）。

## 毛遂自荐

秦围[[邯郸]]，平原君赴楚求救，广招门客随行，[[毛遂]]自荐参加。楚、赵盟约谈判陷入僵局，毛遂仗剑上前，胁迫楚王签约，促成合纵（005-052）。平原君感叹："胜不敢复相天下士矣！遂以毛遂为上客"（005-052）。

## 历史评价

平原君是战国贵族中热心合纵的代表，[[司马光]]记录其事迹，展现了战国末期赵国的政治风貌和合纵外交的操作实态。
