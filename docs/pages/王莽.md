# [[王莽]]

[[王莽]]（前45—23年），字巨君，魏郡元城人，汉元帝皇后之侄，西汉末年外戚专权的集大成者。以谦恭好学、廉洁谦逊之名获取人心，逐步掌握朝政，终于公元9年篡汉建立"[[新朝]]"，改元始建国，推行"托古改制"，终因改革激进、社会大乱，在[[绿林军]]攻入长安后被杀。

## 积累声望

王莽早年以外戚子弟起家，"侍曲阳侯病，因言……莽既拔出同列"（032-055），凭借荐举权贵、揭发奸谋，迅速跻身政界核心。大司马王根病重，力荐莽自代，"以莽为大司马，时年三十八"（032-055）。王莽性格谦逊，广施恩惠，"士多归者，诸侯、州牧守相皆荐莽"（033-022）。

## 逐步专权

汉哀帝死后无嗣，王莽借太皇太后王氏之名，迎立年幼的平帝，"以大司马王莽领尚书事"总揽朝政（033-022）。又荐刘歆入朝，结纳士人，为日后禅代积累学术合法性。

## 篡汉建新

"居摄"期间，王莽先称"假皇帝"，后以符命禅代之说称正统，于公元9年正式篡汉，"莽乃即真天子位，改国曰新，建元曰始建国"（035-031）。改制内容包括：恢复井田、五均六筦、废除奴婢买卖等，然"制度数变，吏民不能从，大众失业"，新政陷入全面混乱。

## 覆亡

天凤至地皇年间，旱涝连发，赤眉、绿林先后起义。更始军攻入长安，"商人杜吴杀莽，取其绶。校尉公宾就斩莽首"（039-027）。传首三处，王莽新朝仅历15年而亡。

## 历史评价

[[司马光]]记录王莽覆汉全程，以其才胜德之典型证明：以欺伪起家、以学术包装政治野心者，终必自取灭亡。王莽改制虽有理想主义色彩，但脱离现实、急于求成，是改革失败的历史教训。
