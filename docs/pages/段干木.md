# [[段干木]]

[[段干木]]，[[战国]]初期魏国著名隐士，[[魏文侯]]礼贤下士的典范人物之一，据传是[[子夏]]的学生，以道德高洁著称。

## 魏文侯礼贤

[[魏文侯]]"每过段干木之庐必式"（001-015），即每次经过段干木的住所，必定扶轼致敬，此举展示了魏文侯尊重贤者的诚意，使"四方贤士多归之"。

## 隐而不仕

段干木不求仕进，隐居不出，却因此得到诸侯的尊重。据说秦国曾因段干木在魏而不敢轻易进攻，体现了"仁义之士可以守国"的儒家理念（055-006）。

## 历史评价

《[[资治通鉴概述|资治通鉴]]》以段干木的例子说明，礼贤下士不仅是个人道德问题，更是治国之道。[[司马光]]通过魏文侯尊师重贤的记录，阐释了贤才对国家的战略价值。
