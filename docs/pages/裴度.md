# [[裴度]]

[[裴度]]（765—839），字中立，河东闻喜人，历仕宪宗、穆宗、敬宗、文宗四朝，以平淮西之功封晋国公，是中唐最重要的政治家之一。

## 平定淮西

元和十年（815年），宰相武元衡遇刺身亡，裴度亦被刺客重伤："又入通化坊击裴度，伤其首，附沟中，度氈帽厚，得不死"（239-055）。即便如此，裴度仍力主讨伐淮西：

> 或请罢度官以安恒、郓之心，上怒曰："如此则朝廷无法！卧裴度于告身上，亦当平贼，况位任如此！"
> （239-058）

裴度亲赴前线宣慰，返朝后力言"淮西必可取之状"（239-051），终使宪宗坚定平叛。唐将李愬雪夜奇袭蔡州，生擒吴元济，叛乱平定。

## 宣慰功勋

裴度每临危机，多次奔赴前线或出使藩镇："遣知制诰裴度至魏博宣慰，以钱百五十万缗赏军士，六州百姓给复一年。军士受赐，欢声如雷"（239-004），深得军心。

## 历史地位

[[司马光]]以裴度平淮西为中唐削藩的样板，强调文臣以忠毅统御武将、坚持到底的重要性，与"才德之辨"主旨相呼应。
