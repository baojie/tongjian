# [[王翦]]

[[王翦]]（?—约前221年），频阳东乡人，[[秦国]]著名将领，战国四大名将之一（与[[白起]]、[[廉颇]]、[[李牧]]并称），灭[[赵]]、[[燕]]、[[楚]]三国的主将，是秦统一六国的最大军事功臣。

## 早期征战

王翦初登历史舞台，与[[桓齮]]、[[杨端和]]联合伐赵，"取鄴，取九城"，"攻阏与、轑阳"（006-092）。

此后多次对赵作战，曾与[[李牧]]相持，遭遇顽强抵抗（006-109，006-121）。

## 灭赵

秦用反间计除掉[[李牧]]后，王翦领兵，"秦破赵，虏赵王迁"（006-121）。

## 讨要封赏自保

王翦率六十万大军伐楚，多次请求秦王赐予良田美宅，表面贪财，实为以此消除秦王对手握重兵大将的疑虑，是中国历史上著名的"功高震主"自保策略。

## 历史评价

王翦是[[秦国]]统一大业的第一功臣。[[司马光]]通过记录其征伐与自保之道，展示了[[秦国]]最后统一六国的历史进程，以及功臣在专制君权下的生存智慧。
