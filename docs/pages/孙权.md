# [[孙权]]

[[孙权]]（182—252年），字仲谋，吴郡富春人，[[孙策]]之弟，东吴开国皇帝。18岁继承江东基业，联合[[刘备]]在[[赤壁之战]]中大败[[曹操]]，确立三国鼎立之势，后称帝建立吴国，在位五十余年，庙号太祖，谥号大皇帝。

## 继承基业

孙权继位之初，"召群僚会议"，面对[[曹操]]势力的压迫，听从[[周瑜]]之策，决意联刘抗曹（064-014）。孙权平鄱阳，整合江东，"[[孙权]]西伐黄祖，破其舟军"（064-021），逐步稳固了江南地盘。

## 赤壁奠基

[[赤壁之战]]前，[[鲁肃]]向孙权分析荆州形势："荆州与国邻接，江山险固，沃野万里，士民殷富，若据而有之，此帝王之资也"（065-046），孙权由此坚定决心，与[[刘备]]结盟，以少胜多，大败曹军，确立了东吴在江南的统治基础。

## 治理东吴

孙权处置内部叛乱果断，"[[孙权]]闻乱，从椒丘还。至丹杨，悉族诛览、员馀党，擢高、婴为牙门"（064-035）。后期孙权多次争夺荆州，最终袭杀[[关羽]]，取得荆州全境。

## 历史评价

[[司马光]]记录孙权的政治智慧与军事决策，将他塑造为一个务实的君主形象。[[曹操]]曾赞"生子当如孙仲谋"。孙权善于借势，在曹、刘两强之间[[合纵连横|纵横]]捭阖，维持东吴数十年稳定，是三国君主中政治最为成熟的一位。
