# [[冉闵]]

[[冉闵]]（？—352年），字永曾，十六国时期冉魏的建立者，谥号武悼天王。他本为后赵收养的汉人，后乘后赵内乱，屠灭石氏，建立冉魏，并颁布"杀胡令"，引发中原的大规模民族冲突，最终为前燕慕容俊所败，被俘遇害。

## 屠灭石氏

后赵内乱时，冉闵趁势而起，屠灭石氏政权。前秦苻洪的谋士麻秋分析道：冉闵、石祗相持，"中原之乱未可平也。不如先取关中，基业已固，然后东争天下"（098-066），由此可见冉闵当时已是中原的核心争夺力量。

## 外交与孤立

冉闵建立冉魏后，试图联合各方，遣使于前燕，慕容俊使人诘问："[[冉闵]]，石氏养息，负恩乱常，罪自己作，与孤何干"（099-006），拒绝其外交请求，并声讨其"弃仁背义，屠灭石氏"之罪（099-005）。

面对围攻，冉闵不屈，使臣常炜慷慨陈词，慕容俊感其忠义，"彼不惮杀身而徇其主，忠臣也"（099-007），释放使臣，但拒绝和解。

## 覆灭与身后

352年，冉闵被前燕军队击败，被俘后遇害。冉魏随之覆灭，中原重归动荡。其颁布"杀胡令"引发的民族仇杀，造成北方人口的大规模减损，是十六国时期最为惨烈的历史片段之一。
