# [[子婴]]

[[子婴]]，[[秦朝]]末代君主，[[秦始皇]]之孙（一说侄），于[[赵高]]被杀后短暂执政，以秦王而非皇帝的名义执政，后向[[汉高祖刘邦]]投降，[[秦朝]]正式灭亡。

## 劝阻诛蒙氏

子婴早年曾有过政治见解，劝阻[[胡亥|秦二世]]诛杀蒙氏兄弟："二世兄子子婴谏曰：'[[王镕|赵王]]迁杀[[李牧]]而用颜聚，齐[[王建]]杀其故世忠臣而用后胜，卒皆亡'"（007-074），以历史为鉴劝谏二世，但未被采纳。

## 杀赵高

[[赵高]]设谋弑杀[[胡亥|秦二世]]后，欲以子婴为傀儡，"乃使郎中令为内应……更立子婴"（008-064）。子婴即位后，斋戒期间设计诛杀[[赵高]]，清除了[[秦朝]]最大的权奸（008-065）。

## 向刘邦投降

[[项羽]]、[[汉高祖刘邦|刘邦]]争相入关，[[汉高祖刘邦|刘邦]]先至："秦王子婴素车、白马，系颈以组，封皇帝玺、符、节，降轵道旁"（009-003）。子婴以亡国之礼向[[汉高祖刘邦|刘邦]]投降，[[秦朝]]灭亡。诸将有人主张杀秦王，[[汉高祖刘邦|刘邦]]以"始怀王遣我，固以能宽容"为由赦免子婴，但子婴后来仍被[[项羽]]所杀。

## 历史评价

子婴是[[秦朝]]最后的努力，其诛[[赵高]]一举显示了政治能力，但积重难返，无力回天（007-074，008-064，009-003）。
