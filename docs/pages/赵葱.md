# [[赵葱]]

[[赵葱]]，[[战国]]末期[[赵国]]将领，因赵王轻信郭开谗言、杀害名将[[李牧]]后出任代将，却无力抵御[[王翦]]大军，最终兵败身死，[[赵国]]随即亡国。

## 代替李牧

[[秦国]]以反间计怂恿赵王杀[[李牧]]，"赵王使赵葱及齐将颜聚代之。[[李牧]]不受命，赵人捕而杀之；废司马尚"（006-121）。赵葱接替了[[赵国]]最优秀的将领，但显然不具备相应的军事能力。

## 兵败战死

[[王翦]]随即大规模进攻，"[[王翦]]击赵军，大破之，杀赵葱，颜聚亡，遂克[[邯郸]]，虏赵王迁"（006-123）。赵葱战死，[[邯郸]]陷落，[[赵国]]灭亡。

## 历史意义

赵葱的兵败是[[李牧]]被杀直接后果的体现。[[司马光]]以此警示：明主应分辨忠佞，杀忠臣而用庸才，是亡国之道（006-121，006-123）。
