# [[晋国]]

[[晋国]]，姬姓，都绛（今山西翼城），是西周至春秋时期最强大的诸侯国之一。晋文公称霸，开"春秋五霸"之局。春秋末期，晋国公室衰微，韩、赵、魏三家大夫专权，最终瓜分晋国，"晋大夫暴蔑其君，剖分晋国，天子既不能讨，又宠秩之，使列于诸侯"（001-007），直接引发了战国格局的形成。

## 三家专权

晋国晚期，赵、韩、魏三家逐步控制国政。[[赵简子]]为赵氏奠基，其子[[赵襄子]]在[[智伯]]攻赵时反击成功，联合[[魏桓子]]、韩康子三家灭智，瓜分智氏土地，由此三家独大，晋国公室名存实亡。

## 三家分晋

周威烈王二十三年（前403年），周天子正式承认韩、赵、魏三家为诸侯，晋国实际上宣告终结。"晋国有难，而无以尹鐸为少，无以晋阳为远，必以为归"（001-009），晋阳作为赵氏根基，是这段历史的关键地理支点。

## 历史意义

[[司马光]]以晋国分裂作为《资治通鉴》的起点，表明"名分"崩坏是一切乱局的根源——天子不能守礼，诸侯大夫自然效仿僭越，直至天下大乱。
