# [[光武帝刘秀]]

[[光武帝刘秀]]（前5—57年），字文叔，南阳蔡阳人，[[汉高祖刘邦]]九世孙，[[新朝]]末年起兵，联合绿林军推翻[[王莽]]，后击败更始帝、赤眉军等各路势力，于公元25年在鄗城称帝，重建[[西汉|汉朝]]，史称"东汉"，庙号世祖，谥号光武皇帝。

## 建武立国

[[资治通鉴概述|资治通鉴]]以"世祖【光武】皇帝上之上建武元年"（040-002）为东汉纪年之始，记录光武帝重建汉室的历程。"【光武】即位之初，群雄竞逐，四海鼎沸，彼摧坚陷敌之人，权略诡辩之士，方见重于世，而独能取……"（040-022）。

## 柔道治国

光武帝即位后，深知民间疾苦，以"柔道"治国，减免赋税，释放奴婢，重视儒学，史称"建武中兴"。[[资治通鉴概述|资治通鉴]]盛赞其治：在"拨乱之后"能休养生息、重建秩序（028-071）。"观【光武】之所以取关中，用是道也。岂不美哉！"（040-075）。

## 历史评价

[[司马光]]对光武帝评价极高，以其"取天下"之道为例，说明王者之兵在于"布陈威德安民"而非一味杀伐。光武帝再造汉室，为东汉近两百年的繁荣奠定基础，是[[资治通鉴概述|资治通鉴]]中最受肯定的开国君主之一。
