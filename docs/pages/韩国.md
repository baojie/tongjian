# [[韩国]]

[[韩国]]，姬姓，[[战国]]七雄之一，由[[三家分晋]]而来，都新郑（今河南新郑），是[[战国]]七雄中国力最弱者，也是最早被[[秦国]]所灭的诸侯国（前230年）。

## 立国根基

韩国由[[韩康子]]一脉传承，"[[韩康子]]生武子，武子生虔，是为景侯"（001-014），经数代积累，景侯时正式称侯，烈侯时称王。

## 韩非的诞生

韩国虽弱，却诞生了法家思想集大成者[[韩非]]："[[韩非]]者，韩之诸公子也，善刑名法术之学，见韩之削弱，数以书干韩王，王不能用"（006-104）。[[韩非]]的思想被秦王[[秦始皇|嬴政]]奉为圭臬，讽刺的是，他为拯救弱韩所著的理论，最终成为灭韩的利刃。

## 灭亡

[[秦始皇|秦王政]]二十年（前230年），秦将内史腾率军灭韩，俘韩王安，"韩王纳地效玺，请为籓臣，使[[韩非]]来聘"（006-104）不过数年前，韩国已以投降求存，终不得免。

## 历史地位

韩国是《[[资治通鉴概述|资治通鉴]]》[[战国]]叙事的重要背景国家，从[[三家分晋]]的参与者到最先覆亡，见证了[[战国]]兼并的全程。
