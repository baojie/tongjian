# [[檀道济]]

[[檀道济]]（？—436年），刘宋开国元勋，杰出军事将领。他从刘裕起兵，历经讨平卢循、北伐后秦等重大战役，战功彪炳；文帝即位后，他参与诛杀权臣徐羡之，位至司空，坐镇南土。然而功高震主，最终遭宋文帝猜忌杀害，临刑大呼"乃坏汝万里长城"，成为历史上忠勇良将遭疑而死的著名悲剧。

## 随军讨平南方叛乱

刘道规遣使入援建康时，檀道济以天门太守身份率军参与，"荆州刺史刘道规遣司马王镇之帅天门太守【檀道济】……入援建康"（115-081）。在枝江之战中，"【檀道济】先进陷陈，谦等大败"（115-083），以前锋突击奠定胜局，展现出卓越的临阵能力。

## 北伐后秦

刘裕大举北伐时，檀道济担任冠军将军，与王镇恶同为北伐主力，"遣龙骧将军王镇恶、冠军将军【檀道济】将步军自淮、淝向许、洛"（117-064）。进军后势如破竹，"王镇恶、【檀道济】入秦境，所向皆捷"（117-070），秦军守将纷纷投降，是刘裕灭后秦战役中最关键的将领之一。

## 渡河破敌

北伐深入时，檀道济渡河作战，"【檀道济】、沈林子自陕北渡河，拔襄邑堡"（118-010），进一步扩大战果。在策略层面，沈林子劝道济放弃强攻蒲阪，"蒲阪城坚兵多，不可猝拔，攻之伤众"（118-013），二人协商配合，展现出高超的军事判断力。

## 参与顾命辅政

文帝刘义隆即位前，檀道济以护军将军身份参与顾命，"护军将军【檀道济】并入侍医药"（119-059），位列朝廷核心。此后又参与诛灭徐羡之等专权大臣，功勋益著，官至司空，是宋初最重要的武将之一。

## 历史评价

[[司马光]]通过檀道济，呈现了南朝开国武将的典型悲剧：军功足以自保，却因威望过盛而成为新君的隐忧；"万里长城"之叹，既是个人命运的控诉，也是南朝政治猜忌文化的缩影——兔死狗烹的历史逻辑，自刘裕以降已成定势。
