# [[班固]]

[[班固]]（32—92年），字孟坚，扶风安陵人，东汉著名史学家，[[班超]]之兄，撰《汉书》，是中国第一部断代正史。

## 撰《汉书》

班固继承父亲班彪的遗志，历二十余年撰成《汉书》，记载西汉一朝的历史。《[[资治通鉴概述|资治通鉴]]》多处引用班固的史论（"班固赞曰"），如评论郦寄卖友（013-060）、论汉初政局（016-121）等。

## 史学贡献

班固的史论以"班固赞曰"的形式保存在《[[资治通鉴概述|资治通鉴]]》中，作为重要的历史参照。[[司马光]]选录班固史评，足见其史学权威地位（018-071）。

## 晚年际遇

班固晚年随窦宪北伐匈奴，立燕然山铭。后因窦氏倒台被株连，死于狱中，《汉书》由其妹班昭续成。

## 历史评价

班固与[[司马迁]]并称"史汉"，是中国"正史"传统的奠基人之一。
