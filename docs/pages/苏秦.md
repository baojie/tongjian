# [[苏秦]]

[[苏秦]]，洛阳人，战国著名[[纵横家]]，与[[张仪]]同为[[鬼谷子]]弟子，以"[[合纵]]"策略著称，曾先后说服六国合纵联盟对抗秦国，位极人臣，身佩六国相印。

## 说秦失败

苏秦早年游说秦惠王以"兼天下之术"，秦王不用（002-091）。苏秦乃转而游说燕、赵、韩、魏、楚、齐等国，逐步形成[[合纵]]联盟。

## 合纵策略

苏秦对各国君主的游说极具针对性：

说燕文公："燕之所以不犯寇被甲兵者，以赵为蔽其南，若无赵，则燕首受秦患"，以地缘安全为纽带（002-091）。

说韩宣惠王："韩地方九百馀里，带甲数十万，天下之强弓、劲弩、利剑皆从韩出"，激发韩国自豪感与危机意识（002-093）。

## 与张仪的关系

[[张仪]]与苏秦"俱事鬼谷先生，学纵横之术，苏秦自以为不及也"。苏秦后来却故意激怒[[张仪]]，使其入秦，以此牵制秦国，保护合纵联盟（002-091-092）。

## 历史评价

苏秦是"[[合纵连横]]"这一外交战略的代表人物，[[司马光]]详细记录了其游说辞令，以展现战国外交博弈的精彩面貌。
