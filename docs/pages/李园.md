# [[李园]]

[[李园]]，赵国人，战国末期楚国权臣，以阴谋手段将妹妹送入楚王宫廷，成功让自己的外甥立为太子，随后在楚王薨逝之际刺杀[[春申君]]灭口，是战国末年宫廷阴谋的典型案例。

## 谋划进献妹妹

楚考烈王无子，[[春申君]]为此忧虑。赵人李园将妹妹进献给春申君，待妹妹有孕后，李园让妹妹劝说春申君将其献给楚王："楚考烈王无子，春申君患之，……乃求为春申君舍人"（006-084）。李园精心设计，最终成功让妹妹进入楚宫，所生之子被立为太子。

## 刺杀春申君

"李园妹为王后，李园亦贵用事，而恐春申君泄其语，阴养死士，欲杀春申君以灭口"（006-085）。楚王病重期间，朱英警告春申君李园必有异动，春申君不信。楚王薨，"李园果先入，伏死士于棘门之内。春申君入，死士侠刺之"（006-085），[[春申君]]被杀，李园随即诛灭春申君全家。

## 历史意义

李园的阴谋揭示了战国宫廷政治的黑暗面：权力更迭时刻，谁能先发制人、谁就能生存。[[司马光]]记此，是对权谋与义理之辨的警示（006-084，006-085）。
