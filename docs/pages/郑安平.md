# [[郑安平]]

[[郑安平]]，魏国人，与[[范雎]]命运深度交织的历史人物——既是范雎的救命恩人，也是范雎晚年失势的重要原因。

## 救范雎逃命

[[范雎]]被[[须贾]]诬陷、几乎被打死后，郑安平冒险救出范雎：

> 魏人郑安平遂操范雎亡匿，更名姓曰张禄。秦谒者王稽使于魏，范雎夜见王稽。（005-014）

郑安平帮助范雎与秦使[[王稽]]接头，让范雎得以秘密入秦，是范雎成功入秦的关键推手。

## 被举荐为将，兵败降赵

范雎在秦国功成之后，提拔郑安平为将领。然而，在围攻[[邯郸]]的战役中，"郑安平为赵所困，将二万人降赵"（005-059），战败投降。按秦国连坐之法，范雎应当受到株连，但秦昭王顾及范雎之功，不予追究。

## 历史意义

郑安平的背叛是范雎政治生涯的重大打击之一，加之[[王稽]]叛秦，范雎的举荐路子全部失败，最终影响了其在秦国的地位（005-014，005-059，006-003）。
