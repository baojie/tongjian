# [[秦国]]

[[秦国]]，嬴姓赢氏，发源于西北边陲，[[战国|战国时期]]逐步崛起为最强诸侯，最终由[[秦始皇]]一统六国。秦国的强盛是《[[资治通鉴概述|资治通鉴]]》开篇[[战国]]史的核心叙事线之一。

## 商鞅变法与崛起

[[秦孝公]]重用[[商鞅]]，推行彻底的法家变法，"[[商鞅|卫鞅]]言于[[秦孝公]]曰：'夫民不可与虑始，而可与乐成……'"（002-020），从土地制度到官爵制度全面改革，使秦国迅速跻身强国之列。

[[司马光]]记载[[秦孝公]]不失信于民，"[[秦孝公]]不废徙木之赏"（002-021），[[商鞅]]以信立法，法令得行，秦国迅速强大。

## 远交近攻

秦国采纳[[范雎]]的"[[远交近攻]]"策略，逐步蚕食邻国。[[长平之战]]（前260年）坑杀赵军四十余万，[[赵国]]由盛转衰，六国再无力抗衡秦国。

## 统一六国

[[秦始皇]]任用[[李斯]]、[[王翦]]等人，以铁腕完成统一，灭韩（前230）、魏（前225）、楚（前223）、燕（前222）、赵（前228，正式灭国前222）、齐（前221），建立[[秦朝]]，奠定中国大一统政治格局。
