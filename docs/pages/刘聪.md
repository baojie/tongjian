# [[刘聪]]

[[刘聪]]（？—318年），字玄明，[[刘渊]]之子，十六国前赵（汉赵）第三位皇帝，谥号昭武帝。他在位期间相继俘虏晋怀帝和晋愍帝，彻底终结西[[李克用|晋王]]朝，是永嘉之乱的直接推手。

## 文武兼备

刘聪颇有才学，能欣赏人才，曾赞赏[[刘琨]]："永明，汉世祖、魏武之流，数公何足道哉！"（085-062），可见其识人之明，连敌对阵营中的才士也得到他的正面评价。

## 攻灭西晋

刘聪率前赵军队多次攻打西晋，"北宫纯等与汉[[刘聪]]战于河东，败之"（086-098），虽有挫折，但整体攻势不可阻挡。[[刘琨]]曾多次请求援军共讨刘聪，"越忌苟晞及豫州刺史冯嵩，恐为后患，不许"（087-045），西晋朝廷内部的猜忌使御敌之策无从落实。

前赵军队接连攻陷[[洛阳]]和长安，先后俘虏晋怀帝和晋愍帝，西晋正式灭亡。"[[刘聪]]窃号于西北，而陛下方高让于东南，此所谓揖让而救火也"（090-035），生动描绘了晋室覆亡前夕的无力处境。

## 历史评价

刘聪终结西晋的历史功过在后世争议颇多。[[司马光]]引用[[姚崇]]驳倒儒生的论据时，以"[[刘聪]]伪主，德不胜妖"（211-094）为反证，将其定性为无德暴君，是西晋自取灭亡的催化剂。
