# [[周世宗]]

[[周世宗]]（921—959年），名柴荣，邢州尧山人，后周太祖郭威养子，在位6年（954—959年）。[[司马光]]对其评价极高，赞曰："若[[周世宗]]，可谓仁矣！不爱其身而爱民；若[[周世宗]]，可谓明矣！不以无益废有益"（292-056），是资治通鉴中少数获得如此高度肯定的皇帝。

## 即位与高平之战

世宗即位之初，北汉勾结契丹入侵，"群臣皆曰：'陛下新即位……不宜轻动，宜命将御之'"（291-090）。[[冯道]]亦力争不可御驾亲征，但世宗力排众议："朕不可不往"，亲率大军出征，以高平一战大破北汉与契丹联军，确立威信。

## 改革内政

世宗即位后大力整顿军政，淘汰老弱，训练精兵，重用能臣，推行均田、减税等改革，使后周国力迅速增强，为统一奠定基础。

## 三征南唐

世宗三次亲征南唐，夺取淮南十四州，逼迫南唐割地称臣，被称为五代中威胁最大的北方君主。北伐契丹时，不幸病重，39岁英年早逝。

## 历史评价

[[司马光]]在资治通鉴末尾专门以"臣光曰"比较后唐庄宗与周世宗，结论是："后唐庄宗、[[周世宗]]皆称英武，二主孰贤？"（294-086），以此作为资治通鉴对五代历史的最终评语，对周世宗的仁明给予最高肯定。
