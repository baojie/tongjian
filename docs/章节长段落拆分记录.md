## 一、拆分逻辑

### 1.1 两种拆分策略

| 策略 | 条件 | 格式 | 示例 |
|------|------|------|------|
| **多年节拆分** | H2 节跨越多个年号 | `### 年名·首要事件` | `### 太初三年·胶东太守延广为御史大夫` |
| **单年长节拆分** | 同一年的内容超长（约 >10 段） | `### 季节·事件` | `### 冬·淮南厉王长自作法令行于其国` |

### 1.2 标题格式

```
### <年号><年>·<事件摘要>
### <季节>·<事件摘要>
```

- 标题前半段：年（如 `天汉元年`）或季节（如 `春`、`秋`、`冬`）
- 标题后半段：从该段落的首个事件描述中提取摘要
- 中间用「·」（间隔号）分隔

### 1.3 执行过程

- 人工配合模型逐卷扫描，识别长段落序列
- 在每个年号行或季节断点处插入 `###` 标题
- 标题插入在第 1 段之前（年号行之上），而非其后
- 只操作 `docs/wiki/pages/` 下已发布的卷页面
- 一卷内若同一王有多个年号，每个新年的第一个段落前插入标题

## 二、常见问题

### 2.1 小节标题位置（已纠正 ✅）

**问题**：初稿中某些小节标题被放在了年号行之后。

**规则**：H3 标题必须放在对应的年号行（如 `[NNN-PPP] 显王三十七年`）**之前**。

```
✅ 正确：
### 从约瓦解·显王末年
[NNN-PPP] 显王三十七年
[NNN-PPP] 事件内容...

❌ 错误：
[NNN-PPP] 显王三十七年
### 从约瓦解·显王末年
[NNN-PPP] 事件内容...
```

### 2.2 年号行王名 wikilink 错误（待修复 🔴）

**问题**：一卷内跨越多位周王时，新王年号行仍链接到旧王。

**典型受影响卷**：
- **第001卷**：威烈王→安王→烈王（已修正 ✅）
- **第003卷**：慎靓王→赧王（仍有问题 ❌）

**第003卷具体问题**：
```
[003-021] 慎靓[[王元]]年    ← BUG：错误链接「王元」
```
赧王段落中的年号应改为 `[[周赧王|赧王]]X年` 格式。

**检查方法**：分节后对每卷中年号行逐一核对，确保王名与实际在位王一致。

### 2.3 标题中的垃圾字符（待修复 🔴）

部分标题携带原文的 OCR 残留字符：

| 位置 | 问题 | 示例 |
|------|------|------|
| 第049卷 | `〓〓` 残留 | `### 秋·〓〓仲长统昌言曰：光武皇帝愠` |
| 第050卷 | `++` 残留 | `### 建光元年·护羌校尉马贤召卢匆++心` |

### 2.4 标题结尾不一致（待修复 🟡）

- 标题后是否加句号/分号不统一
- 部分标题以 `；` 结尾（第001卷、第214卷、第258卷等）
- 部分标题以 `、` 结尾（截断，如 `### 建元三年·代王登、长沙王发、中山王胜、`）

### 2.5 标题摘要截断（待修复 🟡）

标题后半段的事件摘要是从原文取的首句，部分截断后不完整，甚至到句末缺少标点。

### 3.3 注意事项

- **卷第???卷.md 只读**：`wiki/public/pages/第NNN卷.md` 不可直接修改，需通过 `edit_page.py` 脚本（但当前脚本主要用于词条页，章节页的编辑需要特别处理）

- **发布同步**：`docs/wiki/pages/` 下的 H3 标题由 `publish.sh` 发布时从 `wiki/public/pages/` 同步，修改应在 `wiki/public/pages/` 进行
