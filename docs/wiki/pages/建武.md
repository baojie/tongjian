# 建武

**建武**（25—56），[[光武帝刘秀|汉光武帝]][[年号]]，共三十二年，为东汉首个年号。建武元年，刘秀即位于鄗南，改元建武，定都[[洛阳]]。此后削平群雄，先后灭[[赤眉]]、[[隗嚣]]、[[公孙述]]等，至建武十二年统一天下。建武中元二年，光武帝崩于南宫。
