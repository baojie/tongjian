# 石刻

石刻，刻于石上的文字或图像，为古代重要文献载体。

吴得小石刻"皇帝"字（080-052）。北魏太武帝见秦始皇石刻（125-088）。

石刻之制始于秦，始皇东巡，立石颂德，刻辞纪功。汉代石刻大兴，碑碣墓志遍布天下。三国吴末帝皓时，或献小石刻"皇帝"字，吴主大赦改元。及北魏太武帝南征至邹山，见秦始皇石刻，使人排而仆之，以太牢祠孔子。石刻以传之久远，为后世考史之重要依据。
