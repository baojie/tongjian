# 干宝

干宝，字令升，[[东晋]]史学家，著《晋纪》《搜神记》。

干宝论曰："昔[[汉高祖刘邦|高祖]]宣[[皇帝]]，以雄才硕量，应时而起。"（089-085）。干宝所著《晋纪》为晋史名著，其《搜神记》则为志怪小说之祖。

干宝以史才受知于[[王导]]、[[庾亮]]等，为[[东晋]]文坛重镇。
