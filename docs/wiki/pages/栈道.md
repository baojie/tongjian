# 栈道

栈道是秦岭巴山间悬崖峭壁之上架木为道的险路，为川陕之间的交通命脉。楚汉相争时，[[张良]]说[[汉高祖刘邦|汉王]]"烧绝所过栈道，以备[[诸侯]]盗兵，且示[[项羽]]无东意"（009-013），成就了"明修栈道、暗度陈仓"的千古典故。

[[三国]]时期，[[诸葛亮]][[北伐]]频繁使用栈道。[[李严]]督军[[汉中]]时，遇"天[[大雨]]三十馀日，栈道断绝"（071-043），严重影响后勤。[[安史之乱]]中，有人谏言若叛军"烧绝栈道，则[[中原]]之地拱手授贼矣"（218-015）。唐末[[王建]]修栈道以通往来（256-071），可见栈道始终是川陕战略通道的关键所在。
