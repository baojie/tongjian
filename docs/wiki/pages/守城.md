# 守城

守城，防守城池，抵御敌军进攻的防御作战。

[[蒯通]]言武信君秦县无守城（007-085）。[[周苛]]、枞公守城杀[[魏豹]]（010-019）。

守城之要在于上下同心、粮草充足。[[蒯通]]说武信君，谓秦政虐民，各县无守城之意，若不攻而降城，[[传檄]]而千里定。[[项羽]]围[[荥阳]]，汉将[[周苛]]、枞公守城，恐[[魏豹]]反覆，遂杀之。守城之法，[[坚壁]][[清野]]、外结援兵、内抚[[士卒]]，三者不可缺一。历代名将守城，如[[张巡]]守[[睢阳]]，皆以少御众，为世所称。
