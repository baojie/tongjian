# 永始

**永始**（前16—前13），[[汉成帝]]第五个[[年号]]。永始年间，成帝宠[[赵飞燕]]、[[赵合德]]姐妹，立飞燕为皇后。[[王莽]]始封新都侯，折节恭俭，名誉日盛。[[淳于长]]以谋立赵皇后功封侯，骄奢淫逸，后以谋立太子事被诛。[[刘向]]著《列女传》、《新序》、《说苑》上之。
