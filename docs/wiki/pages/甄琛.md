# 甄琛

甄琛，字思伯，[[中山]]无极人。[[北魏]]御史中尉、[[河南]]尹，以执法严刻、附附权势知名。

[[尚书]][[清河]][[张彝]]、[[邢峦]]闻处分非常，亡走，出[[洛阳]]城，为御史中尉[[中山]]甄琛所弹（144-007）。[[侍中]]、领御史中尉甄琛、黄门郎李凭、[[廷尉]]卿阳平王显，素皆谄附于修，至是惧相连及，争助肇攻之（145-089）。初，魏御史中尉甄琛表称："《[[周礼]]》，山林川泽有虞、衡之官，为之厉禁，盖取之以时。"（146-051）

甄琛在宣武帝朝以御史中尉掌监察，附从权臣[[高肇]]，与[[高肇]]合谋陷害[[彭城]]王[[元勰]]。琛又[[上书]]请弛盐禁，宣武帝从之，公私称便。甄琛虽有政才，然以附丽权贵、构陷[[宗室]]为人所讥。
