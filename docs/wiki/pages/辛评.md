# 辛评

辛评，颍川人，袁绍谋臣。

袁绍使高幹及辛评、荀谌、郭图等说韩馥让冀州（060-011）。郭嘉初见袁绍，居数十日，谓绍谋臣辛评、郭图曰："袁公徒欲效周公之下士，而不知用人之机，难矣哉！"遂去之（062-015）。

官渡之战后，郭图、辛评言于袁绍曰审配在位专政，族大兵强，且二子在南，必怀反计（063-032）。辛评兄弟后皆死于邺城之乱。
