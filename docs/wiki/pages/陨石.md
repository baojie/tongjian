# 陨石

**陨石**指坠落地面的流星体残骸，古代史书中多录为灾异之兆。

## 陨石为异

秦有陨石于东郡，或刻其石曰："始皇死而地分"，始皇使御史逐问（007-065）。陨石刻文，起人心之变。

汉世屡记陨石：雍县无云如雷者三，陨石二，黑如黳（022-069）；陨石于梁（029-024）；陨石于亳四（030-028）；陨石东郡八（031-003）。

王莽时陨石于巨鹿二（035-053）。陨石频见，史官录之以为天象。
