# [[青海]]

[[青海]]，古称[[西海]]、鲜水海，位于[[青藏高原]]东北部，是中国最大的内陆[[咸水湖]]。[[唐代]]青海地区是[[吐蕃]]与[[唐朝]]争夺的军事要地，[[大非川]]战役即发生在青海附近（201-027），唐将[[薛仁贵]]在此兵败于吐蕃。

青海湖周边水草丰美，自古为[[羌人|羌]]、[[鲜卑]]、[[吐谷浑]]、吐蕃等游牧民族的活动区域。[[隋]][[唐]]时期，中原王朝多次经略青海，设[[西海郡]]以控制该区域。
