# [[青海]]

[[青海]]，古称[[西海]]、鲜水海，位于[[青藏高原]][[东北]]部，是[[中国]]最大的内陆[[咸水湖]]。[[唐代]]青海地区是[[吐蕃]]与[[唐朝]]争夺的[[军事]]要地，[[大非川]][[战役]]即发生在青海附近（201-027），唐将[[薛仁贵]]在此兵败于[[吐蕃]]。

青海湖周边水草丰美，自古为[[羌人|羌]]、[[鲜卑]]、[[吐谷浑]]、[[吐蕃]]等游[[牧民]]族的活动区域。[[隋]][[唐]]时期，[[中原]]王朝多次[[经略]]青海，设[[西海郡]]以控制该区域。
