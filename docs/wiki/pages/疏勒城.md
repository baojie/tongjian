# [[疏勒城]]

[[疏勒城]]是[[东汉]]时期[[西域]][[车师]]国都城，位于今[[新疆]]奇台一带。[[汉明帝]]永平年间，[[耿恭]]以疏勒城傍有涧水可固，引兵据之（045-088）。[[匈奴]]来攻，断绝涧水，耿恭于城中穿井十五丈不得水，吏士渴乏，至榨马粪汁而饮之。

耿恭率部在疏勒城坚守经年，以寡敌众，最终仅余十三人得还[[玉门关]]。耿恭守疏勒城是东汉经营西域史上最悲壮的战役之一，体现了汉军将士的忠诚与坚韧。
