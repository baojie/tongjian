# 赵绾

赵绾，西汉御史大夫，儒者。

武帝初立，窦婴为丞相，田蚡为太尉，俱好儒术，推毂赵绾为御史大夫（017-014）。赵绾请立明堂以朝诸侯，且荐其师申公。后窦太后好黄老，不悦儒术，赵绾请毋奏事东宫，触怒窦太后，与王臧皆下吏自杀（017-019、017-020）。
