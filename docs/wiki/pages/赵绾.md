# 赵绾

赵绾，[[西汉]][[御史大夫]]，[[儒者]]。

[[武帝]]初立，[[窦婴]]为[[左丞相|丞相]]，田蚡为[[太尉]]，俱好[[儒术]]，推毂赵绾为[[御史大夫]]（017-014）。赵绾请立[[明堂]]以朝[[诸侯]]，且荐其师[[申公]]。后[[窦太后]]好黄老，不悦[[儒术]]，赵绾请毋[[奏事]][[太子詹事|东宫]]，触怒[[窦太后]]，与[[王臧]]皆下吏自杀（017-019、017-020）。

## 世系

赵绾，[[西汉]][[儒者]]。从[[申公]]受《[[诗经|诗]]》。[[汉武帝]]初立，[[窦婴]]、[[田蚡]]推毂绾为[[御史大夫]]（017-014）。绾请立[[明堂]]，触怒[[窦太后]]，下吏自杀（017-019、017-020）。绾[[子孙]]不详于史。
