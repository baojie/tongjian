# 策书

策书，帝王用于册封、责让、免黜之正式文书。较诏书更为郑重，多用于重大人事变动。

汉成帝赐车骑将军王音策书，责让其"何甘乐祸败"，以其纵容王氏奢僭（031-035）。哀帝罢免大司空师丹时，尚书令唐林以策书言辞过深，上疏谏曰"丹经为世儒宗，亲傅圣躬"，请为贤者讳（033-048）。王莽居摄，作铜匮金策书，署曰"赤帝玺某传予黄帝金策书"，托符命以篡汉（036-067）。

> 又赐车骑将军音策书曰："外家何甘乐祸败！而欲自黥、劓，相戮辱于太后前，伤慈母之心，以危乱国家！"
> （031-035）
