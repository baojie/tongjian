# 和峤

和峤，字长舆，[[汝南]][[西平]]人。[[西晋]]名臣，以清直著称。

和峤为[[侍中]]，深受[[武帝]]信任。[[武帝]]欲责[[王济]]，谓峤曰："我将骂济而后官之，如何？"峤对曰："济俊爽，恐不可屈。"（081-081）后拜[[太子]]少保（082-032）。峤家富性吝，[[杜预]]讥其有钱癖。然峤清直有识鉴，朝野推重。

> 帝谓侍中和峤曰："我将骂济而后官之，如何？"峤曰："济俊爽，恐不可屈。"
