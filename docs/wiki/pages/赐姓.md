# 赐姓

赐姓是皇帝赐予功臣或归附者以国姓或其他姓氏的恩荣制度，以示褒宠和笼络。

[[汉高祖刘邦|刘邦]]在楚汉战后，封[[项伯]]等四人为列侯，赐姓刘氏（011-005）。又因[[娄敬]]建议定都长安、[[和亲]][[匈奴]]之功，拜其为郎中，赐姓刘氏（011-021）。[[西汉]]成帝时，班婕妤进侍者李平得幸，亦赐姓曰卫（031-037）。唐代赐姓更盛，[[李勣|徐世勣]]赐姓李氏改名[[李勣]]、[[安禄山]]赐姓等，赐姓成为笼络归附者、增强认同感的重要政治手段。

## 历代赐姓

汉初赐姓以安抚降将和褒奖功臣为主，[[项伯]]赐姓刘氏开创先例（011-005）。[[娄敬]]因献策定都长安，亦得赐姓刘氏（011-021）。[[东汉]]以后赐姓制度继续发展，南北朝时赐姓尤滥，[[宇文泰]]推行[[府兵制]]时广赐[[鲜卑]]姓氏给汉人将领。唐代赐姓最为广泛，归附的蕃将、有功的臣子多被赐以国姓李氏。宋代以后赐姓渐少，但少数民族政权如辽、金、元亦多行此制。

> 封项伯等四人皆为列侯，赐姓刘氏；诸民略在楚者皆归之。
> （011-005）

> 拜娄敬为郎中，号曰奉春君，赐姓刘氏。
> （011-021）
