# 壁垒

壁垒，军营之围墙或防御工事，为固守待敌之要具。

汉王深沟壁垒拒楚（010-008）。司马昭保壁垒以积实资虏（076-048）。

壁垒为军中之保障。楚汉相争，汉王收诸侯还守成皋，深沟壁垒，分卒守徼乘塞，楚人深入敌国八九百里，老弱转粮千里之外。司马昭征毌丘俭，欲保壁垒以积实资虏，王基谏曰："持重非不得之谓也，进而不可犯耳。"壁垒固重要，然不可因守而失机。

> （010-008）
