# 衣冠

衣冠指衣帽[[服饰]]，古代[[服饰]]制度严格体现身份等级。衣冠亦代指[[士族]]、文明礼仪。

[[信陵君]]置酒大会[[宾客]]，侯生摄敝衣冠直上载公子上坐不让（005-055）。[[叔孙通]]谏[[汉惠帝]][[复道]]，言此高帝月出游衣冠之道，子孙奈何乘[[宗庙]]道上行哉（012-084）。

衣冠制度在古代社会具有重要象征意义，[[冠冕]][[衣裳]]的式样、颜色、材质皆有等级规定。衣冠南渡指南晋[[士族]]南迁，"衣冠"成为[[士族]]与[[华夏]]文明的代称。衣冠之治是古代礼制的集中体现。

## 衣冠与礼制

衣冠为身份等级之外在标志，[[帝王]]赐衣冠即为恩宠之象征。[[武帝]]见董偃于窦太主家，赐之衣冠尊而不名（018-027），由是董君贵宠天下莫不闻。[[匈奴]]降者[[金日磾]]为[[武帝]][[养马]]，马肥好，帝异之，即日赐汤沐衣冠拜为马监（019-052），衣冠之赐昭示降虏步入汉廷。[[武帝]]至桥山问[[黄帝]]冢，[[公孙卿]]对以[[黄帝]]仙去群臣葬其衣冠（020-082）。[[暴胜之]]至勃海闻[[隽不疑]]贤，见不疑容貌尊严衣冠甚伟，履屐起迎（021-104）。衣冠由[[服饰]]而礼仪而人物品鉴，成为古代社会身份区分与人伦评价的重要载体。
