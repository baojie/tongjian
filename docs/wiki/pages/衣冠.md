# 衣冠

衣冠指衣帽服饰，古代服饰制度严格体现身份等级。衣冠亦代指[[士族]]、文明礼仪。

[[信陵君]]置酒大会宾客，侯生摄敝衣冠直上载公子上坐不让（005-055）。[[叔孙通]]谏汉惠帝复道，言此高帝月出游衣冠之道，子孙奈何乘[[宗庙]]道上行哉（012-084）。

衣冠制度在古代社会具有重要象征意义，[[冠冕]]衣裳的式样、颜色、材质皆有等级规定。衣冠南渡指南晋[[士族]]南迁，"衣冠"成为[[士族]]与华夏文明的代称。衣冠之治是古代礼制的集中体现。

> 侯生摄敝衣冠，直上载公子上坐不让，公子执辔愈恭。
> （005-055）
