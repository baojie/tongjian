# 李弼

李弼，字景和，辽东襄平人。北周柱国大将军，从[[宇文泰]]创业，为西魏北周元勋。

魏南秦州[[刺史]][[陇西]]李弼说侯莫陈悦曰："贺拔公无罪而公害之，又不抚纳其众，今奉宇文夏州以来，声言为主报仇。"（156-047）弼知悦必败，阴[[遣使]]诣泰，请为[[内应]]。[[宇文泰]]令李弼镇原州（156-051）。

李弼原为侯莫陈悦部将，在[[宇文泰]]与侯莫陈悦之争中，识时务而归泰，为[[宇文泰]]平定关陇的重要助力。西魏大统年间，李弼以功进柱国大将军，为[[府兵制]]八柱国之一。李弼用兵持重，善抚将士，子孙皆为周、隋名臣。其曾孙[[李密]]，为隋末瓦岗军领袖。
