# [[松州]]

[[松州]]是[[唐代]]设置的州，治所在今[[四川]]松潘，隶属[[剑南道]]。松州地处[[岷江]]上游，扼守[[青藏高原]]东缘，是[[唐朝]]与[[吐蕃]]交界地带的军事重镇。

[[唐高祖]]武德年间，[[吐谷浑]]、[[党项]]多次寇扰松州（190-093、190-166）。[[贞观]]年间，唐朝与吐蕃在松州一带多有交战。松州是唐代[[剑南道]]西北边防的前沿阵地，也是茶马古道上的重要节点。
