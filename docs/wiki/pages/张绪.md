# 张绪

张绪，南齐中书令、国子祭酒。以风仪清简著称。

以太子詹事张绪为中书令（135-023）。诏置学生二百人，以中书令张绪为国子祭酒（135-124）。太子詹事张绪领扬州中正，长沙王晃属用吴兴闻人邕为州议曹，绪不许。晃使书佐固请，绪正色曰："此州郡体，宁有榜白之时邪！"（136-132）。

> 以太子詹事张绪为中书令，齐国左卫将军陈显达为中护军。
> （135-023）
