# 京口

京口，六朝军事重镇，今江苏镇江。[[《资治通鉴》]] 载京口兵家必争之事。

[[祖逖]]居京口，纠合骁健，言于睿曰："晋室之乱，非上无道而下怨叛也，由[[宗室]]争权，自相鱼肉"（088-077）。[[庾亮]]以二千人守白石，[[苏峻]]遣军掠京口（094-035）。[[郗鉴]]在京口拒[[苏峻]]（094-044）。[[褚裒]]镇京口（097-011, 098-044）。郗愔镇京口，[[桓温]]常言"京口酒可饮，兵可用"（101-140）。京口要地，去[[都邑]]密迩，自非[[宗室]]近亲，不得镇之（120-023）。[[刘裕]]起自京口，遂成帝业（111-062）。
