# 李从厚

李从厚，后唐闵帝，明宗李嗣源幼子，在位仅四月即为李从珂所废。

以皇子从厚同平章事，充河南尹，判六军诸卫事（275-077）。以从厚为宣武节度使，仍判六军诸卫事（276-040）。皇子从厚纳孔循女为妃（276-073）。朝廷之人皆推从厚而短从荣（276-080）。

## 世系

李从厚（914—934年），[[后唐]]闵帝。父[[李嗣源]]（后唐明宗）。从厚为明宗幼子，封[[宋王]]，即位仅四月即为[[李从珂]]所废，被弑。
