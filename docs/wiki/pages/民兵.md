# 民兵

民兵[[是非]]正规的地方武装力量，于战时临时征发、平时归农，为古代[[兵制]]的重要补充。

[[隗嚣]]将郑兴劝嚣勿自立，言[[西州]]民兵离散，易以辅人难以自建（042-019）。[[马希崇]]募民为兵，澧纠民为兵，有言其咨怨者，澧悉集民兵于[[开元]]寺，绐云犒享，入辄杀之（267-074）。[[周世宗|后周世宗]]以四方分争宜教民战，诏[[团结]]民兵使之习战自卫[[乡里]]（271-010）。

民兵之制历代皆有，汉之[[材官]]、唐之[[团结]]兵、宋之[[乡兵]]、明之民壮皆属之。民兵不脱生产，战时征发，事讫归农，为[[国家]]正规军之重要辅助。

> 今四方分争，宜教民战。且善人畏法禁而奸民弄干戈，是欲偃武而反招盗也。宜团结民兵，使之习战，自卫乡里。
> （271-010）
