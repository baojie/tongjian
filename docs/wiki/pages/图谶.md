# 图谶

图谶是[[谶纬]][[图书]]，以预言[[吉凶]]的图文为特征，盛行于[[两汉]]之际，[[王莽]]、[[光武帝刘秀|光武帝]]皆借图谶论证其政权合法性。

[[王莽]]网罗天下异能之士，通图谶、钟律、[[月令]]、[[兵法]]者皆诣公车（036-017）。蔡少公颇学图谶，言"[[刘歆|刘秀]]当为[[天子]]"（038-074）。[[公孙述]]屡[[移书]][[中国]]，自陈[[符命]]，光武与之书曰"图谶言公孙，即[[司马懿|宣帝]]也"（042-006）。

## 图谶与东汉政治

[[光武帝刘秀|光武帝]]以图谶定天下，政治上倚图谶为决策依据。帝与郑兴议[[郊祀]]，欲以谶断之，兴对"臣不为谶"，帝怒（042-033）。后起[[明堂]]、[[灵台]]、[[辟雍]]，宣布图谶于天下，[[桓谭]]上疏谏曰图谶成于哀平之际，乃虚伪之徒以要世取资（044-072）。淮阳王延与姬兄谢弇作图谶祠祭[[祝诅]]，事下案验，弇及[[司徒]]邢穆皆坐死，所连及死者甚众（045-071）。[[张衡]]上疏请禁图谶，谓图谶成于哀平之际，皆虚伪之徒以欺罔取资（052-005）。[[前秦]]时禁老庄图谶之学，犯者[[弃市]]（103-092）。图谶在[[两汉]]之际盛极一时，至隋唐始遭禁绝，然其政治影响力延及后世。
