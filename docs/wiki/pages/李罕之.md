# [[李罕之]]

[[李罕之]]（？—901年），项城人，唐末五代将领。他原为[[黄巢]]部将，降唐后辗转依附诸葛爽；以[[洛阳]]为据点与[[张全义]]刻臂为盟、相得欢甚；以贪暴轻视[[张全义]]之"田夫"作风，最终引发[[张全义]]背盟；在[[洛阳]]争夺中失败后投奔河东，是五代乱世中"勇而无谋、性复贪暴"这一武将类型的典型代表。

## 以黄巢部将起家，降唐入仕

李罕之以反叛者身份进入正规藩镇体系，"镇海[[节度使]][[高骈]]遣其将张璘、梁缵分道击[[黄巢]]，屡破之，降其将秦彦、毕师鐸、【李罕之】、许勍等数十人"（253-066），以降将身份进入战后格局，辗转成为诸葛爽麾下的怀州[[刺史]]。

## 坚守洛阳，兵尽食绝仍不降

李罕之在困境中展示出军事韧性，"[[洛阳|东都]]留守【李罕之】与[[秦宗权]]将孙儒相拒数月，罕之兵少食尽，弃城，西保渑池"（256-040），在粮食耗尽的极端条件下坚持抵抗数月，展现出作为武将的顽强本色。

## 与张全义刻臂为盟

李罕之以结义方式建立同盟，"初，河阳[[节度使]]【李罕之】与[[河南]]尹[[张全义]]刻臂为盟，相得欢甚"（257-072），以割臂混血的方式与[[张全义]]缔结了盟友关系，但随即暴露出对[[张全义]]价值观的根本轻视，"罕之勇而无谋，性复贪暴，意轻全义，闻其勤俭力穑，笑曰：'此田'"，以鄙夷的态度看待[[张全义]]务农的战略，埋下了最终反目的祸根。

## 历史评价

[[司马光]]通过李罕之，展示了五代粗豪武将的根本局限：以勇力立身而轻视经营，以贪暴取财而失去人心，以嘲笑"此田夫"而最终落败于"田夫"之手——这是乱世中纯粹的军事力量在面对耐心的经济建设时必然失败的历史规律。李罕之的失败，不仅仅是一个军事事件，更是五代"重武轻文"文化内在矛盾的具体演示。
