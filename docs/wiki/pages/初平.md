# 初平

**初平**（190—193），[[汉献帝]]第一个[[年号]]。初平元年，[[董卓]]挟献帝[[迁都]][[长安]]，烧[[洛阳]]宫庙，驱民数百万口西迁，积尸盈路。[[关东]]州郡[[起兵]]讨卓，推[[袁绍]]为盟主，[[曹操]]、[[袁术]]、[[韩馥]]等各[[拥兵]]。[[王允]]与[[吕布]]合谋诛[[董卓]]。[[李傕]]、[[郭汜]]反攻[[长安]]，杀[[王允]]。
