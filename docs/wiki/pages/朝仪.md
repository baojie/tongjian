# 朝仪

朝仪是朝廷礼仪制度的总称，包括[[朝会]]、祭祀、庆典等场合的礼仪规范。

汉[[汉高祖刘邦|高祖]]初定天下，群臣饮酒争功醉或妄呼拔剑击柱，[[叔孙通]]说上征鲁诸生与其弟子共起朝仪（011-043）。[[慕容廆]]以游邃为龙骧长史，命邃创定府朝仪法（090-039）。后燕[[慕容垂]][[改元]]建兴，服色朝仪皆如旧章（105-041）。

[[叔孙通]]所定朝仪以古礼与秦仪参酌而成，奠定了汉代朝仪的基础。朝仪之设立，使君臣名分制度化，是古代政治从草创走向规范的重要标志。

> 叔孙通说上曰："夫儒者难与进取，可与守成。臣愿征鲁诸生，与臣弟子共起朝仪。"
> （011-043）
