# 胡僧祐

胡僧祐，字愿果，南阳人。梁朝武猛将军。

[[侯景之乱]]中，湘东王萧绎遣天门太守胡僧祐精甲二万赴援（164-025）。后晋州[[刺史]]萧惠正辞不堪任，举胡僧祐自代。僧祐时坐忤旨系狱，绎即出之，拜武猛将军，令赴援（164-027）。僧祐与[[王僧辩]]等共平[[侯景]]，进位号（164-029）。

> 湘东王绎遣晋州刺史萧惠正将兵援巴陵，惠正辞不堪，举胡僧祐自代。
> （164-027）
