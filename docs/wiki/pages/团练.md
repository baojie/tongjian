# 团练

团练是唐代地方[[民兵]]组织，亦称团练兵，多由地方百姓临时征集而成，其战斗力不及正规军。

张宥为[[剑南]][[节度使]]，以军政委团练副使[[章仇兼琼]]（214-124）。[[颜杲卿]]用计谓张献诚所将多团练之人，无坚甲利兵（217-054）。[[常山]]团练兵三千人杀[[胡兵]]，执安思义出降（217-071）。[[李光弼]]留[[常山]]太守王俌将团练兵守[[常山]]（218-024）。[[史思明]]攻[[晋阳|太原]]，[[李光弼]][[麾下]][[精兵]]皆赴[[朔方]]，馀团练乌合之众不满万人（219-039）。

[[安史之乱]]后，团练成为地方防御的重要力量，各州置[[团练使]]，渐成藩镇体系之组成部分。团练兵虽装备训练不及正规军，但于[[守城]]御敌多有功绩。

> 足下所将多团练之人，无坚甲利兵，难以当山西劲兵。
> （217-054）
