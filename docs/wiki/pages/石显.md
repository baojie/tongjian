# 石显

石显，字君房，济南人。西汉中书令，元帝时宦官专权的代表人物。与弘恭相表里，把持朝政。

中书令弘恭、仆射石显，自宣帝时久典枢机，明习文法。帝即位多疾，以显久典事，中人无外党（028-015）。上复征周堪、刘更生，欲以为谏大夫；弘恭、石显白，皆以为中郎（028-027）。

石显在元帝时以中书令专权，与弘恭互为表里，排挤萧望之、周堪、刘更生等儒臣。石显为人巧佞，善窥人主之意。元帝病笃，石显专断朝政，公卿畏之。成帝即位后，石显失势，徙为济南太守，不食而死。石显以宦官干政，为西汉中后期政治败坏的重要推手。
