# 斥候

斥候是古代军事中的侦察兵，负责远出侦察敌情、地形，为军队行动提供情报。

[[李广]][[行军]]无部伍行陈，然亦远斥候，未尝遇害（017-072）。汉遣[[屯田]]卒置校尉，张掖酒泉遣骑假司马为斥候（022-072）。[[赵充国]]常以远斥候为务，行必为战备，止必坚营壁（026-011）。

斥候之设古已有之，《左传》即有"斥候"之载。汉代边郡置斥候，[[烽燧]]相望。斥候之远近、多寡，直接关系军队安全，为将者必以斥候为先。

> 充国常以远斥候为务，行必为战备，止必坚营壁，尤能持重，爱士卒，先计而后战。
> （026-011）
