# 南越

南越为秦汉之际[[赵佗]]所建政权，都番禺（今广州）。秦末[[赵佗]]据南海郡自立，汉[[汉高祖刘邦|高祖]]遣[[陆贾]]立佗为南越王，与剖符[[通使]]（012-037）。[[吕后]]时与南越交恶，[[赵佗]]遂自称南越武帝（012-068）。

[[汉文帝]]使[[陆贾]]再使南越，[[赵佗]]去帝号称臣（013-059）。[[汉武帝]]时南越相吕嘉反，武帝遣[[路博德]]、[[杨仆]]等平南越，置九郡（020-052）。南越自[[赵佗]]立国至[[汉武帝]]灭亡，共历五世九十馀年。

## 秦开南越

[[秦始皇]]统一六国后，发诸逋亡人、赘婿、[[贾人]]为兵，略取南越陆梁地，置桂林、南海、象郡（007-053），以谪[[徙民]]五十万人戍五岭，与越杂处。又谪治狱吏不直者筑长城及处南越地（007-056）。秦末大乱，[[赵佗]]诛秦所置长吏，击并桂林、象郡，自立为南越武王（012-037）。[[吕后]]时禁南越关市铁器（013-026），佗遂自称南越武帝，发兵攻长沙（013-028）。汉文再遣[[陆贾]]使越，佗去帝号称臣。南越地处岭外，然自秦至汉，始终与[[中原]]声教相通。
