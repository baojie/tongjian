# 王温舒

王温舒（生卒年不详），[[西汉]]武帝时酷吏。始为广平都尉，择郡中豪敢往吏十馀人以为爪牙，皆把其阴重罪而纵使督盗贼（019-080）。后以河内太守迁为中尉（019-079）。

> 王温舒始为广平都尉，择郡中豪敢往吏十馀人，以为爪牙，皆把其阴重罪，而纵使督盗贼。快其意所欲得，此人虽有百罪，弗法。
> （019-080）

王温舒执法极其严厉，郡中奸猾无所容。是时[[赵禹]]、[[张汤]]以深刻为九卿，义纵以鹰击为治，王温舒亦以酷烈闻名，同属武帝朝酷吏群体。
