# 周兴

周兴，雍州长安人。唐酷吏，以残忍著称。

周兴为尚书都事，与来俊臣之徒效仿索元礼，纷纷继起。兴累迁至秋官侍郎（203-135）。太后有憾于处俊，会奴诬告象贤反，太后命周兴鞫之，致象贤族罪。象贤家人讼冤于监察御史任玄殖，玄殖奏象贤无反状，反被诛（204-023）。太后使周兴等按韩王元嘉等谋反案，迫胁皆自杀（204-036）。后来俊臣告周兴谋反，兴被流放岭南而死。

> 尚书都事长安周兴、万年未来俊臣之徒效之，纷纷继起。兴累迁至秋官侍郎。
> （203-135）
