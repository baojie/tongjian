# 周兴

周兴，[[雍州]][[长安]]人。唐[[酷吏]]，以残忍著称。

周兴为[[尚书]]都事，与[[来俊臣]]之徒效仿[[索元礼]]，纷纷继起。兴累迁至秋官[[侍郎]]（203-135）。[[太后]]有憾于处俊，会奴[[诬告]]象贤反，[[太后]]命周兴鞫之，致象贤族罪。象贤家人讼冤于监察[[御史]]任玄殖，玄殖奏象贤无反状，反被诛（204-023）。[[太后]]使周兴等按[[韩王]][[元嘉]]等[[谋反]]案，迫胁皆自杀（204-036）。后[[来俊臣]]告周兴[[谋反]]，兴被[[流放]][[岭南]]而死。

> 尚书都事长安周兴、万年未来俊臣之徒效之，纷纷继起。兴累迁至秋官侍郎。
> （203-135）
