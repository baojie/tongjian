# 周兴

周兴，雍州长安人。唐酷吏，以残忍著称。

周兴为尚书都事，与[[来俊臣]]之徒效仿[[索元礼]]，纷纷继起。兴累迁至秋官侍郎（203-135）。[[太后]]有憾于处俊，会奴诬告象贤反，[[太后]]命周兴鞫之，致象贤族罪。象贤家人讼冤于监察御史任玄殖，玄殖奏象贤无反状，反被诛（204-023）。[[太后]]使周兴等按韩王元嘉等谋反案，迫胁皆自杀（204-036）。后[[来俊臣]]告周兴谋反，兴被流放[[岭南]]而死。

> 尚书都事长安周兴、万年未来俊臣之徒效之，纷纷继起。兴累迁至秋官侍郎。
> （203-135）
