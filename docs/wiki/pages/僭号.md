# 僭号

僭号指超越本分、擅自称王[[称帝]]的行为，是古代政治中对僭越名号者的贬称。

秦末[[陈胜]]称王，被妻父斥为"怙乱僭号，而傲长者，不能久矣"（008-010），认为[[陈胜]]趁乱越分称王必将不永。东[[光武帝刘秀|汉光武帝]]时，鉴于诸侯王及四夷多有越分称王之举，遂下诏"四夷僭号称王者皆更为侯"（037-006），将擅自称王者降爵为侯。

[[东汉]]中后期，边疆豪帅常有僭号之举，朝廷往往以安抚为主、征讨为辅。如顺帝时建议对边地叛乱的策略中提出，其叛戾乃因"含怨呼天，无所叩诉"，并非"谋主僭号以图不轨"，主张选任贤能牧守即可（058-025）。
