# 禳

禳，[[祭祀]]祈祷以消灾除祸的巫术仪式。

[[高堂隆]]以祈禳对魏[[汉明帝|明帝]]（073-012）。[[朝议]]以禳避之道不可取（097-007）。苻生以修德禳[[星变]]（100-016）。

禳为古人应对[[灾异]]的主要手段之一。魏[[汉明帝|明帝]]时崇华殿灾，问[[高堂隆]]以祈禳之义。隆引《易传》对曰："上不俭，下不节，孽火烧其室。"[[汉明帝|明帝]]乃修德以应之。[[前秦]]苻生时，星象示变，有司请修德以禳之，苻生竟曰[[皇后]]可应丧，[[暴虐]]益甚。禳之本意在于修德省身，而后世或以巫术代之，失其本义。

> 高堂隆以祈禳对魏明帝。
> （073-012）
