# 成都

成都是[[蜀郡]]治所，[[益州]]政治经济中心，[[蜀汉]]、前[[后蜀]]都城，自古有"天府之国"之称。

[[陈胜]]起义时，封[[张耳]]子敖为成都君（007-096），可见成都之名在秦末已显。[[汉成帝]]封舅[[王商]]为成都侯，同日封者五人，世谓之"五侯"（030-050）。[[西汉]]末成都侯[[王商]]尝病，欲避暑，从上借明光宫；后又穿[[长安]]城，引沣水注第中大陂以行船（031-035）。成都自[[秦]]置县以来，即为[[蜀郡]]治所。

> 封张耳子敖为成都君，使使者贺赵，令趣发兵西入关。
> （007-096）

成都地处[[成都平原]]，西部[[岷江]]之利，[[李冰]]修[[都江堰]]以来，[[水旱]]从人，不知[[饥馑]]。[[三国]]时为[[蜀汉]]之都，[[诸葛亮]]治蜀，成都繁荣甲于南方。[[安史之乱]]后玄宗幸蜀，成都一度为唐行在。成都作为[[西南]]政治经济中心，在《[[资治通鉴概述|资治通鉴]]》中为蜀地相关史事之核心地标。
