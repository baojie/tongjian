# 成都

成都是蜀郡治所，[[益州]]政治经济中心，[[蜀汉]]、前后蜀都城，自古有"天府之国"之称。[[陈胜]]起义时，封[[张耳]]子敖为成都君（007-096），可见成都之名在秦末已显。[[汉宣帝]]时"[[侍中]]成都侯金安上"亦以成都为名（025-056）。

[[西汉]]末王氏五侯争以奢侈相尚，成都侯王商尝病，欲避暑，从上借明光宫；后又穿长安城，引沣水注第中大陂以行船（031-035）。王商后为大司马、卫将军（031-061），为成帝朝重臣。[[王莽]]之兴亦与成都侯王商[[上书]]分户邑以封莽有关（031-046）。
