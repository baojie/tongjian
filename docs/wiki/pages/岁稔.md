# 岁稔

**岁稔**指谷物丰收，年成好。[[资治通鉴概述|通鉴]]中岁稔与[[国家]][[仓储]]政策密切相关。

## 常平仓

[[唐朝|唐代]][[常平仓]]之法，岁稔伤农，则增价籴谷以贮之（211-041）。岁稔[[谷贱]]伤农，故[[和籴]]之法增时价什[[二三]]以收粟（214-077）。

## 仓储之弊

然有司竞为苟且，岁稔则不时敛藏，艰食则抑使收籴，遂使豪家[[贪吏]]反操利权（234-020）。良法美意，行久则弊生。
