# 邀遮

**邀遮**指拦阻截击敌军，为古代[[设伏]]中途、阻挡去路的[[军事]]行动。

## 邀遮敌军

诸郡各[[发兵]]邀遮散叛[[羌人]]（049-042）。敕曶、绰于[[南安]]峡口邀遮，元军败果顺[[江东]]下被生获（070-009）。[[突厥]]伺隙邀遮唐军（209-021）。[[张仲武]]谕奚[[契丹]]共邀遮[[回鹘]]那颉啜（246-095）。张璘于郴州[[守险]]邀遮[[黄巢]]（253-079）。
