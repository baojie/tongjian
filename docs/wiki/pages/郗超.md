# 郗超

郗超，字景兴，[[高平]]金乡人。[[东晋]][[名士]]，[[桓温]]谋主，参决[[朝政]]机要。[[郗鉴]]之孙，郗愔之子。

[[桓温]]辟郗超为抚军掾（097-060）。[[桓温]]欲伐燕，郗超曰："道远，[[汴水]]又浅，恐[[漕运]]难通。"温不从，后果败（102-005）。[[桓温]]枋头败后，郗超劝温行废立之事（103-023）。凡[[桓温]]军国大事，必与郗超谋之（101-060）。

郗超少卓荦不羁，有旷世之度。父愔简默冲退而啬于财，超每恣意施与。[[桓温]]府中语曰："髯[[参军]]，短[[主簿]]，能令公喜，能令公怒。"超髯，[[王珣]]短故也。超与温谋废立，权重一时。然临终不以私怨废公义，人以此多之。超早卒，年四十二。

> 大司马温自兗州伐燕。郗超曰："道远，汴水又浅，恐漕运难通。"温不从。
> （102-005）
