# 痛自刻责

## 出处

[[汉武帝]]末年，[[巫蛊之祸]]后，帝知[[太子]]据冤死，痛自刻责（026-041）：

> 上居建章宫，闻太子出亡，怒甚。及太子死，上颇知其冤，乃族灭江充家，焚苏文于横桥上。上怜太子无辜，乃作思子宫，为归来望思之台于湖。天下闻而悲之，上由是痛自刻责。

## 释义

"痛自刻责"意为深切地责备自己。
