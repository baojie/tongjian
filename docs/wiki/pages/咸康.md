# 咸康

**咸康**（335—342），[[晋成帝]][[司马衍]]第二个[[年号]]。咸康年间，[[后赵]][[石虎]]大肆营建宫室，征发民夫，暴虐无道。[[前燕]][[慕容皝]]立为[[燕王]]，迁都[[龙城]]。[[成汉]][[李寿]]改元[[汉兴]]。[[庾亮]]谋北伐，未成而卒。咸康八年，成帝崩。
