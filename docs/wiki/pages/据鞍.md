# 据鞍

**据鞍**指骑在马上按着鞍桥，为古代[[军事]]人物展现[[老将]]雄风或临危[[从容]]的经典姿态。

## 老将据鞍

[[马援]]请行，帝愍其老，援据鞍顾眄，以示可用（044-014）。帝好读书，在舆据鞍，不忘讲道（140-042）。欢据鞍未动，金以鞭拂马，乃驰去（157-102）。

## 据鞍之态

化及战栗不能言，但俯首据鞍称罪过（185-018）。[[太子]]据鞍督兵自若（210-014）。
