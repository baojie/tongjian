# 长江

[[中国]]第一大河，发源于青藏高原，东流入海。自[[三国]]以来为南北政权分界的天然屏障。

## 三国

[[甘宁]]说[[孙权]]曰：竟长江所极，据而有之，此王业也（063-036）。[[赤壁之战]]前，[[周瑜]]谓[[孙权]]曰：[[将军]]大势可以拒操者，长江也（065-048）。[[吕蒙]]劝权取[[关羽]]，全据长江，形势益张（068-036）。

## 六朝

[[东晋]]南渡后，长江为[[南朝]]防御北方[[铁骑]]的[[天险]]。[[王濬]][[楼船]]破吴，吴人以铁锁[[横江]]（081-004）。
