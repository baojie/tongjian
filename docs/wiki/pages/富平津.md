# [[富平津]]

[[富平津]]是[[西晋]]时期[[黄河]]上的重要渡口，位于[[孟津]]附近。[[晋武帝]]时，[[杜预]]以孟津渡险，请建河桥于富平津（080-035）。议者以为殷周所都，历圣贤而不作，质疑其可行性。杜预固请为之，桥成，武帝携百官临会，举觞称杜预之功。

富平津桥是中国历史上最早的黄河浮桥之一，便利了[[洛阳]]与[[河北]]之间的交通，具有重要的军事和经济意义。
