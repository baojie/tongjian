# 地节

**地节**（前69—前66），[[汉宣帝]]第二个[[年号]]。地节年间，[[霍光]]病卒，宣帝始亲政，以[[霍禹]]、[[霍山]]、[[霍云]]等谋反，尽诛霍氏家族，废霍皇后。宣帝励精图治，五日一听事，令群臣各奏封事，以知下情。又置廷尉平，决刑狱，减赋税。
