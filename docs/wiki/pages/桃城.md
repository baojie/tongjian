# [[桃城]]

[[桃城]]位于今[[山东]]汶上一带，是[[东汉]]初年的重要城池。[[汉光武帝]]时，董宪与[[刘纡]]、[[苏茂]]、[[佼强]]去[[下邳]]还[[兰陵]]，使苏茂、佼强助[[庞萌]]围桃城（041-061）。

光武帝时幸蒙，闻桃城被围，乃留辎重，自将轻兵晨夜驰赴，大破庞萌军，桃城围解。此后，[[南朝]][[宋]]时，[[萧承之]]等亦曾争据桃城（122-108）。
