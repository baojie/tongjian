# [[金微山]]

[[金微山]]是[[汉代]]对[[阿尔泰山]]的称呼。[[东汉]][[和帝]]永元三年（91年），[[窦宪]]遣左校尉[[耿夔]]、司马[[任尚]]出[[居延塞]]，围北[[单于]]于金微山，大破之，获其母阏氏、名王以下五千余级，北单于逃走，不知所往，出塞五千余里而还（047-078）。

此役是[[汉]]军对[[匈奴]]作战中出击最远的一次，彻底瓦解了北匈奴的残余势力，迫使其向西迁徙，间接引发了欧洲的"蛮族大迁徙"。
