# 郭泰

郭泰，字林宗，[[晋阳|太原]]界休人。[[东汉]]名士，党锢清流领袖。以品鉴人物、奖掖后进著称，为士林所宗。

[[晋阳|太原]]郭泰少游汝南，先过袁阆，不宿而退，进从宪，累日方还。泰曰："奉高之器，譬诸泛滥，虽清而易挹。叔度汪汪若千顷之陂，澄之不清，淆之不浊，不可量也。"（050-113）[[晋阳|太原]]郭泰曰："孺子之为人，清洁高廉，饥不可得食，寒不可得衣，而为季伟饮酒食肉，此为已知。"（055-004）

郭泰在[[东汉]]桓灵之际，以名士领袖主持清议，品评人物，为士林所重。泰虽在[[党锢之祸]]中免于难，然党祸起时，泰恸哭曰："周德既衰，[[彗星]]见于上，[[荧惑]]守心于下。诗云：'人之云亡，邦国殄瘁。'"泰终身不仕，卒时四方赴吊者万余人，其声望之隆可见。
