# [[梁山]]

[[梁山]]位于今[[陕西]]乾县，[[秦始皇]]曾在梁山建梁山宫。一次始皇在梁山宫望见丞相车骑众，不悦，中人或告丞相，后丞相减损车骑，始皇由此知泄言者，下令追查，无人承认，始皇乃逮捕当时在场者，尽杀之（007-062）。

[[西晋]]元康年间，[[齐万年]]屯兵梁山，有众七万，[[梁王肜]]、[[夏侯骏]]使[[周处]]以五千兵击之，军无后继，处力战而死（082-087）。
