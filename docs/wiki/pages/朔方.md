# 朔方

朔方为[[汉武帝]]所置郡，位于河套西北。武帝用[[主父偃]]之议，"城朔方"，使[[苏建]]发十万人筑朔方城（019-028）。[[卫青]]数出朔方击[[匈奴]]，朔方遂为汉军北进之基地（019-036）。

东汉末朔方为羌胡所据（052-049）。南北朝时[[赫连勃勃]]以朔方为基地建[[赫连勃勃|大夏]]（115-052）。朔方在历代均被视为北疆锁钥，唐代设朔方[[节度使]]，[[郭子仪]]曾任朔方[[节度使]]（217-036）。朔方战略地位的重要性贯穿两千年。

## 置郡始末

主父偃言朔方地肥饶外阻河，蒙恬城之以逐匈奴，省转输戍漕广中国灭胡之本（018-066）。武帝从其议，立朔方郡募民徙之（018-068）。公孙弘数谏以为罢敝中国（018-079）。武帝专力城朔方，罢西夷以聚力量（018-086）。匈奴右贤王数侵扰朔方，卫青将兵出高阙击之（019-007）。朔方之置，为武帝北征匈奴之战略枢纽，自河套而制漠南，汉军始得立足。
