# 达奚武

达奚武，字成兴，代郡人。北周柱国大将军，以善觇敌情、勇为前锋著称，为[[宇文泰]]创业名将。

行台左丞[[苏绰]]、中兵参军代人达奚武亦以为然（157-077）。泰遣须昌县公达奚武觇欢军，武从三骑，皆效欢将士衣服，日暮，去营数百步下马，潜听得其军号，因上马历营（157-100）。[[车骑]]大将军达奚武帅千骑为前驱（158-014）。

达奚武初为[[宇文泰]]部将，以胆略过人、精于侦察著称。沙苑之战前，达奚武深入敌营探听[[高欢]]军号，使[[宇文泰]]得以准确掌握敌情，大获全胜。达奚武历仕西魏北周，官至柱国大将军。子达奚震亦为北周名将。
