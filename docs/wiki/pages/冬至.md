# 冬至

冬至是二十四节气之一，为一年中日影最长之日。古代以冬至为重要节日，天子[[郊祀]]泰一。

公孙卿谓[[汉武帝]]今年得宝鼎，其冬辛巳朔旦冬至与黄帝时等（020-057）。武帝始郊拜泰一于十一月辛巳朔冬至（020-059）。元封元年冬十月行幸泰山，十一月甲子朔旦冬至，祠上帝于[[明堂]]（021-046）。

冬至在古代历法中具有特殊地位，周代以冬至为岁首。汉代以冬至为[[祭天]]之期，天子亲郊泰一。冬至大如年，唐宋以后冬至成为重要民俗节日，合家团聚，祭祖拜贺。

> 十一月辛巳朔冬至，昧爽，天子始郊拜泰一，朝朝日，夕夕月则揖。
> （020-059）
