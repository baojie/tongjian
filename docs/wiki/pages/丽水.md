# [[丽水]]

[[丽水]]在[[唐代]]指今[[金沙江]]（长江上游），流经[[云南]]、[[四川]]、[[西藏]]交界地带。丽水以产黄金著称，《[[蛮书]]》等唐代文献记载其两岸居民以淘金为业，称为"丽水金"。

唐代丽水流域为[[南诏]]所控制，是南诏与[[吐蕃]]、[[唐朝]]三方交往中的重要地理通道和资源产地。[[剑南道]]西南边境与南诏的交界即以丽水为界之一段。
