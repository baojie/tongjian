# 兴平

**兴平**（194—195），[[汉献帝]]第二个[[年号]]。兴平年间，[[李傕]]、[[郭汜]]相攻，[[长安]]城中大乱，献帝播迁。[[曹操]]攻[[吕布]]于[[濮阳]]，[[陶谦]]卒，[[刘备]]领[[徐州]]。[[孙策]]东渡，据[[江东]]。[[三辅]][[大旱]]，谷一斛五十万，人相食，[[白骨]]委积。

> 孝献皇帝丙兴平元年
