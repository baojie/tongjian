# 吕隆

吕隆，后凉末主。[[吕光]]之子，继位后为后秦所逼，出降。

吕隆为后凉主，后秦[[姚兴]]遣将攻凉，隆屡败（112-030、112-037）。隆以百姓饥困，乃降于后秦。后凉自[[吕光]]立国至吕隆而亡（112-045）。

## 后凉末主

[[吕隆]]为后凉主，多杀豪望以立威名，内外嚣然（112-030）。隆与[[沮渠蒙逊]]、[[秃发傉檀]]交兵，互有胜负（112-037）。姑臧大饥，人相食，饿死者十馀万口。民请出城为胡虏奴婢者日有数百，隆恶其沮动众心，尽坑之，积尸盈路（112-060）。南凉王傉檀攻隆于姑臧（112-090）。隆内外交困，乃遣使请降于后秦。后秦[[姚硕德]]表隆为镇西大将军、凉州刺史（113-015）。后凉遂亡，隆后随凉州民徙长安（114-019）。
