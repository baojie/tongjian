# 刘芳

刘芳，字伯文，彭城人。[[北魏]][[太常]]卿，著名经学家，以[[文学]]见亲于孝[[隋文帝|文帝]]。

通直散骑常侍刘芳，缵之族弟也，与给事黄门侍郎[[晋阳|太原]][[郭祚]]，皆以[[文学]]为帝所亲礼，多引与讲论及密议政事（139-098）。如[[李冲]]、[[李彪]]、[[高闾]]、王萧、[[郭祚]]、[[宋弁]]、刘芳、[[崔光]]、[[邢峦]]之徒，皆以文雅见亲，贵显用事（140-042）。

刘芳以经学[[名家]]，尤精礼学，孝[[隋文帝|文帝]]制礼作乐多咨访之。孝[[隋文帝|文帝]]改革汉化，[[朝仪]]典制多出刘芳之手。孝[[隋文帝|文帝]]尝称："刘芳所学，非世人可及。"刘芳历仕孝文、宣武、孝明三朝，官至[[太常]]卿，为[[北魏]]经学之宗。
