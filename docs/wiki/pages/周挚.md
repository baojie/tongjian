# 周挚

周挚，唐代安史之乱叛将。史思明行军司马，官至丞相，后从史朝义。

史思明筑坛于魏州城北，自称大圣燕王，以周挚为行军司马（221-003）。李光弼使韩游瑰将五百骑趣河阳，周挚引兵争河阳，后至不得入（221-010）。史思明自称大燕皇帝，以周挚为相，李归仁为将（221-021）。

思明分兵四道，周挚自胡良济河，会于汴州（221-041）。周挚舍南城，并力攻中氵单，李光弼使荔非元礼出劲兵拒贼（221-048）。周挚复收兵趣北城，光弼登城望贼曰："贼兵虽多，嚣而不整，不足畏也"（221-049）。

后史朝义杀史思明，周挚与许叔冀将后军在福昌，闻变惊倒于地（222-012）。

> 史思明筑坛于魏州城北，自称大圣燕王；以周挚为行军司马。
> （221-003）
