# 刘隗

刘隗，字大连，[[彭城]]人。[[东晋]]司直，以刚直著称。

刘隗为[[丞相]]司直，督运[[令史]]淳于伯被冤杀，血逆流上，观者咸以为冤。刘隗上言伯罪不至死，请免[[从事]][[中郎]]等官（089-095）。后为[[御史]]中丞（090-010）。以[[尚书]][[仆射]][[戴渊]]为[[征西]][[将军]]，刘隗为[[镇北]][[将军]]，各率众镇守（091-070）。隗为[[王敦]]所攻，奔[[石勒]]。

> 丞相司直刘隗上言："伯罪不至死，请免从事中郎周莚等官。"
> （089-095）

## 刚直与出镇

[[元帝]]以刘隗为司直，隗与[[刁协]]并为帝所宠任，欲矫时弊，每崇上抑下，排沮[[豪强]]，为[[王氏]]所疾（088-076）（090-052）。帝畏[[王敦]]之强，引隗等为腹心，稍抑[[王氏]]之权（091-058）。隗为帝谋，出腹心以镇方面，遂以为[[镇北]][[将军]]，[[都督]]青、徐、幽、平四州诸[[军事]]，镇[[淮阴]]（091-060）（091-070）。及[[王敦]][[举兵]]武昌，上疏罪状刘隗，隗为敦所攻，奔[[石勒]]（092-005）（092-006）。
