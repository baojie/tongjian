# 栾布

**栾布**，梁人，[[西汉]]将领。

栾布为[[彭越|梁王]][[彭越]]大夫，使于齐。[[彭越]]被诛，[[枭首]][[洛阳]]，诏有收视者辄捕之。栾布还，奏事越头下，祠而哭之。吏捕以上闻，[[汉高祖刘邦|刘邦]]欲烹之。栾布从容言[[彭越]]之功，[[汉高祖刘邦|刘邦]]壮其义，释之拜为都尉（012-034）。

[[隋文帝|文帝]]时，栾布为将军击[[匈奴]]（015-035）。景帝时吴楚七国反，栾布以将军击齐（016-013），破齐还，并兵引水灌赵城，城坏王遂自杀（016-027）。栾布以忠义立身，哭[[彭越]]而全其节，平七国而立其功。

## 忠义立身

栾布为[[彭越|梁王]][[彭越]]大夫，[[彭越]]被诛，栾布奏事越头下，祠而哭之。[[汉高祖刘邦|刘邦]]欲烹之，布从容言越之功，[[汉高祖刘邦|高祖]]壮其义，释拜都尉（012-034）。[[隋文帝|文帝]]时以将军击[[匈奴]]（015-035）。景帝时吴楚反，栾布击齐，破[[三国]]兵（016-013、016-025），破齐还，引水灌赵城，城坏王遂自杀（016-027）。
