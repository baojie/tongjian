# 高沐

高沐，唐代[[李师道]]判官。忠谏被杀，后获朝廷昭雪。

[[李师道]]叛，高沐在师道军中为判官，与郭晬、李公度等屡谏师道，不从。师道疑沐等为[[内应]]，遂杀沐（237-031）（237-036）。高沐忠谏而死，朝廷知其忠，后加褒赠。

沐之死，皆[[李文会]]等谗构所致。师道既诛，朝廷追赠高沐。
