# 咸宁

**咸宁**（275—280），[[晋武帝]][[司马炎]]第二个[[年号]]。咸宁年间，晋[[大举]]伐[[吴]]，[[王濬]]在[[益州]]造[[楼船]]，[[杜预]]、[[王浑]]等分道并进。咸宁六年，吴主[[孙皓]]降，[[三国]]归晋，[[天下]]重归一统。

咸宁五年，[[王濬]]上疏请伐吴，杜预亦表请，[[张华]]赞成之，武帝乃决意伐吴。咸宁六年，[[王濬]]以[[楼船]]下益州，顺流东下，吴主[[孙皓]]面缚降，[[三国]]遂归一统（081-001）。

## 后世沿用

咸宁亦为[[后凉]]年号。[[吕纂]]即[[天王]]位，大赦，改元咸宁（111-064）。[[唐]]有咸宁太守（216-020）、咸宁王[[浑瑊]]（235-100）。[[唐]][[懿宗]]崩于咸宁殿（252-053）。
