# 旌旗

旌旗是古代以羽毛和旗帜装饰的仪仗旗帜，天子将帅用以指挥军队、彰显威仪。

梁孝王以至亲有功，得赐天子旌旗，从千乘万骑（016-066）。汉武帝勒兵十八万骑，旌旗径千馀里，以见武节威匈奴（020-082）。李陵兵败，尽斩旌旗及珍宝埋地中（021-100）。

韩延寿治城郭，陈钟鼓管弦，设斧钺旌旗（026-041）。旌旗为军队耳目所系，进退攻守皆以旗号为令。
