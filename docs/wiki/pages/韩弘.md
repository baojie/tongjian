# 韩弘

**韩弘**，匡城人，唐宣武[[节度使]]。

宣武[[节度使]]刘合谅薨，军中思[[刘玄佐]]之恩，推其甥韩弘为[[留后]]（235-095）。弘将兵识其材鄙[[勇怯]]，指顾必堪其事。韩弘在宣武掌兵二十馀年，拥[[重兵]]而不反，为[[朝廷]]所重。

韩弘宣武[[节度]]期间，以[[严酷]]御下，然其军为唐廷[[平定]][[藩镇]]的重要力量。及宪宗[[征讨]][[淮西]][[吴元济]]，韩弘[[出兵]]助战。韩弘晚年[[入朝]]，以[[功名]]终。

（按：[[资治通鉴概述|通鉴]]另有隋将韩弘，为隋[[代州]][[总管]]，179-046，与本条[[唐朝|唐代]]韩弘非同一人。）

## 镇宣武

韩弘既为宣武[[节度使]]，[[吴少诚]]叛，弘与[[于由页]]、[[伊慎]]等[[进击]]，屡破之（235-099）。宣武军自[[刘玄佐]]薨凡五作乱，[[士卒]][[骄纵]]。韩弘视事数月尽知主名，召其党三百人，数其罪，尽斩之，一军股栗（235-105）。[[吴少阳]]遣兵掠邻境，弘使谓曰："汝能越吾界而为盗邪！有以相待，毋为空言！"少阳遂不敢犯（236-062）。[[宪宗]]以韩弘为[[淮西]]诸军[[都统]]，然弘乐于自擅，欲倚贼自重，不愿[[淮西]]速平（239-067）。[[朝廷]]以其拥[[重兵]]，加[[司徒]]（239-043）。韩弘以[[宣武]][[强兵]]雄视一方，虽无显功而能[[持重]]自固，为[[中唐]][[藩镇]]中之能者。
