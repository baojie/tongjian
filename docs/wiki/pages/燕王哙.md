# 燕王哙

燕王哙，[[战国]][[燕国]][[君主]]。

燕王哙惑于[[苏代]]之说，专[[任子]]之。鹿毛寿以让[[天下]]之说劝燕王哙，哙遂以国让子之。子之南面行王事，哙老不[[听政]]（003-017）。后[[太子平]]攻子之，齐人伐燕，取子之醢之，遂杀燕王哙（003-024）。

## 世系

燕王哙（？—[[前314年]]），[[战国]][[燕国]][[君主]]。哙惑于[[苏代]]之说，禅位于[[子之]]（003-017）。[[太子平]]攻子之，[[齐]]人伐燕，杀哙（003-024）。齐人杀哙后，燕人共立[[太子平]]，是为[[燕昭王]]（003-032）。
