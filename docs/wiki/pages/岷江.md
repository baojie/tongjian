# [[岷江]]

[[岷江]]是[[长江]]上游重要支流，发源于[[四川]]岷山南麓，流经[[成都平原]]，在[[宜宾]]汇入长江。岷江为成都平原提供了丰富的水源，[[都江堰]]水利工程即建于岷江之上，分水灌溉成都平原。

[[战国]]时期，[[苏秦]]说[[楚威王]]时指出，秦西有[[巴蜀|巴、蜀]]，治船积粟，浮岷江而下，一日行五百余里（003-036），说明岷江是秦军东出[[长江]]的重要水路通道。
