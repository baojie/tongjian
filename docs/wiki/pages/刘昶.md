# 刘昶

刘昶，刘宋[[宗室]]，降[[北魏]]。封丹杨王、[[齐郡]]公，屡引魏军南侵宋朝。

上谓兗州[[刺史]][[垣崇祖]]曰："索虏必以纳刘昶为辞，侵犯边鄙"（135-034）。魏奉丹杨王刘昶入寇，许昶以克复旧业，世胙[[江南]]（135-053）。嘉与刘昶寇寿阳，将战，昶四向拜将士，流涕[[合纵连横|纵横]]，曰："愿同戮力，以雪仇耻"（135-065）。魏刘昶以雨水方降，表请还师（135-072）。刘昶入寇，灵诞请为昶司马，不许（135-118）。[[丹阳]]王刘昶封[[齐郡]]公，加号宋王（137-077）。
