# 鲁国

鲁国为[[西周]]至[[战国]][[诸侯]]国，周公旦[[封国]]，都[[曲阜]]。

## 战国局势

鲁国至[[战国]]已衰弱，四周强邻环伺。[[齐]]屡伐鲁，取最（001-042），又伐鲁（001-069）。鲁虽偶有胜绩，败齐师于平陆（001-053），亦伐齐入[[阳关]]（001-102），然终不能扭转颓势。[[吴起]]尝仕于鲁，齐人伐鲁，鲁欲以为将，起杀妻以求将，大破齐师。然鲁人疑之，起终去鲁归魏（001-016）。
