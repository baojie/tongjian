# 散乐

散乐是民间[[百戏]][[歌舞]]的总称，包含杂技、幻术、俳优等表演艺术。[[隋文帝]]以"散乐非正声"，诏罢散乐，禁民间以此为业（175-025）。然而至[[隋炀帝]]时，大业二年[[突厥]]启民[[可汗]]来朝，[[隋炀帝|炀帝]]大集散乐于[[洛阳]]，征天下奇伎异艺（180-065/079），盛况空前。

[[隋炀帝|炀帝]]又于端午日"大陈[[百戏]]、散乐"，自海内凡有奇伎无不毕集（181-046）。唐代散乐归[[教坊]]掌管，与[[雅乐]]分立，成为宫廷娱乐的重要组成部分。散乐上承[[百戏]]传统、下启宋元杂剧，在中国表演艺术史上具有承前启后的地位。
