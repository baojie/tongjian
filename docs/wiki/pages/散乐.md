# 散乐

散乐是民间百戏歌舞的总称，包含杂技、幻术、俳优等表演艺术。隋文帝以"散乐非正声"，诏罢散乐，禁民间以此为业（175-025）。然而至隋炀帝时，大业二年突厥启民可汗来朝，炀帝大集散乐于洛阳，征天下奇伎异艺（180-065/079），盛况空前。

炀帝又于端午日"大陈百戏、散乐"，自海内凡有奇伎无不毕集（181-046）。唐代散乐归教坊掌管，与雅乐分立，成为宫廷娱乐的重要组成部分。散乐上承百戏传统、下启宋元杂剧，在中国表演艺术史上具有承前启后的地位。
