# 齐襄王

齐襄王，[[战国]][[齐国]][[君主]]，田法章，湣王子。

魏人[[范雎]]使于齐，齐襄王闻其辩口，私赐金、牛、酒（005-014）。齐襄王薨，子建立（005-031）。

## 世系

齐襄王（？—[[前265年]]），[[田]]姓，名法章，[[战国]][[齐国]][[君主]]，[[齐湣王]]之子。[[田单]]复国后迎立为王。襄王薨，子[[齐王建|建]]立（005-031）。
