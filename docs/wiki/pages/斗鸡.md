# 斗鸡

斗鸡是以两鸡相斗为戏的娱乐活动，自战国至唐盛行不衰。[[齐宣王]]好斗鸡，以纪渻子为鸡师（002-094）。[[汉宣帝]]少时喜斗鸡走马，民间多效之（024-027）。

汉武帝时长安"闾里少年，群辈相随，斗鸡走狗"（031-018），斗鸡为市井娱乐之最。唐代斗鸡之风更盛，玄宗尤好之，宫中设鸡坊，选六军小儿五百人驯养斗鸡。斗鸡不仅为娱乐，亦常与赌博相连，为社会文化之缩影。
