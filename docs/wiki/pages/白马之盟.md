# 白马之盟

白马之盟是[[汉高祖刘邦]]与功臣杀白马而盟的[[誓约]]，核心内容为"非刘氏而王者，天下共击之"。

[[汉高祖刘邦|高祖]][[封爵]]之誓曰："使黄河如带，[[泰山]]若厉，国以永存，爰及苗裔。"申以丹书之信，重以白马之盟（021-071）。[[吕后|高后]][[吕后|吕雉]]时，尽差第[[列侯]]位次，藏诸[[宗庙]]，副在有司。

## 历史作用

白马之盟是[[西汉]]立国的根本政治原则之一。[[吕后]]欲王诸吕，问于[[陈平]]、[[周勃]]，[[王陵]]以"高帝刑白马盟曰：非刘氏而王，天下共击之"正色反驳（013-003）。[[吕后]]虽终封诸吕为王，但[[陈平]]、[[周勃]]后来平定[[诸吕之乱]]时，正是以白马之盟为号召，诛灭吕氏，迎立[[文帝]]。

## 崩坏

武帝以后，[[推恩令]]行，[[诸侯]]王势弱，此盟渐弛。至[[王莽]]篡汉，白马之盟完全崩坏。后世每逢权臣谋篡之时，常援引此盟为反对[[篡位]]的合法依据。
