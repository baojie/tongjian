# 白马之盟

白马之盟是[[汉高祖刘邦]]与功臣杀白马而盟的[[誓约]]，核心内容为"非刘氏而王者，天下共击之"。

[[汉高祖刘邦|高祖]][[封爵]]之誓曰："使黄河如带，[[泰山]]若厉，国以永存，爰及苗裔。"申以丹书之信，重以白马之盟（021-071）。[[吕后|高后]][[吕后|吕雉]]时，尽差第[[列侯]]位次，藏诸[[宗庙]]，副在有司。及[[隋文帝|文帝]]、景帝四五世间，[[列侯]]相继。

白马之盟是[[西汉]]立国的根本政治原则之一。[[吕后]]欲王诸吕，问于[[陈平]]、[[周勃]]，[[王陵]]以"高帝刑白马盟曰：非刘氏而王，天下共击之"正色反驳（013-003）。武帝以后此盟渐弛。至[[王莽]]篡汉，白马之盟完全崩坏。
