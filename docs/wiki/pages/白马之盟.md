# 白马之盟

白马之盟是[[汉高祖刘邦]]与功臣杀白马而盟的誓约，核心内容为"非刘氏而王者，天下共击之"。

[[汉高祖刘邦|高祖]]封爵之誓曰："使黄河如带，泰山若厉，国以永存，爰及苗裔。"申以丹书之信，重以白马之盟（021-071）。[[吕后|高后]][[吕后|吕雉]]时，尽差第列侯位次，藏诸宗庙，副在有司。及[[隋文帝|文帝]]、景帝四五世间，列侯相继。

白马之盟是[[西汉]]立国的根本政治原则之一。[[吕后]]欲王诸吕，问于[[陈平]]、[[周勃]]，王陵以"非约也"正色反驳。武帝以后此盟渐弛。至[[王莽]]篡汉，白马之盟完全崩坏。
