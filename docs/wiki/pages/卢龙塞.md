# [[卢龙塞]]

[[卢龙塞]]位于今[[河北]]迁西喜峰口一带，是[[中原]]通往[[东北]]（[[辽西]]、[[辽东]]）的重要关塞。[[东汉]]末年[[曹操]]北征[[乌桓]]时，出卢龙塞，堑山堙谷五百余里，直指[[凡城]]、[[白狼山]]（065-035）。

[[十六国]]时期，[[前燕]][[慕容儁]]遣[[慕容霸]]自东道出徒河，自中道出卢龙塞以伐[[后赵]]（098-059）。[[北魏]]时，幽州刺史[[常景]]自卢龙塞至军都关，皆置兵守险（150-093）。卢龙塞是古代中原政权经略[[辽西]]地区的咽喉要地。
