# 凌统

凌统，[[东吴]]名将，凌操之子。以勇猛善战、孝义双全著称于[[三国]][[东吴]]。

凌统与[[董袭]]俱为前部，各将敢死百人冲突敌舰（065-030）。凌统怨[[甘宁]]杀其父凌操，常欲杀宁，[[孙权]]命统不得仇之令宁将兵屯他所（065-031）。

凌统少年即从[[孙权]]征战，合肥之战中舍死护卫[[孙权]]突围，所部尽没，统独以身免。[[孙权]]以凌统为偏将军，倍加亲信。凌统早逝，[[孙权]]痛惜之。其孝义勇猛之品格为[[东吴]]诸将中的突出者。

> 偏将军董袭与别部司马凌统俱为前部，各将敢死百人，人被两铠，乘大舸，突入蒙冲里。
> （065-030）
