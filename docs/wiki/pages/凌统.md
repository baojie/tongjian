# 凌统

凌统，[[东吴]]名将，凌操之子。以勇猛善战、孝义双全著称于[[三国]][[东吴]]。

凌统与[[董袭]]俱为前部，各将[[敢死]]百人冲突敌舰（065-030）。凌统怨[[甘宁]]杀其父凌操，常欲杀宁，[[孙权]]命统不得仇之令宁将兵屯他所（065-031）。

凌统少年即从[[孙权]]征战，[[合肥]]之战中舍死护卫[[孙权]][[突围]]，所部尽没，统独以身免。[[孙权]]以凌统为[[偏将军]]，倍加亲信。凌统早逝，[[孙权]]痛惜之。其孝义勇猛之品格为[[东吴]]诸将中的突出者。

## 凌统与逍遥津一战

凌统于逍遥津之战中舍死护主，为[[东吴]]忠勇之表率。建安二十年，[[张辽]]率步骑奄至逍遥津北，[[甘宁]]与[[吕蒙]]等力战扞敌，凌统率亲近扶[[孙权]]出围，复还与[[张辽]]战，左右尽死身亦被创，度权已免乃还（067-025）。此役凌统所部尽没，独以身免，[[孙权]]既免，倍加亲信。凌统又以父仇与[[甘宁]]不共戴天，[[孙权]]命不得相仇，令宁将兵屯他所（065-031），统虽怨而不敢违命，其孝勇与服从君命的两难，折射了[[东吴]]诸将间私仇与国事之间的复杂张力。
