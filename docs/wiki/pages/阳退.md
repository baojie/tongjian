# 阳退

**阳退**指佯装撤退以诱敌，为古代军事设伏的常见战术。

## 阳退诱敌

张既平河西，敕精卒为伏，使参军督千馀骑挑战，敕使阳退。胡果争奔之，因发伏截其后，大破之（069-050）。阳退发伏，胡骑大溃。

慕容绍宗将战，谓将士曰：我当阳退，误吴儿使前，尔击其背（160-052）。阳退误敌，前后夹击。

杨行密使李神福拒赵匡凝，神福阳退以示怯，儒军不设备，夜帅精兵袭之（258-073）。阳退示怯，夜袭决胜。

台濛讨田頵，阳退以示弱。頵兵逾濠而斗，濛急击之，頵坠马死（264-052）。阳退以骄，急击勿失。
