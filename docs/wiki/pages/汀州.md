# [[汀州]]

[[汀州]]是[[唐代]]开元年间设置的州，治所在今[[福建]]长汀，隶属[[江南东道]]。汀州地处[[福建]]西部山区，[[汀江]]上游，是唐代开发[[闽]]西地区的重要行政中心。

[[唐德宗]]贞元年间，[[陆贽]]被贬忠州别驾，[[窦参]]贬[[柳州]]司马，[[李巽]]贬汀州长史（235-018）。后[[董晋]]捕斩叛党，[[张惟恭]]亦被械送京师，免死，汀州安置（235-059）。汀州在[[宋代]]是[[客家民系]]形成的重要区域。
