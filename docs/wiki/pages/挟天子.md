# 挟天子

挟天子以令诸侯是中国古代重要的政治策略，指控制皇帝或天子，借其名义号令天下。[[战国]]时[[张仪]]最早向秦惠王提出挟天子以令诸侯的策略。东汉末[[曹操]]迎汉献帝迁都许昌，以此号令天下诸侯，是这一策略最著名的实践。挟天子策略的核心在于利用天子的政治合法性，使自己的行动获得名义上的正当性。

> 仪曰："亲魏，善楚，下兵三川，攻新城、宜阳，以临二周之郊，据九鼎，按图籍，挟天子以令于天下，天下莫敢不听，此王业也。"
> （003-016）

> 荀彧曰："昔晋文公纳周襄王而诸侯景从，汉高祖为义帝缟素而天下归心。今天子蒙尘，将军首唱义兵，诚因此时奉主上以从民望，秉至公以服雄杰。"
> （061-053）
