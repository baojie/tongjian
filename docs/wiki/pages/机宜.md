# 机宜

**机宜**指时机与适宜的策略，关乎进退取舍之道。

## 用人合宜

孙峻为丞相，有媚峻者言曰万机宜在公族，若承嗣为亚公，声名素重，众心所附，不可量也（076-024）。以宗室掌机要为合宜。

## 奏对之才

刘仁轨在百济，前后奏事皆合机宜。唐高宗问刘仁愿，仁愿曰此皆仁轨所为，非臣所及也（201-030）。坐待机宜，必获申蔡之利（238-004），谓把握有利时机。
