# 始元

**始元**（前86—前80），[[汉昭帝]]第一个[[年号]]。始元年间，昭帝年幼即位，[[霍光]]受武帝遗诏辅政。[[匈奴]]与汉和亲，汉廷释放苏武等使节归国。始元六年，召开[[盐铁会议]]，郡国贤良文学与[[桑弘羊]]就盐铁官营等政策展开辩论。[[上官桀]]、[[燕王刘旦]]谋反，被霍光平定。
