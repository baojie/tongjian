# 李固

李固，字子坚，汉中南郑人。[[东汉]][[太尉]]，以忠直不屈著称。因[[立嗣]]之争忤[[梁冀]]，被诬陷而死。

李固以书逆遗[[黄琼]]曰："君子谓伯夷隘，柳下惠不恭。不夷不惠，可否之间，圣贤居身之所珍也。"（051-052）李固，郃之子，少好学，常改易姓名，杖策驱驴，负笈从师，不远千里，遂究览坟籍，为世大儒（051-053）。

李固在汉顺帝时以对策拜议郎，累迁[[太尉]]。固以正直立朝，反对[[梁冀]]专权。冲帝崩后，李固与[[梁冀]]争议立帝，固主立清河王蒜，冀立质帝。质帝被弑后，李固再争[[立嗣]]，冀遂立桓帝而诬杀李固。固临死，遗书敕子孙以忠孝，天下冤之。李固以死节著称，与[[杜乔]]并称"李杜"。
