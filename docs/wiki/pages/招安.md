# 招安

招安，招降安抚。姜师度为[[营田]]使，招安流散，数年间[[仓廪]]充实（211-124）。[[唐武宗]]时遣中使招安城内将士（248-016）。[[王建]]以张琳为邛南招安使，引兵还[[成都]]（258-030）。

> 庆礼清勤严肃，开屯田八十馀所，招安流散，数年之间，仓廪充实，市邑浸繁。
> （211-124）
