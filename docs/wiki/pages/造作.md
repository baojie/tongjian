# 造作

**造作**指官府主持的制造、[[营造]]事务，包括[[兵器]]、[[器物]]、[[宫室]]等各项工程建设。

## 造作俭省

[[汉元帝]]减省[[椒房]]、掖廷用度，服御、舆驾所发诸[[官署]]及所造作，皆如[[竟宁]]以前[[故事]]（030-043）。

## 造作规制

[[国家]][[武备]]，缮治造作，皆度[[大司农]]钱（034-040）。[[楚王英]]与[[方士]]造作[[图书]]，有逆谋，事下[[案验]]（045-052）。[[汉安帝]]诏诸所造作非供[[宗庙]][[园陵]]之用，皆且止（049-045）。
