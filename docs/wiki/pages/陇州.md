# [[陇州]]

[[陇州]]是[[西魏]]至[[隋]][[唐]]时期设置的州，治所在今[[陕西]]陇县，地处[[陇山]]东麓，扼守[[关中]]通往[[陇右]]的咽喉要道。陇州是关中西部的军事重镇和交通枢纽。

[[隋文帝]]曾行幸岐州，至陇州（176-010）。唐初，[[陇州]]刺史[[常达]]击[[薛仁杲]]于[[宜禄川]]，斩首千余级（186-023），为[[唐朝]]平定[[陇右]]立下战功。陇州在唐代为[[陇右道]]东部的门户。
