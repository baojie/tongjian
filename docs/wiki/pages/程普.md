# 程普

程普（生卒年不详），字德谋，[[右北平]]土垠人，[[三国]]时[[东吴]]名将。初从[[孙坚]]起兵讨[[黄巾]]，后随[[孙策]]平定[[江东]]。[[孙策]]死后，程普与[[张昭]]等共辅[[孙权]]。

程普历仕孙氏三世，为[[江东]]元勋[[宿将]]。[[赤壁之战]]时任右[[都督]]，与[[周瑜]]分领左[[右军]]，大破[[曹操]]。程普以年长资深，军中呼为"程公"。性乐善好施，喜[[士大夫]]，在诸将中年辈最高。

## 三世宿将

程普从[[孙策]]讨严白虎，白虎败走（063-013）。讨[[乐安]]，平山越（064-021）。[[赤壁之战]]，程普与[[周瑜]]为左右督，将兵与[[刘备]]并力逆操（065-050）。攻[[曹仁]]，领[[江夏]][[太守]]（066-009）。程普以年长数陵侮[[周瑜]]，瑜折节下之，普后自敬服（066-018）。程普历仕孙氏三世，为[[江东]]元勋[[宿将]]，军中呼为程公，其功在[[江东]]基业。
