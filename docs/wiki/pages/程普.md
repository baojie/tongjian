# 程普

程普（生卒年不详），字德谋，右北平土垠人，[[三国]]时[[东吴]]名将。初从[[孙坚]]起兵讨[[黄巾]]，后随[[孙策]]平定江东。孙策死后，程普与[[张昭]]等共辅[[孙权]]。

程普历仕孙氏三世，为江东元勋宿将。赤壁之战时任右都督，与[[周瑜]]分领左右军，大破[[曹操]]。程普以年长资深，军中呼为"程公"。性乐善好施，喜士大夫，在诸将中年辈最高。
