# 满宠

满宠，字伯宁，山阳昌邑人。曹魏名将，官至征东将军，镇守合肥多年，为[[东吴]]北伐之劲敌。

操以山阳满宠为许令，操从弟洪有宾客在许界数犯法，宠收治之，洪书报宠，宠不听（062-015）。操使许令满宠按彪狱，融与[[荀彧]]皆属宠但当受辞勿加考掠，宠一无所报，考讯如法（062-033）。前将军满宠上疏曰："曹休虽明果而希用兵，今所从道背湖旁江，易进难退，此兵之絓地也。"（071-011）

满宠为官刚正，执法不避权贵。为许令时，[[曹操]]从弟[[曹洪]]之宾客犯法，满宠依法收治，[[曹洪]]说情不从，[[曹操]]闻之亦不责。晚年镇守合肥，屡拒吴军，[[孙权]]深畏之。其用兵持重，善于守御，为曹魏东线柱石。
