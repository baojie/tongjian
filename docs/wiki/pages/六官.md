# 六官

六官指依照《[[周礼]]》设立的六种官职：**天官冢宰**（总揽政务）、**地官[[司徒]]**（掌[[教化]]土地）、**春官宗伯**（掌[[礼乐]][[祭祀]]）、**夏官[[司马]]**（掌军政[[征伐]]）、**秋官[[司寇]]**（掌[[刑狱]][[法律]]）、**冬官[[司空]]**（掌工程建筑），是[[北周]]仿[[周礼]]改革的官制体系。

## 通鉴中的六官

[[前赵]][[刘聪]]设"[[单于]]左右辅，各主六夷十万落，自[[司隶]]以下六官，皆位亚[[仆射]]"（089-006），已见六官之名。[[西魏]][[宇文泰]]命[[苏绰]]、[[卢辩]]"依《[[周礼]]》更定六官"（166-051）。次年"魏初建六官，以[[宇文泰]]为[[太师]]、大冢宰"（166-053）。[[北周]][[武帝]]享[[太庙]]时"班太祖所述六官之法"（168-081）。[[隋文帝]]代周后"除周六官，依汉、魏之旧"（175-007），六官遂罢。

## 历史意义

六官之制是[[宇文泰]]、[[苏绰]]托古[[改制]]的核心内容，以《[[周礼]]》为蓝本重构官制，旨在为[[西魏]]—[[北周]]政权塑造[[正统]]性。隋唐以后的[[三省]][[六部]]制亦受其影响。
