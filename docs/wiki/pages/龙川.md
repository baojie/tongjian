# [[龙川]]

[[龙川]]是[[秦代]]置县，治所在今[[广东]]龙川，隶属[[南海郡]]。秦末，[[南海尉]]任嚣病危，召龙川令[[赵佗]]，嘱其趁秦乱自立，赵佗后据岭南建立[[南越|南越国]]（012-037）。

[[南北朝]]时期，龙川县为[[循州]]辖县，但也是[[北魏]]时[[拓跋]]部讨论迁都时提到的偏远之地（109-018）。龙川在[[唐代]]属[[岭南道]][[循州]]。
