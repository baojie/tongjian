# 辕固

**辕固**，齐人，[[西汉]]初年[[儒生]]，治《[[诗经|齐诗]]》，以直言著称。

[[汉武帝]]时征贤良，辕固年九十馀，亦以[[贤良]]征。公孙弘仄目而事固，固曰："公孙子，务正学以言，无曲学以阿世！"（018-034）。辕固此言直斥公孙弘以儒术缘饰吏事，为千古名训。

辕固曾与[[黄老]]学者[[黄生]]辩论于[[汉景帝]]前，论[[汤武革命]]之是非。又尝以《诗经》教授[[刘德|河间献王]]，为西汉经学传授之重要人物。其《齐诗》之学传于后，与[[申培]]之《鲁诗》、[[韩婴]]之《韩诗》并称三家诗。辕固以直道事君、不阿权贵之品格，为两汉经学家树立了风骨典范。
