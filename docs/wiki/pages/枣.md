# 枣

"枣"在《[[资治通鉴]]》指枣树或地名。秦下兵攻河外，据卷衍、酸枣（003-013）。拔燕、酸枣、虚、桃（004-095）。蒙骜伐魏，取酸枣、燕、虚等二十城（006-064）。冬，雷；桃李华，枣实（012-089）。

> 大王不事秦，秦下兵攻河外，据卷衍、酸枣，劫卫，取阳晋，则赵不南。
> （003-013）

> 王又举甲而攻魏，杜大梁之门，举河内，拔燕、酸枣、虚、桃，入邢。
> （004-095）

> 冬，雷；桃李华，枣实。
> （012-089）
