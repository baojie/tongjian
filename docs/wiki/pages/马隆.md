# 马隆

马隆，[[西晋]][[司马]]督。[[凉州]][[荆州刺史|刺史]]杨欣失羌戎之和，马隆预言其必败，后果如其言（080-077）。[[树机能]]攻陷[[凉州]]，马隆自请讨之，帝许之。隆募能引弓四钧、挽弩九石者，得三千五百人（080-093）。马隆西渡温水，[[树机能]]以众数万据险拒之。隆以山路狭隘，作扁箱车、木屋，转战而前，卒[[平凉]]州（080-104）。

> 帝曰："必能平贼，何为不任，顾方略何如耳！"隆曰："臣愿募勇士三千人，无问所从来，帅之以西，虏不足平也。"
> （080-093）
