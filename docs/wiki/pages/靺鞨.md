# 靺鞨

靺鞨是东北古民族，隋唐之际分粟末、黑水等多部。[[隋炀帝]]征高丽时，高丽王"帅靺鞨之众万馀寇辽西"（178-083），靺鞨与高丽[[结盟]]抗隋。[[宇文述]]九军渡辽时又担心"靺鞨出后"，前后夹击隋军（181-073）。

隋末唐初，靺鞨渠帅突地稽率部归附唐朝，被任命为燕州总管（189-003）。[[李渊]]之时靺鞨与奚、[[契丹]]等部多次入犯，与[[刘黑闼]]、[[窦建德]]等反唐势力联合作战（188-088）。粟末靺鞨后建[[渤海国]]，黑水靺鞨则为女真先民，在历史上扮演重要角色。
