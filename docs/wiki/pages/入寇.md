# 入寇

**入寇**指敌军侵入[[边境]]进行攻击[[劫掠]]，为古代[[史书]]中描述外敌入侵的常用[[军事]]术语。

## 外敌入寇

[[匈奴]]数入寇，[[晋阳]]去塞远，请治马邑（011-037）。遣兵与狐兰支共入寇，击[[车师]]（037-025）。[[单于]]复耻言其计，故赏遂不行。由是大恨，入寇尤深（043-050）。[[辽西]][[鲜卑]]连休等入寇（050-022）。[[代郡]][[鲜卑]]入寇，杀[[长史]]（050-036）。
