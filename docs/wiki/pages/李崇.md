# 李崇

**李崇**，[[北魏]][[扬州]][[刺史]]（注：[[资治通鉴概述|通鉴]]中有两位李崇，一为[[东汉]][[西域都护]]，一为[[北魏]][[扬州]][[刺史]]。本条为[[北魏]]李崇）。

通鉴中"李崇"出现于不同时期。一为[[西汉]]末[[西域都护]]李崇（038-016），随[[王骏]]出[[西域]]，诸国郊迎。另一为[[北魏]]李崇：[[汉武帝|世宗]]景明中为[[扬州]][[刺史]]（147-073），在边镇有威名。

北魏李崇为孝[[隋文帝|文帝]]、宣[[武帝]]时期名臣，屡任方面之任。其在[[扬州]][[刺史]]任上抚纳边民，镇遏南境，为[[北魏]]南疆的稳定做出了重要[[贡献]]。

## 征讨四方

北魏以荆州[[巴]]、[[氐]]扰乱，以镇西大将军李崇为[[荆州]]刺史。崇轻装简从，不烦兵送，恩威并施，边人失和者安之（135-138）。[[武兴]]王[[杨集始]]叛，魏以河南尹李崇为[[都督]]陇右诸军事，将兵讨之。崇槎山分道，出氐不意，表里袭击，群氐皆弃杨灵珍散归，崇进据赤土（141-031、141-034）。鲁阳蛮围湖阳，李崇将兵击破之，斩其渠帅，徙万馀户于幽、并诸州（145-034）。东荆州蛮[[樊素安]]作乱，李崇以镇南将军都督征蛮诸军事，讨破之，生擒素安，进讨西荆诸蛮，悉降之（145-085、145-110）。以功迁[[征南将军]]、[[扬州]][[刺史]]（146-102）。李崇以威略镇边，为[[北魏]]名将。
