# 披靡

**披靡**指敌军如[[草木]]随风倒伏，为古代形容勇猛之师[[所向无敌]]、敌军望风溃散的经典用语。

## 所向披靡

项王大呼驰下，汉军皆披靡，遂斩汉一将（011-005）。辽复还[[突围]]，拔出馀众，权人马皆披靡，无敢当者（067-025）。鸯乃与[[骁骑]]十馀[[摧锋]][[陷陈]]，所向皆披靡，遂引去（076-051）。皝将[[敢死]]数百骑出冲赵兵，[[所向披靡]]，斩获而还（096-015）。木乃帅[[壮士]]四百腾栅而入，秦兵披靡（105-040）。
