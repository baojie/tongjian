# 王肃

**王肃**，字子雍，[[东海]]郯人，[[曹魏]][[经学]]家，[[王朗]]之子。

王肃为[[散骑常侍]]、领[[秘书]]监，数[[上疏]][[言事]]。谏魏[[汉明帝|明帝]]督修[[宫室]]之苛急，以为当务民生（073-015）。[[司马师]]问计于王肃，肃献计使[[毌丘俭]]军心离散（076-045）。王肃以[[经学]][[名家]]，遍注群经，世称"王学"，与[[郑玄]]并立。其女嫁[[司马昭]]，生[[司马炎]]、[[司马]]攸（078-072），为[[西晋]]开创之[[外戚]]。

王肃在[[政治]]上依附[[司马氏]]，为魏晋禅代之际的重要[[人物]]；在学术上为[[经学]]大家，其学派影响深远。

## 谏言献策

王肃在[[曹魏]]历仕[[隋文帝|文帝]]、[[汉明帝|明帝]]、[[齐王]]诸朝，数[[上疏]]陈事。[[魏明帝]]时，王肃以[[散骑常侍]][[上疏]]，言前志有云千里[[馈粮]]士有饥色，请[[朝廷]]慎于[[征伐]]，以安[[百姓]]为先（071-044）。及高贵乡公髦[[即位]]，王肃以[[太常]][[持节]]迎之，正定[[名分]]（076-040）。王肃于[[郊祀]]之礼、[[明堂]]之制亦多所建言，其议为[[朝廷]]所采用。[[晋武帝|司马炎]]为王肃外孙，故晋初[[郊祀]]之礼多从肃议，以[[五帝]]即天帝，自今[[明堂]]、[[南郊]]宜除[[五帝]]座（079-022）。王肃以[[经学]]大家而参议[[朝政]]，其学说与[[制度]]相[[表里]]，影响及于晋初。
