# 杨义臣

杨义臣，代人。隋朔州[[刺史]]，以骁勇善战著称。

[[李景]]被围月余，诏朔州[[刺史]]杨义臣救之。义臣帅马步二万，夜出西陉，悉取军中牛驴，得数千头驱之，[[伏兵]]其后，击破乔钟葵（180-023）。从[[隋炀帝|炀帝]]征[[吐谷浑]]，东屯琵琶峡（181-027）。又从征高丽，与[[宇文述]]趣平壤（182-013）。义臣为隋末名将。

> 李景被围月馀，诏朔州刺史代人杨义臣救之。义臣帅马步二万，夜出西陉。
> （180-023）
