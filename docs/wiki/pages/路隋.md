# 路隋

路隋，[[唐文宗]]朝[[中书侍郎]][[平章事|同平章事]]，因营救[[李德裕]]被贬。

命[[翰林学士]]路隋草遗制（243-089）。以[[翰林学士]]路隋为[[中书侍郎]][[平章事|同平章事]]（243-123）。路隋言于上曰："[[知政事|宰相]]任重，不宜兼金谷琐碎之务。"（244-023）。路隋曰："德裕不至有此。果如所言，臣亦应得罪！"（245-031）。以[[门下侍郎]][[平章事|同平章事]]路隋[[平章事|同平章事]]充[[镇海节度使]]，坐救[[李德裕]]故也（245-033）。

## 世系

路隋（？—约835年），唐[[中书侍郎]]同[[平章事]]、[[镇海]][[节度使]]。其先世不详于史。隋草遗制（243-089），拜相（243-123），坐救[[李德裕]]出为[[镇海节度使]]（245-033）。
