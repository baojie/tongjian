# [[山南东道]]

[[山南东道]]是[[唐代]]十五道之一，与[[山南西道]]同属山南道。玄宗开元二十一年（733年）分天下为十五道，山南东道为其一（213-142）。辖境约当今[[湖北]]西部、[[重庆]]东部及[[湖南]]西北部，治所在[[襄州]]。

山南东道在[[安史之乱]]中具有重要战略地位。[[鲁炅]]任山南东道节度使时，守[[南阳]]以抗[[安禄山]]叛军，城中食尽，饿死者相枕（219-064）。肃宗曾以[[永王璘]]充山南东道、岭南、江南西道节度都使（218-030）。
