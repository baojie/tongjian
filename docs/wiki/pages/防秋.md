# 防秋

**防秋**，[[防御]]秋季入寇，唐时边防[[制度]]。

## 唐代边防制度

防秋为[[唐代]]边防之重要制度。[[开元]]天宝间，已有关中兵集临洮、朔方兵集会州防秋之制，至冬初无寇而罢（213-044）。萧昕论防秋当豫为备，若临事始去则吏卒尚未相识（214-132）。[[大历]]后藩镇岁遣兵诣京西防秋，自赍粮食，所过秋毫不犯（224-142）。[[李忠臣]]将兵二千屯奉天防秋（224-129）。[[硃泚]]遣弟[[朱滔]]将精骑诣[[泾州]]防秋（224-146）。[[郭子仪]]、李抱玉、[[马璘]]、硃泚分统诸道防秋之兵（225-020）。防秋之兵与城内兵合谋作乱，亦时有之（225-034）。防秋制至[[唐末]]犹存。
