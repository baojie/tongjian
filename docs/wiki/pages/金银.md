# 金银

金银即黄金与白银，为古代贵金属，既用作货币流通，亦为[[赏赐]]、贿赂之重要物品。

东[[汉明帝]]时广平、巨鹿、乐成王[[车骑]]朴素无金银之饰，帝以白太后即赐钱各五百万（046-016）。[[班超]]围[[疏勒]]，谢遣骑赍金银珠玉以赂[[龟兹]]，超[[伏兵]]遮击杀之（047-070）。

金银在汉代兼具货币与奢侈品双重属性。黄金为上币，单位为斤；白银为器饰而非币。汉唐[[赏赐]]动辄黄金千斤、白银万两，金银之富为国力象征。

> 广平、巨鹿、乐成王，车骑朴素，无金银之饰，帝以白太后，即赐钱各五百万。
> （046-016）
