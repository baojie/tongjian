# 蒋济

蒋济，字子通，楚国平阿人。曹魏扬州别驾、太尉，历仕四朝，以谋略称。

蒋济为扬州别驾，劝曹操勿徙淮南民（066-004）。蒋济谏文帝勿南征（066-052）。蒋济为太尉，谏齐王芳（068-035）。

蒋济少知名，仕郡计吏。曹操征荆州，济劝操勿徙淮南民，操从之，民皆感悦。文帝时，济为东中郎将，谏伐吴，不从，果无功。明帝时，为护军将军，加散骑常侍。齐王芳时，迁太尉。济每论事，必引经据典，言甚切至。司马懿诛曹爽，济以功封都乡侯，寻卒。
