# 太宁

**太宁**（323—326），[[晋明帝]][[司马绍]][[年号]]。太宁年间，[[王敦]]病笃，明帝乘机下诏讨伐，王敦寻卒，[[王含]]、[[钱凤]]等败死，[[王敦之乱]]平。[[后赵]][[石勒]]南侵，[[祖约]]镇[[寿春]]。明帝在位仅四年而崩，太子[[司马衍]]即位，是为[[晋成帝]]。
