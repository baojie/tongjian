# 崔季舒

崔季舒，字叔正，博陵安平人。[[北齐]]中书侍郎，[[高澄]]心腹，掌察[[孝静帝|东魏孝静帝]]动静。

澄欲置腹心于东魏主左右，擢中兵参军崔季舒为中书侍郎。澄每进书于帝，有所谏请，季舒辄修饰通之（158-135）。澄谓崔季舒曰："[[崔暹]]必造直谏，我亦有以待之。"（159-007）澄使中书黄门郎崔季舒察帝动静，小大皆令季舒知之。澄与季舒书曰："痴人比复何似？痴势小差未？宜用心检校。"（160-042）

崔季舒为[[高澄]]心腹，在东魏朝廷中充当耳目。[[高澄]]权倾朝野，[[孝静帝]]名为天子实同囚徒，崔季舒居中传达，备极荣宠。后仕[[北齐]]，为黄门侍郎。及[[高纬|北齐后主]]时，崔季舒以谏止[[刘禅|后主]]赴[[晋阳]]被诛。
