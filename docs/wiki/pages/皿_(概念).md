# 皿

**皿**为古代[[器物]]。

> （194-103）

> （213-072）

> （216-056）

皿为器皿。[[唐太宗]]营[[昭陵]]，不藏[[金玉]]，人马器皿皆用[[土木]]（194-103）。[[唐玄宗]]敕禁私卖铜[[铅锡]]及以铜为器皿以防盗铸（213-072）。玄宗为[[安禄山]]起第，具幄帟器皿充牣其中（216-056）。[[哥舒翰]]守[[潼关]]，军士散诣[[村落]]求食，[[饮食]]器皿具在（218-013）。[[李抱真]]与[[王武俊]]战，[[回纥]]掠[[馆陶]]器皿而去（229-046）。
