# 段业

段业，京兆人。北凉君主，原为后凉吕光部将，被沮渠蒙逊等推立为主，后为蒙逊所杀。

光与群寮宴，语及政事，参军京兆段业曰："明公用法太峻。"光曰："吴起无恩而楚强，商殃严刑而秦兴。"业曰："明公受天眷命，方当以柔远为怀。"（107-048）以著作郎段业等五人为尚书（108-100）。

段业本为后凉吕光部将，以文吏见用。沮渠蒙逊等叛后凉，推段业为主，建北凉政权。段业在位期间，用蒙逊为将，攻取郡县。然业性多疑，渐忌蒙逊。蒙逊遂杀段业而自立。北凉自段业始，至沮渠蒙逊而强盛，段业仅为一过渡人物。
