# 韩游瑰

韩游瑰，唐中叶将领。[[《资治通鉴》]]载其为邠宁留后，参与平定[[泾原兵变]]，以智略称。

[[郭子仪]]使都游弈使灵武韩游瑰将五百骑前趣河阳，[[周挚]]引兵争河阳，后至不得入而去（221-010）。邠宁留后韩游瑰、庆州刺史[[论惟明]]受诏将兵三千拒[[朱泚]]于便桥，与泚遇于醴泉。游瑰欲还趣奉天（228-059）。朱泚盛兵鼓噪攻南城，韩游瑰曰："此欲分吾力也"，乃引兵严备东北。泚推云梯攻城，游瑰严兵以御（229-014）。
