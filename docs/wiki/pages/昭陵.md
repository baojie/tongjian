# 昭陵

昭陵是[[唐太宗]][[唐太宗|李世民]]的陵墓，位于长安西北九嵕山，[[陪葬]]功臣极多，为唐帝陵之冠。

[[贞观之治|贞观]]十年，文德[[皇后]]先葬于昭陵。将军[[段志玄]]、宇文士及分统士众出肃章门，太宗夜使宫官至二人所，志玄闭门不纳（194-102）。太宗思念[[皇后]]不已，于苑中作层观以望昭陵，引[[魏徵|魏征]]同登。[[魏徵|魏征]]曰"臣昏眊，不能见"，太宗指示之，征曰"臣以为[[陛下]]望献陵，若昭陵则臣固见之矣"，太宗为之泣下，乃毁层观（194-104）。

昭陵以六骏石刻闻名，[[陪葬]]墓多达数百座，包括[[魏徵|魏征]]、[[房玄龄]]、[[李靖]]、[[李勣]]等开国功臣，制度之盛空前绝后。
