# 朱友贞

朱友贞，[[后梁]]末帝，[[梁太祖|朱温]]第四子，以均王[[起兵]]诛友珪[[即位]]。

立子友贞为均王（266-033）。均王友贞为[[洛阳|东都]][[马步都指挥使]]（268-063）。命均王友贞杀友文（268-064）。以均王友贞为[[开封尹]]、[[洛阳|东都]][[鄴都留守|留守]]（268-070）。均王友贞密与谋诛友珪（268-094）。

## 世系

朱友贞（888—923年），[[后梁]]末帝。父[[朱温]]（[[梁太祖|后梁太祖]]），[[朱温|温]]第四子。友贞封均王（266-033），诛[[朱友珪]][[即位]]。与[[李存勖]]（[[后唐庄宗]]）相抗十年。[[后唐]]灭[[后梁]]，友贞自杀。
