# 嘉德殿

**嘉德殿**，[[东汉]][[洛阳]][[南宫]]中的重要殿宇。[[汉安帝|安帝]][[太子]][[刘保|济阴王]]在此夺[[玺绶]][[即位]]；[[汉灵帝]]崩于嘉德殿。

## 通鉴记载

[[汉顺帝]]幸嘉德殿，夺[[玺绶]]收[[阎显]]（051-017）。

[[南宫]]嘉德殿火（054-058）。

[[灵帝|汉灵帝]]崩于嘉德殿，蹇硕欲先诛[[何进]]而立刘协（059-028）。
