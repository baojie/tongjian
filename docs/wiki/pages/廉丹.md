# 廉丹

廉丹，新朝王莽部将，官至更始将军。先后征益州、讨赤眉，最终兵败战死。

王莽遣廉丹与庸部牧史熊大发天水、陇西骑士，广汉、巴、蜀、犍为吏民十万人击益州，不能克（038-014）（038-032）。后与太师王匡东讨赤眉樊崇等（038-067）。

莽诏书让廉丹曰："仓廪尽矣，府库空矣，可以怒矣，可以战矣！将军受国重任，不捐身于中野，无以报恩塞责"（038-072）。丹与王匡攻拔无盐索卢恢，斩首万余级，莽进丹爵为公（038-073）。

廉丹最终与赤眉军交战，兵败战死。
