# 廉丹

廉丹，[[王莽|新朝]][[王莽]]部将，官至更始将军。先后征[[益州]]、讨[[赤眉]]，最终兵败战死。

[[王莽]]遣廉丹与庸部牧史熊大发[[天水]]、[[陇西]]骑士，[[广汉]]、巴、蜀、[[犍为]]吏民十万人击[[益州]]，不能克（038-014）（038-032）。后与太师[[王匡]]东讨[[赤眉]][[樊崇]]等（038-067）。

莽[[诏书]]让廉丹曰："[[仓廪]]尽矣，[[府库]]空矣，可以怒矣，可以战矣！将军受国重任，不捐身于中野，无以报恩塞责"（038-072）。丹与[[王匡]]攻拔无盐索卢恢，斩首万余级，莽进丹爵为公（038-073）。

廉丹最终与[[赤眉]]军交战，兵败战死。

> 莽诏书让廉丹曰："仓廪尽矣，府库空矣，可以怒矣，可以战矣！将军受国重任，不捐身于中野，无以报恩塞责。"
> （038-072）
