# 周迪

周迪，临川人。南朝陈衡州刺史，割据一方。

周文育军中食尽，使人间行遗周迪书，约为兄弟。迪得书甚喜，许馈以粮，文育因得全师（167-012）。周迪在临川，与留异、陈宝应等相连结，闽中豪帅往往立砦以自保（167-051）。迪欲自据南川，乃总召所部八郡守宰结盟，齐言入赴。上恐其为变，厚慰抚之（167-057）。

> 周迪在临川，留异在东阳，陈宝应在晋安，共相连结，闽中豪帅往往立砦以自保。
> （167-051）
