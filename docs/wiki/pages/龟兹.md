# 龟兹

龟兹是[[西域]]北道大国，位于塔里木盆地北缘。[[汉武帝]]时期龟兹成为汉与[[匈奴]]争夺[[西域]]的焦点。贰师将军征[[大宛]]还，将龟兹[[质子]]赖丹入京师，[[霍光]]用[[桑弘羊]]前议以赖丹为校尉[[屯田]]轮台（023-067）。傅介子[[出使]][[大宛]]途经龟兹，责其拦截汉使之罪（023-068）。

常惠奏请击龟兹以报杀校尉赖丹之仇，大将军[[霍光]]许以便宜从事（024-052）。[[乌孙]][[公主]]女嫁龟兹王绛宾，绛宾[[上书]]愿与[[公主]]女俱入朝（025-035）。东[[汉明帝]]时龟兹王[[遣使]]奉献，[[西域]]复通（069-061）。龟兹为汉唐经营[[西域]]之枢纽，唐时设[[安西都护府]]于此。
