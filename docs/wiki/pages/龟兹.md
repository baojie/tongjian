# 龟兹

龟兹是西域北道大国，位于塔里木盆地北缘。汉武帝时期龟兹成为汉与匈奴争夺西域的焦点。贰师将军征大宛还，将龟兹质子赖丹入京师，[[霍光]]用[[桑弘羊]]前议以赖丹为校尉屯田轮台（023-067）。傅介子出使大宛途经龟兹，责其拦截汉使之罪（023-068）。

常惠奏请击龟兹以报杀校尉赖丹之仇，大将军霍光许以便宜从事（024-052）。乌孙公主女嫁龟兹王绛宾，绛宾上书愿与公主女俱入朝（025-035）。东汉明帝时龟兹王遣使奉献，西域复通（069-061）。龟兹为汉唐经营西域之枢纽，唐时设安西都护府于此。
