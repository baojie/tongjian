# [[韶州]]

[[韶州]]是[[唐代]]设置的州，治所在今[[广东]]韶关，隶属[[岭南道]]。韶州地处[[五岭]]中的[[大庾岭]]以南，是[[中原]]翻越大庾岭进入[[岭南]]的第一重镇。[[张九龄]]即韶州[[曲江]]人（214-003），唐玄宗开元名相。

[[武则天]]时，[[李敏]]被流放[[雷州]]，行至韶州，以马缰绞死（202-005）。韶州在唐代以[[曲江]]文化和南北交通枢纽而闻名。
