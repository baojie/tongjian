# 李咸

李咸，东汉太尉。以刚正著称，争皇太后配食桓帝。

以太仆汝南李咸为太尉（056-075）。李咸与胡广、陈蕃并为三司（057-004）。太尉李咸时病，扶舆而起，捣椒自随，谓妻子曰："若皇太后不得配食桓帝，吾不生还矣"，既议，咸竟如议（057-008）。太尉李咸免（057-023）。
