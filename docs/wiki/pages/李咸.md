# 李咸

李咸，[[东汉]][[太尉]]。以刚正著称，争[[皇太后]]配食桓帝。

以[[太仆]][[汝南]]李咸为[[太尉]]（056-075）。李咸与[[胡广]]、[[陈蕃]]并为[[三司]]（057-004）。[[太尉]]李咸时病，扶舆而起，捣椒自随，谓妻子曰："若[[皇太后]]不得配食桓帝，吾不生还矣"，既议，咸竟如议（057-008）。[[太尉]]李咸免（057-023）。
