# 楼船

楼船是古代多层大型战船，为汉代水军主力。汉设楼船将军一职，掌水军。[[汉武帝]]征南越时，"[[马援|伏波将军]]路博德出桂阳，下湟水；楼船将军杨仆出豫章，下浈水"（020-065）。楼船将军杨仆"先陷寻峡，破石门，挫越锋"（020-072），为平南越立下首功。

楼船用于海上作战亦有记载，[[汉武帝]]募天下死罪为兵，"遣楼船将军杨仆从齐浮渤海"以讨朝鲜（021-013）。然楼船士卒条件艰苦，"水居击棹，未战而疾死者过半"（017-060）。[[秦始皇]]使尉屠睢"将楼船之士以攻越"，亦引发重大损失（018-055）。
