# [[洪源谷]]

[[洪源谷]]位于[[凉州]]（今[[甘肃]]武威）附近，是[[唐代]]与[[吐蕃]]交战的重要战场。[[武周]]圣历年间，吐蕃将[[麹莽布支]]寇凉州，陇右诸军大使[[唐休璟]]与战于洪源谷（207-006）。麹莽布支兵甲鲜华，休璟识其不习军事，击败之。

此前，吐蕃[[论赞婆]]归降[[武周]]（206-113），太后宠待甚厚，使其将众守洪源谷。洪源谷是唐代陇右道防御吐蕃的前沿阵地之一。
