# 卫律

**卫律**，父为长水胡人，律本为汉臣，善协律都尉李延年。

卫律[[出使]][[匈奴]]，会李延年被诛，律惧牵连而降[[匈奴]]（021-085）。单于宠信之，封为丁灵王。卫律在[[匈奴]]与[[西汉|汉朝]]对抗中扮演重要角色，曾受单于命治[[苏武]]之事（021-086）。

卫律为汉匈关系中的关键人物。其降[[匈奴]]后多次为单于出谋划策，成为[[西汉|汉朝]]西北边患中的重要对手。然律本胡人之后，在汉为客，在[[匈奴]]为主，其人生选择反映了汉匈边界人物的复杂身份认同。

## 匈奴用事

卫律降[[匈奴]]后，常在单于左右，与[[李陵]]皆贵用事（022-011）。贰师将军[[李广利]]降[[匈奴]]后，尊宠在卫律上，律害其宠，会单于母病，律饬胡巫言先单于怒，遂杀广利（022-076）。单于死，律与颛渠阏氏谋匿丧，矫令立壶衍鞮单于（023-016）。壶衍鞮单于立，国内乖离，卫律为单于谋，与汉[[和亲]]（023-035）。
