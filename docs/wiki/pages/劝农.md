# 劝农

劝农是朝廷督劝农耕的政务，体现重农思想。汉文帝下诏"农，天下之本"，认为田租之赋"于劝农之道未备"，遂免除当年田租（015-033）。汉景帝令郡国"务劝农桑"，以解决民众弃本逐末的问题（016-117）。

儿宽为左内史，"劝农业，缓刑罚，理狱讼"，深得吏民爱戴（020-054）。王莽改制时设大司农部丞十三人分驻各州"劝农桑"（035-046），又遣十一公士分布各地"劝农桑，班时令"（038-006），然徒增纷扰而无实效。
