# 考课

考课是考核官吏治绩的制度，考课结果作为[[黜陟]]升降的依据，为古代吏治的核心环节。

杜恕上疏言今考绩之法废而以毁誉相进退，[[真伪]]浑杂[[虚实]]相蒙。魏[[汉明帝|明帝]]诏[[刘邵]]作《都官考课法》七十二条（073-055）。[[崔林]]引周官考课，言考课之法存乎其人（073-056）。

考课之制源于《周官》六计弊群吏之治。历代考课标准各有不同，唐代以四善二十七最考课官吏。[[司马光]]论考课，以为至公至明则群下之能否焯然形于目中，考课之法不可徒恃条文而贵在得人。

## 考课之法存乎其人

考课之难不在立法，而在得人与执行。杜恕论考绩之法，以为"历六代而考绩之法不著，关七圣而课试之文不垂"，其法可粗依、其详难备举（073-056），制度的精详与实践的粗疏之间[[始终]]存在鸿沟。或论考课之法唐虞所为、[[京房]][[刘邵]]述而修之，然唐虞之官久居其位而可责成，后世官不常任则考课难行（073-058）。[[刘毅]]论[[九品中正]]之弊，指出"当官著效者或附卑品，在官无绩者更获高叙，是为抑功实而隆空名，长浮华而废考绩"（081-074），考课之法虽存，而中正之制反使考课流于形式。考课制度之[[成败]]，不系于条文之多寡，而系于用人是否至公至明。
