# 田布

田布，唐代魏博节度使，田弘正之子。从讨淮西，后讨王庭凑，兵败自杀。

李光颜与田布力战，拒董重质，度仅得入城（240-037）。以左金吾将军田布为河阳节度使（241-069）。起复前泾原节度使田布为魏博节度使，布固辞不获，与妻子宾客诀曰："吾不还矣！"（242-011）。魏博节度使田布将全军三万人讨王庭凑，屯于南宫之南（242-030）。

> 起复前泾原节度使田布为魏博节度使，令乘驿之镇。布固辞不获，与妻子宾客诀曰："吾不还矣！"
> （242-011）
