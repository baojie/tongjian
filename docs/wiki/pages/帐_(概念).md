# 帐

帐，帷帐。秦[[咸阳]]宫观中帷帐[[钟鼓]]美人充之（007-062）。[[项羽]]晨朝[[宋义]]即其帐中斩之（008-048）。[[汉高祖刘邦|刘邦]]见秦[[宫室]]帷帐狗马重宝欲留居之（009-005）。

> 项羽晨朝将军宋义，即其帐中斩宋义头。出令军中曰："宋义与齐谋反楚，楚王阴令籍诛之！"
> （008-048）
