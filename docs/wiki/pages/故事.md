# 故事

**故事**指先例、成例，[[朝廷]]施政的重要参考。

## 汉家故事

汉家常以[[列侯]]尚主（023-020）。故事诸[[上书]]者皆为二封，署其一曰副，领[[尚书]]者先发副封，所言不善屏去不奏（024-074）。

## 观故事知得失

[[魏相]]好观汉故事及便宜[[章奏]]，数条汉兴以来国家便宜行事（025-054）。故事为后世法，循之则治，违之则乱。
