# [[蓟县]]

[[蓟县]]是[[秦]][[汉]]以来燕地的重要县邑，为[[广阳郡]]治所，即今[[北京]]的前身之一。[[战国]]时期为[[燕国]]都城[[蓟_(地名)|蓟]]所在地。[[秦]][[汉]]以后，蓟县历为广阳郡、[[燕国]]、[[幽州]]治所。

[[唐代]]蓟县为[[幽州]][[治所]]，[[安史之乱]]时[[安禄山]]即据幽州起兵。蓟县作为[[北方]]军事重镇，在[[中原]]与[[游牧民族]]的交往中始终具有枢纽地位。
