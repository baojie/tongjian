# 秘书

秘书，宫廷中秘藏之图书典籍，亦指掌管图书之官署。

[[汉成帝]]以秘书颇散亡，使陈农求遗书（030-058）。[[刘向]]父子领校秘书，阅定九流（052-005）。[[荀悦]]为秘书监作《申鉴》（064-047）。

[[汉武帝]]后，广开献书之路，百年之间书积如山。成帝时命[[刘向]]领校秘书，向卒，子歆继之，总群书而奏《七略》，为中国目录学之祖。[[东汉]]置秘书监，掌典[[图籍]]。魏晋以后，秘书省为中央图书机构。秘书之职虽清要，非博学通儒不得任。
