# [[永州]]

[[永州]]是[[隋]][[唐]]时期设置的州，治所在今[[湖南]]永州（[[零陵]]）。地处[[湘水]]上游，南接[[岭南]]，是沟通[[中原]]与[[岭南]]的重要通道。

[[唐代]]著名[[文学]]家[[柳宗元]]曾[[贬谪]]永州[[司马]]，在此留下《永州八记》等名篇，使永州成为[[文学]]史上著名的[[贬谪]]之地。永州也为[[唐朝|唐代]][[江南西道]]属州。
