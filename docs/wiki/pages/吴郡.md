# 吴郡

吴郡，[[东汉]]分[[会稽]]置吴郡，在今[[江苏]]南部和[[浙江]]北部，治[[吴县]]。[[孙坚]]为吴郡[[司马]]（057-035）。[[朱治]]逐[[吴郡太守]]许贡（061-060）。[[孙策]]杀[[吴郡太守]]许贡（063-023）。[[陆逊]]、[[全琮]]、[[顾雍]]皆吴郡人。吴郡为[[孙氏]][[起家]]之地，[[江东]]之[[腹心]]。

### 三国·孙吴

朱治逐[[吴郡太守]]许贡，领[[魏郡太守|太守]]事（061-060）。[[孙策]]杀故[[吴郡太守]]许贡（063-023）。[[陆逊]]，吴郡吴人（068-010）。[[全琮]]，吴郡[[钱塘|钱唐]]人（068-043）。[[顾雍]]，吴郡吴人（069-052）。丹杨邻接吴郡，其势可互为[[应援]]（072-071）。

## 吴郡之义

吴郡在[[通鉴]]中为[[江东]][[腹心]]之地。[[孙坚]]为吴郡[[司马]]（057-035）、[[孙策]]据吴郡（063-023），为[[孙吴]]立国之基。[[陆逊]]（068-010）、[[全琮]]（068-043）、[[顾雍]]（069-052）皆出吴郡，见吴郡为[[江东]][[士族]]之渊薮。[[通鉴]]记吴郡，见[[苏州]]一带自[[东汉]]末至[[南北朝]]在[[政治]]、[[军事]]、人才中之地位。
