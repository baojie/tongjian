# 商贾

商贾是古代对商人的统称，行商曰商，坐贾曰贾。在重农抑商政策下，商贾社会地位低下。

[[鲁仲连]]辞[[平原君]]封赏，谓所贵于天下之士者为人排患释难而无所取，若有所取，是商贾之事也（005-059）。[[韩信]]家贫无行，不得推择为吏，又不能治生商贾，常从人寄食饮（009-017）。[[晁错]]论贵粟疏谓商贾大者积贮倍息，小者坐列贩卖，因其富厚交通王侯，力过吏势（015-024）。

汉初令[[贾人]]不得衣丝、乘车，重租税以困辱之；孝惠、[[吕后|高后]]时复驰商贾之律，然[[市井]]之子孙亦不得仕宦为吏（016-122）。至[[桑弘羊]]行[[平准]][[均输]]之法，始以商贾之术为国家理财（020-089）。

> 商贾大者积贮倍息，小者坐列贩卖，操其奇赢，日游都市，乘上之急，所卖必倍。
> （015-024）
