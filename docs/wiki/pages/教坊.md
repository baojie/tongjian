# 教坊

教坊是唐代始设的宫廷俗乐机构，掌歌舞俳优、散乐百戏。唐高祖时置内教坊于禁中，以按习雅乐（193-117）。玄宗酷爱音乐，更置左右教坊以教俗乐，又选坐部伎子弟三百教于梨园（211-005），教坊与梨园并称。

唐文宗时"教坊宴犒，皆引 fav 人大内，歌舞喧阗"（245-057）。教坊乐工技艺精湛，[[白居易]]《琵琶行》所写琵琶女即"名属教坊第一部"。教坊制度影响深远，后世多沿其制，为宫廷音乐文化的核心机构。
