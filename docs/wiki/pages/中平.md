# 中平

**中平**（184—189），[[汉灵帝]]第四个[[年号]]。中平元年，[[黄巾]]起事，[[张角]]称天公将军，[[张宝]]称地公将军，[[张梁]]称人公将军。灵帝发天下精兵，以[[皇甫嵩]]、[[朱儁]]、[[曹操]]等讨之。黄巾虽平，而朝廷积弱，[[董卓]]、[[袁绍]]等乘势而起。中平六年，灵帝崩，[[少帝]]立。
