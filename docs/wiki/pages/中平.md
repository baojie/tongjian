# 中平

**中平**（184—189），[[汉灵帝]]第四个[[年号]]。中平元年，[[黄巾]]起事，[[张角]]称天公[[将军]]，[[张宝]]称地公[[将军]]，[[张梁]]称人公[[将军]]。灵帝发[[天下]][[精兵]]，以[[皇甫嵩]]、[[朱儁]]、[[曹操]]等讨之。黄巾虽平，而[[朝廷]]积弱，[[董卓]]、[[袁绍]]等乘势而起。中平六年，灵帝崩，[[少帝]]立。
