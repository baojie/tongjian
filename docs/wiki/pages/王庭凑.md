# 王庭凑

王庭凑，[[契丹]]人。唐成德[[节度使]]，杀[[田弘正]]自立，为[[河朔三镇]]之一。

王庭凑杀[[田弘正]]，自称成德留后（242-007）。唐穆宗诏诸道讨王庭凑，不克（242-014）。王庭凑与朱克融连兵拒命（242-016）。

王庭凑初为[[田弘正]]部将，弘正移镇成德，庭凑以成德旧卒怨弘正，乃相率作乱，杀弘正并其家属。庭凑遂自称留后，朝廷不能讨，乃以[[节钺]]授之。庭凑自是据成德，与卢龙朱克融相为表里，朝廷无如之何。庭凑卒，其[[司马师|子元]]逵继立。
