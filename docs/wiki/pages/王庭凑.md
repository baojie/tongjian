# 王庭凑

王庭凑，契丹人。唐成德节度使，杀田弘正自立，为河朔三镇之一。

王庭凑杀田弘正，自称成德留后（242-007）。唐穆宗诏诸道讨王庭凑，不克（242-014）。王庭凑与朱克融连兵拒命（242-016）。

王庭凑初为田弘正部将，弘正移镇成德，庭凑以成德旧卒怨弘正，乃相率作乱，杀弘正并其家属。庭凑遂自称留后，朝廷不能讨，乃以节钺授之。庭凑自是据成德，与卢龙朱克融相为表里，朝廷无如之何。庭凑卒，其子元逵继立。

> 王庭凑杀田弘正，自称成德留后。
> （242-007）
