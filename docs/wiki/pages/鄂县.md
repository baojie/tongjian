# [[鄂县]]

[[鄂县]]是[[秦]][[汉]]置县，属[[江夏郡]]，位于今[[湖北]]鄂州。[[三国]]时期，[[刘备]]用[[鲁肃]]计，进住鄂县之[[樊口]]（065-046），与[[孙权]]联合抗曹。

[[孙权]]于[[黄初]]元年（220年）自[[公安]]迁都于鄂县，改名[[武昌]]，后[[东吴]]亦曾以武昌为都或陪都。[[隋]][[唐]]时期鄂县改为[[鄂州]]治所（今[[武昌]]），是长江中游的重要城市。
