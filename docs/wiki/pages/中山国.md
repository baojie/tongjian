# [[中山国]]

[[中山国]]是[[战国]]时期由[[白狄]]建立的[[中原]][[诸侯]]国，位于今[[河北]]石家庄、[[定州]]一带，都[[灵寿]]。中山国在[[战国]]中期一度强盛，[[魏文侯]]曾遣[[乐羊]]攻灭中山，后中山桓公复国。

[[赵武灵王]][[胡服骑射]]后，终灭中山国。[[汉代]]亦封[[中山王]]，[[汉成帝]]封[[中山王]]舅[[冯参]]为宜[[乡侯]]，益中山国三万户（032-043）。[[东汉]][[光武帝]]以[[常山]]郡益中山国，以郭圣通为中山[[太后]]（043-057）。[[中山]]为[[汉代]]北方重要的同姓[[诸侯王]]国。
