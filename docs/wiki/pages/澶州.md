# [[澶州]]

[[澶州]]是[[唐代]]设置的州，治所在今[[河南]]濮阳，地处[[黄河]]下游，是[[河北道]]与[[河南道]]的交界州。[[晚唐]]初年，[[澶州]]为[[魏博节度使]]辖州。

[[唐僖宗]]时，[[乐行达]]以澶州刺史归[[魏博]]，后为部下立为魏博留后（255-046）。[[五代]]初，[[后梁]]与[[晋]]交战，[[周德威]]攻澶州，刺史[[张可臻]]弃城走（267-134）。澶州在[[宋代]]因[[澶渊之盟]]而闻名。
