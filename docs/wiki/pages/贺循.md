# 贺循

贺循，字彦先，会稽山阴人。东晋中书令，江南士族领袖。

前侍御史贺循起兵于会稽，与周玘、华谭等共讨陈敏（085-028）。周玘、贺循皆散众还家，不言功赏（085-042）。陈敏据江东，以贺循为丹杨内史（086-023）。东晋立，以军咨祭酒贺循为中书令（090-010）。循为江南名士，朝野推重。

> 前侍御史贺循起兵于会稽，庐江内史广陵华谭及丹扬葛洪、甘卓皆起兵以应秘。
> （085-028）
