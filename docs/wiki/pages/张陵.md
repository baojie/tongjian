# 张陵

张陵，东汉桓帝朝尚书，以刚直不阿著称。

元嘉元年正月朔，群臣朝贺，大将军梁冀带剑入省。尚书张陵呵叱令出，敕羽林、虎贲夺剑。冀跪谢，陵不应，即劾奏冀，请廷尉论罪（053-051）。

尚书郎张陵与朱寓、荀绲、刘祐等皆国之贞士，朝之良佐（056-007）。张陵不畏权贵，以直声著于朝廷。
