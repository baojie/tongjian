# 杨谅

杨谅，[[隋文帝]]第五子，封[[汉高祖刘邦|汉王]]。

杨谅曾为[[晋阳|并州]]总管，[[隋文帝|文帝]]以五子杨秀有异谋，废为庶人。秀尝请西岳慈父圣母神兵收[[隋文帝|杨坚]]、杨谅神魂（179-071）。[[隋炀帝|炀帝]]征高丽，以杨谅不肖，师出无功（181-054）。谅后起兵反[[隋炀帝|炀帝]]，兵败降。

> 先帝欲征之久矣，但杨谅不肖，师出无功。当陛下之时，安可不取。
> （181-054）
