# 谷吉

谷吉，[[西汉]]卫[[司马]]，[[出使]][[匈奴]]。

[[郅支单于]]怨汉拥护[[呼韩邪单于|呼韩邪]]，困辱汉使，[[遣使]]奉献求侍子。汉遣卫[[司马]]谷吉送[[郅支单于]]子归，[[御史大夫]][[贡禹]]等谏以为不可，谷吉遂行，为[[郅支]]所杀（028-047）。后[[陈汤]][[矫制]][[发兵]]诛[[郅支单于]]，得谷吉所赍[[帛书]]（029-055），雪谷吉之耻（029-076）。

## 世系

谷吉（？—约[[前44年]]），[[西汉]]卫[[司马]]。其先不详于史。吉为[[卫司马]]，[[汉元帝]]遣吉送[[郅支单于]]子归[[匈奴]]，[[御史大夫]][[贡禹]]谏不可，吉遂行（028-047）。为[[郅支]]所杀。后[[陈汤]]诛[[郅支]]，得吉所赍[[帛书]]（029-055）。子[[谷永]]。
