# 王离

王离，频阳东乡人。秦将，王翦之孙，王贲之子。巨鹿之战被项羽所俘。

王离为秦将，围巨鹿（007-027）。王离为项羽所破，被俘（007-031）。

王离继父祖为秦将。陈胜起兵，秦使离击赵。离围赵王歇于巨鹿。项羽救赵，破釜沉舟，大破秦军。离败走，被擒。或曰："王离，秦之名将也。今为项羽所擒，何也？"答曰："为将三世者必败。以其杀伐多矣，其后受其不祥。"王离之败，秦由此不振。

> 王离为秦将，围巨鹿。
