# 旱灾

旱灾是持续干旱造成的灾害，古时常以天人感应释之。[[汉章帝]]问[[司徒]]鲍昱"何以消复旱灾"，鲍昱以"[[陛下]]始践天位"对，劝帝修德政以弭灾（046-003）。[[南朝]]宋刘义欣修治芍陂，引河水溉田万馀顷，此后民无旱灾之忧（121-121）。

[[唐太宗]]征高丽后，高丽人因"前大驾亲征，国人不得[[耕种]]，继以旱灾"，民太半乏食（198-074）。唐代宗时各地旱灾频发，[[狄仁杰]]建议"当理内以及外，绥近以来远"（208-108）。唐穆宗时江州[[刺史]]李渤上奏"当州今岁旱灾，田损什九"，力谏不可于[[大旱]]中征三十六年前逋负（242-067）。
