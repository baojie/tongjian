# 大中

**大中**（847—860），[[唐宣宗]][[李忱]][[年号]]。大中年间，[[宣宗]]勤于政事，整顿[[吏治]]，用法无私，人称"[[小太宗]]"。[[李德裕]]贬死[[崖州]]，[[牛李党争]]终结。[[张议潮]]在[[沙州]]起义归唐，[[河西]]复归唐室，史称"[[大中之治]]"。大中十三年，[[宣宗]]崩。
