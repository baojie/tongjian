# 举劾

举劾，检举[[弹劾]]。[[江充]]为直指绣衣[[使者]]，举劾无所避（022-027）。[[丞相]]长史[[黄霸]]阿纵[[龚胜]]不举劾而下狱（024-046）。司隶部[[刺史]]举劾苛细，发扬阴私（034-025）。

> 充举劾无所避，上以为忠直，所言皆中意。尝从上甘泉，逢太子家使乘车马行驰道中，充以属吏。
> （022-027）
