# [[永丰仓]]

[[永丰仓]]是[[隋]][[唐]]时期的重要粮仓，位于[[华阴]]（今[[陕西]]华阴），地处[[渭水]]入[[黄河]]处，是[[关中]]东部最大的漕运粮仓。隋朝修建[[大兴城]]后，为保障[[京师]]粮食供应，在[[渭水]]沿线设置永丰仓等大型官仓。

永丰仓是[[隋末]]群雄争夺的战略目标之一。[[李渊]]起兵入[[长安]]时，永丰仓的粮食为其提供了重要的后勤保障。[[唐代]]永丰仓仍为关东[[漕运]]入[[关中]]的重要中转站。
