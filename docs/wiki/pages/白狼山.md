# [[白狼山]]

[[白狼山]]位于今[[辽宁]]喀喇沁左翼一带，是[[东汉]]末年[[曹操]]北征[[乌桓]]时的决战之地。[[建安]]十二年（207年），曹操率军出[[卢龙塞]]，经[[白檀]]、[[平冈]]，在[[凡城]]附近的白狼山与乌桓[[蹋顿]]主力遭遇。

[[张辽]]率先锋突击，在白狼山下斩蹋顿于阵前，乌桓军大溃。此战后，[[幽州]]东北边境平定，曹操得以全力南向与[[孙权]]、[[刘备]]争雄（065-035）。
