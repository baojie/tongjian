# 张希崇

张希崇，[[五代]]后唐卢龙节度使。本书生，为幽州牙将，没于契丹。契丹以张希崇代卢文进为卢龙节度使守平州（276-061）。后为朔方节度使，为政有威信，民夷爱之，兴屯田以省漕运。在镇五年，求内徙，唐潞王以为静难节度使（280-063）。张希崇卒，羌胡寇钞无复畏惮（282-004）。
