# 张希崇

张希崇，[[五代]][[后唐]]卢龙[[节度使]]。本书生，为[[幽州]]牙将，没于[[契丹]]。[[契丹]]以张希崇代[[卢文进]]为卢龙[[节度使]]守平州（276-061）。后为[[朔方]][[节度使]]，为政有威信，民夷爱之，兴[[屯田]]以省[[漕运]]。在镇五年，求[[内徙]]，唐潞王以为静难[[节度使]]（280-063）。张希崇卒，羌胡寇钞无复畏惮（282-004）。

> 朔方节度使张希崇为政有威信，民夷爱之，兴屯田以省漕运。
> （280-063）
