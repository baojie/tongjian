# 苏峻

苏峻，东晋历阳内史。初率众归晋，平[[王敦]]有功，后举兵反晋，攻入[[建康]]。

掖人苏峻帅乡里数千家结垒以自保，远近多附之。曹嶷恶其强，将攻之，峻帅众浮海来奔。帝以峻为鹰扬将军（091-003）。[[王敦]]反，诏征苏峻等入卫京师（093-011）。苏峻等帅精卒万人至，帝夜见劳之（093-021）。[[庾亮]]督苏峻等追[[沈充]]于吴兴（093-023）。

后[[庾亮]]以苏峻兵强，欲夺其兵权，峻遂举兵反，攻入[[建康]]。
