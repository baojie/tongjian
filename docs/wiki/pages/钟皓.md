# 钟皓

钟皓，颍川人，东汉名士，"颍川四长"之一。

钟皓素与荀淑齐名，李膺常叹曰："荀君清识难尚，钟君至德可师。"皓兄子瑾母，李膺之姑也（053-041）。钟皓与同郡荀淑、陈寔皆尝为县长，所至以德政称，时人谓之"颍川四长"（053-103）。

钟皓以德行著称，为颍川士族之代表。
