# 舌

**舌**为人体部位。**舌**，口中辨味发音之器官。公孙吕面长三尺，其舌不容于鼻（004-070）。[[张仪]]为楚相所辱，其妻笑曰："舌在也。"仪曰："足矣。"（011-022）。舌战、舌耕，以言语辩说为事。
