# 阎鼎

阎鼎，西晋太子詹事。聚流民于密，奉秦王业入关据长安，总摄百揆。

前豫州刺史天水阎鼎，聚西州流民数千人于密，欲还乡里。荀籓以鼎有才而拥众，用鼎为豫州刺史（087-085）。阎鼎欲奉秦王业入关，据长安以号令四方，遂行（087-096）。贾疋等奉秦王业为皇太子，以阎鼎为太子詹事，总摄百揆（088-031）。

后阎鼎与京兆太守梁综争权，鼎杀综。麹允等合兵攻鼎，鼎出奔，为氐所杀（088-036）。
