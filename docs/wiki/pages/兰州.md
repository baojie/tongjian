# 兰州

兰州为[[隋代|隋]][[唐代|唐]]州名，治金城（今甘肃兰州），西北边防重镇。[[隋]]时[[突厥]]寇兰州，[[贺娄子干]]败之。[[唐]]初[[吐谷浑]]寇兰州，州兵击走之。[[薛举]]据金城称秦帝，开隋末割据之端。

## 兰州之义

兰州在[[通鉴]]中为[[陇右]]之重镇。[[隋]]时突厥频寇兰州（175-076、175-085），见[[隋]]代西北边防之紧张。唐初[[吐谷浑]]寇兰州（194-010），为唐击吐谷浑之背景。[[吐蕃]]军兰州（211-036），为[[唐]][[蕃]]战争西线之冲。[[契苾何力]]弟为兰州都督（196-049），见唐以[[归附]]诸族领边州之策。[[通鉴]]记兰州，见[[金城]]在[[陇右]]防御体系之枢纽地位。
