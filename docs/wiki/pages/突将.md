# 突将

**突将**指选募[[骁勇]]之士组成的[[精锐]]突击部队，为古代[[军事]]中承担冲锋[[陷阵]]、突袭[[破敌]]任务的重要兵种。

## 突将精兵

丧[[赵云]]、阳群、马玉、阎芝、丁立、白寿、刘郃、邓铜等及曲长、屯将七十馀人，突将、无前、賨叟、青羌、[[散骑]]、武骑一千馀人（071-017）。突将[[王元]]振将作乱，矫令于众曰："来日修[[都统]]宅，各具[[畚锸]]，待命于门。"（222-052）。乃募[[敢死]]士三千人，号曰突将，[[朝夕]]自教习之，使常为行备，欲以袭蔡（240-022）。命[[李祐]]、[[李忠]]义帅突将三千为[[前驱]]（240-038）。[[淄青]][[魏博节度使|节度使]][[薛平]]奏突将马廷崟作乱，[[伏诛]]（242-035）。得[[选兵]]三千人，号曰"突将"（252-004）。卢耽以[[杨庆复]]、摄左[[都押牙]][[李骧]]各帅突将[[出战]]，杀伤蛮二千馀人（252-009）。[[杨庆复]]以右职优给募突将以御之，[[成都]]由是获全（252-091）。
