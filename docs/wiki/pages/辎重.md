# 辎重

辎重是军队携带的物资装备，包括粮草、兵器、营帐等，为后勤保障之核心。

苏秦为从约长，并相六国，北报赵，车骑辎重拟于王者（002-094）。韩信请刘邦假奇兵三万人从间道绝楚军辎重（010-004）。王恢主别从代出击胡辎重（018-007）。

辎重之得失关乎军队存亡，兵法云"军无辎重则亡"，绝敌粮道为常用战术。辎重保护与运输是古代军事组织的重要课题。

> 行数百里，其势粮食必在其后。愿足下假臣奇兵三万人，从间路绝其辎重。
> （010-004）
