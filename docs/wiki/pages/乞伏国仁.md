# 乞伏国仁

乞伏国仁，陇西鲜卑乞伏部首领，西秦建立者。趁前秦淝水败后之机自立，据陇西之地。

秦王坚之入寇也，以乞伏国仁为前将军，领先锋骑。会国仁叔父步颓反于陇西，坚遣国仁还讨之（105-026）。乞伏国仁自称大都督、大将军、单于，领秦、河二州牧，改元建义（106-053）。

乞伏国仁为陇西鲜卑乞伏部首领，前秦时受苻坚之命率部镇守陇西。淝水之战后前秦瓦解，国仁遂据陇西自立，建立西秦政权，都勇士城。国仁招抚诸部，务农积谷，为西秦立国奠定了基础。乞伏氏虽称王，然卒为姚秦所灭。
