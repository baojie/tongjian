# 李德诚

李德诚，[[五代]]杨[[吴将]]领，历仕[[杨行密]]、[[徐温]]、[[徐知诰]]。

[[杨行密]]得李德诚，以宗女妻之（258-013）。德诚与[[张训]]潜入[[扬州]]，灭馀火，得谷数十万斛以赈[[饥民]]（258-088）。[[杨行密]]将[[张训]]、李德诚败[[时溥]]兵于寿河（259-024）。李德诚攻朱[[仁义]]，[[仁义]]以德诚有礼，以爱妾赠之（265-027）。徐知训[[骄倨]]，求李德诚家妓，德诚婉拒之（270-037）。

## 世系

李德诚，[[五代]][[杨吴]]将领。其先世不详于史。[[杨行密]]得德诚，以宗女妻之（258-013）。历仕行密、[[徐温]]、[[徐知诰]]。尝攻[[朱仁义]]，[[仁义]]以德诚有礼，以爱妾赠之（265-027）。德诚[[子孙]]不详于史。
