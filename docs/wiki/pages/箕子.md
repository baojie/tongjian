# 箕子

**箕子**，商末[[宗室]]，[[纣王]]诸父（一说庶兄），封于箕。

## 谏纣被囚

纣王无道，箕子谏而不听，乃被发佯狂为奴而被囚。

## 封于朝鲜

[[周武王]]灭商，释箕子之囚，封[[比干]]之墓，表商容之闾（010-011）。[[班固]]曰：玄菟、[[乐浪]]，本箕子所封。昔箕子居[[朝鲜]]，教其民以[[礼义]]，田蚕织作，为民设禁八条（021-024）。

[[王莽]]时，以[[中山]]王箕子为嗣，封为定安公（035-029），然此乃刘氏[[宗室]]袭箕子之名，非商周之箕子。[[北齐]]时，[[中书侍郎]][[郑颐]]私诱[[祠部尚书]]王昕，以"[[箕子]]为之奴"劝昕曲从（167-099），盖以箕子佯狂为奴之典。

箕子以仁德著于后世，《[[尚书]]·洪范》旧传为箕子为[[周武王|武王]]陈[[天道]]之作。其避地[[朝鲜]]之行，开[[朝鲜]]半岛[[礼义]][[教化]]之始。
