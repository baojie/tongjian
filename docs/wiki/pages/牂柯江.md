# [[牂柯江]]

[[牂柯江]]是[[汉代]]西南地区的重要河流，即今[[贵州]]北盘江及其下游[[红水河]]。[[汉武帝]]建元年间，[[番阳令]][[唐蒙]]出使[[南越]]，南越人食以蜀枸酱，问所从来，答曰道西北牂柯江，广数里，出[[番禺]]城下（018-022）。

唐蒙由此发现蜀地经牂柯江可直通南越，还报武帝，建议开[[夜郎]]道，发巴蜀卒治道，自[[僰道]]指牂柯江（018-023），遂置[[犍为郡]]。牂柯江是汉朝开拓西南夷、经营岭南的重要水路通道。
