# 张嶷

张嶷，字伯岐，巴西人。[[蜀汉]]越巂太守，以绥抚蛮夷、安边保境著称。

汉主以巴西张嶷为越巂太守，嶷招慰新附，诛讨强猾，蛮夷畏服，郡界悉平（074-038）。越巂太守张嶷尝以书戒[[费祎]]曰："昔[[岑彭]]率师，[[来歙]]杖节，咸见害于刺客。"（076-003）。越巂太守张嶷与诸葛瞻书曰："东主初崩，帝实幼弱，[[太傅]]受寄托之重，亦何容易！"（076-023）

张嶷在越巂十五年，治化大行，郡中安靖。后随[[姜维]]北伐，战殁于阵。嶷体恤民情、抚慰蛮夷，深得边民爱戴。其治边之策，宽猛相济，为[[蜀汉]]后期边疆重臣。
