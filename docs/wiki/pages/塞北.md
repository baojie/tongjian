# 塞北

塞北指长城以北的广大地区，自古为游牧民族活动区域。

前燕慕容俊遣[[慕容垂]]、慕容虔等帅步骑八万攻敕勒于塞北，大破之，俘斩十馀万，获马十三万匹、牛羊亿万头（100-059）。南朝[[萧衍|梁武帝]]时，接纳[[侯景]]归降，群臣谏阻，武帝以"得景则塞北可清"为由拒谏（160-012）。塞北与[[中原]]的对抗与融合贯穿中国古代历史，[[匈奴]]、[[鲜卑]]、[[突厥]]、[[契丹]]、蒙古等民族相继在此兴起。

## 突厥与塞北

突厥自幽州还，留屯塞北，更集诸部兵，与北周相约共击北齐（169-067）。唐太宗时，突厥颉利可汗败亡，朝议处置突厥降众，颜师古请置之塞北，分其种落散居州县，教之耕织，以化胡虏为农民，永空塞北之地（193-064）。此前，代州都督张公谨论突厥可诛之状，列其五罪，中有"塞北霜早，糇粮乏绝"一条（193-031）。塞北之兴衰，恒与中原王朝之盛衰相表里。
