# 五行

五行，金木水火土五种基本元素及其运行变化之学说。

汉奏盛德文始五行之舞（024-046）。刘向著洪范五行传论（030-059）。

五行学说在汉代极盛。宣帝时有司奏尊孝武帝庙，奏盛德、文始五行之舞。刘向集合上古至秦汉符瑞灾异之记，推迹行事，著《洪范五行传论》，天子心知向忠精，然终不能夺王氏权。五行生克之说，用以解释朝代更替、灾异变化。汉代以阴阳五行附会经义，董仲舒尤精此道。
