# 屯戍

屯戍是军队在边境[[屯田]]戍守的制度，始于[[汉文帝]][[晁错]]之议。

秦末[[项羽]]平定[[河北]]后，诸侯吏卒繇使屯戍过秦中者，秦中吏卒遇之多无状，积怨甚深（009-006）。[[汉文帝]]时，[[晁错]]上疏建议徙民实边，使远方无屯戍之事，塞下之民父子相保无系虏之患（015-012）。又建言募民徙以实塞下，使屯戍之事益省、输将之费益寡（015-014）。屯戍制度既解决边防军粮补给问题，又减轻内地百姓远程戍守的负担，为历代沿袭。
