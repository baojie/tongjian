# 高德政

高德政，渤海人。[[北齐]]佐命功臣。

[[高澄]]以高德政为黄门侍郎，佐[[高洋]]留守于邺（160-031）。[[高洋]]之为开府也，高德政为管记，由是亲昵，言无不尽（163-032）。德政录在邺诸事，条进于洋，洋令陈山提驰驿赍事条并密书与[[杨愔]]（163-033）。德政参预[[北齐]]禅代之谋。

> 东魏齐王洋之为开府也，勃海高德政为管记，由是亲昵，言无不尽。
> （163-032）

## 禅代元勋

[[高德政]]为[[高澄]]所用，使佐[[高洋]]留守于邺（160-031）。及高洋为齐王，德政为管记，由是亲昵，言无不尽（163-032）。德政录在邺诸事，条进于洋，参预[[北齐]]禅代之谋（163-033）。

德政为侍中、尚书右仆射（167-112）。后德政与[[杨愔]]同为相，愔常忌之。德政数强谏，齐主[[高洋]]不悦，谓左右曰："高德政恒以精神凌逼人。"（167-119）
