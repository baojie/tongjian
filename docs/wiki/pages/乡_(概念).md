# 乡

> （001-060）

> （002-020）

> （002-049）

乡为古代基层行政单位。[[吴起]]言守[[西河]]而秦兵不敢东乡（001-060）。[[商鞅]]变法后[[秦国]]乡邑大治（002-020），又并诸小乡聚集为一县（002-049）。[[苏秦]]说六国[[苏秦合纵|合纵]]，曰六国为一并力西乡而攻秦（002-091）。[[陈胜]]、[[吴广]]屯[[陈胜吴广起义|大泽乡起义]]（007-084）。[[刘邦]]入[[咸阳]]，使人与秦吏行县乡邑告谕之（009-005）。[[项羽]]谓"富贵不归故乡，如衣绣夜行"（009-010）。[[范雎]]说[[秦昭王]]曰莫肯乡秦（005-014）。秦人怜[[白起]]，乡邑皆[[祭祀]]焉（005-058）。
