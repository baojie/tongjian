# 卢文进

卢文进，字国用，范阳人。五代卢龙节度使，后奔南吴。

卢文进为卢龙节度使，叛晋奔吴（268-106）。卢文进在吴，领兵攻中原（269-100）。

卢文进初为刘守光骑将。后唐庄宗时，文进举幽州降。既又叛归契丹，导契丹入寇。后唐明宗时，文进复南奔吴国。吴主杨溥以为卢龙节度使，使帅水军攻中原。文进屡反覆，时人谓之"卢反覆"。然文进在吴，以勇略见称。吴亡，不知所终。

> 卢文进为卢龙节度使，叛晋奔吴。
> （268-106）
