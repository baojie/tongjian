# 升平

**升平**（357—361），[[晋穆帝]][[司马聃]]第三个[[年号]]。升平年间，[[前秦]][[苻坚]]杀[[苻生]]自立，任[[王猛]][[辅政]]，国势日强。[[前燕]][[慕容儁]][[称帝]]，与[[前秦]]争霸[[中原]]。[[桓温]]屡请[[北伐]]，[[朝廷]]忌之。升平五年，穆帝崩，[[司马丕]]立，是为[[晋哀帝]]。
