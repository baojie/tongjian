# 于仲文

于仲文，字次武，[[洛阳]]人。隋右翊卫大将军，[[于谨]]之孙。

[[尉迟迥]]招[[徐州]]总管源雄、东[[郡守]]于仲文，皆不从（174-039）。于仲文率军征讨檀让，以羸师挑战而伪北，让不设备，仲文还击大破之（174-056）。[[隋炀帝|炀帝]]征高丽，于仲文出乐浪道（181-077）。仲文为隋初名将，屡统军征讨。

> 于仲文军至蓼隄，去梁郡七里。檀让拥众数万，仲文以羸师挑战而伪北，让不设备；仲文还击，大破之。
> （174-056）
