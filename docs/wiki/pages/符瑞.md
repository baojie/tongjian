# 符瑞

符瑞是[[祥瑞]]征兆，指帝王受命于天的天象物证，为古代政治合法性的重要理论依据。

[[司马相如]]病且死，有遗书颂功德，言符瑞，劝[[汉武帝]]封泰山（020-080）。[[刘向]]集合上古以来符瑞灾异之记，推迹行事，著其占验，号《洪范[[五行]]传论》（030-059）。[[谷永]]对[[汉成帝]]言王者躬行道德则符瑞并降，失道妄行则妖孽并见（032-017）。

符瑞之说源于天人感应思想，历代君主常借符瑞以彰其德，群臣亦以献符瑞为邀宠之资。汉代符瑞之说最为盛行，后世亦沿用不绝。

> 王者躬行道德，承顺天命，则五征时序，百姓寿考，符瑞并降。
> （032-017）
