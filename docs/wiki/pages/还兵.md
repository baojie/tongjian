# 还兵

**还兵**指回师或退兵，为古代军队改变进攻方向、暂避锋芒或转移兵力的战术行动。

## 还兵待时

[[章邯]]劝[[项羽]]还兵与[[诸侯]]从约共攻秦（008-060）。[[齐王]]闻[[吕氏]]变乃还兵西界待约（013-057）。郑兴劝更始还兵待时，高帝[[征伐]]累年犹以[[汉高祖刘邦|沛公]]行师（041-072）。

## 还兵劫太后

郑飒等出狱还兵劫[[太后]]夺[[玺绶]]（056-031）。虎[[牙将]]军欲还兵，[[御史]]属谏不可（024-050）。
