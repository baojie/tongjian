# 夜遁

**夜遁**指趁夜色掩护撤退，为古代[[军事]]中在不利[[形势]]下利用夜幕掩护脱离接触的重要战术手段。

## 乘夜遁走

[[铜马]]食尽，夜遁，[[萧王]][[追击]]于[[馆陶]]，大破之（039-050）。洪以马与操，操不受。洪曰："[[天下]]可无洪，不可无君！"遂步从操，夜遁去（059-076）。尚夜遁，保[[祁山]]，操复进围之（064-027）。[[进兵]]攻卫，卫等夜遁（067-023）。汉主夜遁，驿人自担烧铙铠[[断后]]，仅得入白帝城（069-068）。
