# 傅燮

傅燮，字南容，北地灵州人。东汉汉阳太守，以忠节死义。凉州之乱中固守孤城，战死殉节。

其护军司马北地傅燮上疏曰："臣闻天下之祸不由于外，皆兴于内。是故虞舜先除四凶，然后用十六相。"（058-049）议郎傅燮厉言曰："斩司徒，天下乃安！"（058-071）

傅燮在黄巾起义时为护军司马，从朱俊击黄巾，以功封侯。后为汉阳太守，正值凉州边章、韩遂之乱。贼围汉阳，城中兵少粮尽。傅燮固守孤城，子干劝其弃城归乡，燮叹曰："世乱不能养浩然之志，食禄又欲避其难乎！"遂战死。傅燮以死全节，为东汉末年少有的忠义之臣。
