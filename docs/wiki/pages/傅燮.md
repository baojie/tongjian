# 傅燮

傅燮，字南容，[[北地]]灵州人。[[东汉]]汉阳太守，以忠节死义。[[凉州]]之乱中固守孤城，战死殉节。

其护军司马[[北地]]傅燮上疏曰："臣闻天下之祸不由于外，皆兴于内。是故虞舜先除四凶，然后用十六相。"（058-049）议郎傅燮厉言曰："斩[[司徒]]，天下乃安！"（058-071）

傅燮在[[黄巾|黄巾起义]]时为护军司马，从朱俊击黄巾，以功封侯。后为汉阳太守，正值[[凉州]]边章、[[韩遂]]之乱。贼围汉阳，城中兵少粮尽。傅燮固守孤城，子干劝其弃城归乡，燮叹曰："世乱不能养浩然之志，食禄又欲避其难乎！"遂战死。傅燮以死全节，为[[东汉]]末年少有的忠义之臣。
