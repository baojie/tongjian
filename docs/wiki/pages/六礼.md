# 六礼

六礼，古代婚礼六项程序：纳采、问名、纳吉、纳征、请期、亲迎。

晋[[遣使]]备六礼迎杜[[皇后]]（095-094）。[[突厥]][[突利|突利可汗]]来逆女教习六礼（178-074）。

六礼之制始于《[[礼记]]》，历代帝王纳后、[[公主]]出降皆依其制。晋成帝聘杜氏为后，[[遣使]]备六礼迎之，[[大赦]]天下。隋时[[突利|突利可汗]]来逆女，[[隋文帝]]舍之[[太常]]，教习六礼，妻以宗女。六礼于民间虽不能尽备，然纳采、亲迎之仪，千百年来相沿不绝。至宋以后，六礼渐简为四礼。

> 晋遣使备六礼迎杜皇后。
