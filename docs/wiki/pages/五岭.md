# [[五岭]]

[[五岭]]指南岭山脉中五个重要关隘的合称，包括大庾岭、骑田岭、都庞岭、萌渚岭、越城岭，横亘于今[[湖南]]、[[江西]]与[[广东]]、[[广西]]交界处，是[[中原]]通往[[岭南]]的必经之路。

[[秦]]统一后，尉[[屠睢]]率军分五路越过五岭征[[南越]]，设[[南海郡|南海]]、[[桂林]]、[[象郡]]三郡。[[汉代]][[汉武帝]]时，[[路博德]]、[[杨仆]]等亦越五岭平定南越。五岭在军事上具有重要的屏障作用，历来为北方政权控制岭南的关键。
