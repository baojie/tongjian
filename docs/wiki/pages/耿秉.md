# 耿秉

耿秉，[[东汉]]名将，[[耿弇]]之侄。[[《资治通鉴》]]载其数与[[窦固]]出击[[匈奴]]，征[[西域]]，以[[勇略]]闻。

[[谒者]][[仆射]]耿秉数上言请击[[匈奴]]（045-066）。耿秉、秦彭率[[武威]]、[[陇西]]、[[天水]]募士及羌、胡万骑出[[张掖]][[居延]]塞（045-068）。遣奉车[[都尉]][[窦固]]、[[驸马]][[都尉]]耿秉、骑[[都尉]]刘张出[[敦煌]]昆仑塞击[[西域]]（045-083）。

> 谒者仆射耿秉数上言请击匈奴，上以显亲侯窦固尝从其世父融在河西，明习边事…
> （045-066）

> 遣奉车都尉窦固、驸马都尉耿秉、骑都尉刘张出敦煌昆仑塞，击西域，秉、张皆去符，传以属固，合兵万四千骑，击破白山… 
> （045-083）
