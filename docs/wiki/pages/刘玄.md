# 刘玄

刘玄，字圣公，汉宗室，更始政权的皇帝，在位仅二年。

刘玄本为流民所立，即帝位后不能节制诸将，内乱不已。诸将议迁都，刘玄以[[建康]]为王者之宅，犹豫未决（094-060）。后[[赤眉]]入关，刘玄投降，不久被杀。

刘玄在位期间，天下大乱，群雄并起。其为人儒弱无断，虽居帝位而不能统御群臣，终至败亡。

## 更始兴亡

刘玄在平林兵中号更始将军，诸将议立刘氏以从人望，遂立为帝（039-004）。入都宛城，刘縯威名日盛，新市、平林诸将阴劝更始除之，更始遂杀刘縯（039-014）。后赤眉西入关，更始不能御，出降，旋被杀（040-056）。
