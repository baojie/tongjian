# 戎服

戎服，军服武服。[[陆机]]闻秀至，释戎服著白帢与秀相见（085-024）。[[太傅]]越戎服入见请讨[[石勒]]（087-050）。汉主[[刘聪]]出畋，以愍帝行[[车骑]][[将军]]，戎服[[执戟]]前导（090-028）。[[苏峻之乱]]，戎服者多死，[[白衣]]者无他（094-009）。[[温峤]]说[[陶侃]]，侃感悟即戎服登舟（094-021）。

> 太傅越既杀王延等，大失众望；又以胡寇益盛，内不自安，乃戎服入见，请讨石勒。
> （087-050）
