# 戎服

戎服，军服武服。陆机闻秀至，释戎服著白帢与秀相见（085-024）。太傅越戎服入见请讨石勒（087-050）。汉主刘聪出畋，以愍帝行车骑将军，戎服执戟前导（090-028）。苏峻之乱，戎服者多死，白衣者无他（094-009）。温峤说陶侃，侃感悟即戎服登舟（094-021）。

> 太傅越既杀王延等，大失众望；又以胡寇益盛，内不自安，乃戎服入见，请讨石勒。
> （087-050）
