# 雅乐

雅乐是古代[[朝廷]][[祭祀]]、[[朝会]]所用的[[正统]][[音乐]]，与郑卫之音相对，被视为[[正声]]。

[[河间王|河间献王]]来朝，献雅乐，对三雍宫及诏策所问三十馀事（018-020）。哀帝罢[[乐府]]官后，[[百姓]]渐渍日久，又不制雅乐有以相变，豪富吏民湛沔自若（033-021）。

东[[汉明帝]]幸[[南阳]]，召校官[[弟子]]作雅乐，奏《鹿鸣》，帝自御埙篪和之（045-040）。[[刘表]]起立[[学校]]，命杜夔作雅乐，夔以"[[将军]]号不为[[天子]]，合乐而庭作之"谏之（062-020）。[[北魏]]世祖克[[统万]]及[[姑臧]]，获雅乐器服工人（137-065）。孝[[隋文帝|文帝]]欲厘正雅乐，自制四器名之曰"通"（145-047）。
