# 景丹

景丹，字孙卿，冯翊栎阳人。东汉骠骑大将军，云台二十八将之一。以上谷突骑助光武帝定河北。

寇恂还，遂与上谷长史景丹及耿弇将兵俱南，与渔阳军合（039-045）。景丹等纵突骑击之，宏等大败。秀曰："吾闻突骑天下精兵，今见其战，乐可言邪？"（039-046）

景丹以上谷长史率突骑投奔光武帝，与渔阳军合兵南下，所过击斩王郎大将无数。景丹所率上谷突骑为天下精兵，在平定河北诸战中屡为先锋。光武帝即位后，景丹拜骠骑大将军，封栎阳侯。丹以病卒于军，光武帝深悼惜之。永平中图画云台，景丹位列二十八将。
