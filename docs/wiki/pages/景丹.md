# 景丹

景丹，字孙卿，冯翊栎阳人。[[东汉]]骠骑大将军，云台二十八将之一。以上谷[[突骑]]助[[光武帝刘秀|光武帝]]定[[河北]]。

[[寇恂]]还，遂与上谷长史景丹及[[耿弇]]将兵俱南，与渔阳军合（039-045）。景丹等纵[[突骑]]击之，宏等大败。秀曰："吾闻[[突骑]]天下[[精兵]]，今见其战，乐可言邪？"（039-046）

景丹以上谷长史率[[突骑]]投奔[[光武帝刘秀|光武帝]]，与渔阳军合兵南下，所过击斩[[王郎]]大将无数。景丹所率上谷[[突骑]]为天下[[精兵]]，在平定[[河北]]诸战中屡为先锋。[[光武帝刘秀|光武帝]]即位后，景丹拜骠骑大将军，封栎阳侯。丹以病卒于军，[[光武帝刘秀|光武帝]]深悼惜之。永平中图画云台，景丹位列二十八将。
