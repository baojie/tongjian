# 永徽

**永徽**（650—655），[[唐高宗]][[李治]]第一个[[年号]]。永徽年间，高宗承贞观遗风，以[[长孙无忌]]、[[褚遂良]]辅政，[[永徽之治]]有贞观之风。[[突厥]]余部反唐，[[程知节]]、[[苏定方]]等讨平之。永徽六年，高宗立[[武则天]]为后，长孙无忌、褚遂良被贬逐，朝政渐入武后之手。
