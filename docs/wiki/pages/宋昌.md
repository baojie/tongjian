# 宋昌

宋昌，[[西汉]]代国[[中尉]]，后拜[[卫将军]]。

[[吕后]]崩，[[吕氏|诸吕]]被诛，群臣迎立代王[[汉文帝|刘恒]]。代王犹豫，宋昌力排众议，以为[[诸吕之乱]]已平，[[天下]]归心，劝代王入京继位（013-062）。代王遂入京，当夜拜宋昌为[[卫将军]]，[[镇抚]]南北军（013-063）。[[隋文帝|文帝]][[论功]]，封宋昌为壮[[诸葛亮|武侯]]（013-076）。

## 世系

宋昌，[[西汉]][[卫将军]]、壮[[诸葛亮|武侯]]。以家吏从[[刘邦]]起事。[[吕后]]崩，[[吕氏|诸吕]]被诛，昌时以[[代国]][[中尉]]力劝[[代王]][[汉文帝|刘恒]]入京继位（013-062），拜[[卫将军]][[镇抚]]南北军（013-063），封壮[[诸葛亮|武侯]]（013-076）。昌[[子孙]]不详于史。
