# 宋昌

宋昌，西汉代国中尉，后拜卫将军。

吕后崩，诸吕被诛，群臣迎立代王刘恒。代王犹豫，宋昌力排众议，以为诸吕之乱已平，天下归心，劝代王入京继位（013-062）。代王遂入京，当夜拜宋昌为卫将军，镇抚南北军（013-063）。文帝论功，封宋昌为壮武侯（013-076）。

## 世系

宋昌，[[西汉]][[卫将军]]、壮武侯。以家吏从[[刘邦]]起事。[[吕后]]崩，诸吕被诛，昌时以[[代国]]中尉力劝[[代王]]刘恒入京继位（013-062），拜[[卫将军]]镇抚南北军（013-063），封壮武侯（013-076）。昌[[子孙]]不详于史。
