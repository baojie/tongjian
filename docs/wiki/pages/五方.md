# 五方

五方指东、南、西、北、**中**五个方位，与[[五行]]（木火金水土）、[[五色]]（青赤白黑黄）、[[五帝]]等体系相配，是[[中国]]古代宇宙观与政治哲学的基石。

## 通鉴中的五方

[[北魏]]迁都[[洛阳]]后，[[甄琛]]上表论治安曰今"五方杂沓，寇盗公行"（147-076），以五方喻各地之人汇聚[[京师]]。[[武则天]]祀[[明堂]]时"先诣昊天上帝座，次[[汉高祖刘邦|高祖]]、[[太宗]]、高宗，次[[魏国]]先王，次五方帝座"（204-050），五方帝为朝廷[[祭祀]]的重要神灵。[[唐_(朝代)|唐]]将[[史敬奉]]破[[吐蕃]]时，以"五方旗"为号辨识中军（232-031）。

## 哲学意义

五方之中，"中央"最为尊贵。[[天子]]居天下之中以治四方，故称"中"国。五方观念贯穿[[中国]]古代政治地理思想。
