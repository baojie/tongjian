# 焉耆

焉耆是西域北道古国，位于塔里木盆地北缘。匈奴西边日逐王置僮仆都尉，常居焉耆、危须、尉黎间，赋税诸国（020-036），其地当丝路要冲。车师太子军宿因不愿为质子入匈奴，亡走焉耆（025-016）。

汉宣帝时召军宿立为车师王，徙车师国民居渠犁（025-053）。[[王莽]]时期西域诸国因莽积失恩信，"焉耆先叛，杀都护但钦"，西域自此瓦解（037-057）。前凉张骏遣将伐西域，焉耆归于姑臧朝贡（095-087）。焉耆在汉唐之际始终为西域重要城邦。
