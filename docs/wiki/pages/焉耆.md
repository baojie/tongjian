# 焉耆

焉耆是[[西域]]北道古国，位于塔里木盆地北缘。[[匈奴]]西边日逐王置僮仆都尉，常居焉耆、危须、尉黎间，赋税诸国（020-036），其地当丝路要冲。[[车师]]太子军宿因不愿为[[质子]]入[[匈奴]]，亡走焉耆（025-016）。

[[汉宣帝]]时召军宿立为[[车师]]王，徙[[车师]]国民居渠犁（025-053）。[[王莽]]时期[[西域]]诸国因莽积失恩信，"焉耆先叛，杀都护但钦"，[[西域]]自此瓦解（037-057）。[[前凉]][[张骏]]遣将伐[[西域]]，焉耆归于[[姑臧]][[朝贡]]（095-087）。焉耆在汉唐之际始终为[[西域]]重要城邦。
