# 焉耆

焉耆是[[西域]]北道古国，位于塔里木盆地北缘。[[匈奴]]西边日逐王置[[僮仆]][[都尉]]，常居焉耆、危须、尉黎间，[[赋税]]诸国（020-036），其地当丝路要冲。[[车师]][[太子]]军宿因不愿为[[质子]]入[[匈奴]]，亡走焉耆（025-016）。

[[汉宣帝]]时召军宿立为[[车师]]王，徙[[车师]]国民居渠犁（025-053）。[[王莽]]时期[[西域]]诸国因莽积失恩信，"焉耆先叛，杀[[都护]]但钦"，[[西域]]自此瓦解（037-057）。[[前凉]][[张骏]]遣将伐[[西域]]，焉耆归于[[姑臧]][[朝贡]]（095-087）。焉耆在汉唐之际[[始终]]为[[西域]]重要城邦。

## 焉耆与唐

唐代焉耆为[[安西]]四镇之一，设焉耆[[都督]]府。[[唐太宗]][[贞观之治|贞观]]年间，焉耆王龙[[突骑]]支[[遣使]][[入贡]]，[[唐朝]]以其地置焉耆镇，与[[龟兹]]、[[疏勒]]、[[于阗]]并为[[安西]]四镇，属[[安西都护府]]管辖。[[玄奘]]西行途经焉耆，《[[大唐西域记]]》记其国。[[吐蕃]]兴起后，焉耆与四镇皆为唐蕃争夺之要地。[[安史之乱]]后，[[唐朝]]势力渐衰，焉耆先为[[吐蕃]]所占，后入[[回鹘]]。焉耆地处丝路北道咽喉，汉唐千年之间，[[商旅]]往来不绝，[[佛教]]艺术繁荣，今存锡克沁千佛洞遗址。
