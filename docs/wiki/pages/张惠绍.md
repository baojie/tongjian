# 张惠绍

张惠绍，字德继，义阳人。南朝梁骁骑将军。

萧衍起兵襄阳，张惠绍为义阳太守。衍筑汉口城以守鲁山，命水军主义阳张惠绍等游遏江中，绝郢、鲁二城信使（144-015）。薜元嗣遣军主沈难当帅轻舸来战，张惠绍等击擒之（144-018）。后为骁骑将军，引兵赴救平乱（145-036）。惠绍在梁初将领中以勇略著称。

> 衍筑汉口城以守鲁山，命水军主义阳张惠绍等游遏江中，绝郢、鲁二城信使。
> （144-015）
