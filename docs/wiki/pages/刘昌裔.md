# 刘昌裔

刘昌裔，唐中叶将领。[[《资治通鉴》]]载其为陈州[[刺史]]，阻[[吴少诚]]攻许州。

[[阳曲]]人刘昌裔说子琳[[遣使]]诣阙请罪（224-084）。[[营田]]副使刘昌裔止陈许[[节度使]][[刘涗]]曰："城中兵足以办贼，但闭城勿与战，不过数日，贼气自衰"（235-094）。刘昌裔登城谓[[刘全义]]曰："[[天子]]命公讨[[蔡州]]，今乃来此，昌裔不敢纳，请舍于城外"（235-121）。

> 营田副使刘昌裔止之曰："城中兵足以办贼，但闭城勿与战，不过数日，贼气自衰，吾以全制其弊，蔑不克矣。"
> （235-094）
