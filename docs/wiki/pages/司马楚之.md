# 司马楚之

司马楚之，字德秀，[[河内]]温人。[[北魏]]征南将军，原[[东晋]][[宗室]]，司马荣期之子。

司马楚之奉父丧归[[建康]]，会宋公[[刘裕]]诛剪[[宗室]]之有才望者，楚之亡命，聚众屯柏谷坞（118-103）。司马楚之屯柏谷坞，[[刘裕]]命康坚守六十日，楚之不得进（118-104）。司马楚之闻魏兵济河，[[遣使]]迎降，魏以为征南将军、[[荆州]][[刺史]]（119-085）。

司马楚之少英果，有才略。值晋宋之际，[[宗室]]诛戮殆尽，楚之亡命汝颍间，聚众万人。后降[[北魏]]，从太武帝征讨，屡立战功。楚之在魏，尚诸王女，拜[[司马睿|琅邪王]]。卒谥贞。楚之虽为晋[[宗室]]而仕魏，然能以功名终，亦一时之杰也。

> 初，司马楚之奉其父荣期之丧归建康，会宋公裕诛剪宗室之有才望者，楚之叔父宣期、兄贞之皆死。
> （118-103）
