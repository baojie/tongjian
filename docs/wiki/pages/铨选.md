# 铨选

铨选，[[唐朝|唐代]][[吏部]][[选官]][[制度]]。玄宗时，[[吏部侍郎]]玄同上言铨选之弊，以为[[人君]]当委任责成，不可下行铨选之事。称周穆王命伯冏为[[太仆]]正，正是委任之意（203-015）。后[[宰相]]以[[陈平]]、[[丙吉]]之例，以为大唐万乘之君岂得下行铨选，请委之有司，停十铨（212-170）。[[五代]]时，[[杨洞潜]]请立[[学校]]、开[[贡举]]、设铨选，[[南汉]]主[[刘岩]]从之（271-015）。

> 昔陈平、邴吉，汉之宰相，尚不对钱谷之数，不问斗死之人；况大唐万乘之君，岂得下行铨选之事乎？
> （212-170）
