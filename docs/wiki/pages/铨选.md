# 铨选

铨选，唐代吏部选官制度。玄宗时，吏部侍郎玄同上言铨选之弊，以为人君当委任责成，不可下行铨选之事。称周穆王命伯冏为太仆正，正是委任之意（203-015）。后宰相以陈平、丙吉之例，以为大唐万乘之君岂得下行铨选，请委之有司，停十铨（212-170）。五代时，杨洞潜请立学校、开贡举、设铨选，南汉主刘岩从之（271-015）。

> 昔陈平、邴吉，汉之宰相，尚不对钱谷之数，不问斗死之人；况大唐万乘之君，岂得下行铨选之事乎？
> （212-170）
