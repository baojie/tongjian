# 斜谷

[[秦岭]]古[[栈道]]之一，为[[关中]]通往[[汉中]]的五条通道之一，北口在眉县，南口在褒城。

## 东汉末年

[[张鲁]]掩杀[[汉中]][[太守]]苏固，断绝斜谷阁，杀害汉使（060-022）。[[刘焉]][[上书]]言"米贼断道，不得复通"。

## 三国

[[曹操]]使[[赵俨]]发兵助[[汉中]][[守御]]，俨护送兵士至斜谷口。殷署军叛，俨率百五十人讨平（067-032）。[[诸葛亮北伐]]屡出斜谷，为魏蜀争战之要道。
