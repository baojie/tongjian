# 永嘉

**永嘉**（145），[[汉冲帝]][[年号]]，仅用一年。永嘉元年，冲帝[[即位]]时年仅二岁，[[梁太后]][[临朝]]，[[梁冀]]为[[大将军]]。[[九江]]贼[[徐凤]]、[[马勉]]等起事，攻掠城邑。[[历阳]]贼[[华孟]]自称黑帝，攻[[九江]]。[[滕抚]]为[[九江]][[都尉]]，击破诸贼。是年冲帝崩。
