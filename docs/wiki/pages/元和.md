# 元和

**元和**（84—87），[[汉章帝]]第二个[[年号]]。元和年间，章帝诏议[[贡举]]法，令郡国举孝廉、明经。[[贾逵]]奉诏撰《[[春秋]]》条例，又作《[[左传]]》解诂。[[班超]]经营西域，降服[[莎车]]、[[疏勒]]。章帝巡狩[[鲁]]地，祠[[孔子]]及七十二弟子。
