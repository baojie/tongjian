# 孟轲

**孟轲**（约前372—前289），邹人，[[战国]]时期[[儒家]]思想家，被尊为"亚圣"。

[[周显王]]三十三年，孟轲见[[魏惠王]]。王曰："叟，不远千里而来，亦有以利吾国乎？"孟子对曰："君何必曰利，[[仁义]]而已矣！"（002-079）。

孟轲之学承[[孔子]]而发，倡性善、仁义、王道。其与魏惠王论义利之辨，为千古名篇。孟子以"仁者无敌"说梁惠王，以"何必曰利"正君臣之义，《[[资治通鉴概述|资治通鉴]]》虽以[[司马光]]之史观取"义利"之说为开篇大旨，然于孟轲与魏惠王问对节取之，以明"礼"与"分"乃治国之本。

孟轲与[[荀卿]]并称先秦儒学的两大宗师，其学说对后世[[宋明理学]]影响至深。元明以降，孟子地位日隆，被奉为"亚圣"，《孟子》跻身[[四书五经|四书]]之列。
