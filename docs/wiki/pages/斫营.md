# 斫营

**斫营**指夜间偷袭敌营，为古代[[军事]]中利用夜色掩护发起突袭、扰乱敌军的常见战术。

## 夜袭斫营

傉檀知吕超必来斫营，畜火以待之（112-045）。乳陈果遣[[壮士]]千馀人来斫营，众火俱起，[[伏兵]][[邀击]]，俘斩无遗（115-095）。昼则遣骑围绕，夜则离彼百里外宿；吴人正有斫营伎（125-060）。永曰："南人好夜斫营，必于渡淮之所置火以记浅处。"乃夜[[分兵]]为二部，伏于营外（141-042）。巡时同贼隙，[[出兵]]击之，或夜缒斫营（217-074）。
