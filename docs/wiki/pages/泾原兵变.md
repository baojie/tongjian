# 泾原兵变

泾原兵变是[[唐德宗]]建中四年（783年），泾原镇兵在[[长安]]哗变，拥立朱泚为帝的重大政治事件。

德宗命泾原诸道兵共七万人讨伐淮西[[李希烈]]，[[制书]]下旬日复为[[鱼朝恩]]所沮，事竟不行（221-089）。后[[郭子仪]]自河中入朝，正值泾原军奏[[仆固怀恩]]引[[回纥]]、[[吐蕃]]十万众将入寇，[[京师]]震骇（223-054）。

泾原兵变后德宗出奔奉天，朱泚在[[长安]]自称[[秦朝|大秦]]皇帝。后赖[[李晟]]等将领力战收复[[长安]]，德宗得以还京。此变标志着唐中央权威的进一步衰落。
