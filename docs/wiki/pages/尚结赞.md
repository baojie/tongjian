# 尚结赞

**尚结赞**，[[吐蕃]][[宰相]]，[[唐德宗]]时期[[吐蕃]]主政者。

尚结赞与唐盟于清水（228-003），又请[[出兵]]助唐收京城（229-055）。然反复无常，[[大举]]寇泾、陇、邠、宁，掠人畜，西鄙骚然（232-027）。尚结赞败于唐将骆[[元光]]，仅而获免，曾谓其徒曰："唐之[[良将]]，[[李晟]]、[[马燧]]、[[浑瑊]]而已"（232-031）。

尚结赞用[[反间]]计，[[离间]][[李晟]]、[[马燧]]与德宗君臣，设计邀[[浑瑊]][[会盟]]于[[平凉]]，[[伏兵]]劫盟，[[浑瑊]]仅以身免。尚结赞为[[吐蕃]][[中兴]]之相，其[[外交]][[谋略]]远胜[[军事]]，以诈术[[合纵连横|纵横]]于唐蕃之间。

## 反复纵横

尚结赞为[[吐蕃]]相，善观时势。时崔汉衡诣[[吐蕃]]发兵助唐收[[长安]]，尚结赞以[[制书]]无[[李怀光]]署名为辞，不肯[[进兵]]（230-006）。[[浑瑊]]出[[斜谷]]，崔汉衡劝[[吐蕃]][[出兵]]，尚结赞以邠军不出为辞，终不[[出兵]]（230-031）。[[朱泚|硃泚]]以[[金帛]]赂[[吐蕃]]，尚结赞受之，遂不助唐，[[李晟]]、[[浑瑊]]屡以为言（231-004）。尚结赞得盐、[[夏州]]，留兵戍之，及闻唐诸军皆闭壁不战，遽引归。崔浣责以负约，尚结赞曰[[吐蕃]]破[[朱泚|硃泚]]未获赏，以此自解（232-051、232-052）。尚结赞之[[纵横]]捭阖，使唐蕃之间和战无常，为德宗朝边患之渊薮。
