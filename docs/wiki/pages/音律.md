# 音律

音律是五音六律的乐理体系，为古代[[礼乐]]制度的基础。[[汉武帝]]定[[郊祀]]之礼，立[[乐府]]，"采诗夜诵，有赵、代、秦、楚之讴，以音律播之"（021-049），以音律协畅天人。[[王莽]]改制定度量，亦以音律为据（022-090）。

魏晋之际，音律之学与[[清谈]][[玄学]]相表里。[[嵇康]]著《声无哀乐论》，以音律自然之理探讨乐之本源。[[司马炎|晋武帝]]时荀勖以音律定尺（088-027），为后世律管定尺之典范。音律之学贯穿古代[[礼乐]]文化的核心。
