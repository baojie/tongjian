# [[朐山]]

[[朐山]]位于今[[江苏]][[连云港]]，南临[[黄海]]。[[南北朝]]时期，朐山为[[海州]]治所所在，地处沿海[[淮北]]地区，是[[南朝]]与[[北朝]]争夺的沿海[[军事]]要地。

[[东魏]]、[[北齐]]据有朐山，以此[[作为]]控扼淮北沿海的据点。朐山周边海域是[[海上丝绸之路]]北线的重要通道。[[唐代]]朐山属[[海州]]，仍为沿海[[军事]]港口。
