# 尉佗

**尉佗**（？—前137），本姓赵，[[真定]]人，[[秦]]末[[南海]]尉，后自立为[[南越]]武王。秦已破灭，佗击并[[桂林]]、[[象郡]]，自立为[[南越]]武王（012-037）。

[[汉高祖刘邦|汉高祖]]遣[[陆贾]]使[[南越]]，说佗去帝号称臣。[[汉文帝]]元年，佗复[[僭号]][[称帝]]，[[隋文帝|文帝]]为佗亲冢在[[真定]]者置守邑，岁时奉祀，召[[陆贾]]为太中大夫使[[南越]]。[[陈平]]、[[周勃]]等议以[[南越]]不宜远征（072-044）。

[[窦融]]据[[河西]]时，或说融当效尉佗故事割据一方（041-059），而融终归[[光武帝刘秀|光武]]。[[漢光武帝|光武]]与[[隗嚣]]书亦以任嚣、尉佗之谋为诫（042-019）。隋末乱世，[[杨袭志]]据岭表，或劝袭志效尉佗之业，袭志怒拒之曰："尉佗狂僭，何足慕也！"（185-035）

尉佗据[[岭南]]称王，传五世九十三岁，至[[汉武帝]][[元鼎]]六年[[南越]]灭。尉佗之事，为秦汉之际割据一方的典型。
