# 奏议

奏议是臣下向[[皇帝]]上奏的[[文书]]，以及朝[[廷议]]政的记录，为古代政治决策的重要依据。

[[班固]]谓[[汉成帝]]时[[公卿]]奏议可述，遭世承平上下和睦（033-009）。五宫[[中郎将]]线纯与[[太仆]][[朱浮]]奏议礼制，以先帝四庙代亲庙（043-073）。

奏议制度在秦汉已趋完备，[[奏章]]有章、表、奏、议之别。奏议是[[朝廷]]决策的重要参考，亦是考察臣下见识与忠诚的途径。历代有识之士的奏议往往切中时弊，具有重要的史料价值。

> 公卿奏议可述。遭世承平，上下和睦。
> （033-009）

## 奏议之用

[[宦官]]疾[[陈蕃]]，[[选举]]奏议辄以中诏谴却（055-051）。[[陈群]]削其草，时人子弟莫能知，朝士乃于《名臣奏议》中见群谏事（073-029）。[[齐王]]攸就国，王铨等奏议谏阻（081-060）。[[尚书]]郎[[孔坦]]奏议[[孝廉]]事（090-063）。[[朝廷]]奏议及百司表疏，[[武后]]时密令参决以分[[宰相]]权（202-054）。[[张文瓘]]性严正，诸司奏议多所纠驳（202-062）。奏议为君臣沟通之[[桥梁]]，直言者以此匡君，奸佞者亦借此害人。
