# 奏议

奏议是臣下向皇帝上奏的文书，以及朝廷议政的记录，为古代政治决策的重要依据。

[[班固]]谓汉成帝时公卿奏议可述，遭世承平上下和睦（033-009）。五宫中郎将线纯与太仆朱浮奏议礼制，以先帝四庙代亲庙（043-073）。

奏议制度在秦汉已趋完备，[[奏章]]有章、表、奏、议之别。奏议是朝廷决策的重要参考，亦是考察臣下见识与忠诚的途径。历代有识之士的奏议往往切中时弊，具有重要的史料价值。

> 公卿奏议可述。遭世承平，上下和睦。
> （033-009）
