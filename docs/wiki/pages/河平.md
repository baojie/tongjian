# 河平

**河平**（前28—前25），[[汉成帝]]第二个[[年号]]。河平年间，黄河决口于[[东郡]]，泛滥兖、豫二州，[[王延世]]以竹落盛石塞决河，功成，成帝改元河平以纪之。[[刘向]]受诏校中秘书，校勘经传、诸子、诗赋。[[匈奴]]遣子入侍。

> （030-040）
