# 外合

**外合**指[[内外]]联合[[夹击]]，为古代[[军事]]中城内守军与城外部队相互配合、协同攻击的重要战术。

## 中外合势

城中亦[[鼓噪]]而出，中外合势，震呼动[[天地]]。莽兵大溃（039-011）。曹、戴二州，[[户口]]完实，[[孟海公]]窃有其地，与郑人外合内离（188-013）。[[吴越]]兵中外合击[[淮南]]兵，大破之，擒其将何朗等三十馀人（267-036）。
