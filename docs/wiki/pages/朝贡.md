# 朝贡

朝贡是周边政权向[[中原]]王朝进献贡品的制度化外交关系，体现了以[[中原]]王朝为中心的天下秩序。

东[[光武帝刘秀|汉光武帝]]时，[[高句丽]]遣使朝贡，帝复其王号（042-044）。随着[[匈奴]]衰弱，[[鲜卑]]、乌桓并入朝贡，边无寇警（044-017）。东晋十六国时期，前凉张骏遣将伐龟兹、鄯善，[[西域]]诸国皆诣姑臧朝贡，远近称之以为贤君（095-087）。朝贡不仅是外交礼节，也是藩属关系合法化的政治标志。
