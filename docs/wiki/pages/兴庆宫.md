# 兴庆宫

兴庆宫是[[唐玄宗]]由藩邸改建的[[离宫]]，开元天宝年间成为朝政中枢。

开元二年，宋王成器等请献兴庆坊宅为[[离宫]]，玄宗制许之，始作兴庆宫。于宫西南置楼，西题"花萼相辉之楼"，南曰"勤政务本之楼"（211-034）。玄宗曾亲耕于兴庆宫侧（213-101）。

兴庆宫在开元天宝年间取代[[大明宫]]成为玄宗听政之所，是开元盛世的政治中心。[[安史之乱]]后玄宗被尊为[[唐高祖|太上皇]]，自蜀还京后居兴庆宫，后被迫[[迁居]]西内。兴庆宫内有龙池、沉香亭等著名景观，李白"云想[[衣裳]]花想容"即作于兴庆宫中。
