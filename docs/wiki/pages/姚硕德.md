# 姚硕德

姚硕德，南安赤亭人。后秦征西将军，姚苌之弟。

后秦主姚苌以姚硕德为使持节、都督陇右诸军事、秦州刺史，镇上邽（106-102）。前秦主苻登攻姚硕德于秦州，后秦主苌自往救之，与登战于胡奴阜，大破之（106-108）。后姚硕德为杨定所逼，退过泾阳，与秦鲁王纂共战，硕德大败，后秦主苌自阴密救之（107-016）。硕德为后秦名将，屡统军征讨。
