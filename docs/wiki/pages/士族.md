# 士族

士族是魏晋南北朝时期特有的社会阶层，指累世官宦、拥有政治经济特权的世家大族。士族垄断文化教育和选官渠道，通过[[九品中正]]制维持其政治地位，形成"上品无[[寒门]]、下品无势族"的局面。东晋时期士族政治达到顶峰，王、谢、庾、桓等大家族轮流执政。南北朝后期士族逐渐衰落，至唐代被新兴的科举官僚阶层取代。
