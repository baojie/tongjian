# 乘间

**乘间**指趁对方有间隙或疏忽时采取行动，为古代[[军事]]与[[政治]][[谋略]]中把握时机的重要方式。

## 乘间进言

王长君乘间入言于帝，帝怒稍解（016-068）。

## 乘间为乱

[[赵充国]]言不令虏得乘间之势（026-019）。[[颍川]]人闻[[陛下]]有事陇蜀故狂狡乘间相诖误（042-039）。马后乘间为帝言之楚狱多滥（045-058）。[[蛮夷]]见郡无[[儆备]]故敢乘间而进（054-087）。
