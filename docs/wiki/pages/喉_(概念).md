# 喉

**喉**为人体部位。**喉**，咽喉，呼吸[[饮食]]之通道。喉为[[要害]]，扼喉即扼敌之[[要害]]。景驹走死，[[项梁]]军据[[要害]]（044-022）。[[班超]]上疏："愿将本国产里千骑，扼喉捣虚。"（051-036）。故自非奉辞伐罪，岂得轻入其喉（051-105）。
