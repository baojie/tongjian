# [[江南东道]]

[[江南东道]]是[[唐代]]十五道之一。玄宗开元二十一年（733年）分天下为十五道，江南道分为江南东道与[[江南西道]]（213-142）。辖境约当今[[江苏]]南部（[[长江]]以南）、[[浙江]]、[[福建]]及[[上海]]，治所在[[苏州]]。

江南东道是唐代经济文化最发达的区域之一，辖下有[[苏州]]、[[杭州]]、[[越州]]、[[福州]]、[[泉州]]等著名州郡。[[安史之乱]]后，唐廷财政收入主要依赖江南东道等南方诸道。
