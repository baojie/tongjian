# 吕僧珍

吕僧珍，字元瑜，东平范人。南朝梁辅国将军，佐萧衍起兵建梁。

中兵参军东平吕僧珍觉其意，亦私具橹数百张（142-038）。衍夜召张弘策、吕僧珍、长史王茂、别驾柳庆远、功曹吉士瞻等入宅定议（143-045）。

吕僧珍为萧衍中兵参军，在萧衍密谋代齐时，僧珍暗中准备攻城器具，为起兵之先备。萧衍举兵后，僧珍率羽林兵从衍东下，攻建康，身先士卒。梁朝建立后，吕僧珍以功拜辅国将军，封侯。僧珍为人谦慎，不以为功自傲。武帝尝谓僧珍："卿有贵相，当为三公。"然僧珍以疾早卒，未及大用。

> 中兵参军东平吕僧珍觉其意，亦私具橹数百张。
