# 禳

禳，祭祀祈祷以消灾除祸的巫术仪式。

高堂隆以祈禳对魏明帝（073-012）。朝议以禳避之道不可取（097-007）。苻生以修德禳星变（100-016）。

禳为古人应对灾异的主要手段之一。魏明帝时崇华殿灾，问高堂隆以祈禳之义。隆引《易传》对曰："上不俭，下不节，孽火烧其室。"明帝乃修德以应之。前秦苻生时，星象示变，有司请修德以禳之，苻生竟曰皇后可应丧，暴虐益甚。禳之本意在于修德省身，而后世或以巫术代之，失其本义。

> 高堂隆以祈禳对魏明帝。
> （073-012）
