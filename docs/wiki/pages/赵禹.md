# 赵禹

赵禹（生卒年不详），[[西汉]]武帝时酷吏。武帝以张汤为太中大夫，与赵禹共定诸律令，务在深文。拘守职之吏，作见知法，吏传相监司。用法益刻自此始（018-028）。

> 上以张汤为太中大夫，与赵禹共定诸律令，务在深文。拘守职之吏，作见知法，吏传相监司。用法益刻自此始。
> （018-028）

赵禹与[[张汤]]以深刻为九卿，然其治尚辅法而行。义纵专以鹰击为治，王温舒择郡中豪敢往吏以为爪牙（019-080）。赵禹参与制定汉武帝时期的严刑峻法，是西汉酷吏政治的代表人物之一。
