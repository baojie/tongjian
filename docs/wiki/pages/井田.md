# 井田

井田是先秦时期的土地制度，以方九百亩为一井，八家各私百亩、共耕公田百亩。[[商鞅]]在[[秦国]]变法时"废井田，开[[阡陌]]"（002-049），标志着井田制的瓦解。[[董仲舒]]曾论秦用[[商鞅]]之法"除井田，民得卖买"，导致富者田连[[阡陌]]、贫者无立锥之地（033-024）。

[[王莽]]推行改制时试图恢复井田制，下令"更名天下田曰王田"，比附井田制度，结果引起社会混乱（037-011）。百姓因"坐卖买田宅、[[奴婢]]、[[铸钱]]"而抵罪者不可胜数（037-030）。[[隗嚣]]部将[[班彪]]论[[王莽]]之政，批评张邯、孙阳"造井田，使民弃土业"（038-061）。井田之制终因不合时宜而未能恢复。
