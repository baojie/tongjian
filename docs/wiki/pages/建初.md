# 建初

**建初**（76—84），[[汉章帝]]第一个[[年号]]。建初年间，章帝诏令轻徭薄赋，与民休息。[[马太后]]临朝，禁抑外戚，不封马氏兄弟为侯。[[班超]]上疏请平西域，章帝遣[[徐幹]]率兵援超。[[曹褒]]定汉礼，章帝诏令依[[叔孙通]]《汉仪》十二篇改制。
