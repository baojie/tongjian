# 谢朏

谢朏，字敬冲，陈郡[[阳夏]]人。[[南朝]]宋齐文人，[[谢安]]之侄。

[[萧道成]]欲引时贤参赞大业，夜召谢朏，屏人与语，久之，朏无言（134-112）。道成以朏有重名，必欲引参佐命，以为左[[长史]]。尝置酒与论魏、晋故事，朏终不为所动（135-006）。及齐[[受禅]]，朏为[[侍中]]，当解[[玺绶]]，阳为不知，不肯解授（135-017）。朏以不附齐篡，世高其节。

> 太傅道成以谢朏有重名，必欲引参佐命，以为左长史。
> （135-006）
