# 刘繇

刘繇，字正礼，[[东莱]]人，[[东汉]][[扬州]][[刺史]]。

侍[[御史]]刘繇素有盛名，[[诏书]]用为[[扬州]][[刺史]]。州旧治[[寿春]]，[[袁术]]已据之，繇欲南渡江（061-031）。[[袁术]]表[[孙策]]为[[折冲]][[校尉]]，使助吴景讨刘繇（061-054）。策攻刘繇[[牛渚]]营，尽得邸阁粮谷战具（061-056）。

刘繇为[[孙策]]所败，走保[[豫章]]，不久病卒。繇在汉末以[[名士]]为牧守，然无御敌之才。

> （061-031）
