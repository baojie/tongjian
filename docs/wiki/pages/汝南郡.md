# [[汝南郡]]

[[汝南郡]]是[[西汉]]置郡，治所在[[上蔡]]（今[[河南]]上蔡），[[东汉]][[魏晋]]时期属[[豫州]]。汝南郡地跨[[汝水]]两岸，是[[中原]]南部人口稠密、经济发达的重要郡。

[[东汉]]末，[[袁绍]]家族号称"四世三公"，门生故吏遍天下，即为汝南[[汝阳]]人。[[南朝]][[宋]]时，[[刘铄]]镇[[寿阳]]，遣陈宪行汝南郡事，守[[悬瓠]]（125-050）。[[唐代]]时，蒋王[[李恽]]曾授汝南郡王，后因诬告自缢（202-043）。
