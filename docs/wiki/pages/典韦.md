# 典韦

典韦，陈留己吾人。曹魏校尉，曹操帐下猛将，以勇力绝伦著称。

曹操募将讨吕布，典韦先登陷阵，勇冠三军（061-024）。曹操征张绣，典韦力战护主，死之（062-023）。

典韦形貌魁梧，膂力过人，好持大双戟。初从张邈，后归曹操，拜为校尉。韦每战必先登陷阵，所挡者无不披靡。濮阳之战，韦冒矢石先登，吕布军为之却。张绣反叛，韦为掩护曹操突围，独守辕门，杀敌甚众，终因重伤而死。太祖闻之，痛哭不已，厚葬之。

> 曹操募将讨吕布，典韦先登陷阵，勇冠三军。
