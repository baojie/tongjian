# 汝州

汝州，唐代州名，在今河南汝州。[[《资治通鉴》]] 载汝州为东都洛阳南面屏障。

## 唐代

唐初张护、李通杀贺若怀广，立汝州别驾窦伏明为主，进逼瓜州（190-109）。太宗命将作大匠[[阎立德]]行清暑之地，作襄城宫于汝州西山（195-086）。高宗多次幸汝州温汤（202-068、202-139）。武则天亦幸汝州温汤（206-128）。
