# 坊州

坊州为[[唐代]]州名，治中部（今陕西黄陵），属鄜坊节度使管辖，为关中北部屏障。

[[武则天]]时萧至忠劾奏[[李味道]]，左迁坊州刺史（207-097）。鄜坊节度使[[杜冕]]屯坊州（223-090）。[[周智光]]素与杜冕不协，遂杀鄜州刺史张麟，坑冕家属八十一人，焚坊州庐舍三千馀家（223-092）。[[郭子仪]]使裨将[[李怀光]]救之，吐蕃寇坊州（225-099）。
