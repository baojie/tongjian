# 口赋

口赋是汉代按人口征收的人头税，儿童所纳。汉仪：民年七岁至十四岁出口赋钱，每人二十钱，以供天子之用。[[汉武帝]]时"重赋于民，民产子三岁则出口赋"（024-003），使百姓不堪重负。

[[王莽]]改制时，责汉室"赋敛无度，竭民财力"，其中亦包括口赋之苛（035-052）。口赋与[[算赋]]（成年丁口之税）同为汉代人头税的基本形式，为国家财政收入的重要来源。口赋之轻重直接影响百姓生计。
