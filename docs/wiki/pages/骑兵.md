# 骑兵

骑兵是骑马作战的兵种，[[战国]][[赵武灵王]]胡服骑射后始成为独立兵种。秦汉之际，骑兵在战争中发挥关键作用。楚汉相争时，[[灌婴]]将骑兵击楚骑于荥阳东，大破之，"楚以故不能过荥阳而西"（009-044）。[[汉文帝]]时条侯[[周亚夫]]"使弓高侯等将轻骑兵出淮泗口"，绝吴楚兵后，塞其饷道（016-019）。

[[汉武帝]]时期，骑兵大规模用于对[[匈奴]]作战，票骑将军[[霍去病]]"骑兵车重与大将军军等"（019-073）。[[赵充国]]在平定西羌时则主张"罢骑兵，[[屯田]]以待其敝"，以节省军费（026-016/019）。
