# 推恩令

推恩令，[[汉武帝]]时期[[主父偃]]提出的削藩政策。规定诸侯王除嫡长子[[继承]]王位外，其余子弟可受封为列侯，由朝廷制定封号。表面是皇帝推恩于诸侯子弟，实则分拆诸侯国、削弱其势力。武帝从[[主父偃]]之策，令诸侯得推恩分子弟（018-065），此后藩国自析，大国分为小国（037-022）。推恩令实施后，"齐分为七，赵分为六，梁分为五，[[淮南]]分为三"（074-063），从根本上解决了汉初诸侯王国尾大不掉的局面。

> 愿陛下令诸侯得推恩分子弟，以地侯之，彼人人喜得所愿。上以德施，实分其国，不削而稍弱矣。
> （018-065）

> 武帝从主父之策，下推恩之令，自是之后，遂以陵夷，子孙微弱，衣食租税，不预政事。至于哀、平，王氏秉权，假周公之事，而宗室诸侯束手俯首，无抗衡之力。
> （074-063）
