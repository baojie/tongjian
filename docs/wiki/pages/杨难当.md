# 杨难当

杨难当，仇池氐王，承兄杨玄之位，据有[[陇右]][[武都]]等地。在南北朝之间周旋，为仇池政权后期重要君主。

以氐王杨难当为冠军将军、[[秦州]][[刺史]]、[[武都]]王（121-091）。暮未请救于氐王杨难当，难当遣将军苻南帅骑三千救之（121-120）。又加北[[秦州]][[刺史]]杨难当征西将军。难当以兄子保宗为镇将军，镇宕昌（122-049）。

杨难当当刘宋与[[北魏]]对峙之际，介于两大之间，时而附宋，时而归魏，以图存国。后为宋将裴方明所败，奔魏。仇池政权自杨腾始，传至杨难当而渐衰，然氐王统系延续数百年，为陇南特殊政治力量。

> 以氐王杨难当为冠军将军、秦州刺史、武都王。
> （121-091）
