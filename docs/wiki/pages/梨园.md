# 梨园

梨园是唐玄宗设立的宫廷音乐舞蹈机构，后世以"梨园"代指戏曲行业。

唐玄宗选乐工数百人，自教法曲于梨园，谓之"皇帝梨园弟子"（211-005）。玄宗御梨园球场，命文武三品以上抛球及分朋拔河（209-062）。

杨氏兄弟宠贵，日遣杨与之选胜游宴，侑以梨园教坊乐（216-057）。安史之乱后梨园衰散，然其影响深远，后世戏曲界仍自称"梨园行"。

> 又选乐工数百人，自教法曲于梨园，谓之"皇帝梨园弟子"。
