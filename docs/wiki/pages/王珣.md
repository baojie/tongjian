# 王珣

王珣，字元琳，琅邪临沂人。东晋尚书左仆射，[[王导]]之孙。以[[文学]]书法知名于时。

温以抚军司马[[王坦之]]为长史。又以征西掾[[郗超]]为参军，王珣为主簿，每事必与二人谋之。府中为之语曰："髯参军，短主簿，能令公喜，能令公怒。"（101-060）以吴郡太守王珣为尚书右仆射（107-108）。

王珣少为[[桓温]]所重，与[[郗超]]并为[[桓温]]谋主。后仕孝武帝，官至尚书左仆射。珣工书，尤精行草，与王献之齐名。晚年与弟王珉并以书名世。珣为人通达，善处权势，在东晋末年的政治纷争中持身以正。
