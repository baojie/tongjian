# 贺兰进明

**贺兰进明**，[[肃宗|唐肃宗]]时[[北海太守]]、[[节度使]]。

[[安史之乱]]中，贺兰进明[[起兵]]讨贼。[[颜真卿]]以书召之[[戮力|并力]]，进明率步骑五千渡河，真卿陈兵迎之，相揖哭于马上（217-079）。进明遣[[录事参军]][[第五琦]]入蜀[[奏事]]，琦言财赋为急，举贺兰进明之荐而为理财名臣（218-051）。

贺兰进明在安史乱中为[[鹰扬]]将领，后仕肃宗朝。然其为人忌刻，与诸将不协，未能成大事。

## 安史乱中

[[贺兰进明]]于[[安史之乱]]中为[[北海太守]]，[[起兵]]讨贼，与[[颜真卿]][[戮力|并力]]（217-079）。遣[[录事参军]][[第五琦]]入蜀[[奏事]]，琦因得理财之名（218-051）。后诣[[行在]]，[[房琯]]荐为[[河南]][[节度使]]（219-009）。及[[张巡]]被围[[睢阳]]，进明在[[临淮]][[拥兵]]不救，致巡[[陷没]]（219-078）。[[房琯]]恶进明，故以[[许叔冀]]副之，二人不协（219-080）。后坐[[第五琦]]党，贬[[溱州]]员外[[司马]]（221-054）。
