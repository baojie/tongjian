# 天险

天险，天然形成的险要地形，一夫当关万夫莫开。

河西天险百年无虞（104-008）。潼关天险刘裕难攻（118-017）。季连据天险之地（145-044）。

天险为兵家必争之地。河西据黄河之险，晋人恃以抗秦。潼关天险，刘裕水军攻之甚难。南齐时刘季连据蜀，以为据天险之地，握强兵，进可匡社稷，退不失作刘备，遂起兵。天险虽足恃，然在德不在险，若失人心，虽金城汤池亦不能守。

> 河西天险百年无虞。
> （104-008）
