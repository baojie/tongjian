# 前凉

前凉是[[十六国]]之一，[[西晋]][[凉州]][[刺史]][[张轨]]所建，都[[姑臧]]（今甘肃[[武威]]），为[[十六国]]中延续最久的汉人政权。

前凉州[[刺史]][[杜邺]]以[[方正]][[对策]]（035-006）。[[隗嚣]]以[[西州]]上[[将军]]自居，以前凉州[[刺史]][[河南]]郑兴为[[祭酒]]（040-031）。前凉州[[刺史]]祝良多所纠罚，[[太守]]令长贬黜将半，功效卓然（054-018）。

前凉历九主七十六年，保据[[河西]]，在[[五胡]]纷争中维持汉文化传统。前凉重视[[儒学]]，保存[[中原]][[典籍]]，为[[北魏]]统一北方[[后汉]]文化的延续提供了重要基础。

> 前凉州刺史祝良，初除到州，多所纠罚，太守令长，贬黜将半，政未逾时，功效卓然，实应赏异。
> （054-018）
