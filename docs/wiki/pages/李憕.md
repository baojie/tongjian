# 李憕

李憕，唐[[东京留守]]，[[安史之乱]]中殉国。

[[河南尹]]达奚珣降于[[安禄山]]，李憕谓[[御史中丞]][[卢奕]]曰："吾曹荷国重任，虽知力不敌，必死之！"遂收[[残兵]]欲战，皆溃去（217-046）。禄山使其党段子光赍李憕、[[卢奕]]、蒋清首徇[[河北]]诸郡，[[颜真卿]]执子光[[腰斩]]以徇（217-050）。后[[朝廷]][[追赠]]李憕，官其[[子孙]]（220-038）。

## 世系

李憕（？—755年），[[唐]][[东京留守]]。其先为[[唐朝|唐]][[宗室]]疏属。[[安史之乱]]中[[洛阳]]陷落，憕与[[卢奕]]同殉国（217-046），首为[[段子光]]所赍徇[[河北]]（217-050）。后[[朝廷]][[追赠]]，官其[[子孙]]（220-038）。憕[[子孙]]不详于史。
