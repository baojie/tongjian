# 投兵

**投兵**指放下武器，为古代军队投降、罢战或[[归顺]]的标志性动作，常与[[解甲]]连用。

## 投兵归顺

淄青军士投兵相顾曰[[明主]]出矣吾属犹反乎（225-143）。[[守城]]者稍稍[[弃甲]]投兵而下，崔彦曾[[故吏]]开门纳官军（251-048）。[[庞勋]]部众投兵迎降，[[朝廷]]欲赦免胁从者（255-032）。

## 解甲投兵

梁晋之战战士悉[[解甲]]投兵而弃之（267-128）。杨思权帅诸军[[解甲]]投兵[[请降]]于潞王（279-011）。
