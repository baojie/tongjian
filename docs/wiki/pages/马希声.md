# 马希声

马希声，楚武安节度使。马殷之子，袭位后去建国之制，复藩镇之旧。

马希声袭位，称遗命去建国之制，复藩镇之旧（277-051）。以武安节度使马希声为武安、静江节度使，加兼中书令（277-055）。马希声慕梁太祖嗜食鸡，日杀五十鸡为膳；居丧无戚容（277-104）。马希声以湖南大旱，命闭诸神祠门，竟不雨（278-006）。马希声卒（278-006）。
