# 义武

义武为[[唐代]][[藩镇]]，治[[定州]]（今[[河北]][[定州]]），领易、定、沧三州，[[张孝忠]]为首任[[魏博节度使|节度使]]。

[[唐德宗]]置义武军[[节度]]于[[定州]]，以易、定、沧三州隶之（227-064）。[[张孝忠]]以[[易州]]归国，诏以孝忠为[[义武节度使]]（231-009）。[[义武节度使]]张升云为义武[[留后]]（233-074）。

> 置义武军节度于定州，以易、定、沧三州隶之。
> （227-064）
