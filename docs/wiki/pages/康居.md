# 康居

康居是中亚游牧大国，位于锡尔河流域。[[张骞]]第二次[[出使]][[西域]]，分遣副使使[[大宛]]、康居、大月氏、[[赫连勃勃|大夏]]、[[安息]]、身毒、[[于阗]]诸国（020-035），康居自此与汉[[通使]]。康居"可五六万"兵力，为[[西域]]北道大国（019-039）。

[[张骞]]报武帝，言康居"兵强，宜厚币结之"（020-036）。大月氏初居敦煌、祁连间，为[[匈奴]]所破，西击塞王，塞王南走，大月氏居其地，后[[乌孙]]击走大月氏，大月氏西徙击[[赫连勃勃|大夏]]，而塞王[[南越]]县度（019-040）。康居在汉与[[西域]]诸国交往中扮演重要角色。

## 张骞初使康居

[[张骞]]首次[[出使]][[西域]]，本欲联大月氏共击[[匈奴]]，道经[[匈奴]]被拘十馀年。间亡西走，至[[大宛]]，[[大宛]]为发导译抵康居，康居传致大月氏（018-083）。大月氏既臣[[赫连勃勃|大夏]]，地肥饶无报胡之心，[[张骞]]不得要领而还。康居为[[西域]]北道枢纽，汉通[[西域]]必经之地。东汉时[[班超]]经营[[西域]]，亦与康居交通，康居在汉与葱岭以西诸国交往中始终为要冲。

> 大宛闻汉之饶财，欲通不得，见骞，喜，为发导译抵康居，传致大月氏。
> （018-083）
