# 康居

康居是中亚游牧大国，位于锡尔河流域。张骞第二次出使西域，分遣副使使大宛、康居、大月氏、大夏、安息、身毒、于阗诸国（020-035），康居自此与汉通使。康居"可五六万"兵力，为西域北道大国（019-039）。

张骞报武帝，言康居"兵强，宜厚币结之"（020-036）。大月氏初居敦煌、祁连间，为匈奴所破，西击塞王，塞王南走，大月氏居其地，后乌孙击走大月氏，大月氏西徙击大夏，而塞王南越县度（019-040）。康居在汉与西域诸国交往中扮演重要角色。
