# 刁协

刁协，字玄亮，[[渤海]]饶安人。[[东晋]][[尚书]][[左仆射]]。

刁协为军咨[[祭酒]]，与王承、卞壶等从[[司马睿]]渡江（087-086）。刁协被周玘所轻，耻恚愈甚（088-066）。以[[镇东]][[长史]]刁协为[[丞相]]左[[长史]]（088-076）。后为[[王敦]]所攻，[[出奔]]而死。

> 以前颍川太守勃海刁协为军咨祭酒，前东海太守王承、广陵相卞壶为从事中郎。
> （087-086）

## 渡江与死难

刁协为[[丞相]]左[[长史]]，后以[[尚书]][[左仆射]]、[[刘隗]]为[[御史]]中丞，共执[[朝政]]（090-010）。[[王敦]]以讨[[刘隗]]、刁协为名举兵，吴兴[[功曹]]徐馥[[矫称]][[周札]]之命，收合徒众，以讨[[王导]]、刁协为名（089-027）。敦兵至[[建康]]，协[[出奔]]，道死。
