# 开元钱

开元钱即开元通宝钱，唐武德四年（621年）始铸，通行唐代近三百年，为唐代主要流通货币。

唐肃宗时，第五琦作乾元钱、重轮钱，与开元钱三品并行。民间争相盗铸，货轻物重，谷价腾踊，饿殍相望。上言者皆归咎于琦，贬为忠州长史（221-054）。诏规定开元钱与乾元小钱皆当十，重轮钱当三十（221-077）。

南唐元宗时铸当十大钱"永通泉货"，又铸"唐国通宝"，与开元钱并行（294-098）。开元钱为唐代代表性货币，其"通宝"名称体系影响了后世千年钱制。
