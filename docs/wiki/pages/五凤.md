# 五凤

**五凤**（前57—前54），[[汉宣帝]]第五个[[年号]]。五凤年间，[[匈奴]]内乱，[[呼韩邪单于]]与[[郅支单于]]争立，五单于并立，互相攻伐。汉廷议者多请乘机灭匈奴，[[萧望之]]独以为当扶弱济危，遣使吊问。宣帝从望之议，存亡继绝，匈奴遂亲汉。
