# 张重华

张重华，字太临，[[前凉]]桓王，[[张骏]]之子。在位时用[[谢艾]]为将，大破[[后赵]][[麻秋]]，保[[凉州]]于危难之际。

赵将军[[王擢]]击张重华，袭武街，执护军曹权、胡宣，徙七千馀户于[[雍州]]（097-084）。张重华将宋秦等帅户二万降于赵。重华以[[谢艾]]为使持节、军师将军，帅步骑三万进军临河（097-098）。张重华使将军牛旋御之，退守枹罕，[[姑臧]]大震（097-099）。

张重华年十六[[嗣位]]，当[[后赵]][[石虎]]大举来攻之际，能用[[谢艾]]、索遐等贤臣良将，屡破赵军，保全[[凉州]]。然重华在位后期，渐好宴乐，不理政事，[[前凉]]由此中衰。其短祚而终，年仅二十七。

> 赵将军王擢击张重华，袭武街，执护军曹权、胡宣，徙七千馀户于雍州。
> （097-084）
