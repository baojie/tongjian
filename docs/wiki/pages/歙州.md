# [[歙州]]

[[歙州]]是[[隋]][[唐]]时期设置的州，治所在今[[安徽]]歙县，隶属[[江南东道]]。辖境约当今[[安徽]]南部[[新安江]]上游地区，[[宋代]]改为[[徽州]]。

歙州地处[[皖南]]山区，是连接[[浙江]]、[[江西]]的重要通道。境内[[新安江]]为[[钱塘江]]上游，[[水运]]便利。歙州（徽州）在[[明清]]时期以[[徽商]]闻名，但在《[[资治通鉴概述|资治通鉴]]》所载的[[隋唐]]时期，歙州尚为[[江南]]偏远之地。
