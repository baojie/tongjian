# 公孙奭

公孙奭，战国韩国大臣，亦见于秦廷。

秦武王欲伐宜阳，甘茂恐公孙奭等阻挠，与之盟于息壤（003-050）。甘茂攻宜阳五月不下，公孙奭果争之（003-052）。后公孙奭与向寿争宜阳事，不能得，由此怨谗甘茂，甘茂惧而奔齐（003-056）。
