# 公孙奭

公孙奭，[[战国]][[韩国]]大臣，亦见于秦廷。

[[秦武王]]欲伐[[宜阳]]，[[甘茂]]恐公孙奭等阻挠，与之盟于息壤（003-050）。[[甘茂]]攻[[宜阳]]五月不下，公孙奭果争之（003-052）。后公孙奭与[[向寿]]争[[宜阳]]事，不能得，由此怨谗[[甘茂]]，[[甘茂]]惧而奔齐（003-056）。

## 世系

公孙奭，[[战国]][[韩国]]大臣。[[秦武王]]欲伐[[宜阳]]，[[甘茂]]恐奭阻挠，与[[秦王]]盟于[[息壤]]（003-050）。[[甘茂]]攻[[宜阳]]五月不下，奭果争之（003-052）。后奭怨谗[[甘茂]]，[[甘茂]]惧而奔[[齐]]（003-056）。奭[[子孙]]不详于史。
