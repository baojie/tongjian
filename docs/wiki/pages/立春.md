# 立春

立春，二十四节气之首。古代于立春日迎春于东郊，行宽赦之政。

[[张敞]]杀舜，会立春，行冤狱[[使者]]出，[[使者]]奏敞贼杀不辜（027-035）。[[周举]]上疏曰："立春以后阴寒失节，宜采纳良臣以助圣化"（051-099）。制自今立春[[亲迎]]春于东郊（214-078）。

> 太常博士王玙上疏请立青帝坛以迎春；从之。冬，十月，辛丑，制自今立春亲迎春于东郊。
> （214-078）
