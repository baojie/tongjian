# 郭璞

**郭璞**，字景纯，[[河东]][[闻喜]]人，[[东晋]]学者、[[术数]]大师。

郭璞[[博学]]多才，通[[经术]]、擅[[辞赋]]、精[[阴阳]][[五行]]。[[元帝]]时以[[著作]]佐郎上疏（091-065），谏帝用刑过差，[[阴阳]]错缪皆繁刑所致。又上疏请因皇孙生而下赦令（092-003）。

郭璞以占验闻名，注《尔雅》《山海经》《穆[[天子]]传》，为魏晋学术集大成者。后为[[王敦]]所杀。郭璞在[[资治通鉴概述|通鉴]]中出现的形象以[[灾异]][[谏诤]]为主，其学术[[贡献]]虽未详述，然足以见其于[[东晋]]思想界的重要地位。

## 术数与谏诤

郭璞以[[阴阳]][[术数]]屡次上疏[[谏诤]]，[[元帝]]时以日中有黑子，上疏言[[阴阳]]错缪皆繁刑所致（091-065），复因皇孙生请下赦令（092-003）。[[王敦]]将举兵伐[[京师]]，使郭璞筮之，卦凶，璞曰"无成"，敦素疑璞助[[温峤]]、[[庾亮]]，遂杀之（093-013）。后[[朝廷]][[追赠]]郭璞等官（093-035）。郭璞尝遇郭含，欲为之筮，含曰："年在天，位在人。修己而天不与者，命也；守道而人不知者，性也。"（096-028）。
