# 董晋

董晋，唐德宗朝门下侍郎同平章事，以方正著称。

祠部郎中虞乡董晋为判官送崇徽公主嫁回纥（224-085）。华州刺史董晋弃州走行在，骆元光引兵复华州（229-012）。以国子祭酒董晋为河北宣慰使（229-032）。董晋方正，可处门下，为宰相（233-049）。门下侍郎、同平章事董晋罢为礼部尚书（234-042）。以东都留守董晋同平章事兼宣武节度使（235-045）。

## 世系

董晋（？—约799年），唐[[门下侍郎]]同[[平章事]]、宣武[[节度使]]，[[虞乡]]人。其先世不详于史。晋方正，为相（233-049），罢为[[礼部]][[尚书]]（234-042），镇宣武擒[[邓惟恭]]（235-059）。薨于镇（235-085）。
