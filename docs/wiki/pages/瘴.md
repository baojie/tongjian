# 瘴

瘴是南方山林中的湿热毒气，古代行军远征之大患。东汉时期南州"水土温暑，加有瘴气"，远征士卒死亡者十必四五（052-043）。南朝陈霸先讨李贲，因"春瘴方起"，士卒死者什六七（158-100）。

隋代周法尚讨岭南，所部遇瘴不能进，只得顿兵衡州（178-057）。唐太宗欲讨冯盎，[[魏征]]谏阻，言"岭南瘴疠险远，不可以宿大兵"（192-055）。唐代狄仁杰上疏论冤狱，指责酷吏将无辜者流放"瘴海"之地（208-126），瘴海遂成贬谪远方的代称。历代用兵岭南，瘴气均为不可忽视的自然障碍。
