# 瘴

**瘴**指南方湿热地区[[山林]]中弥漫的致病之气，又称瘴气。古人认为瘴气是导致南方疫病的主要原因。瘴气之说在古代军事和[[移民]]中影响重大，南方[[征伐]]每受其阻。
