# 王音

王音，西汉外戚，[[王凤]]之从弟。[[《资治通鉴》]]载其敬事王凤，代为大司马车骑将军。

以侍中、太仆王音为御史大夫，王氏愈盛，郡国守相、刺史皆出其门下（030-082）。王音敬凤，卑恭如子，故凤荐之。八月，丁巳，凤薨。以王音为大司马、车骑将军（031-005）。王音既以从舅越亲用事，小心亲职（031-021）。
