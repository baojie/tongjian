# 田绪

田绪，[[唐]]魏博节度使，[[田承嗣]]之子。凶险多过失，[[田悦]]不忍杀，杖而拘之（230-017）。后杀田悦，以魏博兵马使为节度使（230-031）。[[浑瑊]]帅诸军出斜谷，田绪助讨朱泚有功。尚结赞以邠军不出为辞，不肯出兵（230-031）。代宗以嘉诚公主妻田绪（231-051）。

> 兵马使田绪，承嗣之子也，凶险，多过失，悦不忍杀，杖而拘之。
> （230-017）
