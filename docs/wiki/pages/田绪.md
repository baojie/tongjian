# 田绪

田绪，[[唐_(朝代)|唐]]魏博[[节度使]]，[[田承嗣]]之子。凶险多过失，[[田悦]]不忍杀，杖而拘之（230-017）。后杀[[田悦]]，以魏博[[兵马使]]为[[节度使]]（230-031）。[[浑瑊]]帅诸军出[[斜谷]]，田绪助讨[[朱泚]]有功。[[尚结赞]]以邠军不出为辞，不肯[[出兵]]（230-031）。代宗以嘉诚[[公主]]妻田绪（231-051）。

> 兵马使田绪，承嗣之子也，凶险，多过失，悦不忍杀，杖而拘之。
> （230-017）
