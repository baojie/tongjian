# 鼎盛

**鼎盛**指正当壮年或处于全盛时期，常用于形容帝王年富力强。

## 春秋鼎盛

贾谊谏文帝：天子春秋鼎盛，行义未过，德泽有加焉，而诸侯已若此（014-042）。春秋鼎盛，当行仁义。

孔奂谏陈废帝：皇太子春秋鼎盛，圣德日跻，若行废立，臣不敢闻诏（169-129）。鼎盛之年，不当废立。

张廷珪谏玄宗：上春秋鼎盛，宜崇经术，迩端士，尚朴素（211-005）。鼎盛之时，当务根本。

## 鼎盛之戒

王义方谏高宗：皇太子春秋鼎盛，圣道未周，是可言之秋（202-179）。鼎盛之年，辅佐为急。

李绛谏宪宗：陛下春秋鼎盛，方励志太平，宜拒绝方士之说（238-030）。鼎盛之年，勿惑妖妄。
