# 燕惠王

燕惠王，战国燕国君主。

燕惠王为太子时与乐毅有隙，及即位，齐田单纵反间于燕，燕惠王乃使骑劫代乐毅为将，乐毅奔赵（004-073）。燕惠王薨，子武成王立（005-006）。
