# 燕惠王

燕惠王，[[战国]][[燕国]][[君主]]。

燕惠王为[[太子]]时与[[乐毅]]有隙，及[[即位]]，齐[[田单]]纵[[反间]]于燕，燕惠王乃使骑劫代[[乐毅]]为将，[[乐毅]]奔赵（004-073）。燕惠王薨，子武成[[王立]]（005-006）。

## 世系

燕惠王（？—[[前272年]]），[[战国]][[燕国]][[君主]]。父[[燕昭王]]平，子[[燕武成王]]立（005-006）。惠王为[[太子]]时与[[乐毅]]有隙，及[[即位]]中[[田单]][[反间计]]，走[[乐毅]]，使[[骑劫]]代将（004-073）。
