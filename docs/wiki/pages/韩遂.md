# 韩遂

韩遂，字文约，金城人。汉末西凉军阀，本为凉州官吏，被边章等劫持起兵，后独占凉州，与[[马腾]]、[[曹操]]周旋数十年。

金城人边章、韩遂素著名西州，群盗诱而劫之，使专任军政，杀金城太守陈懿，攻烧州郡（058-058）。张温将诸郡兵步骑十馀万屯美阳，边章、韩遂亦[[进兵]]美阳，温与战辄不利（058-079）。韩遂杀边章及北宫[[韩瑗|伯玉]]、李文侯，拥兵十馀万，进围[[陇西]]（058-096）。

韩遂在凉州经营三十余年，先与边章并起，后杀边章兼并其众。与[[马腾]]结盟，旋又相攻。[[曹操]]西征[[关中]]，韩遂与[[马超]]联军拒操，被[[曹操]]用[[离间]]计分化瓦解。韩遂兵败后逃归凉州，为部将所杀。韩遂虽为军阀，然素有名望，在西州羌胡中颇有威信。
