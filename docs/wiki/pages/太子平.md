# 太子平

太子平，燕王哙太子。

燕王哙禅位于子之，国内大乱，太子平与将军市被谋攻子之（003-024）。齐王令章子伐燕，燕士卒不战，城门不闭，齐人杀燕王哙。燕人共立太子平，是为燕昭王（003-032）。

## 世系
