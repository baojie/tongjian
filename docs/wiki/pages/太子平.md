# 太子平

太子平，[[燕王哙]][[太子]]。

[[燕王哙]]禅位于子之，国内大乱，太子平与[[龙骧将军|将军]]市被谋攻子之（003-024）。[[齐王]]令章子伐燕，燕[[士卒]]不战，[[城门]]不闭，齐人杀[[燕王哙]]。燕人共立太子平，是为[[燕昭王]]（003-032）。

## 世系

太子平（？—[[前279年]]），[[姬姓]]，名平，[[战国]][[燕国]][[太子]]，后为[[燕昭王]]。父[[燕王哙]]。[[子之之乱]]中攻子之（003-024），齐人杀[[燕王哙]]后，燕人共立平为[[燕昭王]]（003-032）。子[[燕惠王]]。
