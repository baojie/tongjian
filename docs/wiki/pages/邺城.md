# 邺城

邺城（鄴），战国至[[南北朝]]重要城邑。[[魏国]]时[[西门豹]]治邺，为魏之重镇。[[战国]]末[[秦]]攻拔邺城。

## 通鉴记载

[[魏文侯]]以[[西门豹]]治邺（001-015）。[[信陵君]]窃符救赵，使[[硃亥]]椎杀[[晋鄙]]，夺其兵留壁于邺（005-053、005-055）。邺后归赵（006-075）。秦将[[王翦]]、[[桓齮]]攻邺取九城（006-092）。秦大兴师伐赵，一军抵邺（006-109）。
