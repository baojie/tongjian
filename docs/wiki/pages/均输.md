# 均输

均输是汉代调节物资运输的经济制度，由[[桑弘羊]]创立，令远方各以其物为赋，由官府统一调配运输，以省运费而利国用。

孔仅为大农令，[[桑弘羊]]为大农中丞，稍置均输以通货物（020-033）。均输法行后，民不益赋而天下用饶，弘羊赐爵左庶长（020-089）。汉昭帝时贤良[[文学]]请罢均输官，[[桑弘羊]]难之，以为此国家大业（023-033）。

均输之法改变了汉代郡国[[贡赋]]运输的无序状态，将物资调配权收归中央，既减少运输损耗，又增加财政收入，为[[汉武帝]]时期财政改革之重要组成部分。

> 桑弘羊为大农中丞，稍置均输，以通货物。
> （020-033）
