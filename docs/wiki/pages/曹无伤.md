# 曹无伤

曹无伤，项羽部将，为左尹。告密刘邦，为项羽所卖。

曹无伤为项羽左尹，告密刘邦欲王关中（007-038）。项羽以此怒，欲击刘邦，遂有鸿门之宴。项伯为刘邦说项羽，项羽许诺。及鸿门宴，刘邦谓项羽曰："臣与将军戮力而攻秦，将军战河北，臣战河南。然不自意能先入关破秦，得复见将军于此。今者有小人之言，令将军与臣有郤。"项羽曰："此沛公左司马曹无伤言之。不然，籍何以至此？"刘邦归，立诛曹无伤。

> 曹无伤为项羽左尹，告密刘邦欲王关中。
> （007-038）
