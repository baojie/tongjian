# 水灾

水灾是洪水泛滥造成的灾害，[[资治通鉴概述|通鉴]]历代备载。[[汉武帝]]时黄河决口，田蚡因封邑在[[河北]]而河决南流，"鄃无水灾"，遂阻挠塞河之议，致使梁、楚之地连年被灾（018-010）。后武帝亲临瓠子口，发卒数万人塞决河，筑宣防宫，梁楚之地复宁无水灾（021-005）。

谷永[[上书]]论灾异，指"建始以来，[[日食]]、[[地震]]三倍春秋，水灾亡与比数"，以阴盛阳微释之（031-082）。汉成帝时"比年郡国伤于水灾，禾麦不收"（032-017）。贾让治河[[奏议]]提[[出使]]河入渤海，"乃无水灾"（036-018）。历代水灾记载构成了古代灾害史的重要篇章。
