# 陈俊

陈俊，字子昭，南阳人。东汉强弩将军，云台二十八将之一。从光武帝平定河北。

长史南阳陈俊，光武帝以俊为安集掾（039-042）。强弩将军陈俊言于王曰："贼无辎重，宜令轻骑出贼前，使百姓各自坚壁以绝其食，可不战而殄也。"（040-007）

陈俊从光武帝征战河北，以强弩将军督师，屡立战功。在追击铜马、青犊等农民军时，陈俊献坚壁清野之策，使贼众困饿而降。光武帝即位后，陈俊封列侯，食邑四县。俊善抚士卒，在军有威名。永平中，显宗图画云台二十八将，陈俊位列其中。
