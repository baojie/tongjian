# 黄揆

黄揆，[[黄巢]]之弟，唐末农民军将领。

[[李克用]]将李存贞败黄揆于沙苑（255-042）。[[李克用]]与[[黄巢]]军大战于梁田陂，贼众大败，巢将王璠、黄揆袭华州，据之（255-045）。[[李克用]]进围华州，黄思邺、黄揆婴城固守（255-047）。

## 从黄巢起义

黄揆为[[黄巢]]弟，随巢转战。[[李克用]]进围华州，黄揆与黄思邺婴城固守，后弃城走（255-047, 255-055）。
