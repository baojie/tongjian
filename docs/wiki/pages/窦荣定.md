# 窦荣定

窦荣定，扶风平陵人。隋秦州总管，破突厥阿波可汗。以外戚之亲居将帅之任。

隋秦州总管窦荣定帅九总管步骑三万出凉州，与突厥阿波可汗相拒于高越原，阿波屡败。荣定，炽之兄子也（175-115）。隋以上柱国窦荣定为右武卫大将军。荣定妻，隋主姊安成公主也。隋主欲以荣定为三公，辞曰："卫、霍、梁、邓，若少自贬损，不至倾覆。"（175-135）

窦荣定以外戚贵盛，然谦退不骄。文帝欲拜为三公，荣定固辞，引西汉外戚倾覆之祸为戒，时论称之。荣定在凉州与突厥对垒，以武略建功，为隋初边陲重将。
