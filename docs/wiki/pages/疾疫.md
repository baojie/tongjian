# 疾疫

疾疫是瘟疫等急性传染病的总称，古代频发的重大灾害。[[汉文帝]]诏书承认"间者数年不登，又有水旱、疾疫之灾"，认为朝政有失导致天降灾异（015-051）。[[汉元帝]]因"天下颇被疾疫之灾"，[[诏令]]郡国被灾严重者"毋出今年租赋"（025-050）。

[[赵充国]]论羌事，列举[[出兵]]不利的因素，其一为士卒"离霜露、疾疫、瘃堕之患"（026-019）。[[汉成帝]]以民疾疫，"令太官损膳，减[[乐府]]员，省苑马，以振困乏"（028-007）。[[鲍宣]][[上书]]陈"七死"，其一为"时气疾疫"（034-042），足见疾疫对百姓生命的严重威胁。
