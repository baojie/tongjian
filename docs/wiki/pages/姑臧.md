# 姑臧

姑臧是河西走廊重镇，位于武威（今甘肃武威）。汉武帝置河西四郡，武威郡治姑臧，为丝绸之路必经之地（024-043）。[[窦融]]据河西，以姑臧为基地，保全一方（041-072）。

西晋末中原大乱，[[张轨]]据凉州，以姑臧为都城，史称前凉（087-046）。前凉[[张骏]]在姑臧南作五殿，官属皆称臣，西域诸国朝贡于此（095-087）。[[吕光]]平西域后据姑臧建后凉（107-061），[[沮渠蒙逊]]以姑臧为都建北凉（123-064）。姑臧在五胡十六国时期为西北政治文化中心，也是佛教东传的重要中转站。
