# 姑臧

姑臧是[[河西]]走廊重镇，位于[[武威]]（今甘肃[[武威]]）。[[汉武帝]]置[[河西]]四郡，[[武威]]郡治姑臧，为丝绸之路必经之地（024-043）。[[窦融]]据[[河西]]，以姑臧为基地，保全一方（041-072）。

[[西晋]]末[[中原]]大乱，[[张轨]]据[[凉州]]，以姑臧为都城，史称[[前凉]]（087-046）。[[前凉]][[张骏]]在姑臧南作五殿，官属皆称臣，[[西域]]诸国[[朝贡]]于此（095-087）。[[吕光]]平[[西域]]后据姑臧建[[后凉]]（107-061），[[沮渠蒙逊]]以姑臧为都建[[北凉]]（123-064）。姑臧在[[五胡]][[十六国]]时期为西北政治文化中心，也是[[佛教]]东传的重要中转站。

## 姑臧与凉州政权

姑臧在[[十六国]]时期迭为[[凉州]]诸政权之都城，见证西北政局之兴替。[[前凉]][[张寔]]被左右所杀，其党"轘于姑臧市"（091-045），姑臧作为政[[治中]]心之肃杀可见。[[刘曜]]攻[[凉州]]，扬言百道俱济直抵姑臧，[[张茂]]率众固守，大城姑臧修[[灵钧台]]以御敌（092-071）（092-074）。[[张骏]]继位后，[[晋愍帝]][[使者]]史淑至姑臧，拜骏[[大将军]]、[[凉州牧]]、[[西平公]]（090-004），[[前凉]]以姑臧为中心维系晋室[[正朔]]，号召[[河西]]。[[张骏]]复徙[[陇西]]、[[南安]]民二千余家于姑臧，以实[[河西]]（093-078），姑臧作为区域中心的地位持续强化。自[[前凉]]经[[后凉]]、[[北凉]]，姑臧[[始终]]为[[河西]]政治文化之枢纽，其兴衰与[[十六国]]西北局势紧密相连。
