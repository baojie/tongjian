# 刘屈氂

刘屈氂，西汉左丞相。中山靖王子，武帝末因巫蛊及贰师将军李广利谋立昌邑王事被族诛。

以涿郡太守刘屈氂为左丞相，封澎侯，屈氂，中山靖王子也（022-038）。贰师将军李广利出师，丞相刘屈氂为祖道，送至渭桥，广利曰："愿君侯早请昌邑王为太子，如立为帝，君侯长何忧乎"（022-063）。

> 初，贰师之出也，丞相刘屈氂为祖道，送至渭桥。广利曰："愿君侯早请昌邑王为太子；如立为帝，君侯长何忧乎！"
> （022-063）
