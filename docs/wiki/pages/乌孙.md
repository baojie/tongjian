# 乌孙

乌孙是伊犁河流域的游牧大国，西汉时与汉朝结盟共抗匈奴。张骞第二次出使西域，携厚币至乌孙，欲"令乌孙居浑邪王之地"，结昆弟之好（019-039）。乌孙发译道送骞还，并遣使随骞报谢（020-035）。

汉以江都王女细君为公主妻乌孙昆莫（020-034/037），后以楚王女解忧公主继之。解忧公主在乌孙数十年，使其亲汉疏胡。汉宣帝时乌孙大败匈奴，常惠持金币还赐乌孙贵人有功者（024-052）。乌孙与汉的和亲政策，成为汉朝西域战略的成功典范。
