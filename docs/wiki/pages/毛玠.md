# 毛玠

毛玠，字孝先，[[陈留]]平丘人。曹魏[[丞相]]东曹掾，典选举，以清公著称。

毛玠劝[[曹操]]奉天子以令不臣（060-054）。毛玠与[[崔琰]]并典选举，所举皆清正之士（065-032）。毛玠以清公称，为时所重（066-061）。

毛玠少为县吏，以清公入仕。[[曹操]]领兖州，辟为治中从事。玠语操曰："今天下分崩，宜奉天子以令不臣，脩耕植以畜[[军资]]。"操深纳之。后为东曹掾，与[[崔琰]]并典选举，所举皆清正之士，虽贵宠之臣，无所假借。时人语曰："[[毛公]]举贤，清浊分明。"后坐事免官，卒于家。
