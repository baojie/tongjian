# 服丧

服丧是为死者穿戴[[丧服]]并遵守一定期限的守[[丧礼]]仪，为古代[[丧礼]]的重要组成部分。

河间惠王刘良能修献王之行，母[[太后]]薨服丧如礼，诏益封万户为宗室仪表（033-023）。[[王莽]]令天下吏六百石以上皆服丧三年（036-036）。[[王莽]]为平王服丧，缌缞弁而加麻环绖，如天子吊诸侯服（036-063）。

服丧之制按亲疏分为五服：斩衰、齐衰、大功、小功、缌麻。三年之丧为最重，适用于父母。汉代服丧制度时行时废，至唐宋始臻完备。服丧不仅为家族礼仪，亦具社会教化功能。

> 河间惠王良能修献王之行，母太后薨，服丧如礼；诏益封万户，以为宗室仪表。
> （033-023）
