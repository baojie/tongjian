# 冕

冕是帝王及高级官员的礼冠，上覆冕板、前后垂旒，为古代冕服制度之核心。

[[贾谊]]上疏[[隋文帝|文帝]]，论太子教育当以礼为本："有司齐肃端冕，见之南郊，过阙则下，过庙则趋"（014-050）。[[王莽]]受九锡之礼，受绿韨、衮冕、衣裳等（036-029）。群臣奏请[[王莽]]践祚，"服天子韨冕，背斧依立于户牖之间"，南面朝群臣（036-041）。

东[[汉明帝]]时，东平王刘苍定南北郊[[冠冕]]、车服制度及光武庙登歌、八佾舞数（044-093）。帝及公卿列侯，始服[[冠冕]]、玉佩以行事（044-096）。[[曹操]]受九锡，兗冕之服、赤舄副焉（066-053），后魏王操冕十有二旒，乘金根车（068-005）。冕旒制度沿至明清。

> 东平王苍以为中兴三十馀年，四方无虞，宜修礼乐，乃与公卿共议定南北郊冠冕、车服制度。
> （044-093）
