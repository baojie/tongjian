# 称制

称制是[[太后]]代行[[皇帝]]权力、[[发号施令]]的[[政治]][[制度]]。"制"为[[皇帝]]命令的专称，"称制"即意味着以[[皇帝]]名[[义行]]使最高权力。

[[西汉]]吕[[太后]]在惠帝驾崩后[[临朝]]称制，开[[太后]]称制之先河。惠帝[[太子]][[即位]]年幼，[[吕后]]"[[临朝]]称制"，[[太子]]既葬，即[[皇帝]]位年幼，[[太后]][[临朝]]称制（012-103）。[[吕后]]欲封[[吕氏|诸吕]]为王，问于[[丞相]][[王陵]]、[[陈平]]、[[周勃]]，以"[[太后]]称制，王[[吕氏|诸吕]]，无所不可"为由获支持（013-003）。

[[西汉]]末年，元后王政君亦[[临朝]]称制，扶持[[王莽]]篡汉。[[东汉]]多幼帝[[即位]]，[[太后]]称制更为常见，如[[邓太后]]、梁[[太后]]等。唐代[[武后|武则天]]从称制到改唐为周（[[称帝]]），将称制[[政治]]推向了极致。

称制虽可维持政权稳定，却也常导致[[外戚]]干政、[[权臣]]擅权等[[政治]]问题。
