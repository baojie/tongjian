# 称制

称制是[[太后]]代行皇帝权力、发号施令的政治制度。"制"为皇帝命令的专称，"称制"即意味着以皇帝名义行使最高权力。

[[西汉]]吕[[太后]]在惠帝驾崩后临朝称制，开[[太后]]称制之先河。惠帝[[太子]]即位年幼，[[吕后]]"临朝称制"，[[太子]]既葬，即皇帝位年幼，[[太后]]临朝称制（012-103）。[[吕后]]欲封诸吕为王，问于[[丞相]]王陵、[[陈平]]、[[周勃]]，以"[[太后]]称制，王诸吕，无所不可"为由获支持（013-003）。

[[西汉]]末年，元后王政君亦临朝称制，扶持[[王莽]]篡汉。[[东汉]]多幼帝即位，[[太后]]称制更为常见，如[[邓太后]]、梁[[太后]]等。唐代[[武后|武则天]]从称制到改唐为周（[[称帝]]），将称制政治推向了极致。

称制虽可维持政权稳定，却也常导致[[外戚]]干政、权臣擅权等政治问题。
