# 鲍泉

鲍泉，字润岳，东海人。南朝梁信州刺史。

侯景之乱，湘东王萧绎遣竟陵太守王僧辩、信州刺史鲍泉击湘州。二人入白绎求宽期，绎疑僧辩与誉通，斫之中髀，下狱（162-064）。鲍泉军于石椁寺，河东王萧誉逆战而败，又败于橘洲，誉退保长沙（162-070）。萧绎以鲍泉围长沙久不克，怒之，以王僧辩代为都督，数泉十罪（162-080）。

> 湘东王绎遣竟陵太守王僧辩、信州刺史东海鲍泉击湘州。
> （162-064）
