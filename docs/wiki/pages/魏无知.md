# 魏无知

魏无知，西汉开国谋臣，以荐陈平著称。

陈平亡楚归汉，因魏无知求见汉王刘邦。刘邦召入，赐食（009-037）。后绛侯周勃、灌婴等谗陈平盗嫂受金，刘邦疑之，召让魏无知。无知曰："臣所言者能也，陛下所问者行也"，刘邦乃厚赐陈平（009-045）。后陈平辞功曰："非魏无知，臣安得进？"刘邦复赏魏无知（011-035）。

## 荐贤

陈平弃楚归汉，因魏无知之荐得见高祖刘邦（009-037）。后周勃、灌婴等谗陈平盗嫂受金，高祖疑而召让魏无知。无知对曰："臣所言者能也，陛下所问者行也。今有尾生、孝己之行，而无益胜负之数，陛下何暇用之乎？"（009-045）。高祖善其言，乃厚赐陈平，拜为护军中尉。及陈平功成，辞功曰："非魏无知，臣安得进？"高祖赞其不忘本，复赏魏无知（011-035）。魏无知以荐贤之功，见于史册，为汉初开国谋臣之一。
