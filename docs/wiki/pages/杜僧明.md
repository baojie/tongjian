# 杜僧明

杜僧明，字弘照，广陵临泽人。[[南朝]]陈骁将，[[陈霸先]]帐下先锋，以勇猛著称。

霸先遣主帅杜僧明将二千人顿于岭上（162-094）。杜僧明马被伤，[[陈霸先]]救之，授以所乘马。僧明上马复战，众军因而乘之。路养大败（163-004）。

杜僧明初从广州[[刺史]]萧映，后归[[陈霸先]]。在讨[[侯景]]之役中，僧明与[[周文育]]并为[[陈霸先]]前锋，勇不可当。征蔡路养时，路养妻侄[[萧摩诃]]年十三单骑出战，无人敢当。僧明马被伤，霸先救之，僧明夺马复战，大破路养。杜僧明以勇烈为霸先所重，陈建后封侯。然早卒，未及大用。
