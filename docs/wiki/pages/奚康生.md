# 奚康生

奚康生，[[河南]][[洛阳]]人。[[北魏]]直阁将军，以骁勇有力著称，屡破叛胡。

奚康生直阁将军，勇敢有气力（140-013）。奚康生从征樊安，力战有功（140-082）。

奚康生少壮勇，有膂力，能引强弓。初为军主，从征柔然，率先陷阵。宣武帝时，秦州羌叛，康生从[[崔延伯]]讨平之。康生每战必冲锋在前，矢石所及，无不应弦而倒。后以讨破六韩拔陵有功，累迁直阁将军。然康生性暴虐，好杀降卒，时人以此少之。

> 奚康生直阁将军，勇敢有气力。
> （140-013）

## 骁勇之将

奚康生应募缚筏积柴，因风纵火，烧齐船舰，依烟直进，大破中渚兵（140-013）。击叛胡，追至车突谷，大破之（140-082）。与[[杨大眼]]等救涡阳，裴叔业引军退（141-051）。将羽林一千驰赴寿阳，集城内耆旧宣诏抚赉（143-008）。降御内外，闭城一月以待援（143-012）。救淮阳（146-047），救彭城（146-065）。劝中山王英进据洛水（146-075）。奚康生骁勇为[[北魏]]名将，然性暴虐好杀，时论惜之。
