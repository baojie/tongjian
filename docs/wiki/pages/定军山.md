# [[定军山]]

[[定军山]]位于今[[陕西]]勉县，是[[三国]]时期著名战场。[[汉中之战]]中，[[蜀汉]][[老将]][[黄忠]]在定军山斩杀[[曹魏]]大将[[夏侯渊]]，一举扭[[转战]]局（068-053），使[[刘备]]得以夺取[[汉中]]，进位[[汉中王]]。

定军山也因此成为[[中国]]历史上以少胜多的经典战场之一，后世多有[[文学]]作品演绎其事。
