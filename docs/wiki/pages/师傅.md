# 师傅

**师傅**是古代教导辅佐[[帝王]]或[[太子]]的官员。汉初大臣为[[太子]]择师傅、[[宾客]]，选士之有节行者与居（013-071）。[[司马光]]认为，不置贤师傅而用之典兵，则骄而犯上（014-067）。

## 太子教育

师傅在[[太子]]教育中至关重要。[[汉武]]时[[太子]][[宫中]]事起，[[太子]]惧问[[少傅]]石德，德惧为师傅并诛，遂劝[[太子]]起兵（022-046）。师傅的选择直接影响[[太子]]的品德与国家命运。
