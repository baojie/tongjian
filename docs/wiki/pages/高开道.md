# 高开道

高开道，隋末勃海割据首领。[[《资治通鉴》]]载其初从[[格谦]]起义，后据北平，降唐复叛，自杀死。

格谦将勃海高开道收其馀众，寇掠燕地，军势复振（183-033）。[[窦建德]]、高开道各[[遣使]]招[[李艺]]，艺拒之（186-059）。隋右武卫大将军李景守北平，高开道围之，岁馀不能克（186-072）。营州总管邓暠击高开道，败之（187-045）。李艺告急于高开道，开道帅二千骑救之（188-081）。

> 河间贼帅格谦拥众十馀万，据豆子，自称燕王，帝命王世充将兵讨斩之。谦将勃海高开道收其馀众，寇掠燕地，军势复振。
> （183-033）

> 李艺告急于高开道，开道帅二千骑救之，建德兵引去，开道因艺遣使来降。戊申，以开道为蔚州总管，赐姓李氏。
> （188-081）
