# 颜之推

颜之推，字介，[[琅邪]]临沂人。[[北齐]][[黄门侍郎]]，著名[[文学]]家、教育家。撰《颜氏家训》，为家训之祖。

颜之推论曰：孝昭天性至孝，而不知忌讳，乃至于此（168-102）。[[祖珽]]奏置文林馆，多引[[文学]]之士以充之，以[[中书侍郎]]博陵[[李德林]]、[[黄门侍郎]][[琅邪]]颜之推同判馆事，又命共撰《修文殿御览》（171-067）。

颜之推历仕梁、[[北齐]]、[[北周]]、隋四朝，以[[博学]]善文著称。在[[北齐]]与[[李德林]]等同修《修文殿御览》。之推所撰《颜氏家训》二十篇，以[[儒家]]思想为宗，兼及佛道，论立身治家之道。文风平实，内容丰富，影响深远，被誉为"[[古今]]家训之祖"。其孙[[颜师古]]为唐初著名[[经学]]家。

> 祖珽奏置文林馆，多引文学之士以充之，谓之待诏；以中书侍即博陵李德林、黄门侍郎琅邪颜之推同判馆事。
> （171-067）
