# 颜之推

颜之推，字介，琅邪临沂人。北齐黄门侍郎，著名文学家、教育家。撰《颜氏家训》，为家训之祖。

颜之推论曰：孝昭天性至孝，而不知忌讳，乃至于此（168-102）。祖珽奏置文林馆，多引文学之士以充之，以中书侍郎博陵李德林、黄门侍郎琅邪颜之推同判馆事，又命共撰《修文殿御览》（171-067）。

颜之推历仕梁、北齐、北周、隋四朝，以博学善文著称。在北齐与李德林等同修《修文殿御览》。之推所撰《颜氏家训》二十篇，以儒家思想为宗，兼及佛道，论立身治家之道。文风平实，内容丰富，影响深远，被誉为"古今家训之祖"。其孙颜师古为唐初著名经学家。

> 祖珽奏置文林馆，多引文学之士以充之，谓之待诏；以中书侍即博陵李德林、黄门侍郎琅邪颜之推同判馆事。
> （171-067）
