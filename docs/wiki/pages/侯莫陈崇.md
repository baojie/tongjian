# 侯莫陈崇

侯莫陈崇，字尚乐，[[代郡]]武川人。北周柱国[[大将军]]，[[八柱国]]之一。以单骑擒[[万俟丑奴]]闻名。

直阁[[代郡]]侯莫陈崇单骑入贼中，于马上生擒丑奴，因大呼，众皆披靡（154-008）。泰遣[[都督]]侯莫陈崇帅轻骑一千袭之。崇乘夜将十骑直抵城下，馀众皆伏于近路；归见骑少，不设备。崇即入（156-045）。[[广陵]]王欣、[[赵郡]]公[[李弼]]、[[陇西]]公[[李虎]]、[[河内]]公[[独孤信]]、[[南阳]]公[[赵贵]]、[[常山]]公[[于谨]]、[[彭城]]公侯莫陈崇，谓之[[八柱国]]（163-096）。

侯莫陈崇以勇猛善战著称，在平定[[万俟丑奴]]之乱中一战成名。后从[[宇文泰]]征战，屡立战功，封柱国[[大将军]]。北周初年，侯莫陈崇为[[宇文护]]所忌，以言事忤旨被逼令自杀。侯莫陈氏为代北武川[[豪族]]，子弟多列北周名将。
