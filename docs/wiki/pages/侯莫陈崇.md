# 侯莫陈崇

侯莫陈崇，字尚乐，代郡武川人。北周柱国大将军，八柱国之一。以单骑擒万俟丑奴闻名。

直阁代郡侯莫陈崇单骑入贼中，于马上生擒丑奴，因大呼，众皆披靡（154-008）。泰遣[[都督]]侯莫陈崇帅轻骑一千袭之。崇乘夜将十骑直抵城下，馀众皆伏于近路；归见骑少，不设备。崇即入（156-045）。广陵王欣、赵郡公[[李弼]]、[[陇西]]公[[李虎]]、河内公[[独孤信]]、南阳公[[赵贵]]、常山公[[于谨]]、彭城公侯莫陈崇，谓之八柱国（163-096）。

侯莫陈崇以勇猛善战著称，在平定万俟丑奴之乱中一战成名。后从[[宇文泰]]征战，屡立战功，封柱国大将军。北周初年，侯莫陈崇为[[宇文护]]所忌，以言事忤旨被逼令自杀。侯莫陈氏为代北武川[[豪族]]，子弟多列北周名将。
