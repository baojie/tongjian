# 借商

**借商**指向商人强制借贷，为古代[[朝廷]]财政困难时的非常手段。

## 借商供军

[[德宗|唐德宗]]时，诏借商人钱，令[[度支]]条上。[[杜佑]][[大索]][[长安]]中[[商贾]]所有货，意其不实辄加搒捶（227-052）。借商之政，扰民最甚。

[[康承训]]讨[[庞勋]]，浣借商人钱五万缗以助军（253-049）。借商以充军费，不得已而为之。
