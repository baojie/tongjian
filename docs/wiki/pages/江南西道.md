# [[江南西道]]

[[江南西道]]是[[唐代]]十五道之一。玄宗开元二十一年（733年）分天下为十五道，江南道分为[[江南东道]]与江南西道。江南西道辖境约当今[[江西]]、[[湖南]]大部及[[湖北]]东南部，治所在[[洪州]]（今[[南昌]]）。

[[安史之乱]]中，[[永王璘]]充山南东道、岭南、黔中、江南西道节度都使（218-030）。[[唐代]]后期，江南西道的[[洪州]]为南方军事重镇，[[曹王皋]]曾任江南西道节度使（227-073）。
