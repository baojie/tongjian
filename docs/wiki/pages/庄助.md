# 庄助

庄助，会稽人，汉武帝时中大夫，亦称严助。

武帝初立，庄助以贤良对策，擢为中大夫（017-011）。武帝问伐闽越事，庄助力主兴兵（017-035）。庄助与硃买臣、吾丘寿王、司马相如等并在左右，每令与大臣辩论（017-037）。武帝遣庄助谕意南粤，南粤王称谢（017-065）。汲黯病，庄助为之请告（017-068）。

## 世系

庄助（？—约前122年），亦作[[严助]]，[[西汉]]会稽人。父[[严忌]]（庄忌），[[辞赋家]]。助以[[贤良]]对策擢为[[中大夫]]（017-011），与[[硃买臣]]、[[司马相如]]等并在左右（017-037）。后因与[[淮南王]][[刘安]]交通被诛。助[[子孙]]不详于史。
