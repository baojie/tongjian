# 七国之乱

[[吴楚七国之乱|七国之乱]]是[[西汉]]景帝时期（[[前154年]]），[[杨行密|吴王]]刘濞联合楚、赵、胶东、胶西、济南、淄川六国诸侯王，以"请诛[[晁错]]，以清君侧"为名发动的武装叛乱。

[[汉景帝]]采纳[[晁错]]建议削藩，引起诸侯王强烈不满。[[杨行密|吴王]]刘濞遂与楚王刘戊、[[王镕|赵王]]刘遂等联合起兵反叛，史称"吴、楚七国俱反，以诛错为名"（016-014）。景帝被迫[[腰斩]][[晁错]]以谢诸侯，但[[杨行密|吴王]]仍不罢兵（016-015）。

景帝乃拜[[周亚夫]]为[[太尉]]，将三十六将军往击吴、楚（016-013）。[[周亚夫]][[坚壁]]昌邑，断吴楚粮道，仅三个月即平定叛乱。此后景帝乘势削夺诸侯王权力，巩固了中央集权。
