# 军制

军制是军队编制与指挥体系的总称，为古代军事制度的基础。

[[荀子|荀卿]]与临武君论兵于赵孝成王前，临武君问王者之军制，[[荀子|荀卿]]曰"将死鼓，御死辔，百吏死职，[[士大夫]]死行列"（006-007）。[[汉文帝]]遣[[冯唐]]持节赦魏尚，[[冯唐]]引上古王者遣将之礼曰"阃以外者，将军制之"（015-036）。

古代军制以[[周礼]]为源，[[六军|天子六军]]、[[诸侯]][[三军]]。[[战国]]后军制大变，秦设[[郡县]]兵，汉置南北军，唐行[[府兵]]、[[禁军]]之制。军制之演变反映中央与地方军事权力之消长。
