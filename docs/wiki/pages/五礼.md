# 五礼

五礼是[[中国]]古代五种礼仪的合称，包括：**吉礼**（[[祭祀]]天神地祇）、**凶礼**（[[丧葬]]哀悼）、**军礼**（军旅[[征伐]]）、**宾礼**（[[朝聘]][[会盟]]）、**嘉礼**（婚冠宴飨），是[[周]]代[[礼_(概念)|礼]]制的核心框架。

## 历代修礼

历代王朝均重视五礼修纂。[[南朝齐]]时"诏选[[学士]]十人修五礼，[[丹阳]]尹[[王俭]]总之"（147-088）。[[隋文帝]]命[[牛弘]]修五礼，"勒成百卷"（176-033）。[[隋炀帝]]复诏[[杨素]]、[[苏威]]与[[牛弘]]等"修定五礼"（179-067）。[[唐高宗]]显庆年间所修新礼，"五礼并依《[[周礼]]》行事"（202-096）。[[唐玄宗]]时[[张说]]奏称"[[贞观之治|贞观]]、显庆两曾修纂，前后颇有不同"（213-004）。

## 历史意义

五礼体系贯穿[[中国]]古代礼制史，历代王朝通过修定五礼来确立统治秩序，是中华[[礼乐]]文明的重要载体。
