# 虞潭

虞潭，东晋宗正卿、会稽内史。从平赵王伦之乱，讨王敦，起兵讨苏峻。

主簿淮南赵诱、前秀才虞潭皆曰："赵王篡逆，海内所疾；今义兵四起，其败必矣。"（084-015）。王导、周顗、郭逸、虞潭等三道出战（092-023）。宗正卿虞潭以疾归会稽，闻苏峻反，起兵馀姚以讨充，帝以潭领会稽内史（093-019）。诏以会稽内史王舒行扬州刺史事，吴兴太守虞潭督三吴等诸郡军事（093-087）。

> 宗正卿虞潭以疾归会稽，闻之，起兵馀姚以讨充，帝以潭领会稽内史。
