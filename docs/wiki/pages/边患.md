# 边患

**边患**指[[边境]]遭受外来侵扰的祸患，为古代[[中原]]王朝面临的主要[[军事]]威胁之一。

## 匈奴边患

使[[陆贾]]即授[[玺绶]]，与[[剖符]][[通使]]，使和集[[百越]]，无为南边患害（012-037）。时[[匈奴]]数为边患，[[太子]][[家令]][[颍川]][[晁错]]上言兵事（015-007）。[[鲜卑]]外失暴掠之愿，内无[[功劳]]之赏，[[豺狼]]贪婪，必为边患（047-051）。轲比能部落近塞，[[中国]]人多亡叛归之，道远，故不为边患（069-056）。幽、并二州皆与[[鲜卑]]接，东有务桓，西有力微，多为边患（080-074）。
