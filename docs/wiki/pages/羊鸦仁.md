# 羊鸦仁

羊鸦仁，[[南朝]]梁[[司州刺史]]，[[侯景之乱]]时奉命接[[范雎|应侯]]景。

诏广晋[[县侯]][[泰山]]羊鸦仁等将兵应接[[羊侃]]（152-047）。遣[[司州刺史]]羊鸦仁督将兵三万趣[[悬瓠]]，[[运粮]]食应接[[侯景]]（160-016）。[[侯景]]以羊鸦仁等军未至，[[退保]][[颍川]]（160-021）。羊鸦仁遣[[长史]]邓鸿将兵至[[汝水]]（160-027）。羊鸦仁入[[悬瓠]]城，诏以鸦仁为司、豫二[[州刺史]]（160-034）。羊鸦仁以[[东魏]]军逼，弃[[悬瓠]]还义阳（161-005）。[[侯景]]邀羊鸦仁同反，鸦仁执其使以闻（161-029）。羊鸦仁[[合军]]屯新林王游苑（161-069）。

## 世系

羊鸦仁（？—约548年），[[南朝梁]][[司州]][[刺史]]，[[泰山]]人。其先世不详于史。鸦仁奉命应接[[侯景]]（160-016），入[[悬瓠]]城（160-034）。[[侯景]]邀鸦仁同反，鸦仁执其使以闻（161-029）。鸦仁[[子孙]]不详于史。
