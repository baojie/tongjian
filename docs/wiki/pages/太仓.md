# 太仓

太仓，京师储存粮食的国家大仓，关系朝廷命脉。

齐太仓令淳于意有罪当刑（015-031）。汉文景时太仓之粟陈陈相因（016-122）。[[汉武帝]]太仓甘泉仓满（020-089）。

太仓为秦汉时期国家粮食储备之核心。汉文景之世，太仓之粟陈陈相因，充溢露积于外，至腐败不可食，此为治世之征。武帝时，[[桑弘羊]][[均输平准]]，山东漕粟岁六百万石，太仓甘泉仓满。太仓之盈虚，关乎朝廷之盛衰。汉置太仓令，掌收受[[漕运]]粮食，供百官廪禄及军国之需。
