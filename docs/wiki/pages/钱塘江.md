# [[钱塘江]]

[[钱塘江]]古称[[浙江]]、之江，发源于[[安徽]][[黄山]]，流经[[浙江]][[杭州]]湾入[[东海]]，是[[中国]]东南部最大的河流之一。[[秦]][[汉]]时期，钱唐县即因江得名，[[唐代]]改钱塘。

钱塘江以壮观的钱塘潮闻名，[[八月十八]]潮为天下奇观。在《[[资治通鉴概述|资治通鉴]]》中，钱塘江流域是[[江东]]和[[两浙]]地区的核心地带，[[杭州]]在[[隋唐]]时期因钱塘江水运之便而迅速崛起。
