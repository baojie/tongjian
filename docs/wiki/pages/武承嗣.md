# 武承嗣

武承嗣，[[晋阳|并州]]文水人。[[武后|武则天]]侄，封魏王。

[[太后]]使[[秘书]]监武承嗣赍[[玺书]]慰谕[[刘仁轨]]，仁轨以[[吕后]]祸败之事申规戒（203-057）。[[太后]]御武成殿，皇帝帅王公以下[[上尊号]]。丁卯，[[太后]]临轩，遣礼部[[尚书]]武承嗣册嗣皇帝（203-059）。以礼部[[尚书]]武承嗣为[[太常]]卿、同中书门下三品（203-067）。承嗣请立为[[太子]]，未果，郁郁而卒。

> 太后临轩，遣礼部尚书武承嗣册嗣皇帝。自是太后常御紫宸殿，施惨紫帐以视朝。
> （203-059）
