# 沈恪

沈恪，字子恭，吴兴武康人。[[南朝]]陈吴兴太守。

[[陈霸先]]使中书舍人刘师知引宣猛将军沈恪勒兵入宫，卫送梁主如别宫，恪排闼见王，叩头谢曰："恪身经事萧氏，今日不忍见此。"（167-040）高州[[刺史]]黄法氍、吴兴太守沈恪、宁州[[刺史]][[周敷]]合兵救[[周迪]]（167-083）。上遣左卫将军沈恪代[[留异]]，恪与战而败，退还钱塘（168-108）。
