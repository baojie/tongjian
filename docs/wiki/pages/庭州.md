# 庭州

庭州，[[唐太宗]]置，以可汗浮图城为庭州（195-090），与[[西州]]同属[[安西都护府]]。[[阿史那贺鲁]]率众内属，处之于庭州莫贺城（199-006）。贺鲁招集离散，谋袭取西、庭二州，庭州刺史骆弘义知其谋表言之（199-080）。

> 九月，以其地为西州，以可汗浮图城为庭州，各置属县。
> （195-090）

庭州地处天山以北，[[唐]]置[[北庭都护府]]于此，与[[安西都护府]]（治[[龟兹]]）分治天山南北。[[安史之乱]]后庭州陷于[[吐蕃]]。[[回鹘]]西迁后据其地，为[[高昌回鹘]]之北境。庭州为唐代经营西域的北方军政中心。
