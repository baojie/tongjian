# 桓冲

桓冲，字幼子，谯国龙亢人。东晋车骑将军，桓温之弟。淝水之战前镇守荆州，为东晋西藩屏障。

将军桓冲又败秦丞相雄于白鹿原。冲，温之弟也（099-109）。以征虏将军桓冲为江州刺史（101-068）。桓冲在荆州镇守，与谢玄、谢石等共御前秦。淝水之战前，冲请率军入卫京师，谢安以荆州为重，止之。冲叹曰："谢安乃有庙堂之量，然不闲将略。今大敌至，方游谈不暇，而遣诸不经事少年拒之，众又寡弱，天下事可知矣。"及淝水捷报至，冲惭病卒。
