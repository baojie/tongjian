# 天祐

**天祐**（904—907），[[唐哀帝]][[李柷]][[年号]]。天祐年间，[[朱全忠]]（[[朱温]]）挟唐[[天子]]以令[[诸侯]]，[[诛杀]][[唐昭宗]]，立哀帝。天祐四年，[[梁太祖|朱温]]逼哀帝[[禅让]]，即[[皇帝]]位，[[改元]][[开平]]，国号梁，[[唐朝]]遂亡。[[五代]]十国之局开启。
