# 辛庆忌

辛庆忌，字子真，狄道人。[[西汉]]左将军，以勇武显名，辛武贤之子。

辛庆忌为左将军，以勇武称（031-045）。辛庆忌为国虎臣，[[匈奴]]惮之（032-021）。

辛庆忌少有勇略，以父荫为右校丞。随[[常惠]][[屯田]][[乌孙]]，以功拜侍郎。元帝时，累迁张掖太守，徙酒泉。所在以威武称，[[匈奴]]不敢犯。成帝时，拜左将军。庆忌为国虎臣，[[匈奴]][[西域]]敬其威信。其性与人恭谨，饮食被服尤俭约，然临阵必先士卒。年六十馀卒。

> 辛庆忌为左将军，以勇武称。
> （031-045）
