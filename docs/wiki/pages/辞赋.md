# 辞赋

辞赋是汉代盛行的[[文学]]体裁，渊源于楚辞，以铺陈排比、辞藻华丽为特征。

[[司马相如]]特以辞赋得幸，[[东方朔]]、枚皋不根持论好诙谐，上以俳优畜之（017-037）。汉顺帝时群臣奏言书画辞赋才之小者，匡国治政未有其能（057-066）。

汉代辞赋大家辈出，[[司马相如]]、[[扬雄]]、[[班固]]、[[张衡]]为四大家。赋体以铺张扬厉、体物写志为能事，内容涵盖都城、田猎、山川、物产等。汉赋与唐诗、宋词、元曲并称，代表一代[[文学]]之盛。

> 相如特以辞赋得幸；朔、皋不根持论，好诙谐，上以俳优畜之。
> （017-037）
