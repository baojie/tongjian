# 高阿那肱

**高阿那肱**，[[北齐]][[权臣]]，以谄媚[[刘禅|后主]]得宠。

高阿那肱以[[都督]]帅骑数千击柔然（165-053），虽兵少犹奋击大破之。[[刘禅|后主]]武平时与娄定远、[[赵彦深]]、[[唐邕]]等俱用事，时号"八贵"（170-091）。

高阿那肱为[[高纬|北齐后主]][[高纬]]的宠臣之一，八贵当政加速了[[北齐]]的腐败与灭亡。[[北周]]伐齐，阿那肱居[[晋阳]]不战，致使[[北齐]]军溃。阿那肱降周，后以[[谋反]]诛。

## 八贵用事

高阿那肱以谄佞得幸，与娄定远、[[赵彦深]]、[[唐邕]]等俱用事，时号"八贵"（170-091）。[[刘禅|后主]]年少，阿那肱以谄佞为[[和士开]]所厚，由是有宠，累迁并省[[尚书令]]（170-096）。又与[[和士开]]皆为[[陆令萱]]养子（170-098）。后[[录尚书事]]，总知外兵及内省机密，与[[穆提婆]]、韩长鸾共处衡轴（171-062），进为[[司徒]]（171-090）。宰制[[朝政]]（172-011），[[北齐]]由是日坏。[[北周]]伐齐，阿那肱居[[晋阳]]不战，致使齐军[[溃败]]，遂降周，后以[[谋反]]诛。
