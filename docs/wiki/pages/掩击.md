# 掩击

**掩击**指乘敌不备发起突然袭击，为古代[[军事]]中以隐蔽行动和快速机动打敌措手不及的重要战术手段。

## 出塞掩击

训因发湟中秦、胡、羌兵四千人出塞，掩击迷唐于写谷，破之（047-053）。[[任尚]]将之，缝革为船，置于箄上以渡河，掩击迷唐，大破之（047-056）。[[窦宪]]遣副[[校尉]]阎盘将二千馀骑掩击[[北匈奴]]之守伊吾者，复取其地（047-069）。数[[遣将]]兵出塞，掩击[[北庭]]（048-036）。诩计贼当退，乃潜遣五百馀人于浅水[[设伏]]，候其走路；虏果大奔，因掩击，大破之（049-151）。
