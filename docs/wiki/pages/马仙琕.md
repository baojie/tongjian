# 马仙琕

马仙琕，梁名将。初不附[[萧衍]]，后为梁将，以忠勇著称。

[[萧衍]]东下，豫州[[刺史]]马仙琕拥兵不附衍。衍使其故人姚仲宾说之，仙琕先为设酒，乃斩于军门以徇。衍又遣其族叔怀远往说之，仙琕亦欲斩之（144-072）。后仙琕归梁，为将转战而前，[[兵势]]甚锐。[[元英]]结垒于士雅山，示之以弱，仙琕备战（145-111）。

> 豫州刺史马仙琕拥兵不附衍，衍使其故人姚仲宾说之，仙琕先为设酒，乃斩于军门以徇。
> （144-072）

> 马仙琕救义阳，仙琕转战而前，兵势甚锐。
