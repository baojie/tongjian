# 徐纥

徐纥，字武伯，[[北魏]]胡[[太后]]时[[中书]][[舍人]]。粗有[[文学]]，以谄事权贵得幸，与[[郑俨]]等专权乱政。

徐纥先以谄事[[赵修]]，坐徙[[枹罕]]；后还复除[[中书]][[舍人]]，又谄事[[清河]]王怿；怿死，又谄事[[元叉]]（150-075）。纥与[[城阳]]王徽共毁[[侍中]][[元顺]]于[[太后]]（151-012）。[[尔朱荣]]起兵，徐纥说[[太后]]以[[铁券]]间荣左右，荣闻而恨之（152-016）。[[太后]]召王公议，纥独言[[尔朱荣]]不足惧（152-021）。

及[[尔朱荣]]逼近[[洛阳]]，徐纥矫诏夜开殿门，取厩马十匹，东奔兖州（152-023）。纥遣人告其弟献伯、季产，皆将家属逃去，俱奔[[泰山]]（152-029）。后往依[[泰山]][[太守]][[羊侃]]，劝侃起兵（152-047）。

> 中书舍人乐安徐纥，粗有文学，先以谄事赵修，坐徙枹罕。后还，复除中书舍人，又谄事清河王怿。
> （150-075）
