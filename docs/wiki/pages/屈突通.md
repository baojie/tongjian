# 屈突通

屈突通（？—约628年），[[隋]]末[[唐]]初将领。隋右卫[[大将军]]，为人刚直。检覆群牧，得隐匿马二万馀匹，[[隋炀帝|炀帝]]大怒，将斩太仆卿等千五百人，通谏曰一人死不可复生，况千五百人乎，帝悟，悉免之（178-070）。后奉[[隋炀帝|炀帝]][[玺书]]征[[汉王谅]]（180-012）。唐兵入[[关中]]，屈突通战败被擒，归降[[李渊]]，为唐初名将，凌烟阁功臣之一。

## 归唐

屈突通为隋骁卫[[大将军]]，守[[河东]]。唐兵至，通固守不下（182-029）。后战败，被擒送[[长安]]，[[李渊]]释之。以通为兵部[[尚书]]、蒋国公，慰劳之（182-034）。通由此归唐，为[[唐太宗|秦王世民]]府僚佐，从讨薛仁杲（182-035）。
