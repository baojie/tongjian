# 王僧虔

王僧虔，[[琅邪]]临沂人。南齐书[[法家]]，[[王羲之]]族孙，以书法名世。

王僧虔为南齐[[侍中]]，善书（134-026）。王僧虔论书，为时所重（136-066）。

王僧虔为[[王导]]之裔，[[王羲之]]族孙，少善隶书。[[刘义隆|宋文帝]]见其书，叹曰："非惟迹逾子敬，方当器雅过之。"仕宋为[[尚书令]]。入齐，为[[侍中]]。齐太祖与僧虔赌书，问曰："谁为第一？"对曰："臣书臣中第一，[[陛下]]书帝中第一。"太祖大笑。僧虔著有《笔意赞》，为书学名篇。

> 王僧虔为南齐侍中，善书。
> （134-026）
