# 节钺

节钺是天子授予将帅的专征之权象征，节为符节、钺为斧钺，持节钺者可代天子征伐。汉文帝遣[[周亚夫]]平七国之乱，"拜亚夫为太尉，将三十六将军"，授以节钺（062-008）。

[[诸葛亮]]出师北伐，后主"假亮节钺"，授予征伐全权（064-027）。[[司马懿]]讨辽东，[[曹叡]]亦假节钺（074-024）。节钺的授予和收还在古代政治中具有重大象征意义，代表皇权对军事权力的委托与限制。
