# 赵道德

赵道德，北齐直长、都督，以直言敢谏著称。

直长赵道德以故犊车一乘候于东阁，齐主登车，道德超上抱之，帝叱之，道德曰："何物奴敢逼人！"（163-034）。赵道德以事属黎阳太守房超，超不发书，棓杀其使，齐主善之（163-062）。赵道德进谏："东西两国，强弱力均，彼可擒之而来，此亦可擒之以往。桃枝妄言应诛。"（166-081）。赵道德曰："相王不效周公辅成王，而欲骨肉相夺，不畏后世谓之篡邪？"（168-052）。
