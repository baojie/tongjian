# 丧葬

丧葬，办理丧事安葬死者的礼仪制度，为古代礼制之重要组成部分。

[[王导]]丧葬之礼参用天子礼（096-053）。燕以成哀段后丧葬归龙城（108-106）。

古代丧葬之礼，尊卑有序，天子至庶人各有等差。东晋名相[[王导]]薨，丧葬之礼视汉博陆侯[[霍光]]故事，参用天子之礼，为殊典。北燕[[冯跋]]以章武王宙奉燕垂及段后丧葬于龙城宣平陵。丧葬之厚薄，关乎孝道名节，历代朝廷皆以礼法约束之，然厚葬之风屡禁不止。

> 王导丧葬之礼参用天子礼。
> （096-053）
