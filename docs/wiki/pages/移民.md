# 移民

移民在[[《资治通鉴》]]中多指因灾荒将百姓迁移至丰饶地区就食，以缓解灾情。与[[徙民]]不同，移民更强调因生存需要的人口流动。

[[南北朝]]刘宋时，[[刘义恭|江夏王]][[刘义恭]]议欲芟麦翦苗，移民堡聚，以御[[北魏]]入侵（126-021）。[[北魏]][[高允]]上疏陈汉置[[常平仓]]之制，指出京师粮食不足时移民就丰，既废营生，又于国体有损，建议预储仓粟以安百姓（136-105）。

> 江夏王义恭议欲芟麦翦苗，移民堡聚。
> （126-021）

> 去岁京师不稔，移民就丰，既废营生，困而后达，又于国体，实有虚损。
> （136-105）
