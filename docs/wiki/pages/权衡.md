# 权衡

权衡指称量轻重的器具及标准，历代朝廷常颁定权衡以统一度量。

[[北魏]][[拓跋珪]]命有司平权衡，审度量（110-044）。西魏[[宇文泰]]更权衡度量，命[[苏绰]]损益三十六条之制，颁行之（158-142）。北周武帝议定权衡度量，颁之于四方（173-032）。[[隋文帝]]改度量权衡，并依古式（180-073）。

权衡统一为古代治国要务，度量衡之整齐划一关系赋税征收与商贸公平。[[北魏]]、西魏、北周、隋各朝皆以颁定权衡为整顿政务之始。

> 魏王珪命有司正封畿，标道里，平权衡，审度量；遣使循行郡国，举奏守宰不法者，亲考察黜陟之。
> （110-044）
