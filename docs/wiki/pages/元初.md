# 元初

**元初**（114—120），[[汉安帝]]第二个[[年号]]。元初年间，[[西羌]]连年叛乱，[[邓骘]]率诸将征讨不能克，军费日增。[[鲜卑]]犯塞，[[度辽将军]]击破之。[[刘珍]]等奉诏校定[[东观]][[五经]]、诸子、传记。[[虞诩]]以鲠直著称，不畏权贵。
