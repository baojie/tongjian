# 践阼

践阼，登基[[即位]]。[[王莽]]自以居摄践阼，为功显君服[[天子]]吊[[诸侯]]之服（036-063）。[[刘曜]]谏[[刘曜]]谓[[陛下]]践阼以来已作殿观四十馀所（088-055）。[[庾冰]]以父子相传不让弟，使[[司马睿|琅邪王]]践阼（097-008）。郑遘谓[[唐僖宗]]主上践阼十馀年无大过恶（256-073）。[[唐僖宗]]践阼之始，中外忻忻（257-075）。

> 陛下践阼以来，已作殿观四十馀所，加之军旅数兴，餽运不息，死亡相继。
> （088-055）
