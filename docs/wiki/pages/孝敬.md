# 孝敬

**孝敬**指[[孝顺]]父母、敬爱尊长，是[[儒家]]伦理的核心观念。《[[资治通鉴概述|通鉴]]》在记述史事的同时，也记载了许多孝道故事。

## 汉代

[[汉明帝]]以孝敬著称，[[即位]]后仍恪守子道，朝[[太后]]如仪（056-051）。[[东汉]]诸帝多倡孝道，举[[孝廉]]为选官重要科目。[[汉顺帝]]时，[[张衡]]上疏论孝道[[治国]]之理（065-039）。
