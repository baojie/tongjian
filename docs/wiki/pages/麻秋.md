# 麻秋

麻秋，后赵[[石虎]]麾下大将，以勇猛著称。率军攻[[前凉]]，为[[谢艾]]以少胜多击败。

虎遣将军麻秋讨蒲洪，洪帅户二万降于虎（095-031）。虎遣将军郭太、麻秋帅轻骑二万追辽，至密云山，获其母妻（096-006）。[[王镕|赵王]]虎遣征东将军麻秋帅众三万迎之（096-032）。

麻秋为[[石虎]]帐下骁将，然在与[[前凉]]的战争中屡为[[谢艾]]所破。麻秋之败，不仅因为[[谢艾]]用兵之妙，亦因[[石虎]]连年征伐，民力已疲。麻秋后降于[[前凉]]，寻又谋叛，为[[张重华]]所杀。

> 赵王虎遣征东将军麻秋帅众三万迎之，敕秋曰："受降如受敌，不可轻也。"
> （096-032）
