# 卫绾

卫绾，[[西汉]][[丞相]]。事[[隋文帝|文帝]]、景帝、[[武帝]]三朝，以醇谨无他著称，奏罢申、韩、苏、张之言。

[[河间]]王[[太傅]]卫绾击吴、楚有功，拜为[[中尉]]（016-030）。以[[御史大夫]]卫绾为[[丞相]]（016-106）。[[丞相]]卫绾奏："所举[[贤良]]，或治申、韩、苏、张之言乱国政者，请皆罢"，奏可（017-011）。[[丞相]]卫绾免（017-014）。
