# 卫绾

卫绾，西汉丞相。事文帝、景帝、武帝三朝，以醇谨无他著称，奏罢申、韩、苏、张之言。

河间王太傅卫绾击吴、楚有功，拜为中尉（016-030）。以御史大夫卫绾为丞相（016-106）。丞相卫绾奏："所举贤良，或治申、韩、苏、张之言乱国政者，请皆罢"，奏可（017-011）。丞相卫绾免（017-014）。
