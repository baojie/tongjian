# 纬书

纬书是[[汉代]]附会[[儒家]]经书的[[谶纬]]类[[典籍]]，以神学迷信解释[[经义]]，与"经"相对，盛行于[[东汉]]。

唐初礼官引纬书《[[孝经]]钩命决》言[[封禅]]事，[[褚遂良]]等以为此纬书之曲说，非正经之通义（147-038）。[[唐玄宗]]时礼官奏废纬书六天之义，合方丘[[祭地]]为祀（200-046）。

纬书在[[西汉]]末至[[东汉]]极盛，[[光武帝刘秀|光武帝]]宣布[[图谶]]于[[天下]]，纬学遂为官学。至[[隋文帝]]焚禁纬书，唐时屡禁不绝，然渐趋衰微。

> 郑引《孝经钩命决》云："封于太山，考绩柴燎；禅乎梁甫，刻石纪号。"此纬书之曲说，非正经之通义也。
> （147-038）
