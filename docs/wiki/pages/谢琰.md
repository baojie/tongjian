# 谢琰

谢琰，字瑗度，陈郡阳夏人。东晋辅国将军，谢安之子。淝水之战中为前锋，建功立业。

以徐、兗二州刺史谢玄为前锋都督，与辅国将军谢琰、西中郎将桓伊等众共八万拒之。琰，安之子也（105-010）。谢琰劝石从朱序言，速击秦军（105-013）。

谢琰以谢安之子为淝水之战前锋，从谢玄、谢石大破前秦。琰有父风，为政严明，然性刚愎。后孙恩起义，琰以会稽内史拒之，战败被杀。谢氏三世名臣，谢安最著，谢琰继之，然以刚愎致败。
