# [[朔方郡]]

[[朔方郡]]是[[汉武帝]]元朔二年（前127年）置，治所在朔方城（今[[内蒙古]]杭锦旗北）。[[主父偃]]献策立朔方郡，认为其地"地肥饶，外阻河，蒙恬城之，以逐[[匈奴]]，内省转输戍漕，广中国，灭胡之本也"（018-066）。武帝从其计，使[[苏建]]兴十余万人筑朔方城。

朔方郡的设立是[[汉]]武帝北击匈奴战略的关键一步，使汉朝得以在[[河套]]地区立足，进而经略[[匈奴]]腹地。
