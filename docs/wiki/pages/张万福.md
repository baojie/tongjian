# 张万福

张万福，魏州[[元城]]人。唐金吾[[将军]]，中唐[[忠臣]]，以忠勇[[正直]]著称。

张万福为金吾[[将军]]，忠直敢言（224-078）。张万福谏德宗勿纵[[宦官]]（227-005）。张万福年九十，犹领[[宿卫]]（235-019）。

张万福少[[从军]]，以[[战功]]累迁金吾[[将军]]。德宗时，[[宦官]]专权，[[朝政]]日非。万福每[[廷争]]，言甚切直。帝虽不悦，亦不之罪。[[泾原兵变]]，德宗[[出奔]]奉天，万福率兵护驾，[[昼夜]]不懈。及乱平，万福以老[[乞骸骨]]，帝不许。年九十，犹领[[禁军]][[宿卫]]，时人谓之"老金吾"。卒年九十二。

## 张万福与中唐忠节

张万福以忠勇著称，尤以戡乱安民、稳定漕路为功。平卢[[行军]][[司马]]许杲将卒三千人驻濠州有窥[[淮南]]意，万福摄濠[[州刺史]]，许杲闻其威名即提卒去（224-078）。[[李希烈]]阻兵[[襄阳]]，[[江淮]][[进奉]]船千馀艘泊涡口不敢进，万福驰至立马岸上发[[进奉]]船，淄青将士不敢动（227-005）。德宗时[[谏官]]伏阁请对，万福闻之趋往延英门大言贺曰"[[朝廷]]有直臣，[[天下]]必[[太平]]矣"，遍拜[[谏官]]（235-019）。万福以武将而崇直臣，年九十而[[忠节]]不改，于中唐纷乱之世尤为难能可贵。
