# 退守

**退守**指军队在不利形势下退却以固守待机，保存实力、等待反攻的常见军事策略。

## 退守待援

刘备攻益州，刘璋将刘璝、张任与璋子循退守雒城，备进军围之（066-055）。退守城池，以逸待劳。

前赵刘曜攻陷长安外城，麹允、索綝退守小城以自固，内外断绝，城中饥甚（089-083）。退守孤城，势穷力竭。

张重华使将军牛旋御敌，退守枹罕，姑臧大震（097-099）。退守虽可自保，然军心易摇。

## 退守避锐

石虎将击徐龛，蔡豹退守下邳，为徐龛所败（091-053）。退守非怯，然退而不整，反为所乘。

石生攻郭诵于阳翟，诵与战，大破之，生退守康城（093-006）。战不利则退守，此用兵之常。

吕护退守小平津，中流矢而卒（101-052）。退守之际，主将安危最急。

苻丕被慕容垂围攻，退守中城（105-044）。坚壁退守，以待外援。
