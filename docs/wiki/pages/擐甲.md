# 擐甲

**擐甲**指穿上[[铠甲]]，为古代[[将士]][[出战]]、御驾亲征前的装备准备动作，亦形容处于战时[[戒备]]状态。

## 擐甲亲征

[[灵帝|汉灵帝]]躬擐甲介马称无[[上将军]]（059-018）。

## 擐甲备战

[[费祎]]督[[汉中]]时[[羽檄]]交至人马擐甲（074-067）。[[李荡]]母罗氏擐甲拒战（085-005）。[[司马睿]][[出师]]露次躬擐甲胄（089-095）。石遵擐甲曜兵入自凤阳门（098-029）。
