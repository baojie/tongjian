# 俸禄

俸禄是古代国家发给官吏的薪给，以粮食或钱币形式发放。[[《资治通鉴》]]中俸禄厚薄关乎吏治清浊。

[[西汉]]宣帝诏曰："今小吏皆勤事而俸禄薄，欲无侵渔百姓，难矣！其益吏百石已下俸十五。"（026-040）[[新朝]][[王莽]]以制作未定，上自公侯，下至小吏，皆不得俸禄（038-010）。天下吏以不得俸禄，并为奸利，郡尹、县宰家累千金（038-024）。
