# 阻兵

**阻兵**指依仗武力以对抗朝廷或他人。

## 阻兵仗势

司马光论曹操封魏公：若不明于此，而重焦烂之功，则民利于有乱，俗竞于杀伐，阻兵杖力，干戈不戢矣（067-029）。阻兵仗力，乱之源也。

孙权欲征公孙渊，陆逊谏曰：渊怀贰阻兵，然后致诛，于事为难，不如因其新立先其不意（071-019）。阻兵者难猝制，当乘其未固。

邓艾曰：吴名宗大族皆有部曲，阻兵仗势，足以违命（076-018）。阻兵仗势，吴之积弊。

## 阻兵相守

诸夷阻兵，不可以军惊扰，宜遣恩信吏宣教慰劳（078-031）。阻兵者可以恩信化之。

大晋兵力盛于往时，不于此际平壹四海，而更阻兵相守，使天下困于征戍（080-057）。阻兵相守，非长久之计。
