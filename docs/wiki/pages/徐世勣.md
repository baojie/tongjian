# 徐世勣

徐世勣，字懋功，离狐人。唐初名将，凌烟阁二十四功臣之一，赐姓李氏，后避讳改名为勣。

徐世勣年十七从翟让起兵（183-017）。徐世勣为李密右武候大将军（183-047）。

徐世勣家于卫南，年十七有勇略，说翟让勿侵掠东郡，让从之。后与李密合军，破张须陁，以功授右武候大将军。李密杀翟让时，世勣几被伤，密止之。后随李密归唐，太宗即位，拜并州都督。世勣历事高祖、太宗、高宗三朝，出将入相，累迁至尚书左仆射。高宗立，拜司空。卒谥贞武，配享高宗庙庭。

> 徐世勣年十七从翟让起兵。
> （183-017）
