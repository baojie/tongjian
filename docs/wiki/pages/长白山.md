# [[长白山]]

[[长白山]]位于[[中国]]东北部，是[[辽东]]与[[高句丽]]之间的界山，[[隋]][[唐]]时期称徙太山、太白山。[[隋炀帝]]征[[高句丽]]时，[[来护儿]]率水军经长白山东部海域进兵（181-001）。[[唐代]]灭高句丽后，长白山地区纳入[[安东都护府]]辖境。

长白山也是[[中国]]满族的发祥地，[[清代]]被尊为"龙脉"所在，长期封禁保护。
