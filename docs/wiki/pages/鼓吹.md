# 鼓吹

鼓吹是古代[[军乐]]和[[仪仗]][[音乐]]，以打击[[乐器]]和吹管[[乐器]]为主，用于[[朝会]]、出行、军旅等场合。

[[东汉]][[窦宪]]被收[[印绶]]后，上还所受[[虎贲]]、官骑及诸工技、鼓吹等（048-034）。安帝诏[[太仆]]、[[少府]]减黄门鼓吹以补[[羽林]]士（049-045）。[[曹丕|魏文帝]]使夏侯泰以兵马道从，鸣[[鼓角]]作鼓吹而出（068-003）。

[[西晋]][[刘毅]]为[[司隶]]，纠绳豪贵，皇[[太子]]鼓吹入东掖门，毅[[劾奏]]之（081-047）。[[后赵]][[石虎]]以女骑千人为[[卤簿]]，鸣鼓吹游宴自随（095-102）。[[宋顺帝]]禅位时问"今日何不奏鼓吹"，左右莫应（135-017）。[[萧衍|梁武帝]]诏舆驾始出鼓吹从而不作，还宫如常仪（147-036）。鼓吹制度贯穿魏晋[[南北朝]]至隋唐。

> 毅为司隶，纠绳豪贵，无所顾忌。皇太子鼓吹入东掖门，毅劾奏之。
> （081-047）

## 鼓吹之用途

鼓吹按用途分为[[黄门鼓吹]]、[[骑吹]]、[[横吹]]、[[短箫铙歌]]等。黄门鼓吹用于[[天子]][[朝会]][[仪仗]]（049-045），骑吹用于出行，横吹用于军中[[行进]]，短箫铙歌用于[[凯旋]]。[[窦宪]]收缴鼓吹，示其去势（048-034）。[[曹丕]]以兵马道从作鼓吹以镇服夏侯盛等（068-003）。
