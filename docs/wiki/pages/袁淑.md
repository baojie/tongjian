# 袁淑

袁淑，字阳源，陈郡阳夏人。南朝宋御史中丞。

御史中丞袁淑言于宋文帝曰："陛下今当席卷赵、魏，检玉岱宗；臣逢千载之会，愿上封禅书。"（125-043）太子刘劭将弑文帝，呼袁淑甚急，淑眠不起。劭命登车，辞不上，为劭所杀（127-013）。孝武帝追赠袁淑为太尉，谥忠宪公（127-047）。淑以忠义死节。

> 太子刘劭呼袁淑甚急，淑眠不起。劭停车奉化门催之相续。淑徐起，至车后；劭使登车，又辞不上。
> （127-013）
