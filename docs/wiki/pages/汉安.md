# 汉安

**汉安**（142—144），[[汉顺帝]]第三个[[年号]]。汉安年间，顺帝遣[[张纲]]、[[周举]]等八使分行州郡，巡行风俗，劾奏贪官，时号"八俊"。[[广陵]]贼[[张婴]]据郡反，张纲单车往谕，婴降。[[梁冀]]封侯，权势渐盛。[[匈奴]]、[[鲜卑]]入寇。
