# 太医

太医是宫廷[[医官]]总称，掌皇帝及后宫[[疾病]]诊治。[[汉武帝]]时[[上官桀]]妻父充国为太医监，"阑入殿中，下狱当死"（023-045）。[[汉成帝]]时许[[皇后]]被[[霍氏]]指使女医淳于衍以附子毒杀，淳于衍"取附子并合太医大丸以饮[[皇后]]"（024-049），太医大丸为宫廷御制药剂。

[[王莽]]捕得翟义党王孙庆，使"太医、尚方与巧屠共刳剥之"，以探究五脏脉理（038-015）。[[东汉]][[东海]]恭王刘强病重，[[汉明帝|明帝]]遣"[[使者]]太医乘驿视疾"（044-090）。桓荣[[病笃]]，[[汉明帝|明帝]]遣"太官、太医相望于道"问疾（044-098），太医为帝[[王尊]]崇儒学之表现。
