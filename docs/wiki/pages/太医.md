# 太医

太医是宫廷医官总称，掌皇帝及后宫疾病诊治。汉武帝时上官桀妻父充国为太医监，"阑入殿中，下狱当死"（023-045）。汉成帝时许皇后被霍氏指使女医淳于衍以附子毒杀，淳于衍"取附子并合太医大丸以饮皇后"（024-049），太医大丸为宫廷御制药剂。

王莽捕得翟义党王孙庆，使"太医、尚方与巧屠共刳剥之"，以探究五脏脉理（038-015）。东汉东海恭王刘强病重，明帝遣"使者太医乘驿视疾"（044-090）。桓荣病笃，明帝遣"太官、太医相望于道"问疾（044-098），太医为帝王尊崇儒学之表现。
