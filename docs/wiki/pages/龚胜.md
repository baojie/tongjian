# 龚胜

**龚胜**，字君宾，[[西汉]]光禄大夫，以直谏著称。

龚胜为光禄大夫，与傅喜等固争[[鲍宣]]事（033-051）。胜为司直，郡国皆慎选举（034-044）。[[王莽]]秉政，龚胜与邳汉、卓茂等并[[辞官]]归隐。

[[王莽]]篡汉后，[[遣使]]征龚胜，欲以为[[太子]]师友。龚胜拒不受命，绝食十四日而死，年七十九。胜临终语门人："吾受汉家厚恩，无以报，今老矣，旦暮入地，谊岂以一身事二姓哉！"龚胜为[[西汉]]末忠义之士的典型代表。

## 不事二姓

龚胜为光禄大夫，与傅喜等固争[[鲍宣]]事（033-051）。[[王莽]]秉政，龚胜以莽专政，与邴汉皆[[乞骸骨]]归（035-054）。莽篡汉后，[[遣使]]者奉玺书、安车、驷马迎胜，拜为师友[[祭酒]]。[[使者]]与郡太守、三老、行义、诸生千人以上入胜里致诏，胜称病笃，拒不受命（037-040）。绝食十四日而死，年七十九。
