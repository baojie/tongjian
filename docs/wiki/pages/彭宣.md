# 彭宣

彭宣，字子佩，淮阳阳夏人。[[西汉]]右将军，[[王莽]]时不仕，以清节终。

彭宣为[[大司农]]，以经学称（033-026）。彭宣以不附[[王莽]]，乞骸骨归（033-034）。

彭宣少师事张禹，受《易》《论语》。举[[博士]]，为东平王[[太傅]]。成帝时，累迁[[大司农]]、光禄勋。哀帝时，拜右将军。宣为人恭谨，以经学自守。[[王莽]]秉政，宣[[上书]]乞骸骨，曰："年齿衰朽，不堪任职。"莽虽不悦，而许之。宣归乡里，以寿终。宣当王氏擅权之时，独能引退，世以此高之。

> 彭宣为大司农，以经学称。
> （033-026）
