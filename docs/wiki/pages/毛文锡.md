# 毛文锡

毛文锡，[[前蜀]]翰林[[学士]]承旨、礼部[[尚书]]判枢密院。以[[文学]]受知于[[王建]]。

[[太子]][[宴饮]]，集王宗翰、内枢密使潘峭、翰林[[学士]]承旨高阳毛文锡不至，[[太子]]怒（268-121）。[[太子]]捕潘峭、毛文锡至，拷之几死（268-122）。以内枢密使潘峭为武泰[[节度使]]，翰林[[学士]]承旨毛文锡为礼部[[尚书]]判枢密院（269-019）。毛文锡与飞龙使唐文扆争权（270-005）。

> 太子怒曰："集王不来，必峭与文锡离间也。"
> （268-121）

> 翰林学士承旨毛文锡为礼部尚书，判枢密院。
> （269-019）
