# 王世积

**王世积**，阆熙人，隋上柱国。

[[隋灭陈]]之役，王世积以蕲州[[刺史]]出蕲春，率[[舟师]]出[[九江]]，破陈将纪瑱于蕲口，陈人大骇，降者相继（176-120、177-010）。[[移书]]告谕[[江南]]诸郡，[[豫章]]等诸郡[[太守]]皆诣世积降（177-022）。后与[[周法尚]]讨平[[桂州]]俚帅[[李光仕]]之乱（178-057）。与[[汉高祖刘邦|汉王]]谅并为[[行军]][[元帅]]伐[[高丽]]（178-083）。

王世积为[[隋灭陈]]的重要将领，在[[江淮]]战场[[战功]]显赫，后因事被诛。

## 凉州被诛

王世积以功拜凉州总管。其亲信安定皇甫孝谐有罪，吏捕之，亡抵世积，世积不纳。孝谐配防桂州，遂告世积谋反（178-099）。及王世积被诛，推核之际有宫禁中事，云于高颎处得之，文帝大惊（178-105）。有司请杀颎，文帝曰："去年杀虞庆则，今兹斩王世积，如更诛颎，天下其谓我何！"乃除名为民（178-107）。王世积以平陈功臣而终不免于诛，为隋初猜防诛戮功臣之缩影。
