# 本初

**本初**（146），[[汉质帝]][[年号]]，仅用一年。质帝年八岁即位，[[梁太后]]临朝，[[梁冀]]秉政。质帝聪慧，目梁冀曰"跋扈将军"，梁冀惮之，遂进毒饼弑帝。[[李固]]、[[杜乔]]等议立[[清河王]][[刘蒜]]，梁冀与[[曹腾]]定策立[[刘志]]，是为[[汉桓帝]]。

> （053-002）
