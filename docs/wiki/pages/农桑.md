# 农桑

农桑是农耕与蚕桑的总称，为古代经济的根本。[[贾谊]]上疏[[汉文帝]]，强调"驱民而归之农，皆著于本"，提出"务民于农桑，薄[[赋敛]]，广畜积，以实[[仓廪]]，备水旱"的治国方针（015-022）。[[汉景帝]][[诏令]]郡国"务[[劝农]]桑，益种树"，以解决"末者众，农民寡"的问题（016-117）。

[[汉武帝]]欲扩[[上林苑]]，[[东方朔]]谏曰"绝陂池水泽之利而取民膏腴之地，上乏国家之用，下夺农桑之业"（017-039）。[[龚遂]]为[[渤海]]太守，见齐俗奢侈好末技，"乃躬率以俭约，劝民[[务农]]桑"，使民卖剑买牛、卖刀买犊（025-034）。[[鲍宣]][[上书]]论"七亡"，其中之一即为"苛吏繇役，失农桑时"（034-042）。

## 劝课农桑

历代朝廷以劝课农桑为守令考课之要。[[王莽]]置大司农部丞十三人分驻各州劝农桑（035-046），又遣十一公士分布劝农桑班时令，然徒增纷扰而无实效（038-006）。[[汉明帝]]诏二千石勉劝农桑，罪非殊死须秋案验（046-003），以农事为行政之首。[[光武帝刘秀|光武帝]]将冯异讨关中，敕其散小民令就农桑（040-074）。[[汉明帝|明帝]]以公车受章奏有反支日之忌，帝曰民废农桑远来诣阙而复拘以禁忌岂为政之意哉，遂蠲其制（045-089）。劝课农桑之政，上自诏令、下至守令，贯穿两汉，为古代重农主义之核心。

## 农桑与治乱

农桑之兴废，系乎天下之治乱。[[汉末]]鲍宣论七亡，苛吏繇役失农桑时居其一（034-042）。[[三国]]时[[刘虞]]牧幽州，劝督农桑开胡市，民悦年登，青徐士庶避难归虞者百馀万口（059-077）。[[诸葛亮]]治蜀，劝农讲武、息民休士（072-048），农桑不失其时，蜀中富庶。农桑之业，治世则兴，乱世则废，实为天下盛衰之寒暑表。
