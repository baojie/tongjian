# 建兴

**建兴**（313—317），[[晋愍帝]][[司马邺]][[年号]]。建兴年间，[[西晋]]已[[名存实亡]]，愍帝因守[[长安]]，[[刘曜]]屡攻[[长安]]。[[建兴]]四年，[[刘曜]]陷[[长安]]，愍帝出降，被俘至[[平阳]]，[[西晋]]亡。[[琅琊王]][[司马睿]]在[[建康]]称[[晋王]]，[[改元]][[建武]]，[[东晋]]始。

> （089-002）
