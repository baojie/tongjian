# 绥和

**绥和**（前8—前7），[[汉成帝]]最后一个[[年号]]。绥和年间，成帝罢刺史，置[[州牧]]。[[王莽]]继[[王根]]为大司马，时年三十八。[[赵昭仪]]害死成帝后宫所生子。成帝暴崩，民间归罪赵昭仪，皇太后诏治问，赵昭仪自杀。太子[[刘欣]]即位，是为[[汉哀帝]]。
