# [[槐里]]

[[槐里]]是[[秦]][[汉]]置县，治所在今[[陕西]]兴平，为[[汉代]][[右扶风]]治所。槐里地处[[关中平原]]腹地，[[长安]]以西，[[渭水]]北岸，是京畿三辅之一的重镇。

[[项羽]]入关后，分封诸侯，封[[司马欣]]为[[塞王]]，都[[栎阳]]；[[董翳]]为[[翟王]]，都[[高奴]]；废[[雍王]][[章邯]]守槐里以西。槐里是周秦汉唐时期关中西部的重要政治和军事据点。
