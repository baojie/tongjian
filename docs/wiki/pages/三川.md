# 三川

三川，[[秦]]郡，因境内有[[黄河]]、[[洛水]]、[[伊水]]三川得名，治[[洛阳]]。

[[张仪]]说[[秦惠王]]伐韩，下兵三川攻新城、宜阳，以临二周之郊，据九鼎挟天子以令于天下（003-016）。[[司马错]]与张仪争论于秦惠王前，张仪主伐韩入三川，司马错主伐蜀——三川之战略价值，为秦统一进程中关键议题（003-040）。秦[[蒙骜]]伐韩，取成皋、荥阳，初置三川郡（006-033）。

> 亲魏，善楚，下兵三川，攻新城、宜阳，以临二周之郊，据九鼎，按图籍，挟天子以令于天下。
> （003-016）

三川郡地当中原腹地，自[[秦]]置郡以来即为天下之中。[[汉高祖]]改三川郡为[[河南郡]]，[[洛阳]]为历代都城之所在。三川之得失，关系天下大势。
