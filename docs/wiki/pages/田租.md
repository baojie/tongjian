# 田租

田租是古代按土地征收的赋税，为国家财政收入的重要来源。[[汉文帝]]时期"赐天下民今年田租之半"（013-097），以减轻农民负担。元封元年[[汉武帝]][[巡幸]]各地，[[诏令]]"民田租逋赋，皆贷除之"（020-087）。

[[汉元帝]]因灾害频发，下诏"毋令民出今年田租"（023-015）。[[汉宣帝]]时[[慕容冲|凤皇]]集于胶东、千乘，遂赦天下"勿收田租"（024-041）。[[王莽]]抨击汉制"减轻田租，三十而税一"，却因豪民侵陵导致"厥名三十税一，实什税五"（037-011），揭示田租虽轻而实际负担不减的矛盾。
