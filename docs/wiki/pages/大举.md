# 大举

**大举**指大规模兴兵征伐，通常意味着倾国之力、举国之师。

## 倾国兴师

诸葛亮悉大众十万由斜谷入寇，遣使约吴同时大举（072-050）。大举贵在协同。

司马昭欲大举伐汉，朝臣多以为不可，独钟会劝之（078-009）。大举须有决断。

孙吴大举兵出华里，载太后、皇后及后宫数千人，遇大雪，军士寒冻（079-098）。大举不慎，反伤国力。

## 大举之谋

满宠言孙权大举而来，欲要一切之功，虽不敢至，必当上岸耀兵（072-047）。大举或为虚张声势。

田豫谓贼悉众大举，非图小利，欲质新城以致大军耳，宜听使攻城（072-056）。知敌大举之意，则不为所诱。

杜预言不一大举扫灭，则兵役无时得息（080-057）。大举以毕其功，久战不如一战。
