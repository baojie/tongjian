# 乌震

乌震，后唐将领。从赵王王镕，后降晋王李存勖，为河北道副招讨使，为房知温所害。

以赵将符习为成德节度使，乌震为赵州刺史（271-076）。帝以冀州刺史乌震三将兵运粮入幽州，以震为河北道副招讨，领宁国节度使，屯卢台军（275-081）。朝廷不次擢乌震为副招讨使，讹言益甚；房知温怨震骤来代己（275-086）。
