# 霖雨

**霖雨**指连绵不止的[[大雨]]，古代常被视为灾害，引发水涝、[[道路]]泥泞，影响[[军事]]行动和[[农业]]生产。

## 对军事的影响

秦末，[[大雨]]霖雨，武信君引兵攻[[亢父]]，闻[[田荣]]之急，乃引兵击破[[章邯]]军[[东阿]]下（008-029）。霖雨造成[[道路]]泥泞，常使[[行军]]受阻。

## 作为灾异记录

[[汉代]][[史官]]常记录霖雨等异常天气。冬十二月，雷，霖雨（016-046），被视为反常之兆。治河者言，如有霖雨，旬日不霁，河水必盈溢（030-033），说明霖雨与河患密切相关。
