# 李绅

李绅，字公垂，亳州谯人。唐[[诗人]]、[[宰相]]，以《悯农》诗闻名于世。

李绅与[[元稹]]争进取有隙，[[西川]][[节度使]][[段文昌]]、翰林[[学士]]李绅各以书属所善[[进士]]于钱徽（241-088）。[[李逢吉]]为相，内结知枢密[[王守澄]]，势倾朝野。惟翰林[[学士]]李绅每承顾问，常排抑之（243-017）。后[[韩愈]]、李绅入谢，上各令自叙其事，乃深寤，复以愈为[[吏部侍郎]]，绅为户部[[侍郎]]（243-019）。
