# 李绅

李绅，字公垂，亳州谯人。唐诗人、宰相，以《悯农》诗闻名于世。

李绅与元稹争进取有隙，西川节度使段文昌、翰林学士李绅各以书属所善进士于钱徽（241-088）。李逢吉为相，内结知枢密王守澄，势倾朝野。惟翰林学士李绅每承顾问，常排抑之（243-017）。后韩愈、李绅入谢，上各令自叙其事，乃深寤，复以愈为吏部侍郎，绅为户部侍郎（243-019）。
