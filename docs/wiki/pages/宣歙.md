# 宣歙

宣歙为[[唐代]]方镇，治宣州（今安徽宣城），领宣、歙二州，为江南富庶之地。

[[刘展之乱]]中，宣歙节度使[[郑炅之]]弃城走（221-103）。[[陈少游]]请岁献五万缗，改宣歙观察使（224-032）。宣歙观察使[[薛邕]]盗隐官物以巨万计（226-047）。宣歙观察使[[刘赞]]卒，初[[唐德宗]]以奉天窘乏，还宫以来尤专意聚敛，藩镇多以进奉市恩（235-044）。
