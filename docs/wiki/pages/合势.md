# 合势

**合势**指合并各路兵力，汇聚成强大攻势，为古代[[军事]]中集中力量统一行动的重要原则。

## 中外合势

城中亦[[鼓噪]]而出，中外合势，震呼动[[天地]]（039-011）。

## 合势并力

不与镇恶合势并力，以争[[潼关]]（118-013）。当合势齐力，以取一战之捷（097-092）。须东西合势乃进（076-061）。当合势齐力，以取一战之捷（097-092）。

## 合势摧敌

嚣辄与[[冯异]]合势，共摧挫之（041-042）。分遣锐师与众军合势（155-061）。蓝引军还奔[[临菑]]，[[并兵]]合势（041-065）。
