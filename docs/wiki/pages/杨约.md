# 杨约

杨约，弘农华阴人。[[隋炀帝]]宠臣，[[杨素]]之弟。

杨约与[[杨素]]同鞫勇之功，受赏千段（179-029）。后出为伊州[[刺史]]（179-078）。[[隋炀帝|炀帝]]即位，召杨约来朝，遣约入长安，易留守者，矫称[[汉高祖刘邦|高祖]]之诏，赐故[[太子]]勇死（180-007）。约为[[隋炀帝|炀帝]]心腹，参预夺嫡之谋。

> 出杨约为伊州刺史。炀帝即位，会伊州刺史杨约来朝，太子遣约入长安，易留守者。
> （180-007）
