# 司马德戡

司马德戡，隋虎贲郎将。统领骁果，主谋江都兵变，后为宇文化及所杀。

虎贲郎将扶风司马德戡素有宠于帝，帝使领骁果屯于东城（185-014）。德戡盗御厩马，潜厉兵刃，骁果皆曰："唯将军命！"（185-016）。司马德戡等使令狐行达缢杀炀帝（185-018）。司马德戡窃谓赵行枢曰："化及庸暗，群小在侧，事将必败"（185-032）。化及留人告司马德戡等使讨之（185-024）。

> 虎贲郎将扶风司马德戡素有宠于帝，帝使领骁果屯于东城。
