# 田丰

田丰，字元皓，巨鹿人。汉末[[袁绍]]帐下谋士，以正直不阿、谋略深远著称，后因谏阻[[官渡之战]]为[[袁绍]]所杀。

魏郡[[审配]]、巨鹿田丰并以正直不得志于韩馥，绍以丰为别驾，配为治中（060-012）。田丰说[[袁绍]]曰："[[徙都]]之计既不克从，宜早图许，奉迎天子，动托诏书，号令海内，此算之上者。"（062-041）

田丰多谋善断，然[[袁绍]]不能用其策，终致[[官渡]]之败。田丰在狱中闻绍败，叹曰："若用丰言，何至于此！"绍归后羞见田丰，遂杀之。田丰之死，时人莫不痛惜。
