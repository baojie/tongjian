# 连营

**连营**指营帐相连驻扎，为古代大[[军屯]]驻时的扎营方式，便于联络但也容易被敌军分割。

## 连营进军

[[梁慬]]等连营稍前，[[单于]]恐怖（049-086）。[[徐晃]]连营稍前解[[樊城]]之围（068-037）。

## 连营之弊

[[刘备]]自[[巫峡]][[建平]]连营至[[夷陵]]界[[树栅]]连营七百馀里，[[陆逊]]火烧连营大破之（069-068、069-070）。宇文氏[[士卒]]数十万连营四十里（091-025）。
