# 盟誓

盟誓是中国古代诸侯或政治势力之间缔结同盟的仪式化制度。盟誓仪式通常包含歃血（以血涂口）、宣读盟书、祭告神明等环节，盟书一式多份，一份埋于盟所或沉于河，同盟各方各执一份。[[春秋]][[战国|战国时期]]盟誓最为盛行，诸侯[[会盟]]频繁。此后历代政治军事结盟仍行盟誓之礼，是古代政治外交的重要组成部分。

> 公子行，过辞侯生，侯生曰："臣宜从，老不能，请数公子行日，以至晋鄙军之日北乡自刭以送公子。"公子遂行。
> （064-035）
