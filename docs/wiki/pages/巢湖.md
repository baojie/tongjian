# [[巢湖]]

[[巢湖]]位于[[安徽]]中部，是[[中国]]五大[[淡水湖]]之一，与[[长江]]相通，自古为[[江淮]]地区的重要水域。巢湖流域水网密布，是[[合肥]]以南的重要水上通道。

[[三国]]时期，[[曹操]]曾四越巢湖以攻[[东吴]]（071-017），但皆未成功。[[孙权]]也曾入[[居巢]]湖口，向[[合肥]][[新城]]进攻，众号十万（072-056）。巢湖是[[曹魏]]与[[东吴]]在[[江淮]]地区拉锯争夺的关键水域。
