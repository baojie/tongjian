# 张武

张武，[[西汉]][[代国]][[郎中令]]，[[汉文帝]]旧臣。

[[吕氏|诸吕]]被诛，代王[[汉文帝|刘恒]]被迎立。张武以为汉大臣多谋诈，劝代王[[称疾]]勿往，[[宋昌]]驳之（013-062）。代王入京，拜张武为[[郎中令]]行[[殿中]]（013-063）。后[[匈奴]]入侵，[[汉文帝]]以张武为[[龙骧将军|将军]]屯[[北地]]（015-035、015-066）。张武等受赂[[金钱]]，[[隋文帝|文帝]]发觉后更加[[赏赐]]以愧其心（015-070）。

## 世系

张武，[[西汉]]人。[[吕后]]崩后[[代王]][[汉文帝|刘恒]]被迎立，武时以[[代国]][[郎中令]]劝代王[[称疾]]勿往（013-062），王不听。入京后拜武为[[郎中令]]行[[殿中]]（013-063）。后为[[将军]]屯[[北地]]御[[匈奴]]（015-035）。武受赂，[[隋文帝|文帝]]发觉加赐以愧其心（015-070）。武[[子孙]]不详于史。
