# 张武

张武，西汉代国郎中令，汉文帝旧臣。

诸吕被诛，代王刘恒被迎立。张武以为汉大臣多谋诈，劝代王称疾勿往，宋昌驳之（013-062）。代王入京，拜张武为郎中令行殿中（013-063）。后匈奴入侵，汉文帝以张武为将军屯北地（015-035、015-066）。张武等受赂金钱，文帝发觉后更加赏赐以愧其心（015-070）。
