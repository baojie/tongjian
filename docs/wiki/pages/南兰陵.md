# [[南兰陵]]

[[南兰陵]]是[[南朝]]侨置郡，治所在今[[江苏]]常州一带。[[东晋]]南渡后，[[兰陵]]萧氏侨居南兰陵，[[南朝齐|齐]]、[[南朝梁|梁]]两朝的皇室[[萧氏]]即出自此地。

[[南齐]]末年，[[萧鸾]]（[[齐明帝]]）大肆诛杀[[高帝|高]]、[[武帝]]子孙，遣[[茹法亮]]杀巴陵王子伦，时[[萧子伦]]为南兰陵太守（139-069）。南兰陵是研究六朝侨置郡县制度和[[琅邪王氏|王]]、[[陈郡谢氏|谢]]、萧等士族南迁史的重要地理坐标。
