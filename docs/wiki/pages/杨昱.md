# 杨昱

杨昱，[[北魏]]大臣，[[杨椿]]之子，以忠谏闻名，任[[徐州刺史]]，曾镇守[[荥阳]]。

杨昱以忠谏获免于[[崔亮]]案（146-084）。[[詹事]]杨昱上言请[[太子]][[出入]]有宫臣翼从（147-113）。兼[[中书舍人]]杨昱[[直言]][[不讳]]，奏[[李崇]]等贪秽（149-018）。[[济阴]][[内史]]杨昱被诬陷，考讯百日乃得还任（149-035）。[[黄门侍郎]]杨昱兼[[侍中]]，[[持节]]监[[北海]]王颢军（150-041）。[[黄门侍郎]]杨昱分处降户于冀、定、瀛三州就食（150-091）。魏[[徐州刺史]]杨昱击斩叛民续灵珍（152-048）。魏以[[东南]]道[[大都督]]杨昱镇[[荥阳]]（153-013）。

## 世系

杨昱（？—约530年），[[北魏]]大臣。父[[杨椿]]，兄[[杨昱|杨氏]][[兄弟]]皆为[[北魏]]重臣，[[弘农杨氏]]。昱以忠谏闻（146-084），官至[[黄门侍郎]]，镇[[荥阳]]（153-013）。昱[[子孙]]不详于史。
