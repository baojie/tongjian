# [[鄂州]]

[[鄂州]]，[[隋]][[唐]]时期的重要州郡，治所在今[[湖北]][[武昌]]。地处[[长江]]中游，是连接[[江陵]]与[[江州]]的水路枢纽，具有重要的军事和交通地位。

南梁末，[[萧铣]]据有江陵，以其地将迎[[隋军临江]]，刺史雷长颖以鲁山降唐（189-079）。[[唐代]]鄂州隶属[[江南西道]]，[[武则天]]时曾杀[[宗室]]嗣郑王璥等六人于鄂州（204-069）。玄宗时，瓘贬鄂州别驾（212-128）。
