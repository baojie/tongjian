# [[抚州]]

[[抚州]]是[[唐代]]设置的州，治所在今[[江西]]临川，隶属[[江南西道]]。[[唐太宗]]贞观年间，曾以[[阿史那苏尼失]]为北宁州都督，康苏密为北安州都督，史善应为北抚州都督（193-071），此为北抚州，用于安置归降的突厥部落。

[[唐中宗]]神龙年间，[[冉祖雍]]曾任抚州司仓，上书告[[王同皎]]等谋反案（208-080）。[[唐代]]抚州以临川文化著称，[[王勃]]《[[滕王阁序]]》有"邺水朱华，光照临川之笔"之句即指抚州一带。
