# 莎车

莎车是西域南道名国。西域南道自鄯善傍南山北，"波河西行至莎车，为南道"（020-036），莎车为南道西端枢纽。汉宣帝时莎车王弟呼屠徵杀王自立，冯奉世发诸国兵平之（025-045/046），威震西域。

东汉时莎车与于阗争雄，班超定西域后归于汉。前凉张骏遣将伐西域，"于是西域诸国焉耆、于阗之属，皆诣姑臧朝贡"（095-087）。莎车在丝绸之路南道贸易中地位重要，为东西文化交流的门户。
