# 耿国

耿国，东汉五官中郎将、大司农。耿弇之弟，议纳南匈奴呼韩邪单于降附。

五官中郎将耿国议宜如孝宣故事受南匈奴降，令东扞鲜卑，北拒匈奴（044-013）。大司农耿国上言宜置度辽将军屯五原，以防南匈奴逃亡，朝廷不从，后果如其言（045-025）。

> 五官中郎将耿国独以为："宜如孝宣故事，受之。令东扞鲜卑，北拒匈奴，率厉四夷，完复边郡。"
> （044-013）
