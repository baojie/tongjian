# 孝顺

**孝顺**指孝亲顺应，是[[中国古代]]家庭伦理与社会秩序的基础。

## 汉代

[[汉文帝]]以[[仁孝]]闻于[[天下]]，侍奉薄[[太后]]，药非亲尝弗进（022-050）。[[隋文帝|文帝]]之孝顺不限于家事，更推己及人，施[[仁政]]于[[天下]]。[[汉宣帝]]少时遭变故，然不失孝顺之道，及[[即位]]，[[追尊]]祖考，以孝[[治国]]（052-002）。
