# 永寿

**永寿**（155—158），[[汉桓帝]]第五个[[年号]]。永寿年间，[[梁冀]]被诛。桓帝与[[单超]]、[[徐璜]]、[[具瑗]]、[[左悺]]、[[唐衡]]五[[宦官]]合谋诛[[梁冀]]，收其[[财货]]，没入县官。[[五侯]][[封侯]]，[[宦官]]之势遂炽。[[段颎]]为[[护羌校尉]]，击[[西羌]]，连破之。

> （053-092）
