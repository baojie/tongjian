# 为质

为质，派遣[[宗室]]子弟到他国作人质，[[战国|战国时期]]常见的外交制度。

[[秦昭王|秦昭襄王]]为质于燕（003-053）。楚[[太子]]横为质于秦请救（003-068）。秦泾阳君为质于齐（003-078）。

[[战国]]之时，[[诸侯]][[交质]]盛行。[[质子]]为两国结信之手段，亦为人质之担保。[[秦武王]]死，无子，异母弟稷为质于燕，国人迎而立之，是为昭襄王。楚以[[太子]]横为质于秦，请秦救兵以解[[三国]]之伐。秦泾阳君为质于齐，示两国之好。[[质子]]之制，虽曰结信，然为质者往往生死难卜，其国势[[强弱]]可见一斑。至秦汉一统，此制渐废。

> 秦昭襄王为质于燕。
> （003-053）
