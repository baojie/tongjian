# 郗虑

郗虑，[[东汉]][[御史大夫]]。[[曹操]]使郗虑以光禄勋为[[御史大夫]]。郗虑与[[孔融]]有隙，承[[曹操]]风旨，构成其罪（065-037、065-038）。后[[御史大夫]]郗虑[[持节]]策收[[皇后]][[玺绶]]，以[[华歆]]为副，勒兵入宫收伏后（067-016）。

> 虑承操风旨，构成其罪，令丞相军谋祭酒路粹奏："融昔在北海，见王室不静，而招合徒众，欲规不轨。"
> （065-038）
