# 贾彪

贾彪，东汉名士，太学生领袖之一，与郭泰并为太学诸生之冠。

太学诸生三万余人，郭泰及颍川贾彪为其冠，与李膺、陈蕃、王畅更相褒重（055-048）。党锢祸起，贾彪独闭门不纳岑晊，时人望之。彪曰："传言'相时而动，无累后人。'"（055-054）。

陈蕃既免，朝臣震栗，莫敢复为党人言者。贾彪曰："吾不西行，大祸不解。"乃入洛阳，说城门校尉窦武、尚书魏郡霍谞等，使营解党禁（056-007）。党锢之祸得解，贾彪之力也。
