# 李专美

李专美，[[五代]][[后唐]]端明殿[[学士]]。闵帝赏军乏财，帝责专美，对以"财尽民怨，何能为计"。与议[[石敬瑭]]事，以为不宜[[猜忌]]。[[后晋]]时复为[[赞善]][[大夫]]。[[《资治通鉴》]] 载其事迹。
