# 东晋

东晋（317-420年），[[西晋]]灭亡后由[[司马睿]]在[[建康]]（今南京）重建的[[晋]]朝。[[司马睿]]以安东将[[军镇]][[苻健|建业]]，收贤俊，[[王导]]劝其收揽人心（087-086）。[[永嘉之乱]]后，中国士民多南渡江，东晋偏安[[江左]]。东晋虽开置国学，不及十年辄废（145-122）。江表自东晋以来，[[刑法]]疏缓，[[世族]]陵驾[[寒门]]（177-068）。安史乱时，肃宗虑永王璘据金陵为东晋故事（219-021）。东晋以[[门阀]]政治为特色，王、谢、桓、庾四大家族轮流执政，终为[[刘裕]]所篡。

> 上雅好儒术，以东晋、宋、齐虽开置国学，不及十年辄废之，其存亦文具而已，无讲授之实。
> （145-122）
