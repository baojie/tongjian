# 执金吾

执金吾是汉代掌管京师治安的武官，原为中尉，[[汉武帝]]太初元年更名为执金吾。

武帝时，以执金吾杜周为[[御史大夫]]（022-003）。征和年间，[[江充]]构陷太子据，太子发兵诛充，[[汉武帝]]遣宗正刘长、执金吾刘敢奉策收皇后玺绶（022-049）。执金吾秩中二千石，属官有缇骑、持戟等，掌宫外戒司水火之事，是京师卫戍体系的重要一环。[[光武帝刘秀]]未发迹时曾感慨"仕宦当作执金吾，娶妻当得阴丽华"，足见当时执金吾之显赫。
