# 谏官

谏官，专职规谏君主过失的官员。

汉宣帝出谏官补郡吏（025-043）。萧望之荐名儒以备谏官（028-017）。晋初置谏官（079-016）。

谏官制度为古代政治之重要特色。汉宣帝以谏官补郡国守相，萧望之上疏争之曰："朝无争臣，则不知过。"帝乃征望之入守少府。萧望之复荐名儒茂材以备谏官。及西晋初立，武帝置谏官，以散骑常侍傅玄、皇甫陶为之。傅玄上疏陈士风颓敝，务农积谷，为时所称。谏官虽位卑而责重，以匡正君失、裨益朝政为职。
