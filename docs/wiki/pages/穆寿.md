# 穆寿

穆寿，北魏宜都王，太武帝伐北凉时辅太子晃监国。

魏主进宜都公穆寿爵为王（122-139）。魏主使侍中宜都王穆寿辅太子晃监国（123-067）。魏主敕穆寿曰："吴提与牧犍相结素深，闻朕讨牧犍，吴提必犯塞。"（123-080）。柔然入寇，平城大骇，穆寿不知所为（123-081）。魏主至平城，以无大失亡，故穆寿得不诛（123-085）。命侍中穆寿辅太子决庶政（124-036）。
