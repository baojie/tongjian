# 出使

出使是奉命出使他国的外交行为，为古代邦交之重要形式。

[[范雎]]谓[[秦昭王]]，言[[魏冉|穰侯]]出使不报，[[华阳]]泾阳等击断无讳，四贵备而国不危者未之有也（005-022）。张祎出使[[前凉]]，劝奉[[张骏]]为凉王，祎曰人臣出使苟利[[社稷]]专之可也（097-110）。[[高澄]]诈言出使，虚除[[元康]][[中书令]]（162-073）。

出使之制在[[春秋]][[战国|战国时期]]极为发达，[[苏秦]][[张仪]]等[[纵横家]]即以外交出使闻名。[[汉代]][[张骞]]出使[[西域]]、[[苏武]]出使[[匈奴]]，皆为千古佳话。唐代以后，出使制度更加完备，[[使节]]之选派、礼仪皆有定规。

> 今太后擅行不顾，穰侯出使不报，华阳、泾阳等击断无讳，高陵进退不请，四贵备而国不危者，未之有也。
> （005-022）
