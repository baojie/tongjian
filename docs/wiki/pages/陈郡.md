# [[陈郡]]

[[陈郡]]是[[秦]]置郡，治所在[[陈县]]（今[[河南]]淮阳），[[汉代]]改为[[陈国]]（或淮阳国）。陈郡地处[[中原]]腹地，[[战国]]时期为[[楚国]]都城（[[郢陈]]），[[秦末]][[陈胜]]在此首举义旗，建国号"张楚"。

[[东汉]]末至[[三国]]，陈郡为[[豫州]]辖郡。[[曹操]]在豫州时，举[[陈郡]][[袁涣]]为茂才（062-019）。后使[[梁习]]以别部司马领[[并州]]刺史（065-004）。
