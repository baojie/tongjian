# 华温琪

华温琪，后梁、[[后唐]][[节度使]]。以拒晋有功，历仕忠武、泰源等镇。

帝以晋州[[刺史]]下邑华温琪拒晋兵有功，欲赏之（267-090）。帝以汝州[[防御使]]华温琪权知静胜[[留后]]（271-028）。[[秦州]][[节度使]]华温琪入朝，请留阙下，帝嘉之，除左骁卫上将军（276-037）。

> 帝以晋州刺史下邑华温琪拒晋兵有功，欲赏之。
