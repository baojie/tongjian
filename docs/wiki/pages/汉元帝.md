# 汉元帝

汉元帝刘奭，西汉第十一位皇帝。柔仁好儒，然多病委政宦官，汉室自此由盛转衰。

尚书仆射薛综上疏引昔汉元帝欲御楼船，薛广德请刎颈以血染车之事（072-044）。侍中崔光上表引汉元帝初元中雌鸡化雄之异（145-104）。

元帝为宣帝太子时即好儒术，见宣帝多用文法吏，尝谏曰："陛下持刑太深，宜用儒生。"宣帝作色曰："汉家自有制度，本以霸王道杂之。"及即位，元帝任用萧望之、周堪、刘更生等儒臣，然不能制宦官弘恭、石显，致望之自杀，堪等废黜。元帝优柔寡断，汉政始衰。后世以元帝为西汉由盛转衰之关键。

> 尚书仆射薛综上疏曰："昔汉元帝欲御楼船，薛广德请刎颈以血染车。"
