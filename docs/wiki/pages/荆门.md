# [[荆门]]

[[荆门]]位于今[[湖北]]宜昌长江南岸，与北岸[[虎牙]]山夹江对峙，地势险要。[[东汉]]初年，[[公孙述]]据蜀，遣[[任满]]、[[田戎]]等率军据荆门、虎牙，横江水起浮桥、关楼以拒汉军（042-049）。

[[汉光武帝]]遣[[岑彭]]率水军攻蜀，岑彭装战船数十艘，直冲浮桥，顺风纵火，大破蜀军（042-067），遂克[[夷陵]]，为东汉统一蜀地打开了大门。
