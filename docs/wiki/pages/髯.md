# 髯

"髯"在《[[资治通鉴]]》指胡须。有龙垂胡髯下迎黄帝（020-057）。江、沔间所在起兵以应昌，皆著绛帽，以马尾作髯（085-009）。勒攘袂鼓髯（088-009）。府中为之语曰："髯参军，短主簿，能令公喜，能令公怒"（101-060）。

> 黄帝采首山铜，铸鼎于荆山下，鼎既成，有龙垂胡髯下迎黄帝。
> （020-057）

> 勒攘袂鼓髯曰："张君计是也！"责刁膺曰："君既相辅佐，当共成大功，奈何遽劝孤降！此策应斩。"
> （088-009）

> 又以征西掾郗超为参军，王珣为主簿，每事必与二人谋之。府中为之语曰："髯参军，短主簿，能令公喜，能令公怒。"
> （101-060）
