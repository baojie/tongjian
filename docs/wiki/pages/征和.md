# 征和

**征和**（前92—前89），[[汉武帝]]第十个[[年号]]。征和年间，[[巫蛊之祸]]达到顶峰，[[江充]]诬陷[[太子]][[刘据]]，[[太子]]惧而[[举兵]]诛充，与[[丞相]][[刘屈氂]]战于[[长安]]城中，兵败自杀。[[武帝]]晚年深悔[[征伐]]之事，下《[[轮台罪己诏|轮台诏]]》，"罢劳民、修文教"，[[与民休息]]，为[[武帝]]晚年政策转折点。
