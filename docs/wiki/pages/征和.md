# 征和

**征和**（前92—前89），[[汉武帝]]第十个[[年号]]。征和年间，[[巫蛊之祸]]达到顶峰，[[江充]]诬陷太子[[刘据]]，太子惧而举兵诛充，与丞相[[刘屈氂]]战于长安城中，兵败自杀。武帝晚年深悔征伐之事，下《轮台诏》，"罢劳民、修文教"，与民休息，为武帝晚年政策转折点。
