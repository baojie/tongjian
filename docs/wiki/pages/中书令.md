# 中书令

中书令是[[中书省]]长官，掌机要诏命、典尚书奏事，汉代多以[[宦官]]任职，唐代为真[[宰相]]。

[[汉宣帝]]时，中书令弘恭、仆射[[石显]]久典枢机，明习文法（028-015）。恭死后，[[石显]]为中书令，管尚书，尚书五人皆其党（028-031）（029-012）。[[石显]]擅权，霍山言"中书令出取之，不关尚书"（025-024）。

中书令之职在汉代表面为内廷近臣，实则掌握机要，权力甚重。[[东汉]]以后逐渐演变为[[宰相]]之职，至隋唐成为三省长官之一。

> 中书令弘恭、仆射石显，自宣帝时久典枢机，明习文法。
> （028-015）
