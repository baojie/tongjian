# [[麦城]]

[[麦城]]是[[三国]]时期古城，位于今[[湖北]]当阳东南。[[建安]]二十四年（219年），[[关羽]]率军攻[[樊城]]，[[孙权]]遣[[吕蒙]]袭取[[江陵]]，关羽退走[[夷陵]]后，西保麦城（068-042）。

关羽在麦城孤穷无援，孙权遣使诱降，关羽伪降，立幡旗为象人于城上，趁机遁走，最终为[[吴]]军所擒。麦城之战标志着[[关羽]]之败和[[蜀汉]]失去[[荆州]]，深刻改变了[[三国]]鼎立的格局。
