# 顾雍

顾雍（168-243年），字元叹，吴郡吴县人，[[三国]]时[[东吴]]丞相。[[孙权]]领会稽太守，以雍为丞，行太守事。后累迁大理、奉常，领尚书令，封阳遂乡侯。黄武四年，代[[孙邵]]为丞相，平尚书事。

顾雍为相十九年，选用文武将吏各随其能，心无适莫。孙权敬重之，常咨访国政。雍不饮酒，寡言语，举动时当。孙权尝叹曰："顾君不言，言必有中。"顾雍卒，权素服临吊，谥曰肃侯。
