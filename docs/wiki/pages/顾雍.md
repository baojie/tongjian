# 顾雍

顾雍（168-243年），字元叹，吴郡吴县人，[[三国]]时[[东吴]][[丞相]]。[[孙权]]领会稽太守，以雍为丞，行太守事。后累迁大理、奉常，领[[尚书令]]，封阳遂乡侯。黄武四年，代[[孙邵]]为[[丞相]]，平尚书事。

顾雍为相十九年，选用文武将吏各随其能，心无适莫。[[孙权]]敬重之，常咨访国政。雍不饮酒，寡言语，举动时当。[[孙权]]尝叹曰："顾君不言，言必有中。"顾雍卒，权素服临吊，谥曰肃侯。

## 为相与荐士

吴张温少以俊才有盛名，顾雍以为当今无辈（070-025）。雍为[[丞相]]、平尚书事，寡言举动时当，[[孙权]]尝叹曰："顾君不言，言必有中。"（070-034）。举朝大臣自顾雍以下皆谏，以为不可宠待[[公孙渊]]太厚（072-038）。吕壹告雍过失，[[孙权]]诘责雍（074-014）。雍卒，权素服临吊（074-059）。顾雍为相十九年，选用文武各随其能，心无适莫，有古大臣之风。
