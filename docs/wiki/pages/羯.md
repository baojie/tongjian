# 羯

羯是西晋末入塞的胡人部族，为[[五胡]]之一。西晋末年王浚与[[司马腾]]起兵讨[[司马颖]]，使"乌桓羯硃"率部参战（085-047）。王浚入鄴后，又使"乌桓羯硃"追司马颖至朝歌（085-055）。

上党武乡羯人石勒"有胆力，善骑射"，西晋并州大饥时被掠卖为奴，后聚众起兵，渐成势力（086-009）。"并州诸胡羯多从之"（087-012），石勒最终建立后赵，成为十六国时期的重要政权。羯族在五胡中以勇猛善战著称，后赵亡后渐融入其他民族。
