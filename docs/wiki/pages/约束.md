# 约束

**约束**指军队或政权的法令规章与行为规范，是维持纪律与秩序的基本手段。

## 军令约束

赵括至军，悉更约束，易置军吏，出兵击秦师（005-043）。约束一改，军心遂乱。

祭遵为人，廉约小心，克己奉公，赏赐尽与士卒；约束严整，所在吏民不知有军（042-047）。约束严则军纪整。

萧何守关中，为法令约束，立宗庙、社稷、宫室、县邑（009-052）。约束立则国体定。

## 法度相承

曹参代何为相，举事无所变更，一遵何约束（012-070）。萧规曹随，约束不可轻改。

中行说谓匈奴约束径，易行；君臣简，可久（014-040）。约束简则易行。

刘邦入关，与父老约法三章，馀悉除去秦法，曰："吾所以还军霸上，待诸侯至而定约束耳"（009-005）。定约束以安民心。
