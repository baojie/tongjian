# 郑滑

郑滑为[[唐代]]藩镇，治滑州（今河南滑县），领郑、滑二州。

以汴滑节度使[[李澄]]为郑滑节度使（231-050）。以左龙武大将军[[薛平]]为郑滑节度使，欲为控制[[魏博]]（238-074）。发郑滑、魏博卒凿黎阳古河十四里，以纾滑州水患（239-024）。[[李德裕]]奏乞郑滑五百人、陈许千人以镇蜀（244-043）。
