# 张彝

张彝，字庆宾，[[清河]]东武城人。[[北魏]][[尚书]]、征西将军。以排抑武人致其子被殴，宅第被焚。

[[尚书]][[清河]]张彝、[[邢峦]]闻处分非常，亡走，出[[洛阳]]城，为御史中尉[[中山]][[甄琛]]所弹（144-007）。仆射[[高肇]]、[[秦州]][[刺史]]张彝皆欲尚[[陈留]][[公主]]，[[公主]]许彝而不许肇。肇怒，谮彝于魏主，彝坐沉废累年（145-056）。[[魏徵|魏征]]西将军平陆文侯张彝之子仲瑀上[[封事]]，求铨削选格，排抑武人，不使豫清品。于是喧谤盈路，立榜大巷，克期会集（149-006）。

张彝在孝[[隋文帝|文帝]]、宣武帝朝历任职掌，以门第自高。其子张仲瑀[[上书]]请排抑武人，引发羽林虎贲千余人焚毁张彝宅第，殴伤其子，张彝亦被火烧致死。此事震动朝野，暴露出[[北魏]]文武矛盾之深，实为六镇起义之先声。
