# 李庭望

**李庭望**，[[安禄山]]部将。安禄山以其将李庭望为节度使守[[陈留]]，与[[张巡]]战，屡为所败。

安禄山以李庭望为爪牙（216-064）。以其将李庭望为节度使守陈留（217-042）。李庭望引兵欲东徇地，闻贾贲南击睢阳，不敢进而还（217-048）。李庭望将蕃、汉兵二万馀人东袭宁陵、襄邑，张巡大破之（218-053）。

## 事迹

李庭望，[[安禄山]]部将。安史乱起，安禄山以庭望为爪牙（216-064），令守[[陈留]]（217-042）。庭望欲东徇地，为[[贾贲]]、[[张巡]]所阻（217-048）。后率蕃汉兵二万攻宁陵、襄邑，张巡以短兵三千掩击，大破之（218-053）。
