# 大宛

大宛是中亚古国，位于锡尔河流域，以产汗血马闻名。[[张骞]]通[[西域]]报武帝，言大宛"多善马，马汗血，其先天马子也"（018-083）。汉武欲得大宛天马，[[遣使]]持金马以请，大宛王不肯，遂使贰师将军[[李广利]]征大宛。

[[李广]]利初征不利，武帝"益发兵甲卒"（019-039），再次征伐，围大宛城四十馀日，大宛贵人杀王降汉，汉得善马数十匹、中马以下三千馀匹（019-040）。[[张骞]]分遣副使使大宛、[[康居]]、大月氏、[[赫连勃勃|大夏]]、[[安息]]诸国（020-035）。大宛之役显示了[[西汉|汉朝]]的[[西域]]影响力，也使汗血马传入[[中原]]。
