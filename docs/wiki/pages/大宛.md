# 大宛

大宛是中亚古国，位于锡尔河流域，以产汗血马闻名。[[张骞]]通[[西域]]报[[武帝]]，言大宛"多善马，马汗血，其先天马子也"（018-083）。[[汉武]]欲得大宛天马，[[遣使]]持[[金马]]以请，大宛王不肯，遂使贰师[[将军]][[李广利]]征大宛。

[[李广]]利初征不利，[[武帝]]"益发兵甲卒"（019-039），再次[[征伐]]，围大宛城四十馀日，大宛贵人杀王降汉，汉得善马数十匹、中马以下三千馀匹（019-040）。[[张骞]]分遣副使使大宛、[[康居]]、大[[月氏]]、[[赫连勃勃|大夏]]、[[安息]]诸国（020-035）。大宛之役显示了[[西汉|汉朝]]的[[西域]]影响力，也使汗血马传入[[中原]]。

## 大宛与汉朝西域经营

大宛在[[西汉|汉朝]][[西域]]战略中地位关键。[[张骞]]自[[月氏]]还，备言大宛等诸国风俗物产，[[汉武帝]]乃决意通[[西域]]（019-039）。大宛等[[西域]]诸国兵弱而贵汉财物（019-040），[[张骞]]分遣副使使大宛等诸国，[[乌孙]]发译道送骞还，汉使数十人相望于道（020-035）。自[[玉门]]、[[阳关]]出[[西域]]有南北两道，北道西逾葱岭则出大宛、[[康居]]（020-036）。[[赵破奴]]破[[车师]]，举兵威以困[[乌孙]]、大宛之属（021-018）。大宛既通，[[乌孙]]见汉广大乃益重汉，[[匈奴]]闻[[乌孙]]与汉通怒而欲击之，然大宛、[[月氏]]之属皆事汉，[[乌孙]]于是愿尚汉[[公主]]（021-042）。大宛所产蒲萄、苜蓿由汉使采实以来，[[天子]]种之于[[离宫]]别观旁（021-043）。大宛之通，实为汉通[[西域]]全盘棋局之关键一子。
