# 刘焉

刘焉，[[东汉]][[宗室]]，[[汉鲁恭王]]之后。[[《资治通鉴》]]载其建议改[[刺史]]为[[州牧]]，出任[[益州]]牧，开[[东汉]]末[[割据]]之端。

[[太常]][[江夏]]刘焉见[[王室]]多故，建议以为："四方兵寇，由[[刺史]]威轻，既不能禁，且[[用非其人]]，以致离叛。宜改置[[牧伯]]，选[[清名]]重臣以居其任"（059-006）。州[[从事]][[贾龙]]率[[吏民]]攻相等，数日破走，州界清静，龙乃选[[吏卒]]迎刘焉（059-010）。刘焉在[[益州]]阴图异计，沛人[[张鲁]]自祖父陵以来世为五斗米道，鲁母以鬼道常往来焉家（060-022）。

## 开牧伯之端

[[刘焉]]见[[王室]]多故，建议改[[刺史]]为[[州牧]]，选[[清名]]重臣居其任（059-006）。焉自求为[[益州]]牧，州[[从事]][[贾龙]]率[[吏卒]]迎焉，焉徙治[[绵竹]]，抚纳离叛（059-010）。刘焉在[[益州]]阴图异计，使[[张鲁]]督义众，断绝[[斜谷]]（060-022）。[[马腾]]攻[[李傕]]，刘焉二子范、诞皆死（061-027）。刘焉建议改置[[牧伯]]，开[[东汉]]末[[州牧]][[割据]]之端，其后[[刘表]]、[[袁绍]]等皆以[[州牧]]雄踞一方。
