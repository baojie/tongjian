# 韩当

韩当（？-227年），字义公，辽西令支人，[[三国]]时[[东吴]]名将。初从[[孙坚]]征伐周旋，陷敌擒虏，数被创。后随[[孙策]]平定[[江东]]，拜为校尉。[[孙策]]与[[太史慈]]战于神亭，韩当等十三骑从策（061-056）。

韩当历仕孙氏三世，与[[程普]]、[[黄盖]]等同为[[江东]]元勋宿将。从[[程普]]讨乐安，讨山越，悉平之（064-021）。[[赤壁之战]]中立有战功，后迁偏将军，领永昌太守。[[陆逊]]拒[[刘备]]于夷陵，韩当等五万人拒之（069-040）。当善于[[水战]]，长于弓马，膂力过人，为[[东吴]]倚重之将。[[孙权]][[称帝]]后，拜为昭武将军，封石城侯。
