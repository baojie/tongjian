# 韩当

韩当（？-227年），字义公，辽西令支人，[[三国]]时[[东吴]]名将。初从[[孙坚]]征伐周旋，陷敌擒虏，数被创。后随[[孙策]]平定江东，拜为校尉。

韩当历仕孙氏三世，与[[程普]]、[[黄盖]]等同为江东元勋宿将。赤壁之战中立有战功，后迁偏将军，领永昌太守。当善于水战，长于弓马，膂力过人，为东吴倚重之将。孙权称帝后，拜为昭武将军，封石城侯。
