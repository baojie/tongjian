# 冯奉世

冯奉世，字子明，上党潞人。西汉右将军，出使莎车，威震西域。

冯奉世使西域，以节发兵击莎车（025-046）。冯奉世为右将军，击羌有功（028-072）。冯奉世以功封关内侯（029-003）。

冯奉世少习《春秋》，读兵书。宣帝时，以卫候使西域。值莎车杀汉使，奉世以节发诸国兵万五千人击莎车，拔之。宣帝大喜，议封奉世。萧望之以为不可，乃止。元帝时，陇西羌反，奉世以右将军率兵讨平之，赐爵关内侯。奉世在西域威名甚著，为汉名将。
