# 清暑殿

**清暑殿**，[[东晋]][[建康]][[宫殿]]。[[晋孝武帝]]醉卧清暑殿，为张[[贵人]]所弑，是[[南北朝]]历史转折之[[事件]]。宋初犹存，后大修[[宫室]]。

## 通鉴记载

晋[[刘骏|孝武帝]]醉寝于清暑殿，为张[[贵人]]弑（108-110）。

[[王国宝]]起斋侔清暑殿，帝恶之（108-113）。

晋孝武末始作清暑殿，宋兴无所增改（129-123）。
