# 诸葛瑾

诸葛瑾（174-241年），字子瑜，琅邪阳都人，[[三国]]时[[东吴]]重臣，[[诸葛亮]]之兄。汉末避乱江东，为[[孙权]]所重。曾任长史、绥南将军，转左将军。

诸葛瑾为人谦厚，善于外交。孙权称帝后，拜诸葛瑾为大将军、左都护，领豫州牧。瑾与诸葛亮虽各为其主，而兄弟之情不绝。孙权曾言："孤与子瑜有死生不易之誓。"瑾品格高尚，深受孙权信任。

## 使蜀与辅政

诸葛瑾奉命使蜀，从刘备求荆州诸郡（067-021）。瑾复命，更寻盟好，遂分荆州（067-022）。刘备伐吴，瑾遗笺谏曰："陛下以关羽之亲，何如先帝？"（069-040）。瑾与陆逊等救江陵（069-075）。瑾闻吴主已还，欲急退，逊从容定计（072-058）。诸葛瑾历事孙权，忠谨谦厚，孙权深信任之。
