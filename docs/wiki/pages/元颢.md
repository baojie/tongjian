# 元颢

元颢，字子明，[[河南]][[洛阳]]人。[[北魏]][[宗室]]，[[北海]]王。

元颢取魏铚城而据之（152-059）。后元颢受[[萧衍|梁武帝]]支持，入洛[[称帝]]。及[[北魏]]孝庄帝北走，元颢新入[[洛阳]]，人情未安（153-018）。元颢受制于梁，引梁兵以覆宗国，魏臣多以为贼臣乱子（153-020）。[[尔朱荣]]攻[[洛阳]]，元颢败走，为人所杀。

> 元颢受制于梁，引寇仇之兵以覆宗国，此魏之贼臣乱子也。
> （153-020）
