# 僧寺

僧寺，[[佛教]]寺院。[[北魏]]诏量撤僧寺不急之华，还复百官久折之秩（148-092）。[[太平公主]]与僧寺争碾硙（208-102）。京畿良田美利多归僧寺（224-041）。

> 伏愿量撤僧寺不急之华，还复百官久折之秩。
> （148-092）

## 南北朝僧寺之盛

[[南北朝]]时期[[佛教]]鼎盛，僧寺遍布各地。[[北魏]]末年，[[洛阳]]僧寺多达一千余所，全国[[僧尼]]二百余万，寺院经济膨胀，与[[朝廷]]争夺劳动力与土地资源。[[张普惠]]曾上疏建议量撤僧寺不急之华，还复百官久折之秩（148-092），但收效甚微。唐代僧寺势力依然强大，京畿良田美利多归僧寺（224-041），[[朝廷]]虽多次限制，[[始终]]未能遏制。

## 僧寺与权贵

僧寺常与权贵发生利益冲突。[[太平公主]]与僧寺争碾硙，[[雍州]]司户[[李元纮]]判归僧寺（208-102），可见权贵亦不能完全压制寺院势力。[[唐代宗]]时，胡僧不空官至卿监，势移权贵，京畿良田美利多归僧寺（224-041），[[朝廷]][[敕令]]不得棰曳[[僧尼]]，寺院势力达到顶峰。
