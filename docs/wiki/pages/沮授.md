# 沮授

沮授，广平人。汉末[[袁绍]]帐下奋武将军，监护诸将。其谋略深远，为[[袁绍]]谋主，然绍不能用其策，[[官渡]]败后被[[曹操]]所擒，不降而死。

绍以广平沮授为奋武将军，使监护诸将，宠遇甚厚（060-012）。沮授说[[袁绍]]曰："将军累叶台辅，世济忠义。今朝廷播越，[[宗庙]]残毁，观诸州郡虽外托义兵，内实相图，未有忧存社稷恤民之意。"（061-053）沮授谏[[袁绍]]宜[[务农]]息民，先[[遣使]]献捷天子，若不得通，乃表[[曹操]]隔我王路（063-007）。

沮授在[[袁绍]]军中最为智谋之士，然[[郭图]]、[[审配]]等居中用事，沮授之策多见阻挠。[[官渡]]战前，沮授建议缓进稳守、持久疲敌，绍不从。及绍败，沮授为操所擒，操惜其才欲用之，授曰："叔父、母、弟悬命袁氏，若蒙公灵，速死为福。"遂不屈而死。
