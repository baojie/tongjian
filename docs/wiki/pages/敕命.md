# 敕命

**敕命**是[[皇帝]]的[[诏敕]]命令，为皇权运作的核心工具。

## 敕命之重

[[刘禅]]降魏，得汉主敕命，乃令兵悉放仗（078-029）。敕命一到，[[三军]]解甲。

## 违敕命

[[李元吉]]谮[[秦王]]曰，[[秦王]][[初平]][[洛阳|东都]]，顾望不还，违敕命非反而何（191-109）。[[宋璟]]廷审[[张昌宗]]，[[太后]]遣中使宣敕命赦之（207-126）。敕命为最高指令，违者获罪。
