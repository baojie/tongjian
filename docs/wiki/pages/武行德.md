# 武行德

武行德，[[榆次]]人，[[五代]][[后汉]]将领。为宁国都虞候，部送[[契丹]]铠仗，至河阴与将士谋杀[[契丹]]，据[[河阳]]归[[晋阳]]。为[[河阳节度使]]。[[契丹]]合兵逼[[河阳]]，出战不利，闭城自守。[[《资治通鉴》]] 载其事迹。
