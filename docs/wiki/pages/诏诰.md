# 诏诰

**诏诰**是皇帝颁布的诏书与诰命，为朝廷政令的核心载体。

## 掌诏诰者

徐勉居选官，国史、诏诰、仪体、法律、军旅谋谟皆掌之，二十馀年未尝离左右（145-069）。北周以李德林为内史上士，自是诏诰格式用山东人物（173-028）。

## 诏诰之重

北周诏曰大冢宰晋国公亲则懿昆，任当元辅，自今诏诰及百司文书并不得称公名（169-009）。可见诏诰用字之讲究，一字之尊卑皆关礼制。
