# 储君

储君，[[帝王]]指定的[[继承]]人，即[[太子]]。

[[郑众]]谓[[太子]]储君无外交之义（045-006）。[[诸葛恪]]论储君读书当务其急（074-006）。

储君为国本，历代皆重之。[[汉明帝]]时[[太子]]及[[山阳]]王荆因[[梁松]]请交[[郑众]]，众曰："[[太子]]储君，无外交之义，汉有旧防，蕃王不宜私通[[宾客]]。"其谨守[[法度]]如此。吴[[诸葛恪]]为[[太子]][[师傅]]，论储君读书当务其急，不可效[[博士]]探策讲试以求[[爵位]]。储君之贤否，关乎国运[[兴衰]]，故历代择师教之，选贤辅之。
