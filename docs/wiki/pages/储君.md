# 储君

储君，帝王指定的继承人，即太子。

郑众谓太子储君无外交之义（045-006）。诸葛恪论储君读书当务其急（074-006）。

储君为国本，历代皆重之。汉明帝时太子及山阳王荆因梁松请交郑众，众曰："太子储君，无外交之义，汉有旧防，蕃王不宜私通宾客。"其谨守法度如此。吴诸葛恪为太子师傅，论储君读书当务其急，不可效博士探策讲试以求爵位。储君之贤否，关乎国运兴衰，故历代择师教之，选贤辅之。

> 郑众谓太子储君无外交之义。
> （045-006）
