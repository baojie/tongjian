# 齐康公

**齐康公**，名贷，[[战国]]齐国君主，姜姓齐国末代国君。

[[田和]]迁齐康公于海上，使食一城，以奉其先祀（001-049）。康公卒，田氏遂有齐国（001-082）。

齐康公为姜太公之后，其失国标志着姜氏齐国终结、田氏齐国开始。[[司马光]]于"[[臣光曰]]"论[[田氏代齐]]，深讥周[[天子]]不能守先王之礼，以大夫而命为[[诸侯]]。

## 姜齐之终

[[齐康公]]名贷，[[姜姓]][[齐国]]末代[[国君]]。[[齐相]][[田和]]迁齐康公于海上，使食一城，以奉其先祀（001-049）。康公薨，无子，田氏遂并齐而有之（001-082）。自[[姜太公]]封齐，传至康公，姜氏之祀遂绝。[[司马光]]于"[[臣光曰]]"论[[田氏代齐]]，谓[[周]][[天子]]不能守先王之礼，以大夫而命为[[诸侯]]。齐康公以[[姜姓]]之裔失国海上，田氏遂有齐国（001-082）。
