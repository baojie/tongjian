# 慈爱

**慈爱**指上对下慈祥仁爱之心。《[[资治通鉴概述|通鉴]]》记载了诸多君主以慈爱待臣民的史事。

## 汉代

[[汉文帝]]慈爱[[百姓]]，省刑薄赋，天下颂之。其临终[[遗诏]]，亦以不扰民为念（009-018）。[[汉光武帝]]以柔道[[治国]]，[[宽仁]]待民，慈爱之心溢于政令之中（044-109）。光武之慈爱不仅施于[[百姓]]，亦待功臣以宽厚，故云台诸将皆愿效死。
