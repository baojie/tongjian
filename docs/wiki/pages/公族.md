# 公族

公族指诸侯国君的同[[宗族]]亲。[[战国]]变法常以削弱公族势力为要务，以加强中央集权。

[[吴起]]在楚为相，明法审令，捐不急之官，废公族疏远者以抚养战斗之士（001-060）。[[商鞅]]在秦变法，凌轹公族，残伤百姓，公子虔杜门不出已八年矣（002-074）。越王无彊伐楚失败后，诸公族争立，或为王，或为君，滨于海上（002-087）。

> （001-060）
