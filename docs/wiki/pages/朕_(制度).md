# 朕

朕原为普通第一人称代词，[[秦始皇]]统一天下后定为帝王专用自称，历代沿用。

[[秦始皇]]更号曰"皇帝"，自称曰"朕"，命为"制"，令为"诏"（007-031）。后慕真人，自称"真人"而不称"朕"（007-062）。

[[胡亥|秦二世]]、汉[[汉高祖刘邦|高祖]]、[[汉文帝]]等沿用此称。如二世曰"朕实贤之""朕[[即位]]，二年之间"（008-032）。[[汉高祖刘邦]]谓[[诸侯]]曰"朕自[[汉高祖刘邦|沛公]]以诛暴逆，遂有天下"（012-043）。[[汉文帝]]诏曰"法者，治之正也……朕甚不取"（013-069）。朕作为皇帝自称，使用逾两千年。

> 更号曰"皇帝"，命为"制"，令为"诏"，自称曰"朕"。
