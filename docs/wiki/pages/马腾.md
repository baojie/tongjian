# 马腾

马腾，字寿成，扶风[[茂陵]]人。汉末征西将军，西凉军阀，[[马超]]之父。后入朝为[[卫尉]]，因[[马超]]起兵反曹而[[族诛]]。

耿鄙司马扶风马腾亦拥兵反，与[[韩遂]]合，共推王国为主，寇掠三辅（058-097）。初，[[董卓]]入关，说[[韩遂]]、马腾与共图山东，遂、腾率众诣长安（060-050）。马腾私有求于[[李傕]]，不获而怒，欲举兵相攻，帝[[遣使]]者和解之（061-007）。

马腾雄据西凉，与[[韩遂]]结为异姓兄弟，后渐生嫌隙。[[曹操]]以朝廷名义征马腾入朝为[[卫尉]]，腾遂率家属入邺。后[[马超]]起兵反曹，[[曹操]]诛马腾三族。马氏世代将门，马腾起于[[凉州]]羌胡之间，以勇力知名。

> （058-097）
