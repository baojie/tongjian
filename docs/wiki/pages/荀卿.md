# 荀卿

**[[荀子|荀卿]]**（约前313—前238），名况，赵人，[[战国]]末期[[儒家]]思想家。尝与临武君论兵于[[赵孝成王]]之前，议[[王者]]之兵、[[仁义]]之师（006-004）。其言"凡君贤者其国治，君不能者其国乱；隆礼贵义者其国治，简礼贱义者其国乱"（006-005）。

[[楚国|楚]][[春申君]]以[[荀子|荀卿]]为[[兰陵]]令（006-004）。[[司马光]]于《[[资治通鉴概述|资治通鉴]]》中引[[荀子|荀卿]]论楚之[[得失]]，谓"楚之君诚得其道，臣诚得其人，秦虽强，乌得陵之哉！"（004-023）于[[荀子|荀卿]]"善用之则百里之地可以独立，不善用之则楚六千里而为仇人役"之说深有会心。

[[荀子|荀卿]]主性恶，尚[[礼法]]，与[[孟轲]]性善论相对，同为先秦[[儒学]]大宗。其[[弟子]][[李斯]]、[[韩非]]皆为[[法家]]巨擘。荀学承孔门而开[[法家]]，在[[中国]]思想史上居于枢纽地位。
