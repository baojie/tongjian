# 九宫

九宫是[[中国]]古代[[数术]]与[[历法]]中的重要概念。一为**九宫贵神**（[[太一]]、摄提、轩辕等九位神祇），掌司[[水旱]]，为[[朝廷]][[祭祀]]的重要对象；二为**九宫算数**，与遁甲术、[[历法]]推算密切相关。

## 通鉴中的九宫

[[东汉]]时[[刘秀]]（非[[光武帝刘秀|光武帝]]）论[[图谶]]之伪，列举历律、卦候、九宫、[[风角]]之术"数有征效"（052-005），此为[[资治通鉴概述|通鉴]]最早之九宫记载。[[唐玄宗]]时[[术士]]苏嘉庆上言"遁甲术有九宫贵神，典司[[水旱]]"（215-044），请立坛[[祭祀]]，此后[[朝廷]]四孟月皆祀九宫贵神（215-080）。[[唐肃宗]]亦祀九宫贵神（221-004）。

## 历史意义

九宫在[[中国]]古代从[[数术]]工具发展为官方[[祭祀]]对象。唐代对九宫贵神的尊崇一度达到"礼在昊天上帝下，太清宫、[[太庙]]上"（215-044）的崇高地位。

## 九宫: 数术祭祀与唐代国家宗教的演变

"九宫"在《[[资治通鉴概述|资治通鉴]]》中以两种面目出现——[[数术]]工具与[[国家]][[祭祀]]对象，其演变折射出[[唐代]][[官方]][[宗教]]政策的变化。[[张衡]]上疏批判[[图谶]]之学，以"律历、卦候、九宫、风角，数有征效"（052-005）并列——九宫在[[东汉]]时尚是数术家之术，[[张衡]]虽批评图谶却承认九宫等数术确有征效，暗示了九宫从数术向官方学术过渡的潜流。[[唐玄宗]]时术士苏嘉庆上言"遁甲术有九宫贵神，典司水旱，请立坛于东郊"（215-044）——九宫贵神被正式纳入[[国家]][[祭祀]]体系，其礼秩仅在[[昊天上帝]]之下、太清宫和太庙之上，标志着九宫从民间数术升格为官方[[崇拜]]的完成。此后[[玄宗]]亲祀九宫贵神（215-049），并下制"自今四孟月，皆择吉日祀[[天地]]、九宫"（215-080）——九宫祭祀被制度化、常态化。[[肃宗]]时亦祀九宫贵神（221-004, 221-062），安史乱后[[朝廷]]仍维持九宫祭祀，说明这一信仰已在[[唐廷]][[礼仪]]中扎下了根基。从[[张衡]]的学术讨论到[[玄宗]]的[[国家]]祭祀，九宫地位的跃升是[[唐代]]将民间[[宗教]]资源吸纳为官方[[意识形态]]的典型案例——[[朝廷]]通过将九宫贵神纳入[[祀典]]，既利用了数术的[[民间]]影响力，又强化了[[天子]]沟通[[天地]]的[[宗教]]权威。
