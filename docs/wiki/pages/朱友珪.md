# 朱友珪

朱友珪，[[后梁]]郢王，[[梁太祖|朱温]]第三子，弑父[[篡位]]，在位仅数月。

立皇兄全昱为广王，子友文为博王，友珪为郢王（266-033）。郢王友珪为[[左右]]控鹤[[都指挥使]]，无宠。友珪心不平，益不自安（268-063）。友珪[[矫诏]]称博王友文[[谋逆]]，赖郢王友珪[[忠孝]]将兵诛之（268-065）。友珪即[[皇帝]]位，时[[朝廷]]新有内难，中外人情忷忷（268-066）。

## 世系

朱友珪（884—913年），[[后梁]]郢王。父[[朱温]]（[[梁太祖|后梁太祖]]），[[朱温|温]]第三子。友珪弑父[[篡位]]（268-065），弟[[朱友贞]][[起兵]]讨之，友珪自杀（268-094）。
