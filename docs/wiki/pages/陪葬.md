# 陪葬

陪葬是大臣死后葬入皇帝陵区的殊荣，唐代尤盛。

[[李渊|唐高祖]]武德八年，诏勋戚亡者皆陪葬山陵（195-012）。名臣[[魏徵]]薨，太宗命百官[[九品]]以上赴丧，给羽葆[[鼓吹]]陪葬[[昭陵]]；其妻裴氏以"征平生俭素，今葬以一品羽仪，非亡者之志"辞不受（196-061）。[[长孙无忌]]获罪死后，唐廷追复其官爵，听[[长孙无忌]]丧归陪葬[[昭陵]]（202-040）。陪葬不仅是君臣关系的体现，也是朝廷褒奖功勋的重要方式。

## 唐代陪葬制度

[[李渊|唐高祖]]诏勋戚亡者皆陪葬山陵，陪葬制度始盛（195-012）。[[魏徵]]卒，太宗欲以衡山[[公主]]妻其子叔玉，命百官[[九品]]以上赴丧，给羽葆[[鼓吹]]陪葬[[昭陵]]，裴氏辞以俭素不受一品仪仗（196-061）。[[长孙无忌]]为太宗长孙皇后之兄，以谋反被贬死，后唐廷追复其官爵，听丧归陪葬[[昭陵]]（202-040）。唐代陪葬彰显君臣一体之义，帝王陵园中大臣陪葬墓之多寡，往往体现一朝政治之气象。[[昭陵]]陪葬墓达百余座，为唐代之最。

> 诏勋戚亡者皆陪葬山陵。
