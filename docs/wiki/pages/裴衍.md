# 裴衍

裴衍，[[北魏]]右[[都督]]。率军讨六镇起义军[[鲜于修礼]]。

裴衍为右[[都督]]，率军讨[[鲜于修礼]]（150-096）。衍与[[长孙稚]]等分道进攻，为修礼所拒（150-099）。后衍与元志、[[李神轨]]等皆讨贼有功（151-018）。

> 裴衍为右都督，讨鲜于修礼。
> （150-096）
