# 周舍

周舍，梁右卫[[将军]]。与[[徐勉]]同参国政，俱称贤相。清简过人。

[[汉文帝]]时，[[中尉]]周舍曾为[[将军]]备胡（015-035）。梁时，周舍与[[徐勉]]同参国政。舍[[雅量]]不及勉，而清简过之。两人俱称贤相，常留省内，罕得休下（145-069）。[[武帝]]使[[舍人]]周舍慰劳魏使，曰："战争多年，民物涂炭，吾是以不耻先言与魏朝[[通好]]"（147-042）。

> 周舍同参国政。舍雅量不及勉，而清简过之，两人俱称贤相。
