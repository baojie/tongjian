# 韩嫣

韩嫣，[[汉武帝]][[宠臣]]，[[韩王信]]之孙。

[[武帝]]时韩嫣以巧佞得幸，与[[邓通]]并称。后世论[[宠臣]]之害，常以[[邓通]]、韩嫣为例（035-005、035-021）。韩嫣尝受[[武帝]]宠赐，恃恩[[骄纵]]，终以罪诛（050-111）。

## 世系

韩嫣，[[西汉]][[汉武帝]][[宠臣]]，[[韩王信]]之孙。[[韩王信]]为[[汉初]]所封[[韩王]]，后降[[匈奴]]。嫣以[[巧佞]]得幸，与[[邓通]]并称（035-005、035-021）。恃恩[[骄纵]]，终以罪诛（050-111）。嫣[[子孙]]不详于史。
