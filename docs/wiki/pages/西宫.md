# 西宫

**西宫**为[[汉代]]皇宫建筑群之一，与[[东宫]]相对，常为[[太后]]或[[皇后]]所居。[[汉代]][[未央宫]]、[[长乐宫]]各有东西宫之别。后世"西宫"亦指[[妃嫔]]居所。
