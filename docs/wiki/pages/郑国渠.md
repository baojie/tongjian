# [[郑国渠]]

[[郑国渠]]是[[战国]]时期[[秦国]]修建的大型灌溉渠道。[[韩国]]派遣水工郑国入秦，劝说秦国开凿水渠以消耗其国力，使秦无力东伐。工程进行中计谋被发觉，秦王欲杀郑国，郑国称此举虽为"延韩数岁之命"，但渠成亦为"万世之利"，遂得继续。

渠成后引泾水灌溉关中[[平原]]四方余顷农田，使关中成为沃野，无凶年，为秦统一[[天下]]奠定物质基础。该渠因此被命名为郑国渠，是中国历史上最早的大型水利工程之一，与[[都江堰]]、灵渠并称。
