# 刘寔

刘寔，字子真，平原人，西晋司空，以识鉴著称。

或以问参相国军事刘寔曰钟会、邓艾其平蜀乎。寔曰："破蜀必矣，而皆不还。"客问其故，笑而不答（078-015）。武帝为皇孙遹选僚佐，以散骑常侍刘寔志行清素，命为广陵王傅（082-008）。以右光禄大夫刘寔为司空（083-044）。

刘寔以知几预见闻名，其论钟、邓之事，时人以为神鉴。
