# 太常

太常是秦汉九卿之一，掌[[宗庙]]礼仪，为九卿之首，原名奉常，[[汉景帝]]时改称太常。

秦汉奉常掌[[宗庙]]礼仪，位列诸卿之首，属官有太乐、太祝、太宰、太史、太卜、[[太医]]等（011-045）。[[汉景帝]]中元六年更名太常。

太常负责祭祀天地、[[宗庙]]、社稷等国家大典，兼掌文化教育。汉代太常还负责[[博士]]官的选拔和管理，在国家[[礼乐]]文化体系中居于核心地位。

## 太常任职

汉文帝拜袁盎为太常，使报吴王（016-015）。武帝以太常柏至侯许昌为丞相（017-023）。太常奏公孙弘对策居下，而武帝擢弘为第一（018-033）。孔臧辞御史大夫，愿为太常典臣家业，其礼赐如三公（018-076）。太常虽掌礼仪，然常为重臣迁转阶梯，汉代名臣多由太常入相。
