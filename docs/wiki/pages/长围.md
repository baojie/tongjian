# 长围

**长围**指修筑长达数里甚至数百里的[[围城]]工事，为古代[[军事]]中长期围困敌军[[城池]]的重要战术手段。

## 筑长围守之

尚频为特所败，乃阻长围，缘郫水作营，连延七百里（084-042）。虎筑长围守之。龛[[坚守]]不战（092-014）。宣所舍，辄列人为长围，四面各百里，驱[[禽兽]]，至暮皆集其所（097-107）。不过十旬，取之必矣，何为多杀[[士卒]]以求旦夕之功乎！乃筑长围守之（101-023）。乃筑长围守之，分处老弱于肥乡，筑新兴城以置[[辎重]]（105-046）。
