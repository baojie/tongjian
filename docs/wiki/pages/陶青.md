# 陶青

陶青，[[西汉]][[左丞相|丞相]]，袭封[[开封]]侯。

[[汉景帝]]初年，以[[御史大夫]]陶青为[[左丞相|丞相]]（015-089）。后[[左丞相|丞相]]陶青免，[[周亚夫]]代为[[左丞相|丞相]]（016-054）。

## 世系

陶青，[[西汉]][[丞相]]。父[[陶舍]]，以从[[刘邦]]定[[天下]]功封[[开封侯]]。青[[袭爵]][[开封]]侯。仕[[汉景帝]]，由[[御史大夫]]迁[[丞相]]（015-089）。后以[[丞相]]免（016-054）。青[[子孙]]不详于史。
