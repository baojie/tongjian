# 盟约

盟约是双方缔结的[[盟誓]]条约，为古代邦交与政治联合的重要方式。

[[郦食其]]说[[汉高祖刘邦|刘邦]]，谓[[项羽]][[背盟]]约而杀义帝，天下负之不义之名（010-008）。[[刘縯]]大飨军士设盟约，休卒三日，潜师夜起袭取蓝乡（038-077）。

春秋时期盟会之风最盛，诸侯间以歃血为盟、书于丹书的形式缔结盟约。汉代以后盟约虽不如春秋盛行，但在政治军事联合中仍发挥重要作用。[[背盟]]被视为严重背信行为，有损国际信誉。

> 縯大飨军士，设盟约，休卒三日，分为六部。十二月，晦，潜师夜起，袭取蓝乡，尽获其辎重。
> （038-077）
