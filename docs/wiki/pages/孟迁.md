# 孟迁

孟迁，晚唐[[昭义节度使]]。

[[李克用]]急攻邢州，孟迁食竭力尽，以邢、洺、磁州降于[[李克用]]（258-029）。克用以迁为军城[[都虞候]]（258-046）。后[[李克用]]表孟迁为[[昭义]][[留后]]（261-125），迁为[[魏博节度使|节度使]]（262-013）。后[[梁太祖|朱全忠]]遣[[氏叔琮]]攻[[潞州]]，孟迁降之（262-051）。

## 世系

孟迁，[[唐]]末[[昭义节度使]]。其先世不详于史。[[李克用]]急攻[[邢州]]，迁食竭力尽，以邢、洺、磁三州降（258-029）。克用表迁为[[昭义]][[留后]]（261-125），迁为[[节度使]]（262-013）。后[[朱全忠]]遣[[氏叔琮]]攻[[潞州]]，迁又降之（262-051）。迁[[子孙]]不详于史。
