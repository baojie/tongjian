# [[乐都]]

[[乐都]]位于今[[青海]]乐都，地处[[湟水]]流域，是[[汉代]][[经略]]河湟地区的重要据点。[[西汉]]时置县，属[[金城郡]]。乐都地处[[河西走廊]][[东南]]部与[[青海湖]]之间，控扼[[中原]]通往[[西海郡|青海地区]]的[[交通]]线。

在《[[资治通鉴概述|资治通鉴]]》记载的[[汉代]]至[[十六国]]时期，乐都是[[凉州]]与[[陇西]]之间[[湟水]]流域的重要城镇。
