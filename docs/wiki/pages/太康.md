# 太康

**太康**（280—289），[[晋武帝]][[司马炎]]第三个[[年号]]。太康元年，[[王濬]][[楼船]]入[[建业]]，[[孙皓]]面缚出降，吴亡，天下统一。[[武帝]]怠于政事，耽于酒色，后宫万人。[[王恺]]与[[石崇]]斗富，[[奢侈]]成风。[[刘毅]]谏[[武帝]]不如[[桓灵]]，[[武帝]]不以为忤。
