# 万岁殿

**万岁殿**，[[东汉]]至[[五代]]时期[[洛阳]]重要宫殿。[[后汉隐帝]]崩于万岁殿；[[后周太祖]][[郭威]]亦召将相宴会于此，以安群心。

## 通鉴记载

东汉千秋万岁殿火（055-020）。

后汉隐帝殂于万岁殿（287-088）。

后周太祖召诸军将校至万岁殿庭，亲谕安定（289-047）。

后周太祖与将相食于万岁殿（292-058）。
