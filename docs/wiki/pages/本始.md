# 本始

**本始**（前73—前70），[[汉宣帝]]第一个[[年号]]。宣帝自民间[[即位]]，本始年间褒赏[[霍光]]等定策功臣，[[追尊]]祖母[[卫太子]]妃史良娣、父史皇孙。[[匈奴]]与汉争[[车师]]，汉发兵击之。宣帝兴文教，[[诏令]]举[[贤良]][[方正]]，询求[[治国]]之道。
