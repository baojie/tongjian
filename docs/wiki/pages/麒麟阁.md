# [[麒麟阁]]

[[麒麟阁]]是[[西汉]][[未央宫]]中的著名阁楼建筑。[[汉宣帝]]甘露三年（前51年），为纪念[[中兴]]辅佐之臣，命画工绘制[[霍光]]、[[张安世]]、[[赵充国]]等十一功臣画像于麒麟阁，是为"麒麟阁十一功臣"。此举开创了后世"功臣画像"的传统，[[东汉]]的[[云台二十八将]]、[[唐代]]的[[凌烟阁二十四功臣]]皆仿此制。
