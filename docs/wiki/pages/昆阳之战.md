# 昆阳之战

昆阳之战是[[王莽|新莽]]地皇四年（23年），更始政权的将领[[刘歆|刘秀]]在昆阳（今[[河南]]叶县）以少胜多击败[[王莽]]主力王邑、王寻大军的决定性战役。

更始元年，[[刘歆|刘秀]]与王凤等攻占昆阳等地（039-007）。[[王莽]]遣大[[司徒]]王寻、大[[司空]]王邑率精兵四十二万围攻昆阳，声势浩大。诸将见敌兵盛，皆惶怖欲散归，[[刘歆|刘秀]]力排众议，坚守待援（039-009）。

[[刘歆|刘秀]]率十三骑突围求援，后亲率步骑千余为前锋，猛击敌军中坚。城中守军鼓噪而出，内外合击，莽军大溃。此役奠定了推翻[[王莽|新莽]]政权的基础，也是[[刘歆|刘秀]]崭露头角的成名之战。后世论者常以昆阳之战与[[官渡之战]]相提并论（094-043）。
