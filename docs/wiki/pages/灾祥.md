# 灾祥

灾祥，灾异与[[祥瑞]]，古人以为天人之际感应之征象。

秦祝官有灾祥移过于下（015-030）。庾季才以纬候灾祥劝宣政归权（171-020）。

灾祥之说源于天人感应。秦时祝官有秘祝，遇有灾祥辄移过于下。[[汉文帝]]以为非，诏除之。北周时庾季才以纬候灾祥，言于周宣帝曰"天象示变，宜返政归权"。灾祥之为物，既可以警惕君心，亦可以为奸人所假托。两汉之际，[[谶纬]]蜂起，灾祥之说尤盛。

> 秦祝官有灾祥移过于下。
> （015-030）
