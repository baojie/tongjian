# 铁勒

铁勒，匈奴之遗种，北方游牧部族总称。族类最多，有仆骨、同罗、契苾、薛延陀等部，其酋长皆号俟斤。族姓虽殊，通谓之铁勒，大抵与突厥同俗（180-052）。

北魏时，宇文肱之父与铁勒战死（150-036）。贺拔岳西屯平凉，铁勒斛律沙门等皆附之（156-022）。西魏时，宇文泰率李弼等十二将伐东魏，以铁勒骑为前锋（149-083）。突厥兴起前，铁勒将伐柔然，突厥酋长土门邀击破之（164-035）。
