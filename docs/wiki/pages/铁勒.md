# 铁勒

铁勒，[[匈奴]]之遗种，北方游牧部族总称。族类最多，有仆骨、[[同罗]]、[[契苾]]、[[薛延陀]]等部，其酋长皆号俟斤。[[族姓]]虽殊，通谓之铁勒，大抵与[[突厥]]同俗（180-052）。

[[北魏]]时，宇文肱之父与铁勒战死（150-036）。[[贺拔岳]]西屯[[平凉]]，铁勒[[斛律]][[沙门]]等皆附之（156-022）。[[西魏]]时，[[宇文泰]]率[[李弼]]等十二将伐[[东魏]]，以铁勒骑为前锋（149-083）。[[突厥]]兴起前，铁勒将伐柔然，[[突厥]]酋长土门邀击破之（164-035）。

> 铁勒者，匈奴之遗种，族类最多，有仆骨、同罗、契苾、薛延陀等部，其酋长皆号俟斤。
