# 叔向

叔向，羊舌氏，名肸，晋大夫。[[春秋]][[晋国]]贤大夫，以正直博学著称。

叔向为晋大夫，以贤称（001-035）。

叔向为晋平公大夫。齐使[[晏婴]]使晋，与叔向语，曰："齐政将归陈氏。"叔向曰："然。晋公室亦卑，戎马不驾，卿无军行。"二大夫相与太息。叔向以正直闻于诸侯。楚王问叔向曰："晋其无人乎？"对曰："有之。"[[使者]]至晋，晋以叔向为主宾，楚人服其应对。

> 叔向为晋大夫，以贤称。
> （001-035）
