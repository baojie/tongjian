# [[西明门]]

[[西明门]]是[[西晋]]都城[[洛阳]]的西门之一。[[永嘉]]五年（311年），[[汉]]（[[前赵]]）[[刘聪]]遣[[呼延晏]]、[[刘曜]]、[[王弥]]等率军攻洛阳。刘聪军至宜阳，屯西明门（087-020）。

[[北宫纯]]夜率勇士千余人出攻汉壁，然终未能挡。刘聪军再攻西明门，克之（087-038），遂入洛阳，俘[[晋怀帝]]，纵兵大掠，史称"[[永嘉之乱]]"。西明门的失守标志着西晋灭亡的开端。
