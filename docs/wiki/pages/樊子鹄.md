# 樊子鹄

樊子鹄，北魏骠骑大将军、东南道大行台。从尔朱荣入洛，讨平诸乱。

尔朱荣遣其都督樊子鹄取唐州，子鹄拔平阳，斩元珍及恽（152-040）。魏遣都官尚书平城樊子鹄等讨吕文欣，斩文欣（154-004）。以前御史中尉樊子鹄兼尚书左仆射，为东南道大行台（155-076）。魏东南道大行台樊子鹄围元树于谯城（155-085）。魏以骠骑大将军樊子鹄为青、胶大使（156-020）。齐州刺史侯渊、兗州刺史樊子鹄阴相连结（156-077）。

> 魏东南道大行台樊子鹄围元树于谯城，分兵攻取蒙县等五城，以绝援兵之路。
> （155-085）
