# 东胡

东胡是中国古代东北游牧民族，因居[[匈奴]]（胡）之东得名，[[匈奴]]兴起前活跃于蒙古高原东部。

[[赵武灵王]]胡服骑射，谏者以为"吾国东有齐、中山，北有燕、东胡"，若无骑射之备则何以守之（003-054）。[[战国]]末期，[[赵国]]名将[[李牧]]大破[[匈奴]]，灭襜褴，破东胡，降林胡。单于奔走，十馀岁不敢近赵边（006-056）。

东胡在秦汉之际被[[匈奴]]冒顿单于击破，余部退保[[鲜卑]]山和乌桓山，后演化为[[鲜卑]]和乌桓两大民族。东胡系民族对中国历史影响深远，南北朝时[[鲜卑]]建立多个政权，[[契丹]]、蒙古亦属东胡语族。
