# 李息

李息，郁郅人。西汉材官将军，从卫青击匈奴。

李息为材官将军，从卫青击匈奴（018-007）。李息出代郡击匈奴（018-047）。

李息初以良家子从军。武帝时，为材官将军，从大将军卫青出击匈奴。息出代郡，斩捕首虏有功。又尝为将军，筑城于朔方。息以功赐爵关内侯。李息虽为将，而名亚于卫青、霍去病，然亦为武帝时数出塞之宿将。
