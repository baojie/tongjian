# 神爵

**神爵**（前61—前58），[[汉宣帝]]第四个[[年号]]。神爵年间，[[赵充国]]年七十余，率兵击[[西羌]]，以[[屯田]]之策坐制[[羌人]]，[[罢兵]]留屯，寓兵于农，[[边境]]得安，为后世[[屯田]]之典范。汉廷在[[乌垒]]置[[西域都护]]，[[郑吉]]为第一任[[都护]]，汉始[[号令]][[西域]]。
