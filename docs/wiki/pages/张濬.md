# 张濬

张濬，唐代[[谏议]][[大夫]]。说平卢[[节度使]]王敬武讨[[黄巢]]。

[[王鐸]]遣[[都统]][[判官]]、[[谏议]][[大夫]]张濬往说平卢[[节度使]]王敬武讨[[黄巢]]，濬见敬武，责之曰："公为[[天子]]籓臣，奈何侮慢天使！"敬武愕然，即谢，遣兵从濬而西（255-028）。
