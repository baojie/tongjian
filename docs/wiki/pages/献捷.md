# 献捷

献捷，军队获胜后向[[天子]]进[[献俘]]虏战利品、[[告捷]]之礼。[[袁绍]]欲先[[遣使]]献捷于[[天子]]（063-007）。诏以[[征蜀]]诸将献捷交至，复命[[大将军]]昭进位（078-022）。[[石勒]][[遣使]]奉[[王浚]]首献捷于汉（089-015）。

> 勒至襄国，遣使奉王浚首献捷于汉，汉以勒为大都督、督陕东诸军事、骠骑大将军、东单于。
> （089-015）
