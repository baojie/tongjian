# 督邮

督邮，汉[[代郡]]守属吏，掌督察属县、宣达教令。[[黄霸]]为[[颍川]]太守，许丞老病聋，督邮白欲逐之，霸以许丞廉吏不许（025-066）。宋均为[[九江]]太守，悉省[[掾史]]，闭督邮府内（045-021）。后汉末，[[山阳]]太守翟超以[[张俭]]为东部督邮，举奏中常侍[[侯览]]罪（055-050）。

> 许丞老，病聋，督邮白欲逐之。霸曰："许丞廉吏，虽老，尚能拜起送迎，正颇重听何伤！"
> （025-066）
