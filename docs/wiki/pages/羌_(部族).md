# 羌

羌是古代中国西部的游牧民族，与[[汉高祖刘邦|汉王]]朝长期保持复杂关系。[[汉文帝]]时[[地震]]，"羌道、武都道山崩"（013-011），羌道为[[西汉|汉朝]]管理羌人的县级政区。[[司马相如]]谏猎，言[[车骑]]出行可能惊扰"羌、夷接轸"（017-040）。

[[汉武帝]]通[[西域]]，"欲从羌中归"（018-083），可见羌地为[[西域]]与[[中原]]之间的通道。严安[[上书]]论征伐之弊，言武帝"降羌、僰"（018-055），开边不已。羌人在两汉之际渐成边患，至[[东汉]]中后期更爆发大规模羌乱，成为[[西汉|汉朝]]衰亡的重要原因。
