# 时会

**时会**指时间与际遇的巧合，[[资治通鉴概述|通鉴]]中时会往往关系重大历史转折。

## 天时之合

[[李广]]以百骑遇[[匈奴]]数千骑，时会暮，[[胡兵]]终怪之，不敢击，夜半引兵而去（016-095）。时会乃天时、人事交会之机。

## 人事之会

[[邓艾]]破蜀，[[刘禅]]既降，时会方给[[姜维]]铠仗，[[姜维]]闻变，欲乘机复国（078-046）。时会之巧合，常出人意表。
