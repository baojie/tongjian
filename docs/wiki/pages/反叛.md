# 反叛

**反叛**指臣民或属部起兵反抗朝廷统治，在古代被视作大逆不道的重罪。

## 边民反叛

自西南夷反叛以来，积且十年，郡县距击不已，民力疲敝（038-014）。边民反叛，多因吏治苛暴。

蛮夷数为小吏黠人所见侵夺，穷恚无聊，故致反叛（042-054）。反叛之由，多起于吏治。

## 谋逆不轨

周飏屡谋反叛，左右请杀之。苻坚曰："孟威烈士，秉志如此，岂惮死乎！杀之适足成其名耳"（104-091）。赦反叛以显大度。

谢灵运率众反叛，廷尉奏论正斩刑（122-103）。反叛者必伏诛。

## 防叛之策

凉州刺史袁翻议处柔然降户，请严戒边兵，因令防察，使疏不容反叛，最策之得者也（149-079）。防叛以严边备。
