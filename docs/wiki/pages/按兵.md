# 按兵

**按兵**指止兵不进、持重待机，为古代[[军事]]中以静制动的常用策略。

## 按兵持重

[[诸葛亮]]说[[孙权]]："若能以吴、越之众与[[中国]][[抗衡]]，不如早与之绝；若不能，何不按兵束甲，北面而事之"（065-047）。按兵为审时度势。

[[司马楚之]]谓[[刘裕]]大众远来，当按兵坚阵，勿与交锋，彼求战不得，[[自然]]散走（113-054）。按兵以挫敌锐。

[[李世民]]追薛仁果，[[窦轨]]请且按兵以观之，世民曰："破竹之势，不可失也"（186-042）。按兵与[[进取]]各有时宜。

## 按兵不救

[[刘敬]]玄懦怯，闻[[刘审礼]]战没，按兵不救，[[狼狈]]还走（202-107）。按兵不救，怯懦之失。

[[李孝逸]]与[[李敬业|徐敬业]]战不利，按兵不进，[[监军]][[魏元忠]]劝之乃进（203-091）。按兵不进须有决断。
