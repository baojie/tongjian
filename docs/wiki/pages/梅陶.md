# 梅陶

梅陶，[[东晋]][[尚书]]，以识鉴著称。

咨议[[参军]]梅陶说[[王敦]]曰："[[周访]]与侃亲姻，如左右手，安有断人左手而右手不应者乎！"敦意乃解（089-051）。[[陶侃]]薨，[[尚书]]梅陶与亲人曹识书曰："陶公机神明鉴似魏武，忠顺勤劳似[[诸葛亮|孔明]]，[[陆抗]]诸人不能及也。"（095-052）。
