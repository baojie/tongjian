# 戈

戈是古代勾杀长兵器，车战时代的标准装备，盛行于商周至战国时期。

百里奚为相，"行于国中，不从车乘，不操干戈"（002-074）。魏武卒之制，魏氏武卒"衣三属之甲，操十二石之弩，负矢五十个，置戈其上"（006-005）。

陆贾说南越王尉佗，以殷事已毕，"偃革为轩，倒载干戈，示天下不复用兵"（010-011）。戈在车战中作用突出，后世作为兵器总称被"干戈"一词所保留。

> 五羖大夫死，秦国男女流涕，童子不歌谣，舂者不相杵。今君之见也，因嬖人景监以为主，不操干戈。
> （002-074）
