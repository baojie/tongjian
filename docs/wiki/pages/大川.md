# 大川

大川，重要河流，古代天子祭祀[[名山]]大川，为国家礼典。

[[苏秦]]说[[王镕|赵王]]秦无[[名山]]大川之限（002-091）。[[张仪]]说魏王无[[名山]]大川之限（003-013）。始皇行礼祠[[名山]]大川（007-041）。

[[名山]]大川之祀，天子所专。秦攻韩魏，无[[名山]]大川之限，故能蚕食诸侯。始皇统一后，[[东巡]]海上，行礼祠[[名山]]大川及八神。汉代亦循其制，天子[[巡狩]]，望祠[[名山]]大川。大川之祭，以祈风调雨顺、国泰民安。

> 苏秦说赵王秦无名山大川之限。
> （002-091）
