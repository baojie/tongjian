# 解甲

**解甲**指解除铠甲，为休战或投降的象征性动作。

## 解甲归降

张伯路以军未解甲，不敢归降，法雄曰兵凶器战危事，当以德怀之（049-093）。解甲为归降之信。

杨肇攻陆抗不胜，吴兵追击，肇众凶惧，悉解甲挺走（079-133）。解甲而走，溃败之状。

刘裕讨卢循，斩徐赤特，解甲久之乃出陈（112-024）。解甲整军，再战之备。

## 解甲归顺

萧斌宣令所统，皆使解甲，自石头戴白幡来降（127-045）。解甲以降，白幡为信。

殷孝祖闻废帝已陨，即解甲下标（130-071）。解甲下标，从新主之命。
