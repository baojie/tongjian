# 赵贵

赵贵，字元贵，天水南安人。北周柱国大将军，[[宇文泰]]元从功臣。后以谋诛[[宇文护]]事泄被杀。

魏贺拔岳将讨曹泥，使[[都督]]武川赵贵至夏州与[[宇文泰]]谋之（156-034）。赵贵曰："宇文夏州英略冠世，远近归心，赏罚严明，士卒用命。若迎而奉之，大事济矣。"（156-035）[[宇文泰]]令卫将军赵贵行秦州事（156-051）。

赵贵最早劝[[宇文泰]]代统贺拔岳之众，为[[宇文泰]]创业的关键推手。西魏建立后，赵贵以功封柱国大将军。[[宇文泰]]去世后，赵贵不满[[宇文护]]专权，与[[独孤信]]谋诛之，事泄被杀。赵贵之死，为北周初年权力洗牌的重要事件。
