# 刑法

刑法是古代国家治理的重要工具，[[《资治通鉴》]]中刑法的宽严反映政治清明与否。

[[荀子]]论[[秦国]]之强，谓其以庆赏[[刑罚]]治民，使民要利于上者非斗无由（006-005）。[[汉宣帝]]时好用刑法，任中书官，[[盖宽饶]]上封事曰："方今圣道浸微，[[儒术]]不行，以刑馀为周、召，以法律为《诗》《书》。"（026-031）。

> 故制号政令，欲严以威；庆赏刑罚，欲必以信。
> （006-006）

> 时上方用刑法，任中书官，宽饶奏封事曰："方今圣道浸微，儒术不行，以刑馀为周、召，以法律为《诗》《书》。"
> （026-031）
