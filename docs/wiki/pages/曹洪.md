# 曹洪

曹洪，字子廉，沛国谯人。曹魏都护将军，[[曹操]]从弟。从[[曹操]]起兵，多立战功。

操乃遣扬武中郎将曹洪将兵西迎天子（062-010）。操大喜，乃留曹洪、[[荀攸]]守营（063-031）。

曹洪从[[曹操]]起兵讨[[董卓]]，[[荥阳]]之战[[曹操]]失马，洪以已马授操，曰："天下可无洪，不可无君。"从征[[徐州]]、[[官渡]]，皆有功。曹洪家富而吝啬，[[隋文帝|文帝]][[曹丕]]时以吝啬获罪，无敢救者。卞[[太后]]怒曰："使曹洪今日死，吾明日敕帝废后矣。"乃得免。[[汉明帝|明帝]]时拜卫将军。

> 操乃遣扬武中郎将曹洪将兵西迎天子，董承等据险拒之，洪不得进。
> （062-010）
