# 笏

笏是古代朝臣手持的记事手板，用以记录[[君王]]旨意或备忘，为[[朝仪]][[礼制]]的重要组成部分。

[[三国]][[孙权|吴大帝]][[孙权]]即[[皇帝]]位，[[张昭]]举笏欲褒赞[[功德]]，未及言，[[孙权]]曰："如张公之计，今已乞食矣"，昭大惭伏地流汗（071-023）。[[北周]]初令百官执笏（169-047），后诏内外命妇皆执笏，其拜[[宗庙]]及天台皆[[俯伏]]如男子（174-010）。

[[北周]][[李穆]]一门贵盛，[[子孙]]虽在襁褓悉拜仪同，一门执[[象笏]]者百馀人（175-015）。[[刘行本]]以事忤[[隋文帝]]，置笏于地而退，帝敛容谢之（175-040）。[[柳彧]]治[[杨素]]罪，于阶下端笏整容，素遽下（179-072）。唐代结骨请除一官，"执笏而归，诚百世之幸"（198-108）。笏以材质分品级，[[品官]]用[[象笏]]，卑者用竹木。

> 张昭举笏欲褒赞功德，未及言，吴主曰："如张公之计，今已乞食矣。"昭大惭，伏地流汗。
