# 姚思廉

姚思廉，名简，（一说本名简，字思廉），吴兴武康人。唐史学家，撰《梁书》《陈书》。

姚思廉为太宗朝弘文馆学士（184-046）。阎立本、姚思廉皆为相（189-076）。姚思廉预修梁陈二史（191-125）。

姚思廉为姚察之子。父察在陈朝修梁陈二史未成，临终属思廉继其业。思廉入唐后，受太宗之命，续成父志。贞观中，与魏征同修梁陈二史，思廉据其父旧稿，兼采他书，成《梁书》五十六卷、《陈书》三十六卷。思廉亦为弘文馆学士，预修国史。

> 姚思廉为太宗朝弘文馆学士。
> （184-046）
