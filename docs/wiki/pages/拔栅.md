# 拔栅

**拔栅**指拔除敌军设立的栅栏[[防御]]，为古代军队突[[破敌]][[营垒]]防线的重要战术行动。

## 拔栅攻坚

充、凤至宣阳门，拔栅将战（093-021）。贼皆越堑拔栅，僧珍分人上城（144-051）。[[敢死]]之士拔栅斫桥（146-095）。菩萨等已拔栅而还（154-007）。

## 拔栅突围

董秦帅其众五百，拔栅[[突围]]，降于光弼（221-048）。晟怒，万顷惧，帅众先进拔栅而入（231-011）。
