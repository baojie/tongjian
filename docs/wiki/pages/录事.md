# 录事

录事，录事参军之省称，掌总录众曹[[文簿]]、举弹善恶。[[司马睿]]参佐多避事自逸，录事参军陈頵言于睿，以为洛中承平之时朝士以小心恭恪为凡俗（088-065）。[[前秦]]时，[[张天锡]]会官属谋入朝，禁中录事席仂献计以子[[为质]]（104-008）。苻丕以垣敞为录事参军（104-071）。

> 录事参军陈頵言于睿曰："洛中承平之时，朝士以小心恭恪为凡俗，以偃蹇倨肆为优雅。"
> （088-065）
