# 辛雄

辛雄，北魏三公郎中、吏部郎中。以才学见留，屡上疏言时政得失。

三公郎中辛雄奏理匡，以为"历奏三朝，骨鲠之迹，朝野具知"（149-017）。魏以郎选不精，大加沙汰，唯辛雄等八人以才用见留（149-022）。司空长史辛雄为行台左丞（150-096）。辛雄曰："今裴衍未至，王士众已集，蛮左唐突，挠乱近畿，王秉麾阃外，见可而进，何论别道"（150-097）。辛雄自军中上疏曰："凡人所以临陈忘身，触白刃而不惮者，一求荣名，二贪重赏，三畏刑罚"（150-098）。吏部郎中辛雄上疏以为"夷夏之民相聚为乱，岂有馀憾哉！正以守令不得其人，百姓不堪其命故也"（151-042）。

## 直言时政

辛雄以三公郎中屡上疏论事，理匡冤狱，朝野称之（149-017）。以才学见留，沙汰中独得保全（149-022）。行台左丞任内，上书论赏罚之道，以为临阵忘身者求荣名、贪重赏、畏刑罚三者而已（150-098）。又上疏论天下之乱源于守令不得其人，百姓不堪其命，请慎选牧守（151-042）。
