# 保傅

**保傅**是古代负责教养辅导[[太子]]及贵族的官员。保者护其身体，傅者傅其[[德义]]，合称保傅。

## 教养太子

[[司马光]]论古之明王教养[[太子]]，为之择[[方正]]敦良之士，以为保傅、师友，使[[朝夕]]与之游处，[[左右]]前后无非正人（022-053）。可见保傅在[[储君]]教育中的重要地位。

## 选任之重

[[张皓]]议[[太子]]事，曰宜选[[忠良]]保傅，辅以[[礼义]]（050-137）。[[王莽]]亦引保傅之制，以安[[社稷]]、全保傅为辞（035-044）。历史上保傅的[[选任]]常被视为[[国家]]根本大计。
