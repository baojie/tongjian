# 金吾卫

金吾卫，唐代十六卫之一，掌宫中及京城巡警。高宗时，以右金吾卫将军庞同善为行军总管讨高丽（201-073）。突厥寇边，遣左金吾卫将军曹怀舜屯井陉（202-133）。契丹李尽忠反，以右金吾卫大将军张玄遇等二十八将讨之（205-094）。武则天又以右金吾卫大将军武懿宗为神兵道行军大总管击契丹（206-020）。唐隆政变后，麻嗣宗行左金吾卫中郎将（209-080）。

> 遣左金吾卫将军曹怀舜屯井陉。右武卫将军崔献屯龙门，以备突厥。
> （202-133）
