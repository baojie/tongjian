# [[魏成]]

[[魏成]]（又称魏成子），[[战国]]初期[[魏国]]大臣，[[魏文侯]]的[[丞相]]，以礼贤下士、荐引人才著称。

## 被任命为相

[[魏文侯]]征询[[李克]]用相的意见，[[李克]]以"居视其所亲，富视其所与，达视其所举，穷视其所不为，贫视其所不取"五条标准，指出魏成更合适。[[魏文侯]]依此任命魏成为相（001-015）。

[[翟璜]]听说后"忿然作色"，认为他自己推荐的[[吴起]]、[[西门豹]]等人[[贡献]]更大，[[李克]]则解释：魏成"卑身而贵学"，推荐[[子夏]]、[[段干木]]等[[贤士]]，其格局在[[翟璜]]之上。

## 历史评价

魏成是[[战国]]贤相的典范，他推荐贤才、广纳学者的做法，使[[魏国]]成为[[战国]]初期文化最发达的国家。《[[资治通鉴概述|资治通鉴]]》通过[[魏文侯]]用相的故事，说明用人要看其价值观和格局，而非仅看事功（001-015）。
