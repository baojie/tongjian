# 黄皓

**黄皓**，[[蜀汉]][[刘禅|后主]][[刘禅]]时[[宦官]]，以便僻佞慧受宠信，弄权误国。

[[董允]]秉心公亮，备尽忠益，汉主甚严惮之。黄皓便僻佞慧，汉主爱之。允上则正色规主，下则数责于皓，皓畏允，不敢为非（074-083）。

及[[董允]]卒，[[陈祗]]为[[尚书令]]，祗与黄皓相表里，皓始预政事。[[诸葛瞻]]为都护、卫[[将军]]，共平[[尚书]]事，时中常侍黄皓用事，瞻不能矫正（077-078）。

黄皓用事于中，与右[[大将军]]阎宇亲善，阴欲废[[姜维]]树宇。[[姜维]]知之，乃言于汉主曰："皓奸巧专恣，将败国家，请杀之。"汉主曰："皓趋走小臣耳，往者[[董允]]每切齿，吾常恨之，君何足介乎！"维见汉主宠皓益甚，不敢复言，乃退往沓中（078-004）。

[[曹魏|魏]]军伐蜀，[[姜维]]闻之，[[上表]]求遣左右[[车骑]]分护阳安关口、阴平桥头以防未然。黄皓信巫鬼，谓敌终不自致，启汉主寝其事，群臣莫知（078-010）。[[邓艾]]破[[成都]]，[[诸葛瞻]]子尚叹曰："父子荷国重恩，不早斩黄皓，使败国殄民，用生何为！"策马冒阵而死（078-026）。
