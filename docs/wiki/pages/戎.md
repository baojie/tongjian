# 戎

**戎**为古代民族或方国称谓。

> （002-018）

> （003-016）

> （003-062）

戎为古代西方部族泛称。[[秦孝公]]述穆公修德行武，西霸戎翟（002-018）。[[司马错]]与[[张仪]]争伐蜀，谓"顾争于戎翟，去王业远矣"（003-016）。秦自陇以西有绵诸、绲戎、翟、之戎（006-057）。[[秦穆公]]求士，西取由余于戎，遂霸西戎（006-090）。[[蒙恬]]为秦将，北逐戎人开榆中地数千里（008-060）。[[东方朔]]论[[长安]]形势，谓秦之所以虏西戎兼山东者（017-039）。
