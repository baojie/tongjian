# 贡奉

贡奉，进献贡品。班超上疏言西域诸国贡奉不绝，唯焉耆、龟兹未服从（046-045）。西晋末长安以西不复贡奉朝廷（089-057）。大秦王改元建义，然犹贡奉宋、魏不绝（123-010）。唐西川节度使王播大修贡奉，且以赂结宦官，求为相（241-079）。高骈为盐铁使积年不贡奉，货财在扬州者填委如山（257-015）。

> 门下侍郎、同平章事萧俛，介洁疾恶，为相，重惜官职，少所引拔。西川节度使王播大修贡奉，且以赂结宦官，求为相。
> （241-079）
