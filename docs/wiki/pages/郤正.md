# 郤正

郤正，字令先，蜀汉秘书令，澹然自守，以书自娱。

秘书郎郤正为太子璿师友（074-006）。秘书令郤正久在内职，与宦官黄皓比屋，周旋三十馀年，澹然自守，以书自娱，既不为皓所爱，亦不为皓所憎（077-078）。

刘禅举家东迁洛阳，大臣无从行者，惟郤正及张通舍妻子单身随禅。禅赖正相导宜适，举动无阙，乃慨然叹息，恨知正之晚（078-054）。郤正以忠节终始，为蜀汉遗臣之表。
