# 郤正

郤正，字令先，[[蜀汉]][[秘书]]令，澹然自守，以书自娱。

[[秘书]]郎郤正为[[太子]]璿师友（074-006）。[[秘书]]令郤正久在内职，与[[宦官]][[黄皓]]比屋，周旋三十馀年，澹然自守，以书自娱，既不为皓所爱，亦不为皓所憎（077-078）。

[[刘禅]]举家东迁[[洛阳]]，大臣无从行者，惟郤正及张通舍妻子单身随禅。禅赖正相导宜适，举动无阙，乃慨然叹息，恨知正之晚（078-054）。郤正以忠节终始，为[[蜀汉]]遗臣之表。
