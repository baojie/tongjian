# 步战

**步战**指以[[步兵]]进行战斗，为古代以[[步兵]]为[[主力]]的[[野战]]或[[攻城]]作战方式。

## 步战战术

[[王闳]]与[[张步]]战不胜（040-038）。[[陶侃]][[长史]]曰吾兵不习步战不如急攻石头（094-045）。

## 步战与水战

[[刘胡]]少习步战未闲水斗，步战恒在数万人中（131-074）。[[侯景]][[任约]]据白脊以待僧祐（164-027）。[[薛延陀]]大教步战五人为伍战胜则授马（196-024）。
