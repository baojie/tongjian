# 大明

**大明**（457—464），[[宋孝武帝]][[刘骏]][[年号]]。大明年间，孝武帝诛杀宗室诸王，[[刘义宣]]反，败死。[[沈庆之]]平[[竟陵王]][[刘诞]]之乱。[[祖冲之]]造《[[大明历]]》，上表献之。朝廷以[[颜师伯]]为[[尚书左仆射]]，贿赂公行，政治腐败。大明八年，孝武帝崩。
