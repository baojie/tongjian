# 陈玄礼

陈玄礼，唐龙武大将军，禁军将领。

陈玄礼从李隆基诛韦后（209-075）。玄宗在华清宫，欲夜出游，陈玄礼谏曰："宫外即旷野，安可不备不虞！"上为之引还（216-124）。马嵬之变，陈玄礼整比六军，从玄宗西奔（218-012）。军士杀杨国忠，玄礼逼玄宗缢杀杨贵妃。后从玄宗入蜀。

> 上在华清宫，欲夜出游，龙武大将军陈玄礼谏曰："宫外即旷野，安可不备不虞！陛下必欲夜游，请归城阙。"上为之引还。
> （216-124）
