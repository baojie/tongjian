# 俎豆

俎豆是古代[[祭祀]]用的[[笾豆|礼器]]，俎为置牲肉的[[几案]]，豆为盛食物的高足盘，代指[[祭祀]]礼仪。

[[匡衡]]论[[教化]]与[[刑法]]之关系，谓"为其俎豆、管弦之间小不备，因是绝而不为，是去小不备而就大不备"（032-057）。[[管宁]]至[[辽东]]，讲《诗》《书》，习俎豆，非学者无见也（060-023）。

[[武后|武则天]][[封禅]][[嵩山]]，酌酒实俎豆，登歌皆用[[宫人]]（201-068）。俎豆之设关乎[[礼乐]]文明的传承，废弃俎豆常被视为文化沦丧的标志。

## 俎豆与礼乐

俎豆虽为器物，实为[[礼乐]]文明之象征。[[匡衡]]论[[治道]]，以为[[礼乐]][[教化]]不可因俎豆管弦小不备而废，小不备而去大不备是"惑莫甚焉"（032-057），强调[[礼乐]][[教化]]之精神远重于器物形式。[[管宁]]避乱[[辽东]]，专讲《诗》《书》习俎豆（060-023），以俎豆维系[[礼乐]]于乱世[[边陲]]，俎豆成为文化传承之符码。[[武后|武则天]][[封禅]][[嵩山]]，酌酒实俎豆登歌皆用[[宫人]]（201-068），以俎豆之器行[[封禅]]大典，折射了女主称尊[[礼制]]变革之一端。俎豆由器物而[[教化]]而[[治道]]，层层增益其文化意涵，体现了[[中国]]古代器物与[[礼乐]]制度相为表里的特征。
