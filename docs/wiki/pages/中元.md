# 中元

**中元**（56—57），[[光武帝刘秀|汉光武帝]]最后一个[[年号]]。中元元年，光武帝登[[泰山]]行封禅礼，改元中元。是年，[[群臣]]奏请封禅，光武帝从之，遂登泰山，祭天地。中元二年，光武帝崩于南宫前殿，太子[[刘庄]]即位，是为[[汉明帝]]。
