# 中元

**中元**（56—57），[[光武帝刘秀|汉光武帝]]最后一个[[年号]]。中元元年，[[光武帝刘秀|光武帝]]登[[泰山]]行[[封禅]]礼，[[改元]]中元。是年，[[群臣]]奏请[[封禅]]，[[光武帝刘秀|光武帝]]从之，遂登[[泰山]]，[[祭天]]地。中元二年，[[光武帝刘秀|光武帝]]崩于[[南宫]]前殿，[[太子]][[刘庄]][[即位]]，是为[[汉明帝]]。
