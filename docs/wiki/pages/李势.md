# 李势

李势，[[十六国]]成汉末主。[[桓温]]伐蜀，以李势无道，臣民不附，宜先取之（097-087）。温至[[成都]]，李势夜开东门走，至[[葭萌]]，送降文于温，自称"略阳李势叩头死罪"，舆榇面缚诣军门。温解缚焚榇，送势及[[宗室]]十馀人于[[建康]]（097-094）。李势至[[建康]]，封归义侯。后卒（101-037）。
