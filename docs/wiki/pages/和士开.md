# 和士开

和士开，字彦通，清都人。[[北齐]]给事黄门侍郎，以善握槊、弹琵琶得宠于[[慕容垂|武成帝]]，弄权乱政。

帝之为长广王也，清都和士开发善握槊、弹琵琶有宠，辟为开府行参军，及即位，累迁给事黄门侍郎（168-118）。散骑常侍和士开请止乐，帝怒，挝之（168-132）。和士开复谮元海，帝以马鞭棰元海六十（169-003）。

和士开以弄臣得幸，与[[慕容垂|武成帝]]高湛昼夜狎戏，不恤政事。士开善于迎合，深得帝心，所言无不从。又结交胡[[太后]]，权倾朝野。[[刘禅|后主]][[高纬]]时，为琅邪王高俨所杀。和士开之死，朝野称快，然[[北齐]]政事已不可为矣。
