# 和士开

和士开，字彦通，清都人。[[北齐]][[给事黄门]][[侍郎]]，以善握槊、弹[[琵琶]]得宠于[[慕容垂|武成帝]]，弄权[[乱政]]。

## 弄臣亡国的样本

和士开之得宠，始于[[北齐]]长广王[[高湛]]时期，因善握槊、弹[[琵琶]]而获宠（168-118），至[[高湛]][[即位]]为[[慕容垂|武成帝]]，和士开[[累迁]]至[[给事黄门]][[侍郎]]，[[宠遇]]日隆。其弄权之术，不过"投其所好"四字。

以握槊、[[琵琶]]之戏迎合[[君王]]的游乐之心，以谄媚之言满足[[君王]]的虚荣之欲，使[[慕容垂|武成帝]]须臾之间不得与士开相见（169-020），甚至临终握其手曰"勿负我也"（170-073），其受信之深可见一斑。

和士开的弄权[[乱政]]，构成了[[北齐]][[朝政]]败坏的重要一环。他与[[祖珽]]共为奸谄（169-100），谮害[[河间王]]孝琬（169-153），排挤[[徐之才]]（170-057），勾结[[胡太后]]（170-091），权倾朝野。然而[[北齐]]的亡国并非和士开一人所致。

[[慕容垂|武成帝]]的荒淫、[[刘禅|后主]]的昏聩、朝臣的党争，共同构成了[[北齐]]灭亡的深层原因。和士开不过是这一病态[[政治]]生态中最显眼的病灶而已。[[司马睿|琅邪王]][[东平王俨|高俨]]杀和士开时，朝野称快，但正如[[资治通鉴概述|通鉴]]所揭示的，和士开虽死，[[北齐]]的[[政治]]颓势已不可逆转。

## 权倾朝野

和士开有宠于[[慕容垂|武成帝]]，须臾之间不得与士开相见，或累日不归（169-020）。与[[祖珽]]共为奸谄（169-100），谮[[河间王]]孝琬为草人射之，以草人拟圣躬，孝琏被执（169-153）。[[祖珽]]欲求[[宰相]]，疏士开等罪状，反为所败（170-044）。[[徐之才]]善医，疗上皇疾愈，士开欲得次迁，乃出之为兖[[州刺史]]（170-057）。上皇疾亟，以后事属士开，握其手曰："勿负我也！"遂殂于士开之手（170-073）。士开历官右[[仆射]]、[[左仆射]]（170-067），为世祖所亲狎，[[出入]]卧内，遂得幸于胡后（170-091）。[[高阿那肱]]以[[谄佞]]为士开所厚，[[累迁]]并省[[尚书令]]（170-096）。士开复引[[祖珽]]为谋主，弃旧怨而虚心待之（170-099）。
