# 鲁王

[[西汉]][[诸侯]]王封号，始于鲁元[[公主]]之子张偃。

## 张偃

[[吕后]]封鲁元[[公主]]子张偃为鲁王（013-004）。

## 辅政

[[太后]]为外孙鲁王偃年少孤弱，封[[张敖]]前姬子以辅鲁王（013-054）。

济川王太、淮阳王武、[[常山]]王朝及鲁王张偃皆年少，未之国，居[[长安]]（013-058）。[[吕氏|诸吕]]被诛后，废鲁王张偃（013-059）。徙淮阳王馀为鲁王（016-031）。
