# 獠

獠，古代西南地区少数民族。[[《资治通鉴》]]载獠人自山出。

[[陶璜]]讨降夷獠，州境皆平（079-106）。蜀土先无獠，至是始从山出，自巴西至[[犍为]]、[[梓潼]]，布满[[山谷]]十馀万落（097-086）。[[梁州]][[刺史]]杨亮帅巴獠万馀拒[[前秦]]，战于青谷（103-070）。张育与巴獠[[酋帅]]张重等五万馀人进围[[成都]]（103-081）。
