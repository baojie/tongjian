# 博具

**博具**指[[博弈]]游戏所用的器具。

## 通鉴记载

[[汉哀帝]]时民聚会里巷[[阡陌]]，设张博具，[[歌舞]]祠西王母，至秋乃止（034-031）。[[刘宋]]时[[魏主]]求甘橘及借博具，宋皆与之（125-098）。博具为古代[[博弈]]游戏所用，博戏自汉至[[南北朝]]盛行不衰。
