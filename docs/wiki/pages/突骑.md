# 突骑

突骑，冲锋突阵之[[精锐]][[骑兵]]，汉军对抗[[匈奴]]之重要兵种。

[[晁错]]论[[匈奴]]之长技[[中国]][[轻车]]突骑胜之（015-009）。[[耿弇]]发[[渔阳]][[上谷]]突骑摧乌合之众（039-037）。

突骑为[[汉代]][[骑兵]]之[[精锐]]。[[晁错]]上言兵事，谓[[匈奴]]之众易挠乱，[[中国]][[轻车]]突骑胜之。[[耿弇]]谒光武，言归发[[渔阳]][[上谷]]突骑，以辚乌合之众，如摧枯折腐。光武经营[[河北]]，以[[上谷]]突骑为爪牙，号为天下[[精兵]]。突骑者，冲突敌阵，锐不可当，非[[精锐]]之士不能任。

> 晁错论中国轻车突骑胜匈奴。
> （015-009）
