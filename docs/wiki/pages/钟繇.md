# 钟繇

钟繇，字元常，颍川长社人。曹魏重臣，官至[[太傅]]，封定陵侯。以书法冠绝当世，后世称"钟王"。

黄门侍郎钟繇说[[李傕]]、[[郭汜]]曰："方今英雄并起，各矫命专制，唯曹兖州乃心[[王室]]。"（060-054）[[曹操]]表繇以[[侍中]]守司隶校尉，持节督[[关中]]诸军，特使不拘科制（062-024）。[[曹操]]使司隶校尉钟繇围南单于于平阳（064-012）。

钟繇在[[曹操]]创业时期，镇守[[关中]]，安抚[[马腾]]、[[韩遂]]，使[[曹操]]无西顾之忧。[[隋文帝|文帝]]时拜[[廷尉]]，[[汉明帝|明帝]]时进[[太傅]]，封定陵侯。钟繇工书，尤善隶书，与[[王羲之]]并称"钟王"，其《宣示表》《贺捷表》为楷书经典。晚年以膝疾不便，朝见时特许乘车，备极荣宠。

> 黄门侍郎钟繇说傕、汜曰："方今英雄并起，各矫命专制，唯曹兗州乃心王室。"
