# 垣崇祖

垣崇祖，字敬远，略阳垣道人。[[南朝]]齐朐山戍主，以拒魏功封侯。为南齐边将。

征北参军垣崇祖奔彭城。崇祖，护之之从子也（131-008）。垣崇祖将[[部曲]]奔朐山，据之，[[遣使]]来降；[[萧道成]]以为朐山戍主。崇祖浮舟，于水侧立城以自固（132-023）。

垣崇祖出自将门，从平刘休范之乱，为[[萧道成]]所赏识。后戍守朐山（今连云港），面对[[北魏]]大军压境，崇祖据城固守，水陆设防，屡退魏军。[[萧道成|齐高帝]][[萧道成]]以崇祖为兖州[[刺史]]，封侯。崇祖长期镇守北境，为南齐边防重将。后为[[萧赜|齐武帝]]所疑，被诛。
