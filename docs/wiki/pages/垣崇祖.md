# 垣崇祖

垣崇祖，字敬远，略阳垣道人。南朝齐朐山戍主，以拒魏功封侯。为南齐边将。

征北参军垣崇祖奔彭城。崇祖，护之之从子也（131-008）。垣崇祖将部曲奔朐山，据之，遣使来降；萧道成以为朐山戍主。崇祖浮舟，于水侧立城以自固（132-023）。

垣崇祖出自将门，从平刘休范之乱，为萧道成所赏识。后戍守朐山（今连云港），面对北魏大军压境，崇祖据城固守，水陆设防，屡退魏军。齐高帝萧道成以崇祖为兖州刺史，封侯。崇祖长期镇守北境，为南齐边防重将。后为齐武帝所疑，被诛。
