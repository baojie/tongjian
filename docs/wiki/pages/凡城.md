# [[凡城]]

[[凡城]]位于今[[辽宁]]朝阳一带，是[[十六国]]时期[[前燕]]与[[后赵]]争夺的[[辽西]]要地。[[前燕]][[慕容皝]]分兵讨诸叛城，拓境至凡城（096-018）。[[后赵]][[石虎]]遣[[石农]]率众攻前燕凡城，燕王以悦绾为御难将军，授兵一千守之，赵兵至，将吏皆恐，悦绾率众力战，退赵兵（096-064）。

凡城之战体现了前燕在慕容皝统治时期的军事崛起，为前燕入主中原奠定了基础。
