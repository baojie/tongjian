# [[凡城]]

[[凡城]]位于今[[辽宁]]朝阳一带，是[[十六国]]时期[[前燕]]与[[后赵]]争夺的[[辽西]]要地。[[前燕]][[慕容皝]][[分兵]]讨诸叛城，拓境至凡城（096-018）。[[后赵]][[石虎]]遣[[石农]]率众攻[[前燕]]凡城，[[燕王]]以悦绾为御难[[龙骧将军|将军]]，授兵一千守之，赵兵至，将吏皆恐，悦绾率众[[力战]]，退赵兵（096-064）。

凡城之战体现了[[前燕]]在[[慕容皝]]统治时期的[[军事]]崛起，为[[前燕]]入主[[中原]]奠定了基础。
