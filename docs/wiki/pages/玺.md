# 玺

玺是古代帝王及官员的印信，为权力之象征，秦以后专指皇帝之印。

[[苏代]]谓魏王：秦以地事秦，犹抱薪救火，欲玺者段干子，欲地者秦（004-093）。[[嫪毐]]矫王御玺发兵为乱（006-083）。韩王纳地效玺，请为籓臣（006-104）。

[[秦始皇]]病中令[[赵高]]为书赐[[扶苏]]，书已封在中车府令行[[符玺]]事[[赵高]]处（007-070）。[[赵高]]与[[李斯]]谋，谓上赐长子书及[[符玺]]皆在[[胡亥]]所（007-071）。[[子婴]][[斋戒]]受玉玺，后与子谋诛[[赵高]]（008-065）。[[汉高祖刘邦|刘邦]]至霸上，[[子婴|秦王子婴]]封皇帝玺、符、节降轵道旁（009-003）。[[赵尧]]为[[符玺]]御史（012-023）。玺印制度为历代中央政权之重要信物制度。

> 秦王子婴素车、白马，系颈以组，封皇帝玺、符、节，降轵道旁。
> （009-003）
