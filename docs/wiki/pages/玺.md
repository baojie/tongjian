# 玺

玺是古代帝王及官员的印信，为权力之象征，秦以后专指皇帝之印。

苏代谓魏王：秦以地事秦，犹抱薪救火，欲玺者段干子，欲地者秦（004-093）。嫪毐矫王御玺发兵为乱（006-083）。韩王纳地效玺，请为籓臣（006-104）。

秦始皇病中令赵高为书赐扶苏，书已封在中车府令行符玺事赵高处（007-070）。赵高与李斯谋，谓上赐长子书及符玺皆在胡亥所（007-071）。子婴斋戒受玉玺，后与子谋诛赵高（008-065）。刘邦至霸上，秦王子婴封皇帝玺、符、节降轵道旁（009-003）。赵尧为符玺御史（012-023）。玺印制度为历代中央政权之重要信物制度。

> 秦王子婴素车、白马，系颈以组，封皇帝玺、符、节，降轵道旁。
> （009-003）
