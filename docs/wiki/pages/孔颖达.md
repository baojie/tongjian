# 孔颖达

**孔颖达**，字冲远，[[冀州]]衡水人，唐[[经学]]家、[[国子]][[祭酒]]。

孔颖达为[[儒学]]大师，[[王世充]]令造禅代仪（187-050）。[[唐太宗]]时为[[文学]]馆[[学士]]（189-076），与[[房玄龄]]、[[杜如晦]]等十八人并称"十八[[学士]]"。颖达撰《[[五经]]正义》，[[统一]]南北[[经学]]，为[[唐朝|唐代]]官学标准教材。

孔颖达为[[孔子]]后裔，其《[[五经]]正义》融合南北学统，去取精审，成为[[唐朝|唐代]]科举考试的标准。[[唐朝|唐代]][[经学]]之盛、孔颖达之功为最。其学统沿袭至宋，影响深远。

## 五经正义

孔颖达为[[文学]]馆[[学士]]，与[[房玄龄]]等十八人并称"十八[[学士]]"（189-076）。[[太宗]]问《[[论语]]》"以能问于不能，以多问于寡"之义，颖达具释之，[[太宗]]称善（193-040）。颖达与[[于志宁]]等数[[直谏]][[太子承乾]]，[[太宗]]嘉之，各赐[[金帛]]（194-042、196-077）。后为[[国子]][[祭酒]]，[[太宗]]幸[[国子监]]视[[释奠]]，命颖达讲《[[孝经]]》（195-079）。[[贞观之治|贞观]]中，颖达为[[国子]][[祭酒]]（208-027），撰《[[五经]]正义》颁行[[天下]]，[[统一]]南北[[经学]]，为[[唐朝|唐代]]官学之宗。
