# 朝贺

朝贺是[[诸侯]]群臣朝见[[天子]]的[[礼仪]][[制度]]，为古代[[政治]][[礼制]]的核心内容。

[[秦始皇]]改年，朝贺皆自十月朔，衣服旌[[旄节]]旗皆尚黑（007-032）。汉[[长乐宫]]成，[[诸侯]]群臣皆朝贺，[[谒者]]治礼以次引入殿门（011-045）。[[汉武帝]]与[[张汤]]造白鹿皮币，颜异言[[王侯]]朝贺以苍璧直数千而以皮荐反四十万，[[本末]]不相称（020-021）。

朝贺之制源于[[周礼]]，[[汉代]]以后形成[[正旦]]朝贺大典。朝贺的仪节、位次、[[服制]]皆有严格规定，是体现君臣[[名分]]、[[朝廷]]威仪的重要场合。

## 朝贺与君臣礼制

朝贺不仅是[[礼仪]]仪式，更折射了君臣关系与国际秩序。[[甘露]]年间[[匈奴]][[单于]]来朝，[[丞相]][[御史]]以为其[[礼仪]]宜如[[诸侯王]]位次在下，[[萧望之]]则以[[单于]]非[[正朔]]所加故称敌国，位次宜在[[诸侯王]]上（027-054），朝贺位次之争实为汉匈关系的[[礼仪]]表达。[[匡衡]]上疏论[[正旦]]朝贺之礼，以为"正月初幸路寝[[临朝]]贺置酒以飨万方"关系"[[君子]]慎始"之义（029-086），朝贺为一年[[政治]]之始，不可不慎。南[[匈奴]][[款塞]][[称臣]][[遣使]]朝贺无有二心（030-048），朝贺由内部君臣之礼延伸为华夷秩序之[[制度]]化表达。朝贺[[制度]]从秦定十月朔到汉制[[正旦]]朝贺，其演变折射了大一统王朝[[礼制]]构建的进程。
