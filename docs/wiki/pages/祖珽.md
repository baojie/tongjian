# 祖珽

祖珽，字孝徵，范阳遒人。北齐著作郎，有[[文学]]多技艺而疏率无行。历仕北齐数帝，位至尚书左仆射。

欢乃使仓[[曹参]]军祖珽说[[韦孝宽]]曰："君独守孤城，而西方无救，恐终不能全，何不降也？"孝宽报曰："我城池严固，兵食有馀。"（159-041）陈元康口占使功[[曹参]]军祖珽作书陈便宜（162-073）。齐著作郎祖珽，有[[文学]]，多技艺，而疏率无行（169-099）。

祖珽才学出众，工诗能文，精通[[音律]]，然人品卑下，好货贪财。在玉璧之战中为[[高欢]]效力，后以[[文学]]为北齐诸帝所重。然以贪贿被劾，虽以才得免，终为士林所轻。北齐多才士而少德行，祖珽为其典型。

> 齐著作郎祖珽，有文学，多技艺，而疏率无行。
> （169-099）
