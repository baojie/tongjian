# 祭酒

祭酒为国子监首长，掌儒学教育，始于汉代博士祭酒，后世沿用。

王莽置师友、侍中、谏议、《六经》祭酒各一人，凡九祭酒，秩皆上卿（037-040）。满昌为师友祭酒，劾奏使者序位不当（037-056）。

隗嚣招揽士人，以郑兴为祭酒（040-031）。唐太宗幸国子监，命祭酒孔颖达讲《孝经》（195-079）。祭酒之职位尊望重，常以当世大儒任之。
