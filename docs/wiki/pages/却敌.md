# 却敌

**却敌**指退却敌军，为防御作战之要旨。

## 折冲却敌

[[汉宣帝]]敕边守尉曰[[匈奴]]多变诈，交接得其情则却敌[[折冲]]；应对入其数则反为轻欺（044-045）。却敌之道在于知彼。

## 守险却敌

[[南燕]][[慕容超]]不肯据险却敌，又不肯[[徙民]][[清野]]，延敌入腹坐待攻围，酷似[[刘璋]]（115-019）。却敌当据险而守。
