# 诸葛瞻

诸葛瞻，字思远，琅邪阳都人。[[蜀汉]]卫将军，[[诸葛亮]]之子。

诸葛瞻为[[侍中]]，与[[诸葛恪]]书，恪再攻[[淮南]]，越巂太守[[张嶷]]与瞻书言恪必败（076-023）。后为仆射（077-054），迁都护、卫将军，与董厥共平尚书事。然不能制黄皓，蜀人惜之（077-078）。魏伐蜀，瞻督军拒[[邓艾]]于绵竹，战败而死。

> 汉主以董厥为辅国大将军，诸葛瞻为都护、卫将军，共平尚书事。
> （077-078）
