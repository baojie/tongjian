# 碑文

碑文是刻于石碑上的文字，用于记述功绩、纪念逝者或颁扬圣德，为古代重要文体。

[[唐太宗]]自制碑文并为书石，悼念[[魏徵|魏征]]，叹曰人以铜为镜可以正[[衣冠]]，以古为镜可以见兴替，以人为镜可以知得失（196-061）。

碑文之体始于秦刻石，盛于汉唐。碑文以叙事为主，兼有颂赞，文辞庄重典雅。名臣撰文、[[名家]]书丹者为世所重。碑版之学（金石学）为传统学术重要分支，碑文为考证历史之珍贵资料。

> 上自制碑文，并为书石。上思征不已，谓侍臣曰："人以铜为镜，可以正衣冠，以古为镜，可以见兴替，以人为镜，可以知得失。"
> （196-061）
