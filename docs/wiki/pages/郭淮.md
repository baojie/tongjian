# 郭淮

郭淮，字伯济，太原阳曲人。曹魏雍州刺史，历仕四朝，以拒诸葛亮著称。

郭淮为雍州刺史，拒诸葛亮于渭南（068-023）。郭淮击句芒羌，平之（071-022）。郭淮从司马懿拒诸葛亮，号令明肃（072-005）。

郭淮初举孝廉，仕魏为平原府丞。夏侯渊战殁，淮时为司马，推张郃为帅，三军遂安。文帝践祚，淮领雍州刺史，在关右二十馀年，外御蜀汉，内抚诸羌，威恩并著。诸葛亮伐魏，淮每拒之，号为名将。正元中，迁车骑将军，封阳曲侯。卒谥曰贞。

> 郭淮为雍州刺史，拒诸葛亮于渭南。
