# [[平原郡]]

[[平原郡]]是[[西汉]]置郡，治所在[[平原]]（今[[山东]]平原），属[[青州]]刺史部。平原郡地处[[黄河]]下游，是[[山东]]北部的重要农业区。

[[东汉]]末，[[刘备]]曾为平原相，以此为基础起家。[[隋]][[唐]]时期，平原郡改置[[德州]]。唐初， [[唐高祖]]赠[[刘感]]平原郡公，谥忠壮（186-049）。[[梁师都]]寇[[延州]]，[[段德操]]击败之，以功赐爵平原郡公（187-098）。
