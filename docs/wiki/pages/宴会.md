# 宴会

宴会，帝王设宴款待群臣的礼仪活动，为古代宫廷政治之重要场景。

汉和帝邓后宴会独尚质素（048-109）。汉明帝每宴会论难求政化（049-027）。晋武帝为太子设宴会（080-084）。

宴会非徒为饮食之欢，实为君臣相与、议政论道之场合。汉明帝每宴会则与诸儒论难衎衎，共求政化，期门羽林介胄之士悉通《孝经》。和帝邓后性谦俭，每有宴会，诸姬竞自修饰，后独尚质素。晋武帝为太子设宴，密封尚书疑事令决之，以此试太子之才。

> 汉明帝每宴会论难求政化。
> （049-027）
