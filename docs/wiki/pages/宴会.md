# 宴会

宴会，帝王[[设宴]]款待群臣的礼仪活动，为古代宫廷政治之重要场景。

[[汉和帝]]邓后宴会独尚质素（048-109）。[[汉明帝]]每宴会论难求政化（049-027）。[[司马炎|晋武帝]]为[[太子]][[设宴]]会（080-084）。

宴会非徒为饮食之欢，实为君臣相与、议政论道之场合。[[汉明帝]]每宴会则与诸儒论难衎衎，共求政化，期门羽林介胄之士悉通《孝经》。[[汉和帝|和帝]]邓后性谦俭，每有宴会，诸姬竞自修饰，后独尚质素。[[司马炎|晋武帝]]为[[太子]][[设宴]]，密封尚书疑事令决之，以此试[[太子]]之才。

> 汉明帝每宴会论难求政化。
> （049-027）
