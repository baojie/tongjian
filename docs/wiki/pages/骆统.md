# 骆统

骆统，吴功曹。劝孙权尊贤接士，为孙权所重。

功曹骆统劝权尊贤接士，勤求损益，飨赐之日，人人别进，问其燥湿，加以密意，诱谕使言，察其志趣（063-037）。陆逊与朱然、骆统上言曰："曹丕大合士众，外托助国讨备，内实有奸心，谨决计辄还。"（069-070）。
