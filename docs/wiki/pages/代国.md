# [[代国]]

[[代国]]是[[战国]]时期北方[[诸侯]]国，位于今[[河北]]蔚县、[[山西]]代县一带。[[赵襄子]]灭代国，代地遂属[[赵]]。[[赵武灵王]]置[[代郡]]，是[[赵国]]北疆[[重镇]]。

[[汉代]]封代王，治[[代郡]]，[[刘邦]]曾以[[云中]]、[[雁门]]、[[代郡]]五十三县立兄[[刘喜]]为代王。[[北周]]时，封[[宇文达]]为代王，以[[上党]]郡为代国（173-114）。[[唐代]]代国为公爵封号，以代国公封[[安金藏]]等[[功臣]]（213-112）。
