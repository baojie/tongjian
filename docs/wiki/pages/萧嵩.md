# 萧嵩

萧嵩，梁明帝之孙。唐宰相，河西节度使。

萧嵩为按察使，治河决（212-087）。命尚书左丞萧嵩与京兆、蒲、同等州长官选府兵及白丁十二万，谓之"长从宿卫"（212-121）。以朔方节度使萧嵩为河西节度等副大使。时王君毚新败，河、陇震骇，嵩引裴宽为判官，抚安军心（213-042）。

> 以朔方节度使萧嵩为河西节度等副大使。时王君毚新败，河、陇震骇。嵩引刑部员外郎裴宽为判官。
