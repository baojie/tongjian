# 疏勒

疏勒是[[西域]]北道要国，位于葱岭以东、塔里木盆地西端。自[[车师]]前王廷"随北山循[[河西]]行至疏勒，为北道"（020-036），是丝绸之路北道必经之地。

[[东汉]]初年，[[龟兹]]王攻杀疏勒王，[[班超]]"从间道至疏勒"，擒缚[[龟兹]]所立之王，定疏勒归汉（045-081）。[[耿恭]][[据守]]疏勒城，[[匈奴]]绝涧水以困之，[[耿恭]]于城中穿井得水（045-088），其忠勇感动汉军，发兵迎归，史称"十三壮士归玉门"（046-004）。疏勒自此成为[[西汉|汉朝]]经营[[西域]]的重要基地。
