# 疏勒

疏勒是西域北道要国，位于葱岭以东、塔里木盆地西端。自车师前王廷"随北山循河西行至疏勒，为北道"（020-036），是丝绸之路北道必经之地。

东汉初年，龟兹王攻杀疏勒王，班超"从间道至疏勒"，擒缚龟兹所立之王，定疏勒归汉（045-081）。[[耿恭]]据守疏勒城，匈奴绝涧水以困之，耿恭于城中穿井得水（045-088），其忠勇感动汉军，发兵迎归，史称"十三壮士归玉门"（046-004）。疏勒自此成为汉朝经营西域的重要基地。
