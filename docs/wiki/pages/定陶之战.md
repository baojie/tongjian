# 定陶之战

**定陶之战**是[[胡亥|秦二世]]二年（208年），[[项梁]]率领的楚军在[[定陶]]（今[[山东]]菏泽[[定陶]]区）击败[[章邯]]秦军，旋因轻敌而被[[章邯]]夜袭反杀、[[项梁]]战死的战役。此战是秦末[[反秦起义]]的重要转折点。

## 背景

[[胡亥|秦二世]]二年，[[陈胜]]败亡后，[[项梁]]成为反秦势力的主要领袖。[[项梁]]自[[江东]]渡[[淮河|淮]]而西，先后收[[英布]]、[[蒲将军]]、[[陈婴]]、[[吕臣]]等部，众至六七万人（008-021）。旋即并秦嘉军于胡陵，声势大振。

[[章邯]]在击灭[[陈胜]]后，转攻[[魏王]]魏咎于[[临济]]。[[项梁]]引兵救魏，与[[章邯]]战于[[东阿]]，大破[[章邯]]军（008-029）。

## 定陶破秦

[[项梁]]既破[[章邯]]于[[东阿]]，引兵西进，至[[定陶]]，再破秦军。与此同时，[[项梁]]遣[[项羽]]、[[刘邦]]别攻[[雍丘]]，亦大破秦军，斩[[李斯]]之子[[三川郡]]守[[李由]]（008-036）。

> 项梁已破章邯于东阿，引兵西，北至定陶，再破秦军。项羽、沛公又与秦军战于雍丘，大破之，斩李由。
> （008-036）

## 项梁之死

连战连捷之后，[[项梁]]日益轻敌，面露骄色。[[宋义]]谏曰："战胜而将骄卒惰者，败。今卒少惰矣，秦兵日益，臣为君畏之！"[[项梁]]不听，竟遣[[宋义]][[出使]][[齐国]]（008-036至008-039）。

[[章邯]]在得到[[关中]]的增援之后，趁[[项梁]]不备，于夜间发动突袭，大破楚军于[[定陶]]。[[项梁]]战死，楚军溃散（008-040）。

> 章邯已破项梁，以为楚地兵不足忧，乃渡河，北击赵，大破之。
> （008-040）

## 影响

[[项梁]]之死是[[反秦起义]]的重大挫折。[[项羽]]、[[刘邦]]被迫收缩防线，[[楚怀王]][[熊心]]自[[盱眙]][[迁都]][[彭城]]以自保。[[章邯]]误以为楚地已不足忧，遂引兵北渡[[黄河]]，进攻[[赵国]]，由此引发了决定[[秦朝]]命运的[[巨鹿之战]]。[[宋义]]因预见[[项梁]]之败而获取了楚军领导权，成为[[巨鹿之战]]前楚军的实际统帅。
