# 法服

法服，按礼法制度规定的法定服饰，天子百官各依品级。

刘向谏成帝非法服不可御游（053-054）。杨阜问魏明帝缥绫半袖非法服（073-008）。北魏僧尼依傍法服失佛教本意（107-087）。

法服之制，上自天子冕旒，下至百官朝服，各有等差。汉明帝定冠冕车服制度，法服始备。非郊庙、朝会、公事，不得服法服。魏明帝著衤冒缥绫半袖，杨阜问曰"此于礼何法服也"，帝惭而不答。法服为礼制之外在表征，历代皆以服制辨尊卑、别贵贱。
