# 法服

法服，按礼法制度规定的法定[[服饰]]，天子百官各依品级。

[[刘向]]谏成帝非法服不可御游（053-054）。[[杨阜]]问魏[[汉明帝|明帝]]缥绫半袖非法服（073-008）。[[北魏]][[僧尼]]依傍法服失[[佛教]]本意（107-087）。

法服之制，上自天子[[冕旒]]，下至百官朝服，各有等差。[[汉明帝]]定[[冠冕]][[车服]]制度，法服始备。非郊庙、[[朝会]]、公事，不得服法服。魏[[汉明帝|明帝]]著衤冒缥绫半袖，[[杨阜]]问曰"此于礼何法服也"，帝惭而不答。法服为礼制之外在表征，历代皆以服制辨尊卑、别贵贱。
