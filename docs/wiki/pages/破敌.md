# 破敌

**破敌**指击败敌人，[[军事]]行动的最终目的。

## 激励士气

光武于[[昆阳之战]]激励诸将曰，今若破敌[[珍宝]]万倍，[[大功]]可成；如为所败首领无馀（039-011）。破敌之志在于必胜。

## 据理力争

[[关羽]]争[[荆州]]三郡，责[[鲁肃]]曰乌林之役[[左将军]]戮力破敌，岂得徒劳无一塊土（067-022）。破敌之功当有赏报。
