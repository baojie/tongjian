# 冯素弗

冯素弗，长乐信都人。北燕昌黎尹，[[冯跋]]之弟，佐兄建立北燕。

冯素弗为北燕重臣，[[冯跋]]之弟（114-099）。冯素弗从[[冯跋]]杀慕容熙，推高云为主（114-120）。

冯素弗少慷慨，有智略。从[[冯跋]]起兵，杀后燕主慕容熙，立高云。高云被杀，[[冯跋]]即天王位，以素弗为昌黎尹。素弗在镇，明达政体，勤于庶务。北燕之立，素弗之功居多。素弗早卒，[[冯跋]]哭之恸，曰："天不欲吾成事耶！"北燕卒不能强，亦以素弗死故。

> 冯素弗为北燕重臣，冯跋之弟。
> （114-099）
