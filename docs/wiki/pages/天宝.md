# 天宝

**天宝**（742—756），[[唐玄宗]][[李隆基]]第三个[[年号]]。天宝年间，玄宗怠于政事，以[[李林甫]]、[[杨国忠]]为相，[[政治]]腐败。[[安禄山]]身兼三镇[[陇右节度使|节度使]]，蓄[[谋叛]]乱。天宝十四年，[[安史之乱]]爆发，[[安禄山]]陷[[洛阳]]、[[长安]]，玄宗奔[[蜀]]。[[马嵬坡]]兵变，[[杨贵妃]][[赐死]]。
