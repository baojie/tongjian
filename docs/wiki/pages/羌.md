# 羌

羌是古代中国西部的游牧民族，与汉王朝长期保持复杂关系。汉文帝时地震，"羌道、武都道山崩"（013-011），羌道为汉朝管理羌人的县级政区。司马相如谏猎，言车骑出行可能惊扰"羌、夷接轸"（017-040）。

汉武帝通西域，"欲从羌中归"（018-083），可见羌地为西域与中原之间的通道。严安上书论征伐之弊，言武帝"降羌、僰"（018-055），开边不已。羌人在两汉之际渐成边患，至东汉中后期更爆发大规模羌乱，成为汉朝衰亡的重要原因。
