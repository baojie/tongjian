# 李实

李实，唐司农卿兼京兆尹，为政暴戾，恃恩骄纵，为永贞革新所贬。

山南东道节度判官李实知留后事，性刻薄，裁损军士衣食（234-006）。司农卿李实为德宗所信，实狡险掊克（235-058）。以司农卿李实兼京兆尹，实为政暴戾，上爱信之（236-034）。卢群谓曰："去岁李实怙恩挟贵，气盖一时，公当此时，逡巡路旁。"（236-072）。

## 世系

李实（唐），唐司农卿、[[京兆尹]]。其先世不详于史。实为山南东道节度判官（234-006），为[[唐德宗|德宗]]所信，恃恩骄纵（235-058），兼京兆尹为政暴戾（236-034）。[[永贞革新]]中被贬。
