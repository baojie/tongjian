# 赵王伦

赵王伦，[[西晋]]赵王，[[司马懿]]之子。八王之乱参与者。信用嬖人[[孙秀]]，与雍州刺史解系争军事（082-081）。右军将军赵王伦执兵柄，性贪冒，孙秀等说其讨贾后（083-034）。赵王伦、孙秀将讨贾后，约以鼓声为应（083-038）。后篡位称帝，为诸王所攻，败死。

> 征西大将军赵王伦信用嬖人琅邪孙秀，与雍州刺史济南解系争军事，更相表奏，欧阳建亦表伦罪恶。
> （082-081）
