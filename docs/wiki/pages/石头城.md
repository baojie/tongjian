# 石头城

**石头城**，又称石首城，[[孙权]]于[[建业]]（今[[南京]]）依[[金陵]]山形势所筑之军事要塞，临[[长江|大江]]扼[[淮水]]，为六朝都城屏障。

[[东汉]][[建安]]十六年，[[孙权]]从[[张纮]]言，作石头城，徙治[[秣陵]]，改[[秣陵]]为[[苻健|建业]]（066-042）。[[西晋]]灭吴，石头城入晋。[[东晋]]时，[[苏峻之乱]]中石头城为重要争夺据点。

[[刘裕]]讨[[桓玄]]，屯石头城，立[[留台]]百官，焚[[桓温]]神主于宣阳门外（113-060）。[[卢循]]寇[[建康]]，[[刘裕]]发民治石头城，登城望循军，初见引向新亭，顾左右失色，既而回泊蔡洲乃悦（115-068）。裕以石头城险且淮栅甚固，留[[刘钟]][[据守]]（115-071）。

石头城襟江带淮，控扼[[长江]]下游，六朝更迭之际，凡有兵争，必先据石头城。城以石头山为基，形势[[险要]]，故有"石头虎踞"之称。
