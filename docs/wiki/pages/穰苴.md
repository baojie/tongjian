# 穰苴

穰苴，田氏，齐人。春秋齐将，官至大司马，著《司马兵法》。

穰苴为齐将，以法治军（001-028）。

穰苴本名田穰苴。齐景公时，晋燕伐齐，晏婴荐穰苴。景公以为将军。穰苴曰："臣素卑贱，愿得君之宠臣以为监军。"景公使庄贾往。穰苴与贾约日中会于军门。贾素骄贵，至夕乃至。穰苴曰："将军受命之日则忘其家，临军约束则忘其亲，援枹鼓之急则忘其身。"遂斩贾以徇。晋燕闻之，解去。

> 穰苴为齐将，以法治军。
> （001-028）
