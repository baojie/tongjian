# 边疆

**边疆**为[[国家]][[边境]]地区，关系[[国防]][[安危]]。《[[资治通鉴概述|资治通鉴]]》中多见边疆[[防御]]、镇守与[[治理]]之记载。

## 通鉴记载

[[南朝齐]]时，[[胡谐之]]等议[[谪役]]边疆之法，巧者戍[[缘淮]]各十年，[[百姓]][[怨望]]，乃下诏放宽（137-031）。[[柳仲礼]]久捍边疆，先为[[侯景]]所惮，士马[[精锐]]，无出其前（161-069）。

[[陈子昂]]谏[[武后]]，以为竭人之力、费人之财以爱[[突厥]]数子，使边疆之士不尽力，[[朝廷]]之士不尽忠，则本不固矣（209-016）。[[回鹘]]屡欲侵扰边疆，[[唐武宗]]责其不循理（246-104）。
