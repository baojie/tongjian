# 李昭德

李昭德，京兆[[长安]]人。唐[[宰相]]，[[武后|武则天]]朝[[名相]]，以刚直称。

李昭德为[[宰相]]，谏[[武后|武则天]]立武氏庙（203-023）。李昭德专决[[朝政]]，为[[酷吏]]所害（205-045）。

李昭德少以[[明经]]举。[[武后|武则天]]时，[[累迁]]凤阁[[侍郎]]、[[同平章事]]。昭德性刚直，每[[论事]]必引经据典，不为[[权贵]]屈。时[[武承嗣]]用事，请立武氏[[七庙]]，昭德力[[谏止]]之。后与[[来俊臣]]等不协，为俊臣所构，下狱死。昭德虽死，[[天下]]冤之。

> 李昭德为宰相，谏武则天立武氏庙。
> （203-023）

## 刚直敢言

李昭德深得[[武后|武则天]]信任，密言[[武承嗣]]权太重，[[太后]]感悟，遂罢承嗣[[宰相]]（205-022）。又以[[洛阳|东都]]无外城，始筑之，壮[[京师]]之防（205-007）。昭德长于治事，然恃[[太后]]委遇，颇[[专权]]使气，人多疾之，终为[[来俊臣]]所[[构陷]]（205-065、206-023）。昭德以刚直立朝，虽死于[[酷吏]]之手，[[天下]]冤之。
