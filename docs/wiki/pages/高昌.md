# 高昌

高昌是吐鲁番盆地古国，为西域重要城邦。汉宣帝时[[郑吉]]为西域都护，治乌垒城，高昌为戊己校尉驻地（025-061）。[[李柏]]为[[前凉]]击高昌，高昌遂入凉州版图。

[[麹氏高昌]]自北魏末立国，历南北朝至唐初。唐太宗时高昌王[[麹文泰]]阻绝西域贡路，太宗命[[侯君集]]讨之（190-125）。侯君集破高昌，文泰惊惧死，高昌降，唐置西州（180-052/077/084）。高昌地处丝绸之路中段，汉唐经营西域之枢纽，亦为佛教东传的重要中转站。
