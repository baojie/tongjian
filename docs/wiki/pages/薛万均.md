# 薛万均

[[薛万彻|薛万均]]，[[唐_(朝代)|唐]]初将领。[[薛世雄]]之子。与弟[[薛万彻]]俱以[[勇略]]为[[罗艺]]所亲待。诏以万均为[[上柱国]]、永安郡公（186-059）。[[窦建德]]帅众二十万攻[[幽州]]，[[薛万彻|薛万均]]与[[薛万彻]]帅[[敢死]]士百人从[[地道]]出其背，[[掩击]]之，建德兵溃走，[[斩首]]千馀级（188-082）。

> 薛万均、薛万彻帅敢死士百人从地道出其背，掩击之，建德兵溃走，斩首千馀级。
> （188-082）

## 从征吐谷浑

[[薛万彻|薛万均]]从[[李靖]]征[[吐谷浑]]，由北道[[进兵]]。[[吐谷浑]][[据险]]为寇，万均与诸将击走之（194-070）。又随[[柴绍]]击[[梁师都]]，师都引[[突厥]]兵至城下，万均等[[坚壁]]不出，师都宵遁（192-091）。后为[[殿中]]少监，卒于官。
