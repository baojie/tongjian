# 程务挺

**程务挺**，洺州平恩人，唐[[定襄]]道[[行军]][[总管]]，[[程名振]]之子。

程务挺为[[裴行俭]]副将，以右领军[[中郎将]]从行俭讨[[突厥]]（202-137）。败[[突厥]]馀众于云州（202-149）。务挺以[[勇略]]著称，[[突厥]]甚惮之。后[[裴行俭]]卒，务挺代为[[定襄]]道[[行军]][[总管]]。

[[武后|武则天]][[临朝]]，程务挺以[[宿将]]镇边。及[[徐敬业]][[起兵]]，或谮务挺与敬业通，[[武后]][[遣使]]即军中斩之。[[突厥]]闻务挺死，置酒相庆。务挺之死为[[武后|武则天]]翦除功臣[[宿将]]之一例。

## 破突厥预废立

初，[[裴行俭]]讨[[突厥]][[阿史那伏念]]，遣[[裨将]][[程务挺]]自石[[地道]]掩取伏念[[妻子]][[辎重]]（202-180）。伏念穷窘而降，行俭许以不死。然[[裴炎]]疾行俭之功，奏言伏念为[[张虔勖]]、程务挺所逼而降，竟诛伏念（202-183）。行俭叹为浑、浚争功之再现。程务挺拔自[[偏裨]]，为行俭所识拔，与[[王方翼]]、[[黑齿常之]]等同为一时[[名将]]（203-010）。后务挺从[[裴炎]]、[[刘祎之]][[勒兵]]入宫，[[宣太后]]令，废[[汉宣帝|中宗]]为[[庐陵王]]（203-054）。及[[裴炎]]下狱，程务挺密表申理，由是忤[[武后]]旨。或谮其与[[裴炎]]、[[徐敬业]]通谋，竟被斩于军中（203-100）。
