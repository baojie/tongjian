# 辇

辇，人挽之车，帝王在宫内所乘，亦为天子的代称。

触龙说赵太后恃辇而行（005-029）。天子辇出房乃行朝仪（011-045）。文帝止辇受郎官上书（013-089）。

辇本为宫中代步之车，以人挽行，不用牛马。汉制，天子出房乘辇，群臣奉贺。文帝尝于霸陵欲驰下峻坂，袁盎谏曰"天子不与危同乘"，帝乃止。又尝于郎官上书时止辇受言，言可用采之，不可用置之。后世以"止辇"为帝王纳谏之典。

> 触龙说赵太后恃辇而行。
