# 楚威王

楚威王，[[战国]][[楚国]][[君主]]（前339—[[前329年]]在位）。

[[苏秦]]南说楚威王，称楚为[[天下]]之强国，地方六千馀里，带甲百万（002-094）。楚威王卒，子怀[[王立]]（002-106）。

## 世系

楚威王（？—[[前329年]]），[[芈姓]]，名商，[[战国]][[楚国]][[君主]]。父[[楚宣王]]良夫，子[[楚怀王]]槐立（002-106）。威王在位时灭[[越]]败[[齐]]，[[楚国]]势强。[[苏秦]]说威王称楚为[[天下]]之强国（002-094）。
