# 何曾

何曾，字颖考，西晋司徒、太保。

司隶校尉何曾面质阮籍于司马昭座曰："卿纵情、背礼、败俗之人。"（078-007）。以司空王祥为太尉，征北将军何曾为司徒（078-052）。何曾与王祥、荀顗共诣晋王（078-053）。

何曾以豪奢著称，食日万钱犹云无下箸处。然临终语其子，以家门富贵，恐及于祸。

> 何曾面质籍于司马昭座曰："卿纵情、背礼、败俗之人。"

## 晋室元老

何曾历仕魏晋，为司徒（078-052）。及司马昭进爵晋王，曾与王祥、荀顗共诣之（078-053）。议立晋王世子，曾与裴秀赞司马炎有超世之才，固非人臣之相（078-072）。入晋，进位丞相（079-010），迁太尉（079-013），转太保（079-050），领司徒（080-010），进太傅（080-054），终拜太宰（080-082），为西晋开国元老。
