# 田儋

田儋，狄人，故齐[[王族]]。秦末起兵复齐，为齐王。在楚汉之际的齐国复国运动中首倡大义。

狄人田儋起兵于齐（007-097）。田儋者，故齐[[王族]]也。儋从弟荣，荣弟横，皆豪健，宗强，能得人（007-103）。

田儋在[[陈胜]]起兵后，杀狄令起兵自立为齐王，定齐地。儋引兵救[[魏王咎]]，为[[章邯]]所败，战死。儋虽死，然田氏复齐运动并未终止，其弟[[田荣]]、[[田横]]继之，成为楚汉之际齐国的主要力量。田儋首倡齐国之复，为田氏三齐之始。

> 狄人田儋起兵于齐。
> （007-097）
