# 大夏

大夏是位于中亚阿姆河流域的古国（即希腊化巴克特里亚王国）。张骞第一次出使西域至大月氏，途经大夏，在此见邛竹杖、蜀布，知有身毒可通（018-083）。张骞又分遣副使使大夏诸国（019-039）。

大月氏为[[匈奴]]所破，西徙击大夏而臣之（019-040）。张骞的副使往大夏等国，"各得一二十人"（020-034）。西域自玉门、阳关出，南道西逾葱岭则出大月氏、安息（020-036）。大夏地处丝绸之路要冲，为东西文明交汇之地。
