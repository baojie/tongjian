# 大赦

大赦是皇帝对天下罪犯予以赦免的恩典制度，历代君主即位、[[改元]]、立后、建储等重大事件时经常实行。

[[战国|战国时期]]即有"大赦"之名，如赵主父灭中山后"归，行赏，大赦，置酒，酺五日"（004-012）。秦末汉初，大赦渐成定制，新君即位必赦天下（007-079）。

大赦的适用对象通常为除谋反大逆、贪赃枉法等重罪之外的普通罪犯。各朝对大赦的范围和条件有不同规定，如[[北魏]]律规定"大赦"时"降城亡子不得与焉"（006-041），即降敌叛国者不在赦免之列。

大赦制度体现了君主对臣民的恩德，也起到展示宽仁、收拢民心、庆祝重大典礼的政治功能。但频繁大赦也导致法律权威受损，[[司马光]]对此多有批评。
