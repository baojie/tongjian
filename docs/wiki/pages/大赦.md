# 大赦

大赦是[[皇帝]]对[[天下]]罪犯予以赦免的恩典[[制度]]，历代[[君主]][[即位]]、[[改元]]、立后、建储等重大事件时经常实行。

[[战国|战国时期]]即有"大赦"之名，如[[主父|赵主父]]灭[[中山]]后"归，[[行赏]]，大赦，置酒，酺五日"（004-012）。秦末汉初，大赦渐成定制，新君[[即位]]必赦[[天下]]（007-079）。

大赦的适用对象通常为除[[谋反]][[大逆]]、贪赃[[枉法]]等重罪之外的普通罪犯。各朝对大赦的范围和条件有不同规定，如[[北魏]]律规定"大赦"时"降城亡子不得与焉"（006-041），即降敌叛国者不在赦免之列。

大赦[[制度]]体现了[[君主]]对臣民的[[恩德]]，也起到展示[[宽仁]]、收拢民心、庆祝重大典礼的[[政治]]功能。但频繁大赦也导致[[法律]]权威受损，[[司马光]]对此多有批评。
