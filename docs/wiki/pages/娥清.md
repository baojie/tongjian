# 娥清

娥清，代人。[[北魏]][[振威]][[将军]]。

明[[元帝]]遣[[振威]][[将军]]娥清、[[冀州]][[刺史]]阿薄干将步骑十万屯[[河北]]岸（118-017）。太[[武帝]]遣中领军娥清等将兵七千人南渡河（119-091）。娥清、周几等徇地至[[湖陆]]、[[高平]]，[[民屯]]聚而射之，清等尽攻破[[高平]]诸县，虏掠而还（119-118）。

> 魏主遣中领军代人娥清、期思侯柔然闾大肥将兵七千人会周几、叔孙建南渡河。
> （119-091）
