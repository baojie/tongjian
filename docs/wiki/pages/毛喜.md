# 毛喜

毛喜，字伯武，[[荥阳]]阳武人。[[南朝]]陈[[侍中]]，以忠直[[谏诤]]著称。

毛喜为陈[[侍中]]，尽忠[[辅政]]（169-001）。毛喜谏陈[[刘禅|后主]]勿[[奢靡]]（170-026）。

毛喜少[[好学]]，善属文。陈[[隋文帝|文帝]]时，为[[侍中]]，参掌机密。[[司马懿|宣帝]]时，喜竭诚尽忠，每有[[谏诤]]，必尽其词。[[刘禅|后主]][[即位]]，喜见其[[奢靡]]，谏曰："愿[[陛下]][[节用爱人]]，以保[[社稷]]。"[[刘禅|后主]]不悦，出为[[南康]][[内史]]。喜忧愤卒。陈之亡，人思喜之言。

> 毛喜为陈侍中，尽忠辅政。
> （169-001）
