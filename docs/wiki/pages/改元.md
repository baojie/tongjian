# 改元

改元是皇帝更改[[年号]]的制度，寓意除旧布新、更始革新，始于[[西汉]][[隋文帝|文帝]]、景帝时期，[[汉武帝]]确立[[年号]]制度后成为定制。

汉代改元常以天象[[祥瑞]]或重大事件为契机。如[[汉武帝]]因获白麟而改元元狩，因得宝鼎而改元元鼎。汉哀帝时，待诏贺良等建言"宜急改元易号，乃得延年益寿"（034-012），哀帝从之，改元太初元将，后因无应验而废（034-014）。

历代皇帝即位次年通常改元，亦有在位期间多次改元者。[[武后|武则天]]在位期间改元最为频繁，达十八次之多。宋元以后，帝位更替时改元成为定制，一般一帝一[[年号]]，极少在位中改元。

改元不仅是纪年方式的变化，更是昭告[[天命]]所归、宣示政治革新的重要仪式。[[资治通鉴概述|通鉴]]每卷开首即记[[年号]]改元，以明时代断限（023-044）。
