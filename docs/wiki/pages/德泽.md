# 德泽

**德泽**指以德化[[恩泽]]施于[[百姓]]，是[[儒家]][[仁政]]思想的核心理念。

## 汉代

[[贾谊]]论[[治道]]，主张以德泽化民，反对专任[[刑罚]]。[[汉文帝]]纳其言，德泽流布，天下归心（014-042）。[[董仲舒]]亦倡德治，谓"[[天道]]之大者在[[阴阳]]"，[[治国]]当以德泽为本（015-026）。[[两汉]]之治，德泽为先。
