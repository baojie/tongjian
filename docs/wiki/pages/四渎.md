# 四渎

四渎指中国古代四条独流入海的大川：**江**（[[长江]]）、**河**（[[黄河]]）、**淮**（[[淮河]]）、**济**（[[济水]]），古称"四渎"，与[[五岳]]并列为重要的山川祭祀对象。

## 通鉴中的四渎

[[南朝梁]]修筑浮山堰时，有人认为"四渎，天所以节宣其气，不可久塞"（148-046），以四渎象征天地气脉之通畅。[[北魏]]末年[[尔朱世隆]]使齐州行台尚书房谟"募兵趣四渎"（155-068）。[[武则天]]改元后封洛水神，祭祀比四渎之礼（204-029）。

## 山川祭祀

四渎之神是历代王朝祭祀体系中的重要组成部分，与[[五岳]]并称"岳渎"。天子祭祀四渎，象征对天下河川的统御。
