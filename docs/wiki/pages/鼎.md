# 鼎

鼎是古代青铜礼器，夏商周三代以鼎为国家权力象征，问鼎[[中原]]即争夺天下之意。

[[张仪]]说秦惠王，伐韩据九鼎，[[挟天子]]以令于天下（003-016）。秦武王好以力戏，与孟说举鼎，绝脉而薨（003-053）。[[秦始皇]]过彭城，欲出周鼎泗水，使千人没水求之弗得（007-043）。

[[项羽]]力能扛鼎（007-102）。蒯通说[[韩信]]参分天下，鼎足而居（010-036）。[[淮南]]厉王有材力能扛鼎，袖铁椎杀辟阳侯（014-009）。新垣平言周鼎在泗水中，望东北汾阴有金宝气，以为周鼎将出（015-048）。鼎作为权力象征，其传承关乎天命所归。

> 亲魏，善楚，下兵三川，攻新城、宜阳，以临二周之郊，据九鼎，按图籍，挟天子以令于天下。
> （003-016）
