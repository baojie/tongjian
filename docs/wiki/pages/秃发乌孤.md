# 秃发乌孤

秃发乌孤，河西鲜卑秃发部首领，南凉建立者。趁后凉吕光衰微之际自立，据有河西走廊。

秃发乌孤击乙弗、折掘等诸部，皆破降之，筑廉川堡而都之（108-076）。光遣使者拜秃发乌孤为征南大将军、益州牧、左贤王。乌孤谓使者曰："吕王诸子贪淫，三甥暴虐，远近愁怨，吾安可违百姓之心，受不义之爵？"（108-101）

秃发乌孤为河西鲜卑秃发部首领，秃发即拓跋之异译。乌孤趁后凉吕光病衰之际，击破附近诸部，筑廉川堡为都，后徙居乐都。乌孤拒绝后凉封爵，自称武威王，建立南凉政权。然乌孤在位不久即卒，临终命弟利鹿孤继位。南凉立国十八年，为西秦所灭。
