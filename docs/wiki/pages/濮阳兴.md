# 濮阳兴

濮阳兴，[[陈留]]人。吴[[丞相]]。

吴[[都尉]]严密建议作浦里塘，群臣皆以为难，唯卫[[将军]][[陈留]]濮阳兴以为可成，遂会诸军民就作，功费[[不可胜数]]，[[士卒]]多死亡，民大愁怨（077-070）。吴主孙休以濮阳兴为[[丞相]]（078-005）。休寝疾，口不能言，乃手书呼[[丞相]]濮阳兴入，以子托之（078-064）。[[孙皓]]立，兴被诛。

> 吴主寝疾，口不能言，乃手书呼丞相濮阳兴入，令子出拜之。休把兴臂，把л以托之。
> （078-064）
