# 云中

云中为[[战国]]赵地，[[赵武灵王]]置云中郡（004-045）。秦灭赵后沿置云中郡，[[蒙恬]]北逐[[匈奴]]，云中为防胡重镇（007-041）。[[西汉]]时[[李广]]为云中[[太守]]，[[匈奴]]不敢犯塞（017-022）。

云中地处河套以东，自古以来为农耕与游牧的分界线。汉与[[匈奴]]、唐与[[突厥]]在云中一带反复争夺，为北疆[[军事]]重镇。

[[东汉]][[鲜卑]]其至鞬寇[[居庸关]]，云中[[太守]]成严击之，兵败，[[功曹]]杨穆以身捍严俱殁（050-089）。[[何进]]遣云中人[[张杨]]还[[晋阳|并州]][[募兵]]，会进败，留[[上党]]有众数千人（060-011）。

## 云中军事地位

[[张仪]]说[[燕王]]，言秦下甲云中、[[九原]]，驱赵而攻燕，则[[易水]][[长城]]非燕之有（003-037）。[[赵武灵王]]欲从云中、[[九原]]南袭[[咸阳]]（003-083）。赵置云中、[[雁门]]、[[代郡]]以拒胡（006-057）。汉初以云中、[[雁门]]、[[代郡]]封兄宜信侯喜为代王（011-036）。[[周勃]]悉定[[代郡]]、[[雁门]]、云中地，斩[[陈豨]]于当城（012-045）。云中自古为北疆要地，农耕与游牧在此交汇，历代[[兵家]]必争。
