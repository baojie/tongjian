# 皮日休
**皮日休**（约838–约883年），字袭美，[[襄阳]]（今湖北[[襄阳]]）人，唐末[[诗人]]、散文家。曾参加[[黄巢]]起义军，任翰林学士。
## 生平
皮日休咸通年间[[进士]]及第，曾任著作郎、[[太常]][[博士]]等职。广明元年（880年），[[黄巢]]入[[长安]][[称帝]]，以皮日休为翰林学士（254-019）。[[黄巢]]败亡后，皮日休不知所终。
## 文学成就
皮日休与[[陆龟蒙]]齐名，世称"皮陆"。其诗文多反映唐末社会现实，风格峭拔。著有《皮子文薮》十卷。
