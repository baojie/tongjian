# 高祐

高祐，北魏秘书令。奏改《国书》编年为纪传体，建言止盗之术。

魏秘书令高祐、丞李彪奏请改《国书》编年为纪、传、表、志，魏主从之，祐，允之从祖弟也（136-083）。魏主问高祐曰："何以止盗？"对曰："昔宋均立德，猛虎渡河；卓茂行化，蝗不入境。况盗贼，人也，苟守令得人，治化有方，止之易矣。"（136-084）。
