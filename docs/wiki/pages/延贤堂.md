# 延贤堂

**延贤堂**，位于[[华林园]]内的重要[[殿堂]]。[[南燕]]主[[慕容备德]]宴群臣于此；[[南朝宋]][[皇帝]]亦常临延贤堂[[听讼]]理政。

## 通鉴记载

[[南燕]]主备德宴群臣于延贤堂（112-039）。

宋帝临延贤堂[[听讼]]，自是每岁三讯（120-081）。

宋帝出[[华林园]]延贤堂，召[[岑文本|景仁]]议诛讨（123-114）。
