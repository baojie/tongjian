# 胡骑

胡骑，胡人组成的[[骑兵]]部队，以其强劲的机动能力著称。

[[汉高祖]]困平城胡骑解去（011-048）。[[李广]]以数十骑御胡骑（016-095）。

胡骑为汉军劲敌。[[汉高祖]]被[[匈奴]]围于平城，胡骑四面围攻，帝用[[陈平]]计得出。[[李广]]为右北平守，尝出猎，胡骑大至，广令士卒解鞍以示不走，胡骑不敢击。胡骑长于[[骑射]]，来去如风，[[西汉|汉朝]]以长城[[守御]]、以[[和亲]]怀柔、以[[骑兵]]对抗。汉武设八校尉，胡骑校尉掌池阳胡骑，即所以制胡之策。
