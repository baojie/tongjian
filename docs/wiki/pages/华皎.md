# 华皎

华皎，晋陵人。[[南朝]]陈湘州[[刺史]]。

华皎为寻阳太守，遣兵逆击叛军，尽获其船仗（168-119）。湘州[[刺史]]华皎闻[[韩子高]]死，内不自安，缮甲聚徒，抚循所部，启求广州以卜朝廷之意（170-018）。陈顼遣[[章昭达]]、[[程灵洗]]等共袭华皎。皎兵败奔[[江陵]]，依后梁（170-020）。
