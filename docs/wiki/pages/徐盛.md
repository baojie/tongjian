# 徐盛

徐盛，字文向，[[琅邪]]莒人。[[三国]]吴安东将军，以勇武著称。

徐盛初为芜湖令，收[[蒋钦]]屯吏，表斩之。[[蒋钦]]不记前嫌，每称徐盛之善（068-003）。吴黄武元年，[[刘备]]伐吴，[[孙权]]以[[陆逊]]为大[[都督]]，督徐盛、朱然、[[潘璋]]等五万人拒之（069-040）。后[[孙权]]与[[曹丕|魏文帝]][[曹丕]]使臣邢贞盟，贞有骄色，徐盛忿愤，谓同列曰："盛等不能奋身出命，为国家并许、洛，吞巴、蜀，而令吾君与贞盟，不亦辱乎！"（069-051）

> 中郎将琅邪徐盛忿愤，顾谓同列曰："盛等不能奋身出命，为国家并许、洛，吞巴、蜀，而令吾君与贞盟，不亦辱乎！"
> （069-051）
