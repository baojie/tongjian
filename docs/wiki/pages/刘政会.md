# 刘政会

刘政会，唐初大臣，凌烟阁二十四功臣之一。从晋阳起兵，后为刘武周所虏，密表陈武周形势。

刘文静引开阳府司马胙城刘政会入立庭中，称有密状（183-070）。渊以刘政会及武城崔善为、太原张道源为户曹（184-006）。卫尉少卿刘政会在太原，为武周所虏，政会密遣人奉表论武周形势（187-094）。刘政会配享凌烟阁（196-068）。
