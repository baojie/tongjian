# 天竺

天竺，[[汉代]]至唐代对印度之称呼。[[佛教]]自天竺传入中土。

后[[汉明帝]]闻[[西域]]有神曰佛，[[遣使]]天竺求其道，得其书及[[沙门]]以来（045-028）。[[后赵]][[石勒]]、[[石虎]]奉天竺僧[[佛图澄]]，敬事之，[[佛图澄]]预言[[成败]]数有验（095-080）。[[前秦]][[吕光]]征[[西域]]，天竺[[沙门]]鸠摩罗什劝光东归（106-018）。[[南朝]]宋时，师子王与天竺迦毘黎王月爱皆[[遣使]][[入贡]]（121-032）。隋末，大理卿骨仪本天竺[[胡人]]也（182-039）。

> 帝闻西域有神，其名曰佛，因遣使之天竺求其道，得其书及沙门以来。
> （045-028）
