# 天竺

天竺，汉代至唐代对印度之称呼。佛教自天竺传入中土。

后汉明帝闻西域有神曰佛，遣使天竺求其道，得其书及沙门以来（045-028）。后赵石勒、石虎奉天竺僧佛图澄，敬事之，佛图澄预言成败数有验（095-080）。前秦吕光征西域，天竺沙门鸠摩罗什劝光东归（106-018）。南朝宋时，师子王与天竺迦毘黎王月爱皆遣使入贡（121-032）。隋末，大理卿骨仪本天竺胡人也（182-039）。

> 帝闻西域有神，其名曰佛，因遣使之天竺求其道，得其书及沙门以来。
> （045-028）
