# 秦治驰道

秦治驰道是[[秦始皇]]在统一六国后下令修筑全国性交通干线的重大工程。始皇二十七年（前220年），秦"治驰道于天下"（007-038），以首都咸阳为中心，向全国各地辐射，形成中国历史上第一个全国性交通网络。

## 背景

秦始皇二十六年（前221年）灭齐，六国统一。为了加强对全国的统治、便于军队调动和政令传达，秦始皇下令修筑驰道。驰道作为秦帝国的重要基础设施，与车同轨、书同文等政策相配套。

## 驰道工程

驰道以咸阳为中心，东通燕、齐（今河北、山东），南达吴、楚（今江苏、湖北），主要干线宽约五十步（约合今70米），道旁每隔三丈植青松一株，路基夯实，路面平整。同期秦始皇还修建了自咸阳至骊山的甬道、甘泉前殿等配套设施。

## 作用

驰道使秦的政令能够迅速传达全国，加强了中央对地方的控制。同时也促进了各地的经济文化交流。驰道体系为后世汉唐等王朝的道路建设奠定了基础，成为中国两千多年驿道制度的开端。
