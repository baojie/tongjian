# 累迁

**累迁**指官员因政绩或资历连续多次升迁，为古代[[官僚]]体系中常见的晋升路径。

## 累迁至高位

[[杨震]]年五十馀始仕，累迁[[荆州刺史]]、[[东莱]][[太守]]（049-081）。[[陈蕃]]因面[[称帝]]违，拜五官[[中郎将]]，累迁[[大鸿胪]]（054-043）。[[黄皓]]始预政，累迁至中常侍，操弄威柄，终以覆国（074-083）。

## 累迁循吏

[[司马]]舒[[对策]]升第，累迁[[后将军]]钟毓[[长史]]（078-023）。张俶以谮白得宠，累迁司直[[中郎将]]，[[封侯]]（080-072）。累迁既可为贤臣进身之阶，亦可为[[佞幸]]窃权之途。
