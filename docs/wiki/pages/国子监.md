# 国子监

国子监是国家最高教育机构，掌儒学教育，隋唐时正式设立，历代沿用。

[[唐太宗]]幸国子监，观释奠，命[[祭酒]]孔颖达讲《孝经》，赐[[祭酒]]以下至诸生高第帛有差（195-079）。天宝年间置广文馆于国子监，以教诸生习[[进士]]者（216-039）。

杨绾上贡举条目，国子监举人令[[博士]]荐于[[祭酒]]，[[祭酒]]试通者升之于省，如乡贡法（223-004）。国子监在唐代教育体系中发挥核心作用，其兴衰直接反映王朝文教政策之走向。
