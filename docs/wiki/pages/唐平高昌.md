# 唐平高昌

**唐平高昌**是[[贞观之治|贞观]]十四年（640年），[[李世民|唐太宗]]命[[侯君集]]率军讨灭[[高昌]]国的战役。唐朝在[[高昌]]故地置西州，标志着唐朝正式将[[西域]]东部纳入行政管辖。

## 背景

[[高昌]]王[[麹文泰]]与西[[突厥]]勾结，阻断[[西域]]商路，侵扰唐朝边境。[[李世民|唐太宗]]徵[[麹文泰]]入朝，文泰称疾不至（195-004）。

## 进军

[[贞观之治|贞观]]十四年，[[李世民|唐太宗]]命[[侯君集]]为交河道[[行军]]大总管，率步骑数万[[讨伐]][[高昌]]。唐军至碛口，[[麹文泰]]闻讯惊惧而死。

## 克城

子麹智盛继位后[[出兵]]拒战，被唐军击败。唐军兵临[[高昌]]城下，麹智盛出降。唐得[[高昌]]三州五县二十二城，户八千，口三万七千（195-006至195-008）。

## 结果

唐朝在[[高昌]]置西州，设[[安西都护府]]，作为唐朝经营[[西域]]的基地。[[高昌]]之平为唐朝后来击败西[[突厥]]、控制[[西域]]奠定了基础。[[魏徵]]等[[谏臣]]认为[[高昌]]荒远，置兵镇守加重百姓负担，太宗不纳（195-008）。
