# 于阗

于阗是[[西域]]南道大国，以产玉闻名。[[张骞]]通[[西域]]时分遣副使使[[大宛]]、[[康居]]、大[[月氏]]、[[赫连勃勃|大夏]]、[[安息]]、身毒、于阗及诸旁国（020-035），于阗自此与汉[[通使]]。[[西域]]南道自[[鄯善]]傍南山而行至于阗，于阗有河"一出葱岭，一出于阗"合流注盐泽（020-036）。

东[[汉明帝]]时"[[鄯善]]、[[龟兹]]、于阗王各[[遣使]]奉献"，[[西域]]复通（069-061）。[[前凉]][[张骏]]遣将讨[[西域]]，"于是[[西域]]诸国[[焉耆]]、于阗之属，皆诣[[姑臧]][[朝贡]]"（095-087）。于阗自[[西汉]]至唐始终为[[西域]]重要邦国，唐时设毗沙[[都督]]府，至五代犹通[[中原]]。
