# 于阗

于阗是西域南道大国，以产玉闻名。张骞通西域时分遣副使使大宛、康居、大月氏、大夏、安息、身毒、于阗及诸旁国（020-035），于阗自此与汉通使。西域南道自鄯善傍南山而行至于阗，于阗有河"一出葱岭，一出于阗"合流注盐泽（020-036）。

东汉明帝时"鄯善、龟兹、于阗王各遣使奉献"，西域复通（069-061）。前凉张骏遣将讨西域，"于是西域诸国焉耆、于阗之属，皆诣姑臧朝贡"（095-087）。于阗自西汉至唐始终为西域重要邦国，唐时设毗沙都督府，至五代犹通中原。
