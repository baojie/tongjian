# 唐穆宗

唐穆宗李恒（795年—824年），[[唐朝]]第十一位[[皇帝]]，宪宗之子。

## 即位

[[元和]]十五年（820年），宪宗被[[宦官]]陈弘志所弑，穆宗[[即位]]（241-043）。

## 河朔再叛

穆宗在位期间荒于政事，游幸无常，[[赐予]]无度。长庆初年，[[河朔三镇]]再度叛乱（242-002），[[朝廷]][[用兵]]无功，[[河北]]地区遂失控制。穆宗因击毬惊吓得疾，不久崩逝（242-040）。
