# 李良

李良，秦末赵将。从[[武臣]]略赵地，攻取[[常山]]、[[晋阳|太原]]，后为秦将[[章邯]]所败，叛赵降秦。

[[王镕|赵王]]使李良略[[常山]]，张黡略[[上党]]（007-096）。赵李良已定[[常山]]，还报[[王镕|赵王]]，[[王镕|赵王]]复使良略[[晋阳|太原]]，至石邑，秦兵塞[[井陉]]，未能前（008-007）。赵[[张耳]]、[[陈馀]]收其散兵击李良，良败，走归[[章邯]]（008-014）。

> 赵李良已定常山，还报赵王。赵王复使良略太原，至石邑，秦兵塞井陉，未能前。
