# 李良

李良，秦末赵将。从武臣略赵地，攻取常山、太原，后为秦将章邯所败，叛赵降秦。

赵王使李良略常山，张黡略上党（007-096）。赵李良已定常山，还报赵王，赵王复使良略太原，至石邑，秦兵塞井陉，未能前（008-007）。赵张耳、陈馀收其散兵击李良，良败，走归章邯（008-014）。
