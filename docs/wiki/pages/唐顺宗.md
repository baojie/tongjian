# 唐顺宗

唐顺宗李诵（761年—806年），[[唐朝]]第十三位[[皇帝]]，德宗之子。

## 即位与永贞革新

[[贞元]]二十一年（805年），德宗崩，顺宗[[即位]]（236-056）。时顺宗失音，不能决事，常居宫中施帘帷，独[[宦官]][[李忠]]言、昭容牛氏侍左右（236-059）。顺宗任用[[王叔文]]、[[王伾]]、[[柳宗元]]、[[刘禹锡]]等人推行[[永贞革新]]，抑制藩镇和[[宦官]]势力，加强中央集权。但改革因[[宦官]]集团反对而失败，顺宗被迫内禅于宪宗（237-033）。
