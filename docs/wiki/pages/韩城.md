# [[韩城]]

[[韩城]]是[[战国]]时期[[魏国]][[城邑]]，[[秦]]置县，位于今[[陕西]]韩城，地处[[黄河]]西岸，扼守[[关中]][[东北]]部的秦晋通道。韩城为[[秦]]、[[魏]]两国长期争夺之地，[[秦惠文王]]时，秦攻魏，取韩城。

[[汉代]]韩城属[[左冯翊]]。[[司马迁]]即[[夏阳]]（今韩城一带）人，韩城因此有"[[史圣]]故里"之称。
