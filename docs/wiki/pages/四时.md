# 四时

四时指春、夏、秋、[[冬_(概念)|冬]]四季，是[[中国]]古代[[历法]]与时间观念的基本框架，亦是政治哲学中"顺天应时"思想的基础。

## 通鉴中的四时

[[苏秦]]说[[齐宣王]]从亲抗秦时，请[[山东]]之国"奉四时之献，以承大王之明诏"（002-094），四时[[朝贡]]为[[诸侯]]之礼。[[蔡泽]]以"四时之序，成功者去"说[[范雎]]退位让贤（006-003）。[[陈平]]论[[宰相]]之责时称"上佐[[天子]]，理[[阴阳]]，顺四时"（013-077），视四时调和为[[宰相]]天职。[[贾谊]]论政则称"庆赏以劝善，[[刑罚]]以惩恶，行此之令，信如四时"（014-051）。

## 四时与农时

[[晁错]]论农民之苦曰"春不得避风尘，夏不得避暑热，秋不得避阴雨，冬不得避寒冻，四时之间亡日休息"（015-024），四时与[[农事]]紧密结合，是古代民生的基本节律。
