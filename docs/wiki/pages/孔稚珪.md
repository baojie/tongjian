# 孔稚珪

孔稚珪，字德璋，会稽山阴人。南朝齐廷尉，以文学律学知名。撰《律文》二十卷。

廷尉山阴孔稚珪上表，以为："《律》文虽定，苟用失其平，则法书徒明于帙里，冤魂犹结于狱中。"（137-066）中丞孔稚珪奏王融险躁轻狡，招纳不逞，诽谤朝政（138-043）。

孔稚珪在齐武帝时掌廷尉，精于律令，撰《律文》二十卷。稚珪以为律法虽定，然吏用失其平则法为空文，时有"法书徒明"之论。亦能文章，与江淹、谢朓等游。齐明帝时，稚珪奏诛王融，以此颇受讥议。稚珪仕齐至太子詹事。
