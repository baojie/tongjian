# 孔稚珪

孔稚珪，字德璋，[[会稽]][[山阴]]人。[[南朝]]齐[[廷尉]]，以[[文学]]律学知名。撰《律文》二十卷。

[[廷尉]][[山阴]]孔稚珪[[上表]]，以为："《律》文虽定，苟用失其平，则法书徒明于帙里，冤魂犹结于狱中。"（137-066）中丞孔稚珪奏[[王融]]险躁轻狡，招纳不逞，[[诽谤]][[朝政]]（138-043）。

孔稚珪在[[萧赜|齐武帝]]时掌[[廷尉]]，精于[[律令]]，撰《律文》二十卷。稚珪以为律法虽定，然吏用失其平则法为空文，时有"法书徒明"之论。亦能[[文章]]，与[[江淹]]、[[谢朓]]等游。齐[[汉明帝|明帝]]时，稚珪奏诛[[王融]]，以此颇受讥议。稚珪仕齐至[[太子]]詹事。

> 廷尉山阴孔稚珪上表，以为："《律》文虽定，苟用失其平，则法书徒明于帙里，冤魂犹结于狱中。"
> （137-066）
