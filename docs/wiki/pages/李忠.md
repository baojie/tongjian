# 李忠

李忠，字仲都，东莱黄人。[[东汉]]右大将军，云台二十八将之一。从[[光武帝刘秀|光武帝]]定[[河北]]，封信都侯。

秀拜[[任光]]为左大将军，信都都尉李忠为右大将军（039-041）。王郎遣将攻信都，大姓马宠等开城内之。更始遣兵攻破信都，秀使李忠还行太守事（039-046）。

李忠在[[光武帝刘秀|光武帝]]信都起兵时为右大将军，与[[任光]]分统左右。信都为王郎所攻时，城内大姓马宠叛变，李忠家属被执。王郎以李忠母妻为人质招降，忠斩叛将来使，不以私废公。光武闻而嘉之，拜忠为太守。李忠后从光武平定[[河北]]，封列侯。永平中图画云台，李忠位列二十八将。

> 秀拜任光为左大将军，信都都尉李忠为右大将军。
> （039-041）
