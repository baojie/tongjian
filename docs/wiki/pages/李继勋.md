# 李继勋

李继勋，后周大将，从征后蜀拔秦州，从征南唐战于寿春，入宋后封王。

蜀主以左千牛卫上将军李继勋为秦州宣慰使（286-016）。蜀李继勋与兴州刺史刘景攻固镇拔之（286-032）。侍卫步军都指挥使李继勋营于寿州城南，唐刘仁赡出兵击之（293-034）。河阳节度使李继勋为右卫大将军（293-087）。

## 世系

李继勋（？—约977年），[[后周]]、[[北宋]]将领。其先世不详于史。继勋从[[后周世宗]]征[[南唐]]。入[[北宋]]为昭义[[节度使]]，封王。继勋[[子孙]]不详于史。
