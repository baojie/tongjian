# 摄政

摄政是君主年幼或不能理政时，由宗室重臣、[[太后]]或[[外戚]]代行国政的制度。摄政源于周公辅成王的故事，历代沿用为政治惯例。

[[西汉]][[霍光]]受武帝遗诏辅佐昭帝，处伊尹、周公之位摄政擅权（023-013）。[[王莽|新莽]]末年，[[申屠刚]]以周公摄政为例劝谏[[更始|更始帝]]纳贤听谏，指出"成王幼少，周公摄政，听言下贤，均权布宠"方得天下归心（035-044）。[[东汉]]多幼帝即位，[[太后]]临朝[[称制]]与大臣摄政并存。

摄政制度在维护政权稳定方面有其积极作用，但也常引发权力斗争。如[[王莽]]以安汉公摄政，周公摄政七年之说为其篡汉提供了合法性辩解（036-019）。后世如清代多尔衮、醇亲王载沣等均以摄政身份执掌朝政。
