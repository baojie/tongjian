# 卢毓

卢毓，魏吏部[[尚书]]、[[司空]]。[[卢植]]之子，以选举贤能著称。

[[曹操]]以卢毓为法曹议令史。毓，植之子也（065-032）。[[侍中]]卢毓进谏，帝乃解（073-028）。帝诏吏部[[尚书]]卢毓曰："选举莫取有名，名如画地作饼，不可啖也。"毓对曰："名不足以致异人而可以得常士"（073-055）。卢毓论人及选举，皆先性行而后言才（073-059）。帝问吏部[[尚书]]卢毓："谁可为[[司徒]]者？"毓荐处士[[管宁]]（074-005）。
