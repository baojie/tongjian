# 卢毓

卢毓，魏吏部尚书、司空。卢植之子，以选举贤能著称。

曹操以卢毓为法曹议令史。毓，植之子也（065-032）。侍中卢毓进谏，帝乃解（073-028）。帝诏吏部尚书卢毓曰："选举莫取有名，名如画地作饼，不可啖也。"毓对曰："名不足以致异人而可以得常士"（073-055）。卢毓论人及选举，皆先性行而后言才（073-059）。帝问吏部尚书卢毓："谁可为司徒者？"毓荐处士管宁（074-005）。
