# 魏国

魏国为战国七雄之一，据有中原腹地。[[《资治通鉴》]]中魏国[[魏文侯]]率先推行变法，招贤纳士，开战国霸业之先河。

[[吴起]]浮西河而下，中流谓吴起曰："美哉山河之固，此魏国之宝也！"（001-060）。[[北魏]]时，[[元顺]]斥[[徐纥]]："此魏之宰嚭，魏国不亡，此终不死！"（151-012）。
