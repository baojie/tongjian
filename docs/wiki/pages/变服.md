# 变服

**变服**指改变服饰，常指因[[习俗]]变革、[[丧礼]]或隐秘行动而改变衣着。著名的"变服[[骑射]]"即[[赵武灵王]]推行[[胡服骑射]]之改革。变服亦指[[帝王]]因[[灾异]]而变服修省，或为[[隐匿]]身份而改装。
