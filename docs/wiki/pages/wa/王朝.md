# [[王朝]]

《[[资治通鉴概述|资治通鉴]]》以编年体记载了自[[周威烈王]]至[[后周]]多个[[王朝]]的兴衰历程，包括[[秦朝|秦]]、[[汉朝|汉]]、[[三国]]、[[两晋]]、[[南北朝]]、[[隋]]、[[唐]]、[[五代十国]]等主要[[王朝]]。

> 臣光曰：臣闻天子之职莫大于礼，礼莫大于分，分莫大于名。
> （001-004）

通鉴通过记录各[[王朝]]的建立、鼎盛与衰亡，揭示了[[王朝]]更替的内在规律。司马光认为[[礼]][[名分]]是维系[[王朝]]长治久安的根本，而[[王朝]]衰落的共同特征则是[[纲纪]]败坏、[[名器]]滥授。
