# 崔延伯

崔延伯，博陵人。[[北魏]]征西将军，骁勇善战，以镇压莫折天生之乱闻名。

崔延伯既破[[秦州]]叛羌，乃于[[安定]]与[[杨大眼]]等合势，整勒[[部曲]]，号令严明（148-040）。崔延伯率众击莫折天生，大破之，降者十馀万（150-055）。

崔延伯少以勇略闻，初从[[征伐]]，为统军，累迁征西将军。孝[[汉明帝|明帝]]时，六镇乱起，莫折天生据陇东，延伯受诏讨之。延伯每战必先士卒，[[勇冠三军]]，然轻敌寡谋，终为[[万俟丑奴]]所袭，战殁。延伯死后，[[关中]]大扰，朝廷深惜之。
