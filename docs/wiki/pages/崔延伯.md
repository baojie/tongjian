# 崔延伯

崔延伯，博陵人。[[北魏]][[征西]][[将军]]，[[骁勇]]善战，以镇压[[莫折天生]]之乱闻名。

## 骁勇殒身

崔延伯为[[北魏]][[征西]][[将军]]，与[[萧宝寅]]讨[[莫折天生]]于黑水，率众五万，大破之，降者十馀万（150-055）。又引兵会宝寅于[[安定]]，甲卒十二万，军威甚盛，与[[万俟丑奴]]相持（150-077）。初，[[崔亮]]攻[[硖石]]，延伯守[[下蔡]]，取车轮去辋削锐接竹为桥，夹淮为营，以遏水势（148-040）。延伯虽[[骁勇]]，然轻敌寡谋，终为丑奴所袭，战殁。

崔延伯既破[[秦州]]叛羌，乃于[[安定]]与[[杨大眼]]等合势，整勒[[部曲]]，[[号令]]严明（148-040）。崔延伯率众击[[莫折天生]]，大破之，降者十馀万（150-055）。

崔延伯少以[[勇略]]闻，初从[[征伐]]，为统军，累迁[[征西]][[将军]]。孝[[汉明帝|明帝]]时，六镇乱起，[[莫折天生]]据陇东，延伯受诏讨之。延伯每战必先[[士卒]]，[[勇冠三军]]，然轻敌寡谋，终为[[万俟丑奴]]所袭，战殁。延伯死后，[[关中]]大扰，[[朝廷]]深惜之。
