# 宗室

宗室，指[[天子]]或[[诸侯]]的同姓[[宗族]]。秦汉以来，宗室成员享有[[封爵]]、[[赏赐]]等特权，但亦受[[朝廷]]约束。

[[秦孝公]]用[[商鞅]]，宗室非有[[军功]]论，不得为属籍（002-020）。周，天下之宗室也；齐，韩之与国也（003-016）。王及宗室所[[赏赐]]者，尽以与军吏[[士大夫]]（005-042）。

> 宗室非有军功论，不得为属籍。明尊卑爵秩等级，各以差次名田宅、臣妾、衣服。
> （002-020）
