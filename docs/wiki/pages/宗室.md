# 宗室

宗室，指天子或诸侯的同姓宗族。秦汉以来，宗室成员享有封爵、赏赐等特权，但亦受朝廷约束。

秦孝公用商鞅，宗室非有军功论，不得为属籍（002-020）。周，天下之宗室也；齐，韩之与国也（003-016）。王及宗室所赏赐者，尽以与军吏士大夫（005-042）。

> 宗室非有军功论，不得为属籍。明尊卑爵秩等级，各以差次名田宅、臣妾、衣服。
> （002-020）
