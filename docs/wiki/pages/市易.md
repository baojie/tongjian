# 市易

市易是古代官府主导的市场贸易管理。[[王莽]]改制，推行五均六筦，于长安及五都设五均司市师，掌市易[[平准]]（037-023）。五均之意在于平抑物价、防止奸商囤积居奇。

唐代市易由市令管理，长安东市、西市为全国最大商业中心。[[唐德宗]]时"市易蕃息，[[商旅]]辐凑"（216-076）。[[白居易]]《卖炭翁》"一车炭，千馀斤，宫使驱将惜不得"，即反映宫市对市易秩序的破坏（269-073）。市易制度的健全与否直接关系国计民生。
