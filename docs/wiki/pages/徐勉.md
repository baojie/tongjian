# 徐勉

徐勉，字修仁，东海郯人。梁朝名相，以勤勉著称。

徐勉初与王融相识，融欲引为党，勉知其必败，避之（138-044）。梁武帝以徐勉及周舍同参国政，两人俱称贤相（145-069）。勉精力过人，为吏部尚书，虽文案填积，坐客充满，应对如流，手不停笔（146-104）。勉居官清廉，不营产业，世称贤相。

> 上以约轻易，不如尚书左丞徐勉，乃以勉及右卫将军汝南周舍同参国政。
> （145-069）
