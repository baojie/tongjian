# 张苍

**张苍**，阳武人，[[西汉]][[丞相]]，[[律历]]大家。

[[天下]]既定，[[汉高祖刘邦|刘邦]]命[[萧何]]次[[律令]]，[[韩信]]申[[军法]]，张苍定章程，[[叔孙通]]制[[礼仪]]（012-054）。张苍好书，博闻，尤邃[[律历]]。[[隋文帝|文帝]]时以[[御史大夫]]为[[丞相]]（014-020）。张苍奏定[[律令]]，以[[髡钳]][[城旦]]舂代黥、劓等[[肉刑]]（015-032）。

张苍以为汉得水德，鲁人[[公孙臣]]以为汉当土德，苍以为非是（015-038）。后竟罢相。张苍为汉初[[制度]]建设的核心人物，[[律历]]章程多出其手。

## 定律历

张苍从[[汉高祖刘邦|刘邦]]定[[天下]]，受命定章程（012-054）。[[隋文帝|文帝]]时为[[丞相]]（014-020），与[[御史大夫]]冯敬奏定[[律令]]，以[[髡钳]][[城旦]]舂代黥、劓、斩左趾等[[肉刑]]（015-032）。又奏[[英布|淮南王]]长废[[先帝]]法，当[[弃市]]（014-034）。苍以为汉得水德，鲁人[[公孙臣]]以为当土德，及[[黄龙]]见[[成纪]]，苍由此自绌，遂免相（015-038、015-040、015-056）。
