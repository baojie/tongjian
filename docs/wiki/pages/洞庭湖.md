# [[洞庭湖]]

[[洞庭湖]]位于[[湖南]]北部，[[长江]]中游南岸，是中国历史上最重要的湖泊之一。[[战国]]时期即为楚国云梦泽的一部分，[[唐代]]以后逐渐成为[[中国]]第二大淡水湖。

洞庭湖在《[[资治通鉴概述|资治通鉴]]》中多次出现，是[[江陵]]、[[岳阳]]之间水路交通的关键水域。其"八百里洞庭"的浩渺景象也成为中国古代诗文中的经典意象。
