# 海运

海运是海上运输方式，唐代在东北亚军事后勤中发挥重要作用。

青州刺史刘仁轨坐督海运覆船，以白衣从军自效（200-112）。会讨百济，仁轨当浮海运粮，遭风失船，丁夫溺死甚众（201-075）。

安禄山以海运使刘道玄摄景城太守，贾载、穆宁共斩道玄，获甲仗五十馀船（217-050）。海运在唐代逐渐成为南方物资北运的重要补充。
