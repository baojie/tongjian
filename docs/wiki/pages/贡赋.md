# 贡赋

贡赋是地方向中央进献土特产与方物的制度。战国时[[孟子]]言"夏后氏五十而贡"（002-042），贡为古老的赋税形式。汉初许民铸钱，文帝时更除盗铸令，而地方贡赋为中央财政收入之重要来源（067-038）。

三国时期各国以贡赋维持财政（121-067）。[[北魏]]时"于时国家殷富，库藏盈溢"，贡赋为之主要来源（163-037）。唐代贡赋制度完备，各州以土产为贡，按等第定额，丰富了中央物资供应。贡赋在古代国家财政体系中占有重要地位。
