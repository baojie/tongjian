# [[王景仁]]

[[王景仁]]（？—？），本名李茂章，吴越王[[钱镠]]麾下将领，后入梁改名。他以宁国[[节度使]]身份被梁主任为北面行营都招讨使；以柏乡之战统帅梁国主力却被诱敌之计击败；是后梁在[[河北]]争夺战中遭遇决定性挫折的直接当事人。

## 以宁国节度使入仕后梁

王景仁以吴越旧将身份进入梁国体制，"吴越王镠遣宁国[[节度使]]【王景仁】奉表诣[[大梁]]，陈取[[淮南]]之策。[[岑文本|景仁]]即茂章也，避梁讳改焉"（267-003），以[[避讳]]改名的方式完成了从吴越藩镇到后梁朝廷体制的过渡。

## 以北面都招讨使统帅大军攻晋

王景仁获得了梁国最高的军事任命，"以宁国[[节度使]]、同平章事【王景仁】充北面行营都指挥招讨使，潞州副招讨使[[韩勍]]副之，以[[李思安]]为先锋将，趣上党"（267-110），以统帅主力大军攻晋的最高军事职务，参与了梁晋争霸的核心决战。

## 被周德威以诱敌之计大败于柏乡

柏乡之战是后梁的重大转折，"丁亥，[[周德威]]与别将史建瑭、[[李嗣源]]将精骑三千压梁垒门而诟之，【王景仁】、[[韩勍]]怒，悉众而出"（267-127），被[[周德威]]以激将法逼出营垒，在野战中被以逸待劳的晋军彻底击溃，梁国在[[河北]]的军事优势从此一蹶不振。

## 历史评价

[[司马光]]通过王景仁，展示了柏乡之战中战略决策失误的典型：以优势兵力居守本可稳操胜券，却因受到[[周德威]]的言语激怒而放弃地利主动出战。"怒，悉众而出"四字，概括了这场决定梁晋战略态势转折的关键失误。柏乡之败不只是一次军事挫折，更标志着后梁主动进攻[[晋国]]、争夺[[河北]]的战略意图的彻底破产。
