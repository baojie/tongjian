# 大旱

大旱是严重干[[旱灾]]害，古代常以天人感应视角将大旱与朝政得失相联系。

汉惠帝时夏大旱，江河水少，谿谷水绝（012-091）。[[汉文帝]]十四年春大旱（014-063）。[[汉文帝]]后元六年夏四月大旱蝗，令诸侯无入贡弛山泽减诸服御发仓庾以振民（015-068）。

大旱在古代被视为上天示警，帝王多因[[旱灾]]而减膳、撤乐、理冤狱、求直言。[[董仲舒]]天人感应学说强化了灾异与政治的联系，大旱之应对成为检验君德与吏治的重要标尺。

> 夏，四月，大旱，蝗。令诸侯无入贡；弛山泽，减诸服御，损郎吏员；发仓庾以振民；民得卖爵。
> （015-068）
