# 公孙贺

公孙贺，字子叔，北地义渠人。西汉轻车将军，武帝时丞相。公孙昆邪之子。

公孙贺为轻车将军，从卫青击匈奴（018-007）。公孙贺为丞相，以谨慎称（018-038）。

公孙贺少为骑士，从军有功。武帝时，为轻车将军，从卫青出塞击匈奴，以功封南奅侯。后代石庆为丞相，封葛绎侯。贺虽为相而无所建树，唯以谨慎自守。其子敬声以罪下狱，贺追捕朱安世以赎子罪。安世反告贺父子为巫蛊，贺遂被族诛。贺为相十二年，以祸终。
