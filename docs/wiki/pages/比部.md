# 比部

**比部**为[[尚书省]]所属机构，掌审计勾检、核查账目及收支。比部[[郎中]]、[[员外郎]]为实际负责人。唐代比部隶属于[[刑部]]，定期审核[[太府寺]]的出纳账目，为古代审计[[制度]]的重要组成部分。
