# [[建筑]]

《[[资治通鉴概述|资治通鉴]]》记载了历代的[[建筑]]活动，从[[秦始皇]]修筑[[长城]]与[[阿房宫]]到[[隋炀帝]]开凿[[大运河]]、建造[[洛阳]]城，从[[唐长安]]城的营建到[[五代]]各[[都城]]的兴废。

> 分天下为三十六郡，郡置守、尉、监。
> （007-034）

通鉴对[[建筑]]的记载侧重于[[政治]]层面：[[宫殿]]的修建反映[[皇帝]]的奢俭、[[城墙]]的加固反映[[边防]]形势、[[都城]]的选址反映[[战略]]格局。通鉴也记录了[[桥梁]]、[[水利]]等民生[[建筑]]的营建。
