# 张翰

张翰，字季鹰，吴国人，西晋名士。

齐王司马冏辟张翰为东曹掾（084-029）。张翰、顾荣皆虑及祸，翰因秋风起，思菰菜、莼羹、鲈鱼鲙，叹曰："人生贵适志耳，富贵何为！"即引去（084-057）。

张翰以莼鲈之思弃官归隐，为千古名士风度之典范。
