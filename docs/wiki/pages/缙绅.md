# 缙绅

缙绅代指古代官员、[[士大夫]]阶层，本义为插笏（缙）于绅（腰带）的装束，为官者的标志。

[[贾谊]]论[[风俗]]之弊，谓侈而无节则不可赡，民离本而徼末，缙绅者不惮为诈（018-055）。唐代[[甘露之变]]后，[[宦官]]气焰嚣张，[[郑覃]]、[[李石]]曰："训、注诚为乱首，但不知训、注始因何人得进？"[[宦者]]稍屈，缙绅赖之（245-071）。

[[李存勖|后唐庄宗]]时，诸伶出入[[宫掖]]，侮弄缙绅，群臣愤嫉莫敢出气（272-054）。[[杜牧]]注《[[孙吴|孙子]]》论武道废弛，"使缙绅之士不敢言兵，或耻言之"（244-085）。缙绅作为[[士大夫]]阶层的代称，盛行于后世文献。

> 奢侈而无节，则不可赡，民离本而徼末矣。末不可徒得，故缙绅者不惮为诈。
> （018-055）
