# 妾

妾，女子谦称。[[孙吴|吴起]]为卒吮疽，卒母曰"妾不知其死所矣"（001-016）。[[聂政]]姐嫈哭尸曰"以妾尚在之故，重自刑以绝从"（001-037）。秦[[商鞅变法]]明尊卑爵秩，以差次名田宅臣妾衣服（002-020）。

> 其姊嫈闻而往哭之，曰："是轵深井里聂政也。以妾尚在之故，重自刑以绝从。"
> （001-037）
