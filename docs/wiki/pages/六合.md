# 六合

六合指上、下、东、南、西、北六个方向，总括[[天地]]四方，泛指整个宇宙空间或普天之下。

## 通鉴中的六合

[[贾谊]]《[[过秦论]]》名句"秦以区区之地致万乘之权，招[[八州]]而朝同列，百有馀年，然后以六合为家"（009-004），以六合为家喻秦一统[[天下]]。[[晋]]初[[荀顗]]论[[宰相]]不得久在外曰"今[[天下]]已定，六合为家"（081-059）。[[前燕]][[慕容德]]上疏称[[先帝]]"志平六合"（101-148）。[[苻坚]]对[[王猛]]亦言"朕方混六合为一家"（103-076）。

## 六合与九州

六合为空间概念，[[九州]]为地理区划，二者常配合使用。[[王猛]]称[[苻坚]]"威烈振乎[[八荒]]，声教光乎六合，[[九州]]百郡，十居其七"（103-088），以六合[[九州]]同颂一统之功。
