# 永光

**永光**（前43—前39），[[汉元帝]]第二个[[年号]]。永光年间，[[石显]]擅权日甚，[[京房]]以灾异推言石显之奸，被石显诬为与[[张博]]通谋，弃市。[[贾捐之]]建议弃[[珠崖郡]]，元帝从之，罢珠崖，专事关中。[[冯奉世]]击[[西羌]]，破之。[[韦玄成]]为丞相，议罢郡国宗庙。
