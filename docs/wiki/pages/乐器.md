# 乐器

乐器是古代演奏[[音乐]]的器具总称，[[周礼]]分为[[八音]]：金、石、丝、竹、匏、土、革、木。[[汉武帝]]立[[乐府]]，广集各地乐器与[[乐人]]（024-020）。[[汉宣帝]]时"乐器、鼓车"充列宫庭，为[[郊祀]][[礼乐]]之备。

[[南北朝]]时期，[[西域]]乐器大量传入[[中原]]，[[龟兹]][[琵琶]]、箜篌、筚篥等与[[中原]]乐器交融（124-104）。隋代[[牛弘]]、辛彦之等更定[[雅乐]]，参酌[[古今]]乐器[[制度]]（175-025, 176-016）。[[唐太宗]]时[[祖孝孙]]斟酌南北乐器，考以古音，成大唐[[雅乐]]。乐器之演变乃中华[[音乐]]史的缩影。

## 乐器与政治

乐器在古代政[[治中]]具有重要的象征意义。[[汉文帝]]时，[[贾山]]言[[治乱]]之道，以乐器之[[得失]]系于[[政治]]之清浊。[[汉武帝]]发[[乐府]]乐器，引[[昌邑]][[乐人]]击鼓歌吹作俳倡（024-020），[[乐府]]之[[兴衰]]可见[[国政]]之[[得失]]。[[公孙述]]据蜀，[[益州]]传送述瞽师、郊庙乐器、葆[[车舆]]辇至[[洛阳]]，于是法物始备，[[东汉]][[礼乐]]由是复兴（043-026）。[[刘裕]]入[[建康]]，收[[图书]][[器物]]封闭[[府库]]，有金饰乐器，裕问[[臧熹]]"卿得无欲此乎"，熹正色以对（113-060）。乐器之于[[国家]]，非独为[[声乐]]之具，实为[[礼法]]秩序之象征。故[[王莽|新朝]]定鼎，必先定[[礼乐]]；国将衰，则礼坏乐崩。
