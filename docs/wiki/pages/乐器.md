# 乐器

乐器是古代演奏[[音乐]]的器具总称，周礼分为八音：金、石、丝、竹、匏、土、革、木。[[汉武帝]]立[[乐府]]，广集各地乐器与乐人（024-020）。[[汉宣帝]]时"乐器、鼓车"充列宫庭，为[[郊祀]][[礼乐]]之备。

南北朝时期，[[西域]]乐器大量传入[[中原]]，[[龟兹]]琵琶、箜篌、筚篥等与[[中原]]乐器交融（124-104）。隋代[[牛弘]]、辛彦之等更定[[雅乐]]，参酌古今乐器制度（175-025, 176-016）。[[唐太宗]]时祖孝孙斟酌南北乐器，考以古音，成大唐[[雅乐]]。乐器之演变乃中华[[音乐]]史的缩影。
