# 乐器

乐器是古代演奏音乐的器具总称，周礼分为八音：金、石、丝、竹、匏、土、革、木。汉武帝立乐府，广集各地乐器与乐人（024-020）。汉宣帝时"乐器、鼓车"充列宫庭，为郊祀礼乐之备。

南北朝时期，西域乐器大量传入中原，龟兹琵琶、箜篌、筚篥等与中原乐器交融（124-104）。隋代牛弘、辛彦之等更定雅乐，参酌古今乐器制度（175-025, 176-016）。唐太宗时祖孝孙斟酌南北乐器，考以古音，成大唐雅乐。乐器之演变乃中华音乐史的缩影。
