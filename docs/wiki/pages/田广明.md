# 田广明

田广明，字子公，西汉将领。以击叛军、征蛮夷有功，官至御史大夫、祁连将军。

故城父令公孙勇与客胡倩等谋反，诈称光禄大夫，淮阳太守田广明觉知，发兵捕斩之（022-065）。益州蛮夷反，汉遣大鸿胪田广明击之（023-023）。以功赐爵关内侯（023-039）。武都氐人反，田广明将三辅、太常徒击之（023-041）。

昌邑王废立之际，田广明时为御史大夫，谓太仆杜延年曰："《春秋》之义，以功覆过。当废昌邑王时，非田子宾之言，大事不定"（024-045）。后以祁连将军率四万余骑出西河击匈奴（024-047）。

> 故城父令公孙勇与客胡倩等谋反，倩诈称光禄大夫，言使督盗贼；淮阳太守田广明觉知，发兵捕斩焉。
> （022-065）

> 大发兵，遣御史大夫田广明为祁连将军，四万馀骑，出西河。
> （024-047）
