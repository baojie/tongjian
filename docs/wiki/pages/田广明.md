# 田广明

田广明，字子公，[[西汉]]将领。以击叛军、征[[蛮夷]]有功，官至[[御史大夫]]、祁连将军。

故城父令公孙勇与客胡倩等[[谋反]]，诈称光禄大夫，淮阳太守田广明觉知，发兵捕斩之（022-065）。[[益州]][[蛮夷]]反，汉遣大鸿胪田广明击之（023-023）。以功[[赐爵]][[关内侯]]（023-039）。[[武都]]氐人反，田广明将三辅、[[太常]]徒击之（023-041）。

昌邑王废立之际，田广明时为[[御史大夫]]，谓太仆[[杜延年]]曰："《[[春秋]]》之义，以功覆过。当废昌邑王时，非田子宾之言，大事不定"（024-045）。后以祁连将军率四万余骑出[[西河]]击[[匈奴]]（024-047）。

> 故城父令公孙勇与客胡倩等谋反，倩诈称光禄大夫，言使督盗贼；淮阳太守田广明觉知，发兵捕斩焉。
> （022-065）

> 大发兵，遣御史大夫田广明为祁连将军，四万馀骑，出西河。
> （024-047）
