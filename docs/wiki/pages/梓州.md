# [[梓州]]

[[梓州]]是[[唐代]]设置的州，治所在今[[四川]]三台，隶属[[剑南道]]。唐代中后期置[[东川节度使]]，治梓州，与[[西川节度使]]分治剑南，是控制[[四川盆地]]东部的军政重镇。

梓州地处[[涪江]]中游，水陆交通便利，是连接[[成都]]与[[巴蜀]]东部的重要枢纽。[[安史之乱]]后，唐廷以梓州为东川治所，与西川成都形成东西呼应之势，在维护西南稳定中发挥了重要作用。
