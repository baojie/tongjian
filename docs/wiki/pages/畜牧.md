# 畜牧

畜牧，牲畜饲养业，为古代北方游牧民族之主要产业。

李牧大纵畜牧诱匈奴（006-056）。匈奴以畜牧为生（014-039）。

战国李牧为赵将守雁门，大纵畜牧人民满野，匈奴小入佯北不胜，单于大率众来入，牧为奇阵大破之。匈奴以畜牧为生，得汉食物皆去之，以示不如湩酪之便美。畜牧之盛衰，关于游牧民族之强弱。汉与匈奴和市，以金帛易畜产，亦以此制其命。
