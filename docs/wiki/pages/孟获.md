# 孟获

**孟获**，[[三国]][[蜀汉]]南中豪帅。

[[永昌]][[太守]]吕凯闭境拒吴，孟获诱扇诸夷，诸夷皆从之，与[[硃褒]]、[[高定]]等叛应[[东吴]]（070-015）。[[诸葛亮]]南征，孟获收吕闿余众以拒亮。获素为夷汉所服，[[诸葛亮]]募生致之，既得，使观于营陈之间。亮纵之使更战，七纵七擒而亮犹遣获，获止不去，曰："公，天威也，南人不复反矣！"（070-037）

[[诸葛亮]]七擒孟获为[[平定]]南中的关键一战，自此南中[[安定]]，为[[蜀汉]][[北伐]]提供了稳定的后方。

## 南中归心

[[诸葛亮]]南征，[[孟获]]收[[吕闿]]余众以拒亮。亮募生致之，七纵七擒而亮犹遣获，获止不去，曰"公，天威也，南人不复反矣"（070-037）。初，[[永昌]][[太守]][[吕凯]]闭境拒[[东吴]]，孟获诱扇诸夷，诸夷皆从之，与[[硃褒]]、[[高定]]等叛应吴（070-015）。自获[[归降]]，南[[中平]]定，[[诸葛亮]]乃得专力[[北伐]]。孟获[[七擒七纵]][[故事]]，为[[三国]]传奇之经典。
