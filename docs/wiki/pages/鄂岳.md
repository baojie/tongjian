# 鄂岳

鄂岳为[[唐代]]方镇，治鄂州（今湖北武昌），领鄂、岳二州，为长江中游军事重镇。

[[唐德宗]]命[[萧复]]充山南东、西、荆湖、淮南、江西、鄂岳、浙江东、西、福建、岭南等道宣慰安抚使（229-051）。以鄂岳观察使[[韩皋]]为奉义节度使（237-005）。每岁赋税倚办止于浙江东、西、宣歙、淮南、江西、鄂岳、福建、湖南八道四十九州（237-072）。鄂岳观察使[[郗士美]]遣僚属以事过[[伊宥]]之境，因告以凶问，先备篮舆即日遣之（238-036）。

> 命萧复充鄂岳等道宣慰、安抚使。
