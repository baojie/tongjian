# 裴蕴

裴蕴，河东闻喜人。隋御史大夫，掌选事，为"选曹七贵"之一。

裴蕴与苏威、宇文述、虞世基、裴矩等同参掌选事，时人谓之"选曹七贵"（180-058）。裴蕴希旨，奏括天下乐家子弟皆为乐户（180-065）。裴蕴以民间版籍脱漏尚多，奏令貌阅，若一人不实则官司解职（181-035）。

裴蕴初仕陈为直阁将军，陈亡归隋，累迁太常少卿。蕴深识帝意，凡炀帝所欲，无不曲意逢迎。貌阅之法，使户口大增，国家财赋赖以充盈。然蕴性多诡诈，善于附会，为"选曹七贵"中最得炀帝宠信者。宇文化及弑炀帝，蕴亦被杀。

> 民部侍郎裴蕴以民间版籍，脱漏户口及诈注老小尚多，奏令貌阅，若一人不实，则官司解职。
> （181-035）
