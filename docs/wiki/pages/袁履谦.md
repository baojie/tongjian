# 袁履谦

袁履谦，唐[[常山]][[长史]]，[[安史之乱]]中与[[颜杲卿]]共举义旗。

[[安禄山]]至藁城，[[常山]][[魏郡太守|太守]][[颜杲卿]]力不能拒，与[[长史]]袁履谦往迎之，禄山使仍守[[常山]]（217-039）。[[颜杲卿]]以禄山命召[[李钦凑]]，使袁履谦等携[[酒食]]往劳之，醉而斩之（217-054）。后[[史思明]]攻[[常山]]，城陷，杲卿及袁履谦等被执送[[洛阳]]，禄山杀之（217-064）。后[[朝廷]][[追赠]]袁履谦等（220-038）。

## 世系

袁履谦（？—756年），[[唐]][[常山]][[长史]]。其先世不详于史。[[安史之乱]]中与[[常山太守]][[颜杲卿]]共举义旗，斩[[李钦凑]]（217-054）。城陷被执送[[洛阳]]，骂[[安禄山]]而死（217-064）。后[[朝廷]][[追赠]]（220-038）。履谦[[子孙]]不详于史。
