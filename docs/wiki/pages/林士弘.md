# 林士弘

林士弘，饶州鄱阳人。隋末南方起义军首领，自称[[桓玄|楚帝]]。

林士弘从乡人操师乞起义，师乞中流矢死，士弘代统其众，与隋将刘子翊战于[[彭蠡]]（183-025）。张善安袭破士弘，士弘徙居南康（184-064）。[[萧铣]]与林士弘各遣人招[[交趾]]太守丘和，和不从（185-034）。士弘据有[[江南]]数郡，为隋末南方重要割据势力。

> 鄱阳贼帅操师乞自称元兴王，以其乡人林士弘为大将军。
> （183-025）
