# [[李嗣源]]

[[李嗣源]]（867—933年），本名邈吉烈，字若虚，[[沙陀]]人，[[李克用]]养子，后唐明宗，[[五代]][[后唐]]第二位[[皇帝]]。他起自[[军旅]]，以勇猛谨慎著称，924年登基后推行[[节俭]]、整顿[[吏治]]，是[[五代]]十国时期少有的[[明君]]，在位十年[[政治]]相对清明。

## 军中英武

李嗣源早年在[[李克用]][[麾下]]征战，以沉稳著称。"[[李嗣源]]性谨重廉俭，诸将相会，各自诧[[勇略]]，嗣源独默然，徐曰：'诸君喜以口击贼，嗣源但以手击贼耳'"（258-062），以行动代替言语的风格令人折服。

在对抗[[后梁]]的战争中，李嗣源屡建[[军功]]，"会[[横冲]]都将[[李嗣源]]以所部兵至，谓嗣昭曰：'吾辈亦去，则势不可支矣，我试为公击之'"（261-087），关键时刻主动[[请战]]，是晋军的主要支柱之一。

[[晋阳]]被围时，李嗣源与[[李嗣昭]]"数将[[敢死]]士夜入[[氏叔琮]]营，[[斩首]][[捕虏]]，汴军惊扰"（263-012），[[守城]][[克敌]]，忠勇双全。

## 即位与仁政

[[唐庄宗|后唐庄宗]][[唐庄宗|李存勖]]末年[[政治]]腐败，926年李嗣源被拥立为帝，是为后唐明宗。他[[即位]]后减轻[[赋税]]，整顿[[吏治]]，注重[[农业]]生产，使[[五代]]乱世中出现了一段相对[[安定]]的局面。

## 历史评价

[[司马光]]对李嗣源多有正面评价，视其为[[五代]]难得的实践[[仁政]]的[[君主]]。其"以手击贼"的务实作风，是那个时代武将出身[[帝王]]的难得品格。
