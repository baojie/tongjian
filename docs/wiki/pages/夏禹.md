# 夏禹

**夏禹**，姒姓，名文命，[[鲧]]之子，[[夏]]朝建立者。禹以治水功绩受[[舜]][[禅让]]，为[[三代]]圣王之一。

[[东汉]]末，[[荀悦]][[上书]][[曹操]]，以夏禹玄圭之赐喻明君[[赐赉]]有制（047-039）。[[汉灵帝]]时，[[吕强]]谏曰宜惟夏禹卑宫、[[汉文帝|太宗]]露台之意，以尉下民之劳（057-127）。[[西晋]][[齐王冏]]大兴第舍，[[顾荣]]谏以"唐、虞茅茨，夏禹卑宫"（084-054）。

[[北魏]][[孝文帝]][[太和]]十八年至[[龙门]]，[[遣使]]祀夏禹（141-018）。[[唐代]][[狄仁杰]]以吴、楚多淫祠，奏焚其一千七百馀所，独留夏禹、吴太伯、[[季札]]、[[伍子胥|伍员]]四祠（204-028）。

禹在[[中国]]古代政治文化中为勤政爱民、卑宫菲食之典范。后世[[帝王]]或臣下每以夏禹卑宫、尧舜茅茨为君德之楷模，亦有以禹贡[[九州]]论天下形势之要。
