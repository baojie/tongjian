# 赦罪

赦罪，赦免罪犯，古代朝廷施恩之制度。

[[汉高祖]]立[[太子]]赦罪人（009-048）。[[匈奴]]募降者赦罪（021-087）。[[张奂]]请宽赦罪隶（054-018）。

赦罪为古代帝王施恩之要政，即位、立[[太子]]、[[改元]]、灾异等皆有赦。[[汉高祖]]立[[太子]]刘盈，赦罪人。武帝时[[苏武]]使[[匈奴]]，单于募降者赦罪，武不屈。汉末[[张奂]]上疏请宽赦罪隶，扫除更始，则胡马可不窥长城。赦罪虽曰施恩，然赦数过频，则奸宄无所惩，故明君慎之。

> 汉高祖立太子赦罪人。
> （009-048）
