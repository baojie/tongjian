# 李国昌

李国昌，沙陀人。唐振武节度使，沙陀酋长，晋王李克用之父。

李国昌为振武节度使，以功赐姓名（251-052）。李国昌与其子李克用据云州，唐不能制（252-028）。

李国昌本名朱邪赤心，沙陀酋长。唐懿宗时，从康承训讨庞勋，以功拜振武节度使，赐姓名李国昌。国昌在边，威名甚著。然与其子克用皆桀骜难制。僖宗时，父子据云州，朝廷讨之，国昌奔鞑靼。黄巢乱起，国昌父子复起，卒平黄巢。国昌卒，克用袭位，后为晋王。
