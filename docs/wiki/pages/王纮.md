# 王纮

王纮，[[北齐]]库直、[[都督中外诸军事|都督]]、[[开府仪同三司]]，以冒刃御贼、[[直言]]敢谏著称。

库直王纮冒刃御贼（162-073）。[[都督中外诸军事|都督]]王纮曰："有大乐，亦有大苦。长夜之饮，不寤国亡身陨，所谓大苦！"（166-080）。[[开府仪同三司]]王纮[[谏议]]陈师（171-077）。齐使[[开府仪同三司]]王纮聘于周（171-092）。军正王纮曰："齐失[[纪纲]]，于兹累世。天奖周室，一战而扼其喉。"（172-078）。

## 世系

王纮，[[北齐]]库直、[[都督]]、[[开府仪同三司]]。其先世不详于史。纮冒刃御贼（162-073），以[[直言]]敢谏称。[[北周]]伐齐，纮为军正（172-078）。纮[[子孙]]不详于史。
