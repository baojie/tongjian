# 王纮

王纮，北齐库直、都督、开府仪同三司，以冒刃御贼、直言敢谏著称。

库直王纮冒刃御贼（162-073）。都督王纮曰："有大乐，亦有大苦。长夜之饮，不寤国亡身陨，所谓大苦！"（166-080）。开府仪同三司王纮谏议陈师（171-077）。齐使开府仪同三司王纮聘于周（171-092）。军正王纮曰："齐失纪纲，于兹累世。天奖周室，一战而扼其喉。"（172-078）。
