# [[檀乡]]

[[檀乡]]是[[东汉]]初年的乡邑，位于今[[山东]]境内。[[更始]]年间，[[赤眉军]]余部[[刁子都]]为其部曲所杀，余党与诸贼会于檀乡，号"檀乡贼"，寇掠[[魏郡]]、[[清河]]（040-042）。

魏郡大吏李熊弟陆谋反城迎檀乡，被魏郡太守[[铫期]]察觉，及时处置。檀乡贼是东汉初年[[河北]]地区的一支重要流民武装，最终被[[光武帝]]所平定。
