# 博士

博士是古代学官名，掌经学传授，兼议典礼政事。秦时已有博士官，[[秦始皇]]巡游湘山，"问博士曰：'湘君何神？'"（007-043），博士掌通古今以备顾问。焚书议起，[[李斯]]奏"非博士官所职，天下有藏《诗》、《书》、百家语者，皆诣守、尉杂烧之"（007-057），可见博士在秦代享有特殊学术地位。

汉代博士制度大备，[[贾谊]]年二十馀被召为博士（013-081），[[隋文帝|文帝]]爱其辞博，一岁中超迁至太中大夫。[[董仲舒]]"少治《春秋》，孝景时为博士"（017-011），学者皆师尊之。[[汉文帝]]使博士刺《六经》中作《王制》，谋议巡狩、[[封禅]]事（015-046），博士亦参与国家典礼制定。
