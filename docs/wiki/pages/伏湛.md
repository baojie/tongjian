# 伏湛

伏湛，琅邪人，东汉初年名臣，官至大司徒。

更始时任平原太守，天下兵起，湛独抚循百姓，境内以安（040-015）。光武帝即位，以伏湛为大司徒（041-011）。光武欲亲征彭宠，伏湛谏止之（041-031）。

## 世系

伏湛（？—约37年），字惠公，琅邪人。其先为[[伏胜]]（伏生），[[西汉]]初《[[尚书]]》学者。父[[伏理]]，以《[[诗经|诗]]》授[[汉成帝]]。湛仕[[东汉]]为[[大司徒]]，封阳都侯。子[[伏隆]]。
