# 伏湛

伏湛，[[琅邪]]人，[[东汉]]初年名臣，官至大[[司徒]]。

更始时任[[平原太守]]，[[天下]]兵起，湛独抚循[[百姓]]，境内以安（040-015）。[[光武帝刘秀|光武帝]][[即位]]，以伏湛为大[[司徒]]（041-011）。光武欲亲征[[彭宠]]，伏湛[[谏止]]之（041-031）。

## 世系

伏湛（？—约37年），字惠公，[[琅邪]]人。其先为[[伏胜]]（伏生），[[西汉]]初《[[尚书]]》学者。父[[伏理]]，以《[[诗经|诗]]》授[[汉成帝]]。湛仕[[东汉]]为[[大司徒]]，封阳都侯。子[[伏隆]]。
