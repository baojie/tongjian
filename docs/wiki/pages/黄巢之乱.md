# 黄巢之乱

黄巢之乱是唐末（875—884年）由[[黄巢]]领导的大规模农民起义，波及全国大半，沉重打击了唐朝统治。

[[黄巢]]，曹州冤句人，以贩私盐为业，善[[骑射]]，喜任侠，粗涉书传。乾符元年（874年），[[王仙芝]]起义于长垣，[[黄巢]]聚众数千人响应（252-098）。乾符五年（878年）[[王仙芝]]战死后，[[黄巢]]被推举为起义军领袖，号"[[黄巢|冲天大将军]]"。

[[黄巢]]率军流动作战，转战南北。广明元年（880年）攻入长安，即皇帝位，国号大齐，年号金统。中和四年（884年），黄巢与李克用等大战于汴州，不利而退（253-005）。后在[[李克用]]、[[时溥]]等藩镇势力的围攻下被迫撤离长安，退至狼虎谷，为外甥林言所杀。黄巢之乱历时十年，使唐朝中央权威彻底衰落，加速了唐末藩镇割据和五代十国局面的形成。
