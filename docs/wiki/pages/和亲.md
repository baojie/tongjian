# 和亲

和亲是[[中原]]王朝与周边民族政权通过[[通婚]]联姻建立和平关系的外交政策，始于[[西汉]]初年。

汉[[汉高祖刘邦|高祖]]七年（[[前200年]]）白登之围后，[[刘敬]]向[[汉高祖刘邦|高祖]]建议以嫡[[公主]]嫁[[匈奴]]单于，缔结[[婚姻]]以固和平（012-009）。此后历代沿袭，成为汉匈关系的重要纽带。[[匈奴]]单于遣[[汉书]]曰："前时，皇帝言和亲事，称书意，合欢。"（014-037）

然而和亲并不能完全阻止[[匈奴]]侵扰。[[汉文帝]]时[[贾谊]]上疏陈御[[匈奴]]之策，力主断绝和亲、发兵征讨（015-016）。[[汉宣帝]]以后，随着[[西汉|汉朝]]国力增强和[[匈奴]]衰弱，和亲政策逐渐被军事进击所取代。

和亲政策也被后世王朝用于处理与其他民族政权的关系，如唐代与[[吐蕃]]的和亲（文成[[公主]]入藏），成为[[中原]]王朝民族外交的重要传统。
