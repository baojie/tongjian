# 案甲

**案甲**（亦作"[[按甲]]"）指[[按甲]]止兵、暂不交战，为古代[[军事]]中持重待机的策略。

## 案甲息师

[[诸葛恪]]欲兴师伐魏，诸将不可，恪曰："不如案甲息师，观隙耐劝。且兵者大事，众苟不悦，君独安之"（076-011）。案甲息师，[[以逸待劳]]。

魏将攻赭阳，诸将欲案甲不战以疲之，李佐独[[昼夜]]攻击，[[士卒]]死者甚众（140-026）。案甲不战与急攻各有[[得失]]。

吴频岁[[称兵]]，而犹案甲养士，未果致讨者，诚以[[百姓]]疲劳故也（073-042）。案甲养士，以待[[民力]]。
