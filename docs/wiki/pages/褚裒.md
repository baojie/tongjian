# 褚裒

褚裒，字季野，河南阳翟人。东晋卫将军，康献皇后之父。以名士风流著称。

豫章太守褚裒与殷浩等皆以识度清远，善谈《老》、《易》，擅名江东（095-054）。征豫章太守褚裒为待中、尚书。裒自以后父，不愿居中任事，苦求外出（097-016）。

褚裒出自阳翟褚氏，为东晋一流名士。其女为康帝皇后，裒以外戚居职，然谦退自抑，不愿居朝中掌权，力求外任。后以北伐后赵之举，兵败而还，惭恨卒。褚裒之风度，为江东士族所宗。
