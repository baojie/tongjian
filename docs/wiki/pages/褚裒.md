# 褚裒

褚裒，字季野，[[河南]]阳翟人。东晋卫将军，康献[[皇后]]之父。以名[[士风]]流著称。

豫章太守褚裒与[[殷浩]]等皆以识度清远，善谈《老》、《易》，擅名[[江东]]（095-054）。征豫章太守褚裒为待中、尚书。裒自以后父，不愿居中任事，苦求外出（097-016）。

褚裒出自阳翟褚氏，为东晋一流名士。其女为康帝[[皇后]]，裒以[[外戚]]居职，然谦退自抑，不愿居朝中掌权，力求外任。后以北伐后赵之举，兵败而还，惭恨卒。褚裒之风度，为[[江东]][[士族]]所宗。
