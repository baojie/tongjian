# 邢杲

邢杲，北魏流民起义首领。帅河北流民十万馀户反于青州，自称汉王，为元天穆、尔朱兆所破。

前幽州平北府主簿河间邢杲帅河北流民十万馀户反于青州之北海，自称汉王，改元天统（152-044）。魏使征虏将军韩子熙招谕邢杲，杲诈降而复反（152-057）。魏诏于晖还师讨邢杲（152-062）。魏诏上党王天穆讨邢杲（153-009）。魏元天穆将击邢杲，众皆曰："杲众强盛，宜以为先"（153-011）。魏上党王天穆及尔硃兆破邢杲于济南，杲降，送洛阳，斩之（153-012）。
