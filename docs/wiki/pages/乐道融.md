# 乐道融

乐道融，东晋参军。虽事王敦而忿其悖逆，说甘卓起兵讨敦，事败而死。

敦遣参军丹杨乐道融往邀甘卓，道融虽事敦而忿其悖逆，乃说卓曰："主上亲临万机，自用谯王……"（092-011）。都尉秦康与乐道融说卓曰："今分兵断彭泽，使敦上下不得相赴，其众自然离散，可一战擒也。"（092-034）。
