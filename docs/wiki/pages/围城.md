# 围城

围城是中国古代战争中围攻敌方城邑的战术，包括长期围困和强攻两种主要方式。围城战往往旷日持久，攻守双方在城墙上下展开殊死搏斗。[[资治通鉴概述|通鉴]]中记载了大量围城战例，从[[战国]]秦赵[[长平之战]]后的[[邯郸]]之围，到东汉[[董卓]]、[[曹操]]之间的各次攻城战役，围城战术贯穿中国古代战争史。
