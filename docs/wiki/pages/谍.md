# 谍

**谍**，古代指秘密侦察敌情之人，即[[间谍]]。

[[战国]]时[[李牧]]守边，习[[骑射]]谨烽火多[[间谍]]，约[[匈奴]]入盗即急入收保（006-055）。[[资治通鉴概述|通鉴]]中"[[间谍]]""谍人""间使"等共现于战术、外交场合，是古代军事活动中不可或缺的情报手段。

## 谍战

用谍为兵家要务。[[曹操]]围樊城，[[满宠]]建议遣谍通仁，以内外合力破敌（068-037）。吴[[诸葛恪]]远遣谍人观相径要，欲图寿春（074-060），魏[[司马懿]]遂将兵入舒迎击。[[邓艾]]伐蜀，先遣[[间谍]]探其虚实。军无耳目，虽众亦危，吴主[[孙皓]]时边备松弛，[[间谍]]不行，军无耳目（075-104），此为亡国之兆。[[北魏]]侦谍至南朝，被擒而纵还，委曲善问之（125-059），以[[间谍]]为情报之耳目。

> 今边壤之守，与贼相远，贼设罗落，又特重密，间谍不行，耳目无闻。
> （075-104）
