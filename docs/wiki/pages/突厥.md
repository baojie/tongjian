# 突厥

突厥是中国北方重要的游牧民族，姓阿史那氏，世居金山（阿尔泰山）之阳，为柔然铁工。六世纪中叶突厥强大，建立突厥汗国。隋时分裂为东、西突厥。唐初东突厥为患，[[贞观之治|贞观]]四年（630年）为唐所灭。西突厥至显庆年间亦平。突厥对隋唐政治、军事格局影响深远。

## 兴起

突厥本西方小国，至其酋长土门始强大。土门邀击铁勒，破之，尽降其众五万馀落（164-035）。土门恃其强盛，求婚于柔然，柔然头兵[[可汗]]怒，土门遂杀柔然使，求婚于西魏（159-006）。

## 与北朝关系

突厥木杆[[可汗]]在位时，西破嚈哒，东走[[契丹]]，北并契骨，威服塞外（166-050）。周武帝欲与突厥连兵伐齐，许纳其女为后（169-028）。周[[杨忠]]与突厥三[[可汗]]以十万骑会攻齐（169-035）。
