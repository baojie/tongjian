# 四海

四海是中国古代对普天之下的统称，源自"四方有海"的地理想象，后引申为整个国家乃至全天下。

## 通鉴中的四海

《[[资治通鉴概述|资治通鉴]]》开篇即言"夫以四海之广，兆民之众，受制于一人，岂非以礼为之纲纪哉"（001-004），以四海之广喻天下之大。[[司马光]]论信义曰"古之王者不欺四海，霸者不欺四邻"（002-021）。[[荀子]]论仁人之兵则曰"四海之内若一家"（006-007）。[[董公]]说[[汉高祖刘邦|汉王]]伐[[项羽]]之辞曰"四海之内莫不仰德，此三王之举也"（009-038）。

## 语义演变

先秦时期，"四海"指环绕中原的四方海域，带有地理想象色彩。秦汉以后，"四海"渐成国家的代称，"四海为家"、"四海升平"成为常见政治修辞。
