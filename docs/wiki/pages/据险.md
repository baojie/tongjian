# 据险

**据险**指占据险要地形以防守，为古代防御战的基本战术。

## 据险守要

董承等人据险拒曹洪，洪不得进，曹操乃遣董昭说杨奉（062-010）。据险则敌不能进。

刘晔劝曹操乘取汉中之势取蜀，操不从，曰蜀已据险守要，不可犯矣（067-024）。据险守要，敌不可犯。

刘放论用兵之道，谓吴、蜀据险守要，泛舟江湖，皆难卒谋（070-007）。据险之国，不可急图。

## 据险观衅

陈泰论姜维之来，曰不如据险自保，观衅待敝，然后进救（076-062）。据险自保，以逸待劳。

吴君臣相保六十载，设令列船津要，坚城据险，横行之计其殆难捷（075-104）。据险固守，老师费财。
