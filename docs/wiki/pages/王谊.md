# 王谊

王谊，北周内史、大司空，参与伐齐之谋，后为行军元帅讨平司马消难。

周主与内史王谊谋伐齐（172-026）。遣内史王谊监诸军攻平阳城（172-073）。王谊等以为去必不免（172-095）。以大司空王谊为襄州总管（173-088）。周以柱国王谊为行军元帅讨司马消难（174-041）。王谊帅四总管至郧州，司马消难以鲁山、甑山来降（174-063）。王谊遣诸将讨平叛蛮（174-065）。
