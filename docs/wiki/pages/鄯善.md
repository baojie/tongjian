# 鄯善

鄯善即古[[楼兰]]国，为[[西域]]南道名国。自玉门、[[阳关]]出[[西域]]，"从鄯善傍南山北，波[[河西]]行至[[莎车]]，为南道"（020-036），鄯善为南道门户。[[傅介子]][[出使]][[大宛]]，途经[[楼兰]]、[[龟兹]]，责其拦截汉使之罪（023-068），此后[[楼兰]]王被杀，更名鄯善。

鄯善王尉屠耆降汉，报告[[楼兰]]为[[匈奴]][[反间]]之状（023-068）。东[[汉明帝]]时鄯善王[[遣使]]奉献，[[西域]]复通（069-061）。[[前凉]][[张骏]]遣将伐[[西域]]，鄯善归于[[姑臧]][[朝贡]]（095-087）。鄯善位于丝绸之路南道要冲，在汉唐中外交流史上占有重要地位。
