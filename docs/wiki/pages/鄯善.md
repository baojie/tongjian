# 鄯善

鄯善即古楼兰国，为西域南道名国。自玉门、阳关出西域，"从鄯善傍南山北，波河西行至莎车，为南道"（020-036），鄯善为南道门户。傅介子出使大宛，途经楼兰、龟兹，责其拦截汉使之罪（023-068），此后楼兰王被杀，更名鄯善。

鄯善王尉屠耆降汉，报告楼兰为匈奴反间之状（023-068）。东汉明帝时鄯善王遣使奉献，西域复通（069-061）。前凉张骏遣将伐西域，鄯善归于姑臧朝贡（095-087）。鄯善位于丝绸之路南道要冲，在汉唐中外交流史上占有重要地位。
