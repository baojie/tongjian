# 武德

**武德**（618—626），[[唐高祖]][[李渊]][[年号]]，唐朝开国[[年号]]。武德元年，[[李渊]]在[[长安]][[称帝]]，[[改元]]武德。武德年间，唐军削平群雄：[[李世民]]平[[薛仁杲]]、[[刘武周]]、[[窦建德]]、[[王世充]]，[[李靖]]平[[萧铣]]、[[辅公祏]]。武德九年，[[玄武门之变]]，[[李世民]]杀[[太子建成]]、[[李元吉|齐王元吉]]，[[汉高祖刘邦|高祖]]传位[[太宗]]。
