# 权贵

**权贵**指权门显贵之家。《[[资治通鉴概述|通鉴]]》通篇记载权贵之[[兴衰]]，其规律为：权贵之家多不能长久，恃权而骄者尤速败。

## 西汉

[[霍光]]以[[大将军]][[辅政]]，权倾朝野，[[霍氏]]家族满门显贵。然光死后三年，[[霍氏]]以[[谋反]]被族灭（050-021）。[[王莽]]以[[外戚]][[起家]]，谦恭下士，终篡汉自立，然其新政失败，身死国灭（051-028）。
