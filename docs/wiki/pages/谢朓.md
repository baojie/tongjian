# 谢朓

谢朓（464-499年），字玄晖，陈郡阳夏人，[[南朝]]齐著名[[诗人]]。与[[范云]]、萧琛、[[任昉]]、[[王融]]、[[萧衍]]、[[沈约]]、陆倕并以[[文学]]见亲待，号曰"八友"（136-003）。

> 记室参军范云、萧琛、乐安任昉、法曹参军王融、卫军东阁祭酒萧衍、镇西功曹谢朓、步兵校尉沈约、扬州秀才吴郡陆倕，并以文学，尤见亲待，号曰八友。
> （136-003）

谢朓后为宣城太守，以山水诗闻名，与[[谢灵运]]并称"大小谢"。其诗风清新流丽，对唐代诗歌影响深远。齐东昏侯时因不愿从逆而被下狱死。
