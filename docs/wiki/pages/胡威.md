# 胡威

胡威，[[西晋]][[前将军]]、[[青州]][[刺史]]、[[尚书]]。[[胡质]]之子，以清慎著称。

[[前将军]][[青州]][[刺史]][[淮南]]胡威卒（081-036）。威为[[尚书]]，尝谏时政之宽。帝曰："[[尚书]]郎以下，吾无所假借。"威曰："臣之所陈，岂在丞、郎、[[令史]]，正谓如臣等辈，始可以肃化[[明法]]耳。"（081-036）。[[凉州]]人遣[[主簿]]胡威诣[[长安]]请留王尚（114-047）。

## 清慎谏诤

胡威父[[胡质]]以清忠著称，威承父风，以清慎闻名。胡威仕魏尝为[[徐州]][[刺史]]，入[[西晋]]为[[尚书]]。尝谏[[晋武帝]]时政过宽，[[武帝]]曰"[[尚书]]郎以下吾无所假借"，威对曰"臣之所陈岂在丞郎[[令史]]，正谓如臣等辈始可以肃化[[明法]]"，其言婉而直（081-036）。后威为[[前将军]]、[[青州]][[刺史]]，所至以清约率下。又[[凉州]]人遣[[主簿]]胡威诣[[长安]]请留[[刺史]]王尚，威流涕以陈，其见重于人如此（114-047）。胡氏父子三世清慎，为魏晋名臣之冠。
