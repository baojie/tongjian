# 胡威

胡威，西晋前将军、青州刺史、尚书。胡质之子，以清慎著称。

前将军青州刺史淮南胡威卒（081-036）。威为尚书，尝谏时政之宽。帝曰："尚书郎以下，吾无所假借。"威曰："臣之所陈，岂在丞、郎、令史，正谓如臣等辈，始可以肃化明法耳。"（081-036）。凉州人遣主簿胡威诣长安请留王尚（114-047）。
