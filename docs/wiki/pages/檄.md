# 檄

檄，古代征召、声讨的军事[[文书]]，传檄可定千里。

[[蒯通]]说武信君传檄而定千里（007-093）。[[韩信]]谓三秦可传檄而定（009-018）。[[汉高祖]]以[[羽檄]]征天下兵（012-025）。

檄文始于[[战国]]，盛于秦汉。其文体骈散兼行，辞气激昂，多以敌罪为辞，申己方正义。[[汉高祖]]以[[羽檄]][[征兵]]，[[羽檄]]者，插羽以示紧急。魏晋以降，檄文成为重要军事文体，[[骆宾王]]《讨[[武后|武曌]]檄》为其名篇。檄亦为官府征召之文，称檄召。

> 蒯通说武信君传檄而定千里。
> （007-093）
