# 厌胜

厌胜，以符咒、埋物等方式压制邪祟或诅咒仇敌的巫术。

陈[[皇后]]祠祭厌胜事发（018-027）。汉以单于宿[[上林苑]]为厌胜（035-023）。[[王莽]]铸威斗以厌胜众兵（038-019）。

厌胜之术在汉代宫闱中屡见不鲜。陈[[皇后]]以女巫楚服祠祭厌胜，事觉，牵连诛者三百馀人。[[王莽]]于新末造威斗，以五石铜为之，欲以厌胜众兵。[[汉元帝]]以单于来朝，以太岁厌胜所在，舍单于[[上林苑]]蒲陶宫。厌胜虽为巫术小道，然其祸福所系，往往震动宫闱朝野。
