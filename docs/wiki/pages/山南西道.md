# [[山南西道]]

[[山南西道]]是[[唐代]]十五道之一，与[[山南东道]]同属山南道。玄宗开元二十一年（733年）分天下为十五道（213-142），山南西道为其一。辖境约当今[[陕西]]南部、[[四川]]东部及[[重庆]]西部，治所在[[梁州]]（后升[[兴元府]]）。

山南西道是控制秦蜀之间交通的战略要地，在[[安史之乱]]及[[唐代]]后期的藩镇格局中具有重要地位。
