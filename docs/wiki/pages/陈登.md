# 陈登

陈登，字元龙，东汉广陵太守，从曹操破吕布。

陶谦卒，陈登率州人迎刘备。备未敢当，陈登曰："今欲为使君合步骑十万，上可以匡主济民，下可以割地守境。"（061-028）。后从曹操，广陵太守陈登率郡兵为操先驱，进至下邳攻吕布（062-042）。以功加伏波将军（062-046）。

陈登以雄才大略著称，刘备尝称"元龙湖海之士，豪气不除"。陈登年三十九卒，时人惜之。
