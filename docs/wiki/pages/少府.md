# 少府

少府是秦汉[[九卿]]之一，掌皇帝私库及宫廷事务。

少府负责管理山海池泽之税，以充天子私用，与国家财政（归[[大司农]]）分途。属官极为庞杂，有[[尚书]]、[[符节]]、[[太医]]、太官、汤官、导官、[[乐府]]、若卢、考工室、左弋、居室、甘泉居室、左右[[司空]]、东织、西织、东园匠等十六官令丞。

[[章邯]]曾任少府（007-095），秦末天下大乱，[[章邯]]以少府身份率骊山徒人拒[[陈胜]]军。[[汉武帝]]时，少府有六丞，属官益繁（013-063）。[[东汉]]以后，少府职权逐渐被[[尚书]]台、[[宦官]]侵夺，至隋唐其职掌并入殿中省。
