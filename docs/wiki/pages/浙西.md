# [[浙西]]

[[浙西]]是[[唐代]]对浙江西道节度使辖区的简称，治所在[[润州]]（今[[镇江]]），辖[[润州]]、[[常州]]、[[苏州]]、[[湖州]]、[[杭州]]、[[睦州]]等州，约当今[[江苏]]南部和[[浙江]]北部。

[[东晋]]时，[[王舒]]已监浙东军事，[[虞潭]]监浙西军事（094-031）。[[唐]]时浙西为重要财赋之地，[[安史之乱]]后，[[张镐]]曾檄浙东、浙西等军共救[[睢阳]]（220-020）。浙西在唐代后期为[[镇海军]]节度使辖区。
