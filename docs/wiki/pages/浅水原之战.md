# 浅水原之战

浅水原之战是唐初秦王[[唐太宗|李世民]]与[[秦国|西秦]][[薛举]]、薛仁杲父子之间的关键战役，决定了[[关中]]西部霸权的归属。

武德元年，[[薛举]]进攻泾州，[[唐太宗|李世民]]率军征讨。[[薛举]]潜师掩其后，双方战于浅水原，唐八总管皆败，士卒死者什五六，大将军慕容罗睺、李安远、刘弘基皆没（185-064）。后[[薛举]]病死，其子薛仁杲继位。

[[唐太宗|李世民]]吸取教训，[[坚壁]]不出六十余日，待仁果粮尽将士离心，命行军总管梁实营于浅水原以诱之，大破薛仁杲军（186-042）。此役奠定了唐王朝在[[关中]]的稳固统治。
