# 高隆之

高隆之，本徐氏养子。东魏尚书右仆射，[[高欢]]命以为弟。

高隆之恃欢势骄狎公卿，南阳王宝炬殴之（155-081）。[[高欢]]引兵入滏口，魏主使高隆之帅步骑十万会[[丞相]]欢于[[晋阳|太原]]，因以隆之为[[丞相]]军司（155-084）。以行台尚书[[司马子如]]为尚书左仆射，与右仆射高隆之、[[侍中]]高岳、[[孙腾]]留邺，共知朝政（156-079）。

> 侍中河南高隆之，本徐氏养子，丞相欢命以为弟，恃欢势骄狎公卿。
> （155-081）
