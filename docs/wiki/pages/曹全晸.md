# 曹全晸

曹全晸，淄州人。唐天平[[节度使]]，讨[[黄巢]]有功。

曹全晸为淄州[[刺史]]，讨诛天平军[[牙将]]崔君裕（253-069）。[[黄巢]]北趣[[襄阳]]，[[刘巨容]]与江西[[招讨使]]曹全晸合兵屯荆门以拒之。全晸以轻骑逆战，阳不胜而走，贼追之，伏发，大破[[黄巢]]（253-085）。以功为天平[[节度使]]兼东面副[[都统]]（253-117）。

> 黄巢北趣襄阳，刘巨容与江西招讨使淄州刺史曹全晸合兵，屯荆门以拒之。
> （253-085）
