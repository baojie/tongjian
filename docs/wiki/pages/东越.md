# [[东越]]

[[东越]]是[[秦]][[汉]]时期[[东南]]沿海的[[越人]]政权，建都[[东冶]]（今[[福建]][[福州]]），辖境约当今[[福建]]大部。[[秦]]末天下乱，东越助[[刘邦]]灭[[项羽]]，受封为王。

[[汉武帝]]建元年间，[[闽越]]、东越屡相攻伐，武帝遣[[王恢]]讨[[东越]]（016-012）。东越王[[郢]]发兵拒汉，后其弟[[馀善]]杀郢以降。后[[吴王]][[刘濞]]败亡，保东越（016-024）。武帝元鼎年间，东越王馀善反汉，武帝遣大军平定，徙其民于[[江淮]]间，东越遂灭。
