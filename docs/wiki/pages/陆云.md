# 陆云

陆云，字士龙，吴郡人，西晋文学家，陆机之弟。

长沙王乂以陆云为右司马（084-062）。宦官孟玖有宠于大将军司马颖，欲用其父为邯郸令，右司马陆云固执不许，曰："此县，公府掾资，岂有黄门父居之邪！"（085-024）。陆云以守正不阿著称。

陆云与兄陆机齐名，时称"二陆"。年四十二为司马颖所杀。
