# 坟墓

坟墓指墓穴与封土，为祖先遗骸安葬之所。古人视坟墓为慎终追远、[[孝道]]传承之象征。

[[郦食其]]劝[[汉高祖刘邦|刘邦]]，言[[天下]]游士离其[[亲戚]]弃坟墓去[[故旧]]从[[陛下]]游者，徒欲日夜望咫尺之地（010-011）。[[陆贾]]说[[赵佗]]，[[足下]][[中国]]人[[亲戚]]昆弟坟墓在[[真定]]，今反天性弃[[冠带]]欲与[[天子]][[抗衡]]（012-037）。[[南越]]王[[赵佗]]闻[[父母]]坟墓已坏削[[兄弟]][[宗族]]已诛论，更自号为帝（013-079）。

古代丧[[葬礼]]仪繁重，坟墓规模体现身份等级。[[汉代]]盛行厚葬，立碑树阙，至魏晋始倡[[薄葬]]。祖宗坟墓为[[家族]]凝聚力之所系，掘人坟墓被视为极端侮辱。

> 天下游士，离其亲戚，弃坟墓，去故旧，从陛下游者，徒欲日夜望咫尺之地。
> （010-011）
