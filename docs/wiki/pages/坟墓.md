# 坟墓

坟墓指墓穴与封土，为祖先遗骸安葬之所。古人视坟墓为慎终追远、孝道传承之象征。

[[郦食其]]劝[[汉高祖刘邦|刘邦]]，言天下游士离其亲戚弃坟墓去故旧从[[陛下]]游者，徒欲日夜望咫尺之地（010-011）。[[陆贾]]说[[赵佗]]，[[足下]][[中国]]人亲戚昆弟坟墓在[[真定]]，今反天性弃冠带欲与[[天子]][[抗衡]]（012-037）。[[南越]]王[[赵佗]]闻父母坟墓已坏削兄弟[[宗族]]已诛论，更自号为帝（013-079）。

古代丧[[葬礼]]仪繁重，坟墓规模体现身份等级。[[汉代]]盛行厚葬，立碑树阙，至魏晋始倡[[薄葬]]。祖宗坟墓为家族凝聚力之所系，掘人坟墓被视为极端侮辱。

> 天下游士，离其亲戚，弃坟墓，去故旧，从陛下游者，徒欲日夜望咫尺之地。
> （010-011）
