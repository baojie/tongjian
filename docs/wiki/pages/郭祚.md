# 郭祚

郭祚，字季祐，[[晋阳|太原]][[晋阳]]人。[[北魏]][[吏部]][[尚书]]、[[侍中]]，掌官吏[[铨选]]，以清谨爱惜官位著称。

通直[[散骑常侍]][[刘芳]]与[[给事黄门]][[侍郎]][[晋阳|太原]]郭祚，皆以[[文学]]为帝所亲礼（139-098）。[[黄门侍郎]]郭祚曰："[[山水]]者，仁智之所乐，宜复修之。"帝曰："魏[[汉明帝|明帝]]以奢失之于前，朕岂可袭之于后。"（140-042）[[侍中]]郭祚兼[[吏部]][[尚书]]。祚清谨，重惜官位，每有铨授，虽得其人，必徘徊久，然后下笔（142-066）。

郭祚以[[文学]]受知于孝[[隋文帝|文帝]]，与[[刘芳]]并为孝[[隋文帝|文帝]]亲近文臣。孝[[隋文帝|文帝]]南迁[[洛阳]]、推行汉化，郭祚参与其间。其掌[[铨选]]，慎重官位，为时所称。后为宣[[武帝]]所杀，朝野冤之。
