# 周盘龙

周盘龙，北[[兰陵]]人。[[南朝]]齐军主，以[[骁勇]]善战、父子[[陷阵]]闻名。

辅师[[将军]]任农夫、马军主[[东平]]周盘龙等将兵自石头济淮（133-107）。上遣领军[[将军]]李安民为[[都督]]，与军主周盘龙等救之。盘龙之子奉叔以二百人陷陈（135-098）。

周盘龙在[[南齐]]以勇将著称，父子皆[[骁锐]]无敌。周盘龙与子周奉叔并随[[萧道成]]起兵，在平[[袁粲]]、拒魏诸战中[[战功]]卓著。一次魏军[[围城]]，周奉叔率二百人[[陷阵]]不得出，周盘龙单骑突入重围，父子[[合纵连横|纵横]]奋击，魏军披靡。时人语曰："奉叔以二百人[[陷阵]]，盘龙以单骑救之，真父子[[将军]]也！"

> 盘龙之子奉叔以二百人陷陈，魏人围之。盘龙奋矟大呼，单骑突入重围，直攻魏主。
> （135-098）
