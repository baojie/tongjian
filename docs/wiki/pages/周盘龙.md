# 周盘龙

周盘龙，北兰陵人。[[南朝]]齐军主，以骁勇善战、父子陷阵闻名。

辅师将军任农夫、马军主东平周盘龙等将兵自石头济淮（133-107）。上遣领军将军李安民为[[都督]]，与军主周盘龙等救之。盘龙之子奉叔以二百人陷陈（135-098）。

周盘龙在南齐以勇将著称，父子皆骁锐无敌。周盘龙与子周奉叔并随[[萧道成]]起兵，在平[[袁粲]]、拒魏诸战中战功卓著。一次魏军[[围城]]，周奉叔率二百人陷阵不得出，周盘龙单骑突入重围，父子[[合纵连横|纵横]]奋击，魏军披靡。时人语曰："奉叔以二百人陷阵，盘龙以单骑救之，真父子将军也！"

> 盘龙之子奉叔以二百人陷陈，魏人围之。盘龙奋矟大呼，单骑突入重围，直攻魏主。
> （135-098）
