# 戴至德

**戴至德**，唐右[[仆射]]、[[户部]][[尚书]]，[[戴胄]]之兄子。

西台[[侍郎]]戴至德同东西台三品（201-089）。后为右[[仆射]]、[[户部]][[尚书]]。至德在相位，以谨慎著称。[[张文瓘]]评其"行过于名"。

戴至德为高宗朝[[宰相]]，历经[[龙朔]]至[[总章]]年间。其在相位参与[[朝政]]议论，然风格较[[温和]]，不如[[郝处俊]]之刚直。至德在相位期间的主要[[贡献]]在于维持[[朝政]]稳定。其叔[[戴胄]]为[[太宗]]朝名臣。

## 高宗朝相

[[唐高宗]]时，戴至德以西台[[侍郎]]同东西台三品（201-089）。时[[刘仁轨]]与至德分掌[[左右仆射]]，[[张文瓘]]称至德"行过于名"（202-016）。至德改右[[仆射]]，同[[中书门下]]三品（202-062）。卒，赠[[开府仪同三司]]（202-120）。至德在高宗朝为[[宰相]]，以谨慎[[温和]]、不尚虚名著称。
