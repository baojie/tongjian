# [[资治通鉴概述]]

《资治通鉴》，北宋[[司马光]]主持编纂，历时十九年（1065-1084），是中国第一部编年体通史巨著。

| 项目 | 内容 |
|------|------|
| 作者 | [[司马光]]主编，[[刘攽]]、[[刘恕]]、[[范祖禹]]协助 |
| 体裁 | 编年体通史 |
| 时间跨度 | 公元[[前403年]]（[[周威烈王]]二十三年）— [[公元959年]]（[[后周世宗]]显德六年） |

| 完成年份 | 北宋神宗元丰七年（1084年） |

## 书名含义

书名出自宋神宗赐名，意为"鉴于往事，有资于治道"，即通过历史记录为治国提供借鉴。

## 编纂特色

[[司马光]]创立"臣光曰"评论体例，在史实记载之后附上自己的史论，体现[[儒家]]治国理念和历史观。全书以礼义纲常为核心价值，强调君臣之道、德才兼备的用人原则。

## 进书表

元丰七年（1084年），[[司马光]]完成全书，向宋神宗进呈，自述编纂动机：

> 臣光言：先奉敕编集历代君臣事迹，又奉圣旨赐名《资治通鉴》，今已了毕者。伏念臣性识愚鲁，学术荒疏……每患迁、固以来文字繁多，自布衣之士，读之不遍，况于人主，日有万几，何暇周览？
> （294-109）

[[司马光]]坦承以历代[[史书]]繁浩，帝王难以尽读为由，以简明编年形式提炼治国要略，供帝王参阅。

## 开篇旨意

全书以[[三家分晋]]开篇，第一篇"臣光曰"即论礼义之本：

> 臣闻天子之职莫大于礼，礼莫大于分，分莫大于名。
> （001-004）

[[司马光]]借[[周威烈王]]承认[[三家分晋|三晋]]僭越一事，点明全书核心主旨：礼义纲常是国家治乱存亡之本，此后所有事件皆在此框架下呈现。
