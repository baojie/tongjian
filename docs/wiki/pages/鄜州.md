# 鄜州

鄜州为[[唐代]]州名，治洛交（今陕西富县），关中以北之屏障。

[[段德操]]击[[梁师都]]，鄜州刺史鄜城壮公梁礼战没（187-098）。鄜州道行军总管段德操击梁师都，至[[夏州]]，俘其民畜而还（190-094）。[[唐太宗]]以[[尉迟敬德]]为鄜州都督（195-050）。尉迟敬德表乞骸骨（196-066）。
