# 务农

务农，从事农业生产，为古代立国之本。

[[晁错]]论使民务农在于贵粟（015-024）。[[汉景帝]]欲天下务农蚕（016-112）。[[龚遂]]劝民务农桑（025-034）。

务农为古代四民之首。[[晁错]]上疏[[隋文帝|文帝]]，谓方今之务莫若使民务农，务农在于贵粟，贵粟之道在于使民以粟为赏罚。景帝诏曰："朕亲耕，后亲桑，欲天下务农蚕，素有蓄积，以备灾害。"[[龚遂]]为渤海太守，见民好末技不田作，乃躬率以俭约，劝民务农桑。务农则粟多，粟多则国强。
