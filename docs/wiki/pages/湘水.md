# 湘水

湘水，[[长江]]支流，源出[[广西]]，北流经[[湖南]]入[[洞庭湖]]。[[《资治通鉴》]]载孙刘争[[荆州]]，以湘水为界分[[长沙]]、[[江夏]]、[[桂阳]]以东属[[孙权|权]]，[[南郡]]、[[零陵]]、[[武陵]]以西属[[刘备|备]]（067-022）。

> 遂分荆州，以湘水为界；长沙、江夏、桂阳以东属权，南郡、零陵、武陵以西属备。
> （067-022）

湘水为[[湖南]]四水之首，自[[秦]]置[[长沙郡]]以来即为中原通向岭南的交通孔道。[[汉武帝]]平[[南越]]，[[伏波将军]][[路博德]]出桂阳下湟水，楼船将军[[杨仆]]出豫章下横浦，皆沿湘水南下。湘水流域所经[[长沙]]、[[衡阳]]、[[零陵]]等地，为古代荆楚腹地。
