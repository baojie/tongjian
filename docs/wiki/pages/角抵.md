# 角抵

角抵本为摔跤角力之戏，秦汉时兼指各类杂技表演。秦二世在甘泉宫"作角抵俳优之观"（008-042），以角抵为娱乐。汉武帝元封三年作角抵戏，"三百里内皆来观"（021-019），角抵规模盛大。

汉武帝元封六年再作角抵于上林苑（021-043），招待四夷使者。汉宣帝时诏乐府减乐人，罢角抵诸戏（026-006）。范子讦奏种暠，言其在度辽将军任上"徵发羌人作角抵戏"（043-117），可见角抵亦为边塞娱乐。角抵在中国表演艺术史上占有一席之地。
