# 葛婴

葛婴，秦末[[陈胜]]部将。从[[陈胜]]起义，徇蕲以东，后立襄强为[[楚王]]，闻陈王已立，杀襄强还报，[[陈胜]]诛之。

[[陈胜]]令[[符离]]人葛婴将兵徇蕲以东，攻铚、酂、苦、柘、谯，皆下之（007-084）。葛婴至东城，立襄强为[[楚王]]，闻陈王已立，因杀襄强还报，陈王诛杀葛婴（007-090）。
