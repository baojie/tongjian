# 元凤

**元凤**（前80—前75），[[汉昭帝]]第二个[[年号]]。元凤年间，[[霍光]]专断朝政，[[上官桀]]、[[桑弘羊]]与[[燕王刘旦]]谋诛霍光，事泄被杀。[[匈奴]]犯塞，汉遣[[范明友]]、[[赵充国]]等出击。朝廷推行轻徭薄赋，与民休息，国用饶足。
