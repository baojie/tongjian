# 露布

露布，不封缄之公开文书，用于报捷或直陈时政。因其公开性质，亦为直言之士所用。

后汉时，白马令甘李云露布上书，移副三府，揭露梁冀专权之状，言辞激烈（054-037）。晋时，齐王司马冏破赵王司马伦将路始等，露布至，伦乃复遣兵拒之（084-019）。前秦时，太子苻泓击斩姚厥，诸将请露布表言广其首级，泓不许，曰："主上委吾后事，不能式遏寇逆，当责躬请罪，尚敢矜诞自矜乎！"（115-010）

> 白马令甘陵李云露布上书，移副三府曰："梁冀虽持权专擅，虐流天下，今以罪行诛，犹召家臣扼杀之耳。"
> （054-037）
