# 苏绰

苏绰，字令绰，京兆武功人。西魏行台左丞，[[宇文泰]]谋主。创制立法，定文案朱出墨入之制。

泰用武功苏绰为行台郎中，居岁馀，泰未之知也，而台中皆称其能，有疑事皆就决之（157-022）。[[丞相]]泰引卢柔为从事中郎，与苏绰对掌机密（157-064）。

苏绰少好学，博览群书，尤精算术。[[宇文泰]]用为行台郎中，未几以知人善任著称。绰为泰创制：定计账、[[户籍]]之法，制文案程式朱出墨入，定[[律令]]、定官制。又著《六条诏书》：先治心、敦教化、尽地利、擢贤良、恤[[狱讼]]、均赋役，为西魏治国纲领。[[宇文泰]]令百司习诵，非通六条者不得居官。苏绰虽位不极，然实西魏制度之祖。

> 泰用武功苏绰为行台郎中，居岁馀，泰未之知也，而台中皆称其能，有疑事皆就决之。
> （157-022）
