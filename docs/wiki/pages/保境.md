# 保境

**保境**指保卫[[疆域]]领土，为古代地方政权或守将以[[固守]]本土为核心的[[防御]]战略。

## 保境安民

若魏以礼遇主上，则保境而降不晚也（078-055）。但能[[闭关]]保境足矣，平秦非吾事也（101-148）。岂特保境安民而已哉（102-030）。仲堪志在保[[全军]]保境，不欲弃州逆走（111-060）。[[先王]]临终，殷勤戒汝：深慎[[用兵]]，保境宁民，以俟天时（119-021）。
