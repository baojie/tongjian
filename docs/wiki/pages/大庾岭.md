# [[大庾岭]]

[[大庾岭]]是[[五岭]]之一，位于今[[江西]]与[[广东]]交界处。大庾岭道是[[中原]]经江西通往[[岭南]]的最重要通道，[[唐代]]张九龄曾主持开凿大庾岭新路，使其成为南北交通要道。

[[南朝]][[陈]]霸先自[[始兴]]发兵，过大庾岭（163-004），进军[[建康]]，最终建立[[陈|陈朝]]。[[唐末]]，[[黄巢]]起义时，[[李係]]曾自大庾岭趣[[广州]]（253-079）。大庾岭在军事上是从中原进入岭南的咽喉要道。
