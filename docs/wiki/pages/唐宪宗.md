# 唐宪宗

[[宪宗|唐宪宗]]李纯（778年—820年），[[唐朝]]第十位[[皇帝]]，顺宗之子。

## 即位

永[[贞元]]年（805年），顺宗内禅，宪宗[[即位]]（236-088）。

## 元和中兴

宪宗[[即位]]后励精图治，任用[[杜黄裳]]、[[裴度]]、[[武元衡]]等贤相，整顿中央集权，对跋扈藩镇采取强硬政策（237-002）。先后平定[[西川]][[刘辟]]、镇海[[李锜]]、[[淮西]][[吴元济]]、淄青[[李师道]]等（237-073），[[成德]][[王承宗]]、[[卢龙]][[刘总]]亦归顺中央，史称"[[元和中兴]]"。晚年宪宗渐骄，好[[神仙]]，求[[长生]]，终被[[宦官]]陈弘志所弑（241-037）。
