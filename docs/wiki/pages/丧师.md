# 丧师

**丧师**指军队遭受重大损失或全军覆没，为古代军事中指战败损兵、伤亡惨重的军事灾难。

## 丧师辱国

今违命出战，若丧师负国，死无益矣（078-018）。非但丧师，殆将亡国（119-021）。崇丧师徒，罪当万死（175-120）。张知运丧师，斩之以徇（211-106）。禄山失律丧师，于法不可不诛（214-034）。云南数丧师（217-016）。李全忠既丧师，恐获罪（256-039）。国家连年丧师，疆土日蹙（270-025）。
