# 陈翔

陈翔，[[东汉]][[御史]]中丞。[[党锢]]"八及"之一，被[[宦官]]陷害入狱。

[[太仆]][[颍川]][[杜密]]、[[御史]]中丞陈翔及[[陈寔]]、[[范滂]]之徒二百馀人被诬下狱（055-055）。[[张俭]]、翟超、[[岑晊]]、[[苑康]]及[[山阳]][[刘表]]、[[汝南]]陈翔、[[鲁国]]孔昱、[[山阳]]檀敷为八及，及者，言其能导人追宗者也（056-050）。

> 张俭、翟超、岑晊、苑康及山阳刘表、汝南陈翔、鲁国孔昱、山阳檀敷为八及，及者，言其能导人追宗者也。
