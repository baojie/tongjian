# 卫尉

卫尉是秦汉[[九卿]]之一，掌[[宫门]]警卫，统领[[宫城]][[卫士]]。

汉[[汉高祖刘邦|高祖]]时，帝欲召见[[齐王]][[田横]]，横[[称臣]]恐不敢[[奉诏]]。[[汉高祖刘邦|高祖]]乃诏卫尉[[郦商]]曰："[[齐王]][[田横]]即至，人马从者敢动摇者，致族夷"（011-018）。[[相国]][[萧何]]因请[[上林苑]]空地为[[民田]]被[[汉高祖刘邦|高祖]]下[[廷尉]][[械系]]，王卫尉侍前，问[[萧何]]何罪，言论颇为[[萧何]]辩解（012-049）。

卫尉秩中二千石，属官有公车[[司马]]、[[卫士]]令等，统领[[未央]]、[[长乐]]等[[宫卫]]士，是皇宫安全的关键防线。[[汉代]]卫尉与[[执金吾]]（掌[[京师]]治安）并称，分管宫[[内外]]安全。

## 卫尉与宫廷安全

卫尉掌[[宫门]]警卫，在权力[[更迭]]之际有举足[[轻重]]之势。[[诸吕之乱]]，[[太尉]][[周勃]]使[[曹参|平阳侯]]告卫尉"毋入[[相国]]产殿门"（013-059），[[吕产]]不得入殿为乱，卫尉听命于[[功臣]]集团，遂定刘氏[[社稷]]。[[隋文帝|文帝]]时[[南阳]][[直不疑]]为卫尉（016-106），[[武帝]]时[[李广]]为卫尉屯[[云中]]御[[匈奴]]（017-072），[[韩安国]]亦以卫尉屯[[渔阳]]备胡（018-041），卫尉兼领兵出征之任，非仅守门之官。卫尉由[[宫禁]][[守卫]]而兼[[军旅]]之职，反映了[[汉代]]中央[[官制]]中宫廷[[宿卫]]与[[国家]][[军事]]之间的职能交叉。
