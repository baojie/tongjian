# 卫尉

卫尉是秦汉九卿之一，掌宫门警卫，统领宫城[[卫士]]。

汉[[汉高祖刘邦|高祖]]时，帝欲召见齐王[[田横]]，横称臣恐不敢奉诏。[[汉高祖刘邦|高祖]]乃诏卫尉[[郦商]]曰："齐王[[田横]]即至，人马从者敢动摇者，致族夷"（011-018）。相国[[萧何]]因请上林苑空地为民田被[[汉高祖刘邦|高祖]]下[[廷尉]]械系，王卫尉侍前，问[[萧何]]何罪，言论颇为[[萧何]]辩解（012-049）。

卫尉秩中二千石，属官有公车司马、[[卫士]]令等，统领未央、长乐等[[宫卫]]士，是皇宫安全的关键防线。汉代卫尉与[[执金吾]]（掌京师治安）并称，分管宫内外安全。
