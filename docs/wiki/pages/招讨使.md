# 招讨使

招讨使，唐代临时设立的征讨军事长官，以平乱征伐为任，事毕即罢。

开元中，以杨思勖为黔中道招讨使，讨奚州蛮覃行璋，生擒斩首三万级（212-135）。安史之乱中，颜真卿以堂邑之功让于贺兰进明，进明得加河北招讨使（217-079）。防河招讨使李铣承制以董秦为平原太守（219-042）。安庆绪部将能元皓降，以为河北招讨使（220-057）。后以崔光远为荆襄招讨使，充山南东道处置兵马都使（221-040）。

> 以监门卫大将军杨思勖为黔中道招讨使，将兵击之。
> （212-135）
