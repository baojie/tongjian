# 姚弋仲

姚弋仲，南安赤亭人。[[后赵]]羌酋，[[姚苌]]之父。率部东徙，为[[后赵]]重将，后秦政权的奠基人。

南安赤亭羌姚弋仲东徙榆眉，戎、夏襁负随之者数万（088-048）。以赤亭羌酋姚弋仲为平西将军，封平襄公（092-068）。

姚弋仲为南安羌人豪帅，[[永嘉之乱]]后率部东迁，部众数万。先后附于[[前赵]]、[[后赵]]。[[石虎]]时弋仲为[[后赵]]重将，屡立战功。弋仲性鲠直，敢直言。[[后赵]]末，[[冉闵]]乱华，弋仲命子[[姚襄]]率众击之。弋仲临终谓襄曰："汝才十倍于闵，若不早除，必为后患。"弋仲虽未及建国，然为后秦奠基人，其子[[姚苌]]始建后秦。
