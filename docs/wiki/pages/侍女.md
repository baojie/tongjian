# 侍女

侍女，侍奉贵族起居的年轻女仆。

唐昭宗醉杀侍女数人（262-027）。吴越钱镠置侍女振纸警枕（270-084）。

侍女为古代贵族家庭之常设仆役。唐昭宗酒后醉杀黄门、侍女数人，宦官刘季述以此为由，率兵废帝。吴越王钱镠勤于政事，置粉盘于卧内，有所记则书盘中。或寝方酣，外有白事者，令侍女振纸即寤。其后钱俶纳土归宋，以吴越所积献于宋廷，侍女亦随之入汴。
