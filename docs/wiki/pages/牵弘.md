# 牵弘

**牵弘**，[[西晋]]将领，官至[[扬州]][[刺史]]、[[凉州]][[刺史]]。

牵弘为[[陇西]][[太守]]时，从[[邓艾]]伐蜀，[[邀击]][[姜维]]于强川口（078-019）。[[钟会]]取[[成都]]后，牵弘等领蜀中诸郡（078-028）。晋初为[[扬州]][[刺史]]，吴[[丁奉]]入涡口，牵弘击走之（079-084）。

后迁[[凉州]][[刺史]]，讨秃发[[树机能]]。众胡皆内叛，与[[树机能]]共围牵弘于[[青山]]，弘军败而死（079-101）。[[陈骞]]尝言[[胡烈]]、牵弘皆勇而无谋，强于自用，非绥边之材，后果如其言（079-102）。

## 边塞殒身

[[牵弘]]为[[西晋]]将领，初从[[邓艾]]伐[[蜀]]有功，[[邀击]][[姜维]]于强川口（078-019）。[[晋]]初为[[扬州]][[刺史]]，击走[[吴]]将[[丁奉]]（079-084）。后迁[[凉州]][[刺史]]，讨[[秃发树机能]]，众胡皆内叛，与[[树机能]]共围弘于[[青山]]，弘军败而死（079-101）。[[陈骞]]尝言[[胡烈]]、牵弘皆勇而无谋，强于自用，非绥边之材，果如其言（079-102）。牵弘以[[边塞]]殒身，与[[胡烈]]同为晋初西陲[[败亡]]之将。
