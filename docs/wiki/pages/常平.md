# 常平

常平，古代平粜平籴之粮食仓储制度。耿寿昌建言令边郡筑[[常平仓]]，谷贱增贾而籴，谷贵减贾而粜，民便之（027-029）。元帝时罢[[盐铁]]官、[[常平仓]]（028-045）。[[北魏]]亦置[[常平仓]]以救匮乏（136-105、140-084）。[[侯景]]乱梁，食石头常平诸仓既尽，军中乏食（161-051）。

> 令边郡皆筑仓，以谷贱增其贾而籴，以利农，谷贵时减贾而粜，名曰常平仓。
> （027-029）
