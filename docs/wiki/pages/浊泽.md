# [[浊泽]]

[[浊泽]]是[[战国]]时期的沼泽湖泊，位于今[[山西]]运城一带。[[周烈王]]元年（前375年），[[韩]]、[[赵]]、[[魏]]三家分晋后，韩灭[[郑国]]，魏、赵之间在浊泽发生冲突，[[魏武侯]]与[[赵敬侯]]、[[韩哀侯]]战于浊泽（001-070）。

浊泽之战是战国初期诸侯兼并过程中的一场重要战役，反映了魏国在战国初期称霸中原的局面。
