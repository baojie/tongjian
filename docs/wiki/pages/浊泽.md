# [[浊泽]]

[[浊泽]]是[[战国]]时期的沼泽湖泊，位于今[[山西]]运城一带。[[周烈王]]元年（[[前375年]]），[[韩]]、[[赵]]、[[魏]][[三家分晋]]后，韩灭[[郑国]]，魏、赵之间在浊泽发生冲突，[[魏武侯]]与[[赵敬侯]]、[[韩哀侯]]战于浊泽（001-070）。

浊泽之战是[[战国]]初期[[诸侯]][[兼并]]过程中的一场重要[[战役]]，反映了[[魏国]]在[[战国]]初期称霸[[中原]]的局面。
