# 徭役

徭役是古代百姓为国家无偿提供的体力劳动，涵盖筑城、修路、漕运、戍边等多种形式，是历代赋役制度的核心组成部分。汉代官吏"擅兴徭役，饰厨传，称过使客，越职逾法以取名誉"（025-050），可见徭役征发之滥。[[汉文帝]]时"人无徭役，岁比登稔，百姓殷富"（045-048）被视为太平盛世的标志。

边疆地区徭役尤重，[[赵充国]]在湟中[[屯田]]时指出若大兵不出，"徭役复更发也"（026-021）。东[[光武帝刘秀|汉光武帝]]以舂陵乡为章陵县，"世世复徭役，比丰、沛"（042-003），以优待故里。北燕[[冯跋]]"省徭役，薄赋敛"，勤于政事，燕人悦之（116-019）。
