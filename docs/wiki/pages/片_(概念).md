# 片

**片**，指扁薄之物，亦为量词。[[李陵]]令军士人持二升粝、一片冰（021-100）。[[张奂]]耻以片言自及微效（054-092）。[[忠义]]之士，一片丹心。片亦指片段、部分，如片纸只字。
