# 贞元

**贞元**（785—805），[[唐德宗]][[李适]][[年号]]。贞元年间，德宗采纳[[陆贽]]之言，姑息藩镇，[[河北]]诸镇割据之势已成。[[李希烈]]称帝，[[朱泚]]据长安。[[吐蕃]]屡寇西陲。德宗晚年信用[[裴延龄]]等聚敛之臣，陆贽被贬。贞元二十一年，德宗崩，[[唐顺宗]]立。
