# 李可举

**李可举**，晚唐卢龙节度使。乾符年间，以卢龙留后李可举为节度使（252-115）。诏昭义、幽州诸镇与吐谷浑、沙陀酋长合兵讨李国昌父子于蔚州（253-059）。李可举与吐谷浑都督赫连铎共讨沙陀，李克用拒可举于雄武军（253-112）。

> 卢龙节度使李可举、吐谷浑都督赫连鐸共讨沙陀。
> （253-112）

## 沙陀之役

李可举为[[晚唐]][[卢龙节度使]]，与[[赫连铎]]、[[李琢]]联兵讨[[李国昌]]、[[李克用]]父子。李可举遣[[韩玄绍]]邀击李克用于[[药儿岭]]，大破之，杀七千馀人（253-119）。然[[李克用]]后势复振，赫连铎与李可举战不利（254-105）。李可举为[[幽州]]藩镇之代表人物，与沙陀[[李克用]]之争夺为[[晚唐]]河北政局之关键。
