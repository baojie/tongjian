# 李可举

**李可举**，[[晚唐]][[卢龙节度使]]。[[乾符]]年间，以[[卢龙]][[留后]]李可举为[[魏博节度使|节度使]]（252-115）。诏[[昭义]]、[[幽州]]诸镇与[[吐谷浑]]、[[沙陀]]酋长合兵讨[[李国昌]]父子于蔚州（253-059）。李可举与[[吐谷浑]][[都督中外诸军事|都督]]赫连铎共讨[[沙陀]]，[[李克用]]拒可举于雄武军（253-112）。

> 卢龙节度使李可举、吐谷浑都督赫连鐸共讨沙陀。
> （253-112）

## 沙陀之役

李可举为[[晚唐]][[卢龙节度使]]，与[[赫连铎]]、[[李琢]]联兵讨[[李国昌]]、[[李克用]]父子。李可举遣[[韩玄绍]][[邀击]][[李克用]]于[[药儿岭]]，大破之，杀七千馀人（253-119）。然[[李克用]]后势复振，赫连铎与李可举战不利（254-105）。李可举为[[幽州]][[藩镇]]之代表[[人物]]，与[[沙陀]][[李克用]]之争夺为[[晚唐]][[河北]]政局之关键。
