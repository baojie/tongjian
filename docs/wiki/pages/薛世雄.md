# 薛世雄

薛世雄，隋右翊卫将军。[[《资治通鉴》]]载其从征[[辽东]]，孤军度碛击伊吾。

帝以右翊卫将军河东薛世雄为玉门道[[行军]]大将，与突阙启民[[可汗]]连兵击伊吾，师出玉门，启民不至，世雄孤军度碛（181-018）。右翊卫将军薛世雄出沃沮道，从征[[辽东]]（181-077）。萨水之败，高丽追围薛世雄于白石山，世雄奋击，破之，由是独得免官（181-084）。

> 帝以右翊卫将军河东薛世雄为玉门道行军大将，与突阙启民可汗连兵击伊吾，师出玉门，启民不至。世雄孤军度碛，伊吾请降。
> （181-018）
