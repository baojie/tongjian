# 韦坚

韦坚，京兆万年人。唐陕郡太守，领江淮租庸转运使。

韦坚为长安令，以干练升陕郡太守，领江、淮租庸转运使（215-010）。[[宇文融]]既败，言利者稍息。及[[杨慎矜]]得幸，于是韦坚、[[王鉷]]之徒竞以利进（215-011）。韦坚引浐水抵苑东望春楼下为潭，以聚江、淮[[运船]]，役夫匠通[[漕渠]]，发人丘垄，自江、淮至京城（215-030）。后以党累贬死。

> 以长安令韦坚为陕郡太守，领江、淮租庸转运使。
> （215-010）
