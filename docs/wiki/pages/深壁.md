# 深壁

**深壁**指加高加固[[壁垒]]，与"[[高垒]]""[[深沟]]"连用，为[[防御]]战术[[中坚]]守不出的重要手段。

## 深壁勿战

[[汉高祖刘邦|汉王]]曰："汉与楚相距[[荥阳]]数岁，汉常困。愿[[君王]]出[[武关]]，项王必[[引兵]]南走。王深壁勿战。"（010-020）。深壁以疲敌。

不如深壁，令[[齐王]]使其信臣招所亡城（010-031）。深壁以守，招亡以攻。

皆[[高垒]]深壁，勿与战，益为[[疑兵]]（059-076）。深壁以观变。

## 深壁以待

[[司马师]]命诸军皆深壁[[高垒]]，以待东军之集（076-050）。深壁以待援。

今当深壁[[高垒]]，以听后诏（101-131）。深壁以候朝命。
