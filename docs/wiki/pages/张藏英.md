# [[张藏英]]

[[张藏英]]（？—？），[[范阳]]人，[[五代]][[后周]]边将，原为[[契丹]]知户台[[军事]]。他以熟悉[[契丹]][[边境]]地形的特殊优势[[归降]][[后周]]；向[[周世宗]]详述北方地形[[要害]]，提出募边人[[骁勇]]、厚给[[禀赐]]以守边的策略；是[[后周]]北方边防体系建设中重要的情报与战略来源。

## 以契丹知户台军事身份归降后周

张藏英以特殊背景[[归附]]，"[[沧州]]奏[[契丹]]知户台[[军事]][[范阳]]【张藏英】[[来降]]"（291-055），以熟悉[[契丹]]内部军政的特殊身份[[归降]][[后周]]，其所掌握的[[边境]]情报立即成为[[后周]]北方[[防御]]体系的宝贵来源。

## 向世宗详述地形要害，提出募边人策略

张藏英以专业的[[边境]]知识为[[后周]]提供战略建议，"帝召德州[[刺史]]【张藏英】，问以[[备边]]之策，藏英具陈地形[[要害]]，请列置[[戍兵]]，募边人[[骁勇]]者，厚其[[禀给]]，自请将之"（292-033），以详细阐述地形、提出募当地[[骁勇]]为兵并亲自统率的完整方案，为[[周世宗|后周世宗]]的北方边防战略提供了重要参考。

## 历史评价

[[司马光]]通过张藏英，展示了[[五代]][[边境]]战略中情报与经验的价值：一个熟悉敌方内部地形、[[习俗]]和[[军事]]部署的[[归降]]官员，往往比纸面上的战略规划者更有实用价值。

张藏英的"具陈地形[[要害]]"，是[[五代]][[边境]][[战争]]中以[[人才]]情报优势弥补[[军事]]实力差距的典型案例。他亲自请将的精神，也体现了[[五代]]边将以身作则的武将文化。
