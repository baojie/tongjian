# 叔孙建

叔孙建，代人。[[北魏]]相州[[刺史]]，为胡、魏所畏服。

叔孙建为[[拓跋珪]]外朝大人，掌[[宿卫]]及参军国谋议（106-065）。明元帝时，[[刘裕]]伐姚泓，假道于魏。魏将尉建弃城而走，明元帝遣叔孙建与公孙表自河内向枋头，斩尉建于城下（117-072）。叔孙建前在[[晋阳|并州]]，为胡、魏所畏服，诸将莫及（117-067）。

## 经略青兗

叔孙建为北魏名将，明元帝命与长孙嵩各简精兵，伺刘裕西过，自济渡河，南侵彭、沛（118-026）。后讨西山丁零翟蜀洛支等，平之（118-046）。太武帝初，叔孙建将兵自平原济河，徇青、兗（119-090），入临淄，所向城邑皆溃（119-103）。又率三万骑逼东阳城，城中文武才一千五百人，垣苗悉力固守，时出奇兵击魏，破之（119-116）。
