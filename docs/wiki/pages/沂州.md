# [[沂州]]

[[沂州]]是[[南北朝]]至[[唐代]]设置的州，治所在今[[山东]][[临沂]]。地处[[沂水]]流域，[[淮河]]以北，是[[徐州]][[东南]]方向的重要[[屏障]]和连接[[山东]]与[[江淮]]的[[交通]][[要冲]]。

[[南北朝]]时期，沂州地处[[南朝]]与[[北朝]]交界地带，[[军事]]地位尤为重要。[[唐代]]沂州属[[河南道]]，是[[山东]]南部的[[政治]]、[[经济]]中心之一。
