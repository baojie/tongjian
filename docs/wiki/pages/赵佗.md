# 赵佗

赵佗，秦末汉初[[南越]]国创建者，原为秦南海尉。汉[[汉高祖刘邦|高祖]]立为[[南越]]王，[[吕后]]时称[[南越]]武帝，[[汉文帝]]时复去帝号称臣。

[[汉文帝]]遣[[陆贾]][[出使]][[南越]]，立赵佗为[[南越]]王，与剖符[[通使]]，使和集[[百越]]（012-037）。[[吕后]]时赵佗以兵威财物赂遗闽越、西瓯、骆，役属焉，东西万馀里，乘黄屋左纛，[[称制]]与中国侔（013-078）。[[汉宣帝]]时[[萧望之]]谓方今四夷宾服，南无赵佗、吕嘉之难（030-030）。

赵佗治[[南越]]，和辑[[百越]]，推广[[中原]]文化，使[[南越]]国成为汉初南方重要力量。其入朝称臣，为[[西汉|汉朝]]统一南方奠定基础。

> 诏立秦南海尉赵佗为南粤王，使陆贾即授玺绶，与剖符通使，使和集百越，无为南边患害。
> （012-037）
