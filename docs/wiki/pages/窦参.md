# 窦参

窦参，字时中，[[唐德宗]]朝同平章事，阴狡而愎，恃权而贪。

[[李泌]]荐窦参通敏，可兼[[度支]][[盐铁]]（233-049）。窦参恶[[陆贽]]，解其翰林学士之职（233-076）。以私憾毁福建观察使吴凑，德宗始恶参之诬（233-078）。

窦参为[[度支]]转运使时许班宏一年归使职，岁余无归意，二人遂有隙（234-007）。参阴狡而愎，恃权而贪，每迁除多与族子窦申议之，申招权受赂，时人谓之"喜鹊"（234-008）。参请从汴人之请以李士宁为[[节度使]]（234-009），德宗渐不能堪，终贬参。

> 窦参阴狡而愎，恃权而贪，每迁除，多与族子给事中申议之。
> （234-008）
