# 王彪之

王彪之，字叔虎，琅邪临沂人。东晋吏部尚书，王导从子。以识度清正、明于典故著称。

吏部尚书王彪之言于会稽王昱曰："此属皆自为计，非能保社稷，为殿下计也。"（099-030）会稽王昱问于王彪之曰："郊祀应有赦否？"彪之曰："自中兴以来，郊祀往往有赦，愚意常谓非宜。"（099-032）

王彪之出自琅邪王氏，历仕元帝至孝武帝六朝，以精通朝典、持正不阿著称。桓温废海西公时，彪之正色立朝，不阿附权势。谢安辅政时，亦多咨访之。彪之之持重，为东晋名臣之冠。
