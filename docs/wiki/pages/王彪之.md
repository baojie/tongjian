# 王彪之

王彪之，字叔虎，[[琅邪]]临沂人。[[东晋]]吏部[[尚书]]，[[王导]]从子。以识度清正、明于典故著称。

吏部[[尚书]]王彪之言于[[会稽]]王昱曰："此属皆自为计，非能保[[社稷]]，为殿下计也。"（099-030）[[会稽]]王昱问于王彪之曰："[[郊祀]]应有赦否？"彪之曰："自中兴以来，[[郊祀]]往往有赦，愚意常谓非宜。"（099-032）

王彪之出自[[司马睿|琅邪王]]氏，历仕元帝至孝武帝六朝，以精通朝典、持正不阿著称。[[桓温]]废海西公时，彪之正色立朝，不阿附权势。[[谢安]]辅政时，亦多咨访之。彪之之持重，为[[东晋]]名臣之冠。
