# [[沂水]]

[[沂水]]发源于[[山东]]沂源，流经[[临沂]]，南入[[江苏]]注入[[黄海]]，是[[山东]]南部最大的河流之一。[[沂州]]即因沂水得名。

[[南北朝]]时期，沂水是[[北魏]]与[[南朝]]交界的天然屏障。[[薛安都]]等曾建议以[[彭城]]、[[下邳]]、沂水一线防御北魏（132-018）。[[隋]]末，东海贼帅[[彭孝才]]转掠沂水（182-072），彭城留守[[董纯]]讨擒之。
