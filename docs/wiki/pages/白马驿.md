# [[白马驿]]

[[白马驿]]是[[唐代]][[滑州]]白马县（今[[河南]]滑县）的驿站。[[唐昭宣帝]]天祐二年（905年），[[朱全忠]]（[[朱温]]）聚[[裴枢]]等及朝士贬官者三十余人于白马驿，一夕尽杀之，投尸于[[黄河]]（265-048）。

[[李振]]因屡举进士不中，深疾搢绅之士，劝朱全忠曰："此辈常自谓清流，宜投之[[黄河]]，使为浊流！"全忠笑而从之。白马驿之祸标志着[[唐代]]士族政治势力的终结，史称"白马驿之祸"或"白马之祸"。
