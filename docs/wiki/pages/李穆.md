# 李穆

李穆，北周至隋名将，于邙山之战救[[宇文泰]]性命，功勋卓著。

赵将李穆纳拓跋翳槐于大宁，其故部落多归之（095-113）。邙山之战，[[宇文泰]]与[[高欢]]合战，泰马中流矢惊逸，泰坠地，[[东魏]]兵追及之。[[都督]]李穆下马，以策抶泰背骂曰："笼东军士！尔曹主何在，而独留此？"追者不疑其贵人，舍之而去（158-016）。[[宇文泰]]欲追[[高欢]]，李穆曰"[[高欢]]破胆矣，速追之可获"，泰不听（157-102）。

官至[[太保]]、太师，周以柱国安武公李穆为大[[司空]]（169-094），与齐公宪将兵趣宜阳（170-105），为[[太保]]（171-021）。入隋，李穆一门执象笏者百余人，贵盛无比（175-015）。
