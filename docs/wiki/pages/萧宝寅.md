# 萧宝寅

萧宝寅，南齐[[汉明帝|明帝]]之子，[[萧衍]]篡齐后奔[[北魏]]，请兵伐梁，为[[北魏]]名将。

萧宝寅伏于魏阙之下请兵伐梁，虽暴风大雨终不暂移（145-067）。魏主以萧宝寅、陈伯之皆受任城王澄节度（145-073）。萧宝寅[[歃血]]为盟，誓死以报梁之仇（145-038）。

萧宝寅是南北朝之际由南入北的宗室将领典型。其请兵伐梁的执着，体现了南齐宗室对[[萧衍|梁武帝]][[萧衍]]篡位的不甘。萧宝寅在[[北魏]]受到重用，屡次率军南征，成为连接南北政权的人物。

> 萧宝寅伏于魏阙之下，请兵伐梁，虽暴风大雨，终不暂移。
> （145-067）
