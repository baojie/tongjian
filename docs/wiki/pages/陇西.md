# 陇西

陇西为[[战国]]秦置郡，因在陇山以西得名。[[秦昭王|秦昭襄王]]灭义渠戎，置陇西、[[北地]]、[[上郡]]（004-040）。[[李信]]为陇西人，秦将（007-026）。[[李广]]亦陇西成纪人（017-022），世为秦将。

[[汉武帝]]置[[河西]]四郡后，陇西渐失边郡地位，但仍为西陲重镇。[[隗嚣]]据陇西以抗[[东汉]]（041-061）。[[诸葛亮]]北伐以陇西为首要目标（071-020）。陇西为[[关中]]的西部屏障，历代为兵家必争。

## 陇西战事

秦[[白起]]攻赵，又使[[司马错]]发陇西兵因蜀攻楚黔中（004-067）。秦灭义渠后于陇西、[[北地]]、[[上郡]]筑长城以拒胡（006-057）。始皇巡陇西、[[北地]]，至鸡头山（007-038）。[[汉高祖刘邦|刘邦]]定雍地，以为中地、[[北地]]、陇西郡（009-049）。陇西北接胡地，南控[[巴蜀]]，自古为秦之右臂，据陇西则可制[[关中]]，失陇西则[[关中]]危。
