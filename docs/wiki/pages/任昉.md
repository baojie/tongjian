# 任昉

任昉，字彦升，[[乐安]]人。[[南齐]]梁[[文学]]家，[[竟陵]]八友之一。

[[竟陵]]王萧子良开西邸招[[文学]]之士，任昉与[[萧衍]]、[[沈约]]、[[谢朓]]、[[王融]]、[[萧琛]]、[[范云]]、陆倕并游，号曰"八友"（136-003）。任昉为[[御史]]中丞，[[奏劾]]不避权贵（145-009）。

任昉少聪敏，七岁能属文。永明中，为[[竟陵]]王[[记室]][[参军]]，与[[沈约]]齐名，时人号曰"任笔沈诗"。昉好交游，奖掖后进，士林多出其门。[[萧衍|梁武帝]]立，拜[[御史]]中丞。昉于学无所不窥，尤长于[[文章]]，为一代[[文宗]]。卒谥曰敬。
