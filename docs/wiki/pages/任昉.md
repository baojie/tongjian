# 任昉

任昉，字彦升，乐安人。南齐梁文学家，竟陵八友之一。

竟陵王萧子良开西邸招文学之士，任昉与萧衍、沈约、谢朓、王融、萧琛、范云、陆倕并游，号曰"八友"（136-003）。任昉为御史中丞，奏劾不避权贵（145-009）。

任昉少聪敏，七岁能属文。永明中，为竟陵王记室参军，与沈约齐名，时人号曰"任笔沈诗"。昉好交游，奖掖后进，士林多出其门。梁武帝立，拜御史中丞。昉于学无所不窥，尤长于文章，为一代文宗。卒谥曰敬。
