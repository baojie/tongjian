# 昌义之

昌义之，字叔远，历阳乌江人。[[南朝]]梁[[徐州]][[刺史]]，以守钟离拒魏功封侯。

[[徐州]][[刺史]]历阳昌义之与魏平南将军陈伯之战于梁城，义之败绩（146-044）。北[[徐州]][[刺史]]昌义之拔梁城（146-055）。

昌义之为[[萧衍]]帐下将领，从[[萧衍]]起兵。[[萧衍|梁武帝]]时，昌义之历任北[[徐州]]、南[[徐州]][[刺史]]，镇守边境。南梁[[北伐]]时，义之与魏军交战有胜有负。[[钟离之战]]中，昌义之率三千人固守钟离，魏军数十万围攻，百计不能克，义之坚守待援，为[[韦睿]]等援军至大破魏军创造了条件。义之以勇毅善守著称。

> 乙丑，徐州刺史历阳昌义之与魏平南将军陈伯之战于梁城，义之败绩。
> （146-044）
