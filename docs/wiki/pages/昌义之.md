# 昌义之

昌义之，字叔远，[[历阳]]乌江人。[[南朝]]梁[[徐州]][[刺史]]，以守[[钟离]]拒魏功[[封侯]]。

[[徐州]][[刺史]][[历阳]]昌义之与魏平南将军陈伯之战于梁城，义之败绩（146-044）。北[[徐州]][[刺史]]昌义之拔梁城（146-055）。

昌义之为[[萧衍]]帐下将领，从[[萧衍]]起兵。[[萧衍|梁武帝]]时，昌义之历任北[[徐州]]、南[[徐州]][[刺史]]，镇守边境。南梁[[北伐]]时，义之与魏军交战有胜有负。[[钟离之战]]中，昌义之率三千人固守[[钟离]]，魏军数十万围攻，百计不能克，义之坚守待援，为[[韦睿]]等[[援军]]至大破魏军创造了条件。义之以勇毅善守著称。

> 乙丑，徐州刺史历阳昌义之与魏平南将军陈伯之战于梁城，义之败绩。
> （146-044）

## 守钟离

[[钟离之战]]为[[南朝]]梁与[[北魏]]间决定性大战。魏[[中山]]王[[元英]]率百万之众攻[[钟离]]，城中守兵才三千人。昌义之督帅将士，随方抗御。魏人以车载土填堑，使众负土随之，严骑蹙其后。又昼夜苦攻，分番相代，坠而复升，莫有退者（146-090）。义之率[[士卒]]以死拒守，城垣屡坏辄补，魏师百计不能克。及[[韦睿]]、[[曹景宗]]等[[援军]]至，大破魏军，魏人争投水死者十馀万。[[韦睿]]遣报义之，义之悲喜不能答语，但叫曰"更生！更生！"（146-095）。昌义之以三千弱卒抗百万之师，为[[南朝]][[守城]]战之经典。
