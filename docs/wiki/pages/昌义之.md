# 昌义之

昌义之，字叔远，历阳乌江人。南朝梁徐州刺史，以守钟离拒魏功封侯。

徐州刺史历阳昌义之与魏平南将军陈伯之战于梁城，义之败绩（146-044）。北徐州刺史昌义之拔梁城（146-055）。

昌义之为萧衍帐下将领，从萧衍起兵。梁武帝时，昌义之历任北徐州、南徐州刺史，镇守边境。南梁北伐时，义之与魏军交战有胜有负。钟离之战中，昌义之率三千人固守钟离，魏军数十万围攻，百计不能克，义之坚守待援，为韦睿等援军至大破魏军创造了条件。义之以勇毅善守著称。

> 乙丑，徐州刺史历阳昌义之与魏平南将军陈伯之战于梁城，义之败绩。
> （146-044）
