# 崔宏

崔宏，字玄伯，[[清河]]东武城人。[[北魏]][[吏部]][[尚书]]，佐道[[武帝]]定制度。

崔宏为[[北魏]][[吏部]][[尚书]]，典掌机要（110-095）。崔宏劝道[[武帝]]立[[学校]]（111-032）。

崔宏少俊迈，有俊才。初仕[[后燕]]，后归[[北魏]]。道[[武帝]]闻其名，以为[[吏部]][[尚书]]，命与周澹、[[晁崇]]等共定官制、[[律令]]。宏博通[[古今]]，于制度尤为精详。道[[武帝]]每有大事，必咨宏而后行。宏性[[俭约]]，不治产业，卒赠[[司空]]。[[崔浩]]即其子。

> 崔宏为北魏吏部尚书，典掌机要。
> （110-095）
