# 北周武帝

**北周武帝**宇文邕（543—578），[[北周]]第三位[[皇帝]]，[[宇文泰]]第四子。在位期间励精图治，灭[[北齐]]统一北方，为后来[[隋_(_朝代)|隋]]统一全国奠定基础。

[[资治通鉴概述|通鉴]]载周主邕勤于政事，亲自耕[[籍田]]（170-008），尊母为[[太后]]（170-021）。周主祀[[南郊]]、如武功以巡视[[疆土]]（170-048、170-051）。

## 灭齐与改革

[[北周]][[亲迎]][[突厥]]木杆[[可汗]]女为[[皇后]]，与[[突厥]][[和亲]]以稳定北方（170-052）。周主勤勉[[治国]]，享[[太庙]]、如[[醴泉]]宫、巡岐阳（170-058至170-081）。宇文邕在位期间推行[[府兵制]]改革、灭佛运动，增强国力，最终于建德六年灭[[北齐]]，统一北方。惜英年早逝，未能完成统一[[天下]]之大业。

## 武帝政略

宇文邕以英武之姿承[[宇文泰]]基业，内修政理，外结[[突厥]]，志在混一[[四海]]。其勤政见于[[资治通鉴概述|通鉴]]所载：耕[[籍田]]以[[劝农]]（170-008），祀[[南郊]]以敬天，巡武功以固边防（170-048）。[[亲迎]][[突厥]][[皇后]]以[[和戎]]（170-052），享[[太庙]]以修礼（170-058），巡[[醴泉]]以观风（170-081）。[[北齐]][[刘禅|后主]][[高纬]]昏暴失政，宇文邕乘间伐之，一举克[[邺城|邺]]，齐主奔[[青州]]被擒（173-004），[[北齐]]遂亡。惜邕在位仅六年即病殂，其子[[宇文赟|宣帝]]荒淫，[[北周]]国祚遂为[[杨坚]]所夺。宇文邕若得享天年，以周之强乘势平[[南陈]]，[[天下]]统一或不必待[[隋_(朝代)|隋]]。史家每以邕之早逝为憾，谓其"功成身殒，命也夫"。
