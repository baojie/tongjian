# 柏梁台

柏梁台是[[汉武帝]][[元鼎]]二年（前115年）筑以承露求仙的高台，以柏木为梁得名。

[[武帝]]起柏梁台，作承露盘高二十丈、大七围，以铜为之。上有仙人掌以承露，和玉屑饮之，云可以[[长生]]（020-028）。[[武帝]]东至海上求神者莫验，乙酉，柏梁台灾（021-046）。柏梁台[[火灾]]后，[[武帝]]听[[越人]]勇之之言更大规模地兴建了[[建章宫]]。

柏梁台在[[文学]]史上亦有重要地位，相传[[武帝]]与群臣在柏梁台联句赋诗，每人一句，句句押韵。此体后被称作"柏梁体"，是七言古诗的重要源头之一。

## 柏梁体

柏梁台联句之体相传始于[[武帝]]与群臣赋诗台上，人各一句，每句用韵，后世称为柏梁体。此体在[[南北朝]]隋唐间颇为流行，[[萧衍|梁武帝]]、唐[[汉宣帝|中宗]]、[[唐德宗]]等皆有君臣联句之作。柏梁体对七言诗的发展产生过重要影响，然其[[真伪]]历有争议，学者多疑传世柏梁诗为后人拟作。
