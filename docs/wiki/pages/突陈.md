# 突陈

**突陈**指冲入敌军阵中突[[破敌]]阵，为古代[[骁勇]]战将以个人武力或[[精锐]]小队冲[[破敌]]阵的重要战术手段。

## 骁将突陈

[[北宫]]纯募[[勇士]]百馀人突陈，弥兵大败（086-097）。峻乃舍其众，与数骑北下突陈，不得入（094-046）。秦[[英布|淮南王]]生[[单骑]]突陈，[[出入]]以十数，杀伤晋[[将士]]甚众（099-109）。安都怒，脱[[兜鍪]]，解铠，[[单骑]]突陈，所向无前（125-090）。直阁[[将军]]席豪发愤突陈而死（144-052）。
