# 柳裘

柳裘，[[北周]]御饰[[银青光禄大夫|大夫]]，与[[刘昉]]、[[郑译]]共谋引[[隋文帝|杨坚]][[辅政]]，入隋为佐命[[功臣]]。

御饰[[银青光禄大夫|大夫]]柳裘与[[刘昉]]、[[郑译]]等谋引[[隋文帝|杨坚]][[辅政]]（174-018）。坚使[[内史]][[银青光禄大夫|大夫]]柳裘诣穆，为陈[[利害]]（174-038）。[[隋文帝]]曰："微[[刘昉]]、[[郑译]]、卢贲、柳裘、皇甫绩等，则我不至此。然此等皆反覆子也。"（178-030）。

## 世系

柳裘（？—约590年），[[北周]][[御饰]][[大夫]]。其先为[[河东柳氏]]。裘与[[刘昉]]、[[郑译]]等谋引[[杨坚]][[辅政]]（174-018），入[[隋]]为佐命[[功臣]]。[[隋文帝]]曰："微[[刘昉]]、[[郑译]]、卢贲、柳裘、皇甫绩等，则我不至此"（178-030）。
