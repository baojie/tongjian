# 夜郎

夜郎为西南夷古国，位于今贵州西部。[[汉武帝]]遣[[唐蒙]]通夜郎，置犍为郡（017-036）。夜郎王曾问汉使"汉孰与我大"，此即"夜郎自大"典故所出。[[司马相如]]等通西南夷，蜀地吏民多以为不便（017-045）。

汉成帝时夜郎王兴举兵反，[[陈立]]为牂柯太守，诛兴而平其乱（029-069）。夜郎自[[西汉]]纳入版图后，逐渐融入华夏体系，但其独特的地域文化特色延续至今。

## 通夜郎

[[唐蒙]]风晓[[南越]]，知蜀枸酱出夜郎，有牂柯江可通番禺（018-022）。蒙乃[[上书]]请通夜郎，以精兵从夜郎浮江而下制越。武帝拜蒙为中郎将，遂见夜郎侯多同，约为置吏（018-023）。汉通西南夷后，谏者以为徇西夷朝夜郎为穷兵之祸（018-055）。后罢西夷，独置南夷、夜郎两县一都尉（018-086）。夜郎侯自以为道远不知汉广大，问汉孰与我大（019-040）。
