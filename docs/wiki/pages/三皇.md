# 三皇

三皇是中国古代传说中的三位远古帝王，具体所指说法不一。通常指**天皇**、**地皇**、**人皇**（或称**燧人氏**、**伏羲氏**、**神农氏**），代表华夏文明从火食、渔猎到农耕的三个起源阶段。

## 通鉴中的三皇

[[秦始皇]]统一六国后，"自以为德兼三皇，功过五帝，乃更号曰'皇帝'"（007-031）。[[陆贾]]称[[汉高祖刘邦|汉高祖]]"继五帝、三皇之业，统理中国"（012-037）。[[后汉]]时[[梁冀]]一门，"前后七侯，三皇后，六贵人"（054-026），"三皇"在此为"三任皇后"之省。[[北魏]]节闵帝诏书追溯古制，以"三皇称'皇'，五帝称'帝'，三代称'王'"（155-007），为三皇五帝三代之等级序列。

## 历史地位

三皇是华夏文明的神话源头，虽不见于信史，但作为文化符号贯穿中国古代政治哲学，皇帝称号即源于"德兼三皇、功过五帝"之语。
