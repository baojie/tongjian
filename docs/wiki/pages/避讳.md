# 避讳

避讳是古代对君主、尊长之名避而不称的礼制，秦汉至唐严格执行，违反者可能获罪。

[[西汉]][[杜钦]]批评大将军[[王凤]]以燕王、盖主之事疑上心，避讳[[吕后]]、[[霍光]]而不肯称，内有管蔡之萌、外假周公之论（030-082）。南北朝梁代[[徐勉]]该综百氏，皆为避讳，为人严谨（146-104）。避讳制度影响深远，古代文献因避讳而改字、改名、改官名者不可胜数，如汉避讳改"邦"为"国"，唐避讳改"民部"为"户部"等。

## 避讳之例

秦汉避讳最严。[[秦始皇]]名政，正月改称端月；汉[[汉高祖刘邦|高祖]]名邦，改邦为国；[[隋文帝|文帝]]名恒，改恒山为常山；武帝名彻，改彻侯为通侯。[[东汉]]以后避讳制度日渐严密，至唐代尤甚。[[唐太宗]]名世民，故改"世"为"代"、改"民"为"人"；高宗名治，改"治"为"理"。避讳之学遂成为读古书之一大关隘，历代学者多有考辨。
