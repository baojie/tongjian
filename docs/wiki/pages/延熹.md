# 延熹

**延熹**（158—167），[[汉桓帝]]第六个[[年号]]。延熹年间，[[宦官]]擅权，[[李膺]]、[[陈蕃]]等[[名士]]与[[太学]]生[[郭泰]]、[[贾彪]]等交结，抨击[[宦官]]，号为"党人"。[[张奂]]征[[匈奴]]，[[段颎]]讨[[东羌]]，皆建[[功勋]]。延熹九年，[[党锢之祸]]起，[[李膺]]等二百余人被诬为党人，下狱[[禁锢]]。
