# 九原

九原为[[战国]]赵地，[[赵武灵王]][[胡服]][[骑射]]后拓边设九原（003-050）。秦灭六国后置九原郡，修筑[[直道]]自云阳至九原（007-030）。[[蒙恬]]北逐[[匈奴]]，以九原为防胡前线（007-041）。

[[汉武帝]]时九原为[[朔方]][[刺史]]部辖地（020-045）。九原地处河套地区，是[[中原]]王朝与游牧民族对峙的前沿，秦汉经营北疆的战略要地。

## 九原军事

[[张仪]]说燕王，言秦下甲[[云中]]、九原，驱赵而攻燕（003-037）。[[赵武灵王]]欲自[[云中]]、九原南袭[[咸阳]]（003-083）。[[蒙恬]]除[[直道]]，道九原抵云阳，堑山堙谷千八百馀里（007-060）。始皇崩，辒车从[[井陉]]抵九原，从[[直道]]至[[咸阳]]发丧（007-072）。武帝遣[[公孙贺]]出九原二千馀里至浮沮井而还（020-078）。九原为秦汉北征之始途，凡出塞击胡，多以此为发兵之地。
