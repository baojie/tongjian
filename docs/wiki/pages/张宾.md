# 张宾

张宾，字孟孙，赵郡南和人。十六国后赵谋主，石勒军师，常自比张良。

石勒以张宾为谋主，集衣冠人物别为君子营（087-012）。张宾好读书，阔达有大志，常自比张子房（087-013）。石勒谋保据江汉，张宾以为不可（087-061）。

张宾初为石勒参军都尉，及勒僭号，拜大执法。宾机不虚发，算无遗策，勒甚重之。每事必咨而后行，宾亦竭诚尽节，佐勒定河北，灭前赵，称霸中原。宾卒，勒哭之恸，曰："天不欲吾成事耶？何夺吾右侯之早也！"石勒之兴，宾功最多，时人谓之"张子房"再出。

> 汉安东大将军石勒寇巨鹿、常山，众至十馀万，集衣冠人物，别为君子营。以赵郡张宾为谋主。
> （087-012）
