# 奄蔡

奄蔡是中亚游牧古国，位于里海沿岸，为[[西汉|汉朝]]通[[西域]]所及最西的诸国之一。[[张骞]][[出使]][[西域]]，言奄蔡"在[[康居]]西北可二千里，行国，与[[康居]]大同俗"，"控弦者十馀万"（019-039）。[[西域]]北道"西逾葱岭则出[[大宛]]、[[康居]]、奄蔡焉"（020-036）。

奄蔡在汉使足迹所至的最西端，与[[罗马帝国]]（[[秦朝|大秦]]）已相去不远。奄蔡在后世史料中亦名阿兰聊，为东西文明交流的最远端之一。汉与奄蔡的交往标志着丝绸之路向西延伸至里海沿岸，具有重要的文明史意义。
