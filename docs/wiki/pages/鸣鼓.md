# 鸣鼓

**鸣鼓**指击鼓发出信号，为古代军队进攻、示威或传递[[军令]]的重要手段。

## 鸣鼓进军

[[周亚夫]]出[[武关]]击鸣鼓，[[诸侯]]闻之以为[[将军]]从天而下（016-018）。光武令盆子[[君臣]]列观，[[勒兵]]鸣鼓相攻[[决胜]]负（041-007）。[[公孙述]]募[[敢死]]士鸣鼓[[挑战]]潜遣[[奇兵]]（043-007）。

## 鸣鼓攻敌

[[班超]]令十人持鼓藏虏舍后，见火然皆当鸣鼓大呼（045-069）。汉诏遣兵屯[[河内]]冲要皆设鸣鼓以备羌寇（049-129）。
