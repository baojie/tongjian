# 晋安

两晋[[南朝]]郡名，属[[扬州]]，治今福建省福州市一带。

## 两晋

[[苻雄]][[葬礼]]依晋安平献王故事（099-120）。晋安为[[东南]]沿海重要郡治。

## 南朝

[[刘裕]]追[[卢循]]至晋安，屡破之（113-020）。[[刘义隆|宋文帝]]时，晋安襄侯刘道产为[[雍州]][[刺史]]，善为政（124-016）。[[刘骏|孝武帝]]立皇子子勋为晋安王（129-040）。

## 郡望

晋安在南北朝时已为[[东南]]文化重镇，隋唐以后日盛。
