# 王坦之

王坦之，字文度，[[晋阳|太原]][[晋阳]]人。东晋[[侍中]]，与[[谢安]]齐名。以毁[[桓温]]居摄九锡之诏著称。

温以抚军司马王坦之为长史。坦之，述之子也（101-060）。[[谢安]]尝与左卫将军王坦之共诣[[郗超]]，日旰未得前，坦之欲去，安曰："独不能为性命忍须臾邪？"（103-035）

王坦之与[[谢安]]并为孝武帝朝重臣。[[桓温]]病重时讽求九锡，坦之与[[谢安]]共缓其事，及温死，锡命遂寝。坦之为人刚正，与[[谢安]]之宽和相济。时人谓[[谢安]]能容、王坦之能断。坦之著有《废庄论》，反对当时[[清谈]]虚浮之风。
