# 王坦之

王坦之，字文度，太原晋阳人。东晋侍中，与谢安齐名。以毁桓温居摄九锡之诏著称。

温以抚军司马王坦之为长史。坦之，述之子也（101-060）。谢安尝与左卫将军王坦之共诣郗超，日旰未得前，坦之欲去，安曰："独不能为性命忍须臾邪？"（103-035）

王坦之与谢安并为孝武帝朝重臣。桓温病重时讽求九锡，坦之与谢安共缓其事，及温死，锡命遂寝。坦之为人刚正，与谢安之宽和相济。时人谓谢安能容、王坦之能断。坦之著有《废庄论》，反对当时清谈虚浮之风。
