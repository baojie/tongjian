# 玺绶

玺绶是官印及其系绶带的总称。汉制官秩高低以玺绶材质颜色严格区别。秦王子婴"奉天子玺绶"降[[刘邦]]（012-037），以示亡国。[[汉武帝]]时列侯"金印紫绶"，二千石"银印青绶"（018-027）。

[[王莽]]即位时逼太后王政君交出传国玺，政君"投之地"，玺遂缺一角（022-049）。玺绶制度是古代官阶的外在标志，其授予与收缴标志着官职的任命与罢免，在官僚制度中具有重要的符号意义。
