# 周举

周举，[[东汉]]名臣。[[《资治通鉴》]]载其与[[杜乔]]等分行[[州郡]]，数[[进谏]]言，以[[正直]]闻名。

[[陈蕃]]及同郡周举常相谓曰："时月之间不见黄生，则鄙吝之萌复存乎心矣"（050-113）。[[司徒]]掾[[汝南]]周举谓[[李郃]]曰："昔瞽瞍常欲杀舜，舜事之逾谨；郑武姜[[谋杀]]庄公，庄公誓之黄泉，[[秦始皇]]怨母失行…"（051-022）。[[司徒]]掾周举说[[硃伥]]曰："[[朝廷]]在西钟下时，非[[孙程]]等岂立！今忘其大德，录其小过。如[[道路]][[夭折]]，帝有杀[[功臣]]之讥"（051-037）。

## 荐贤举能

周举才学优深，[[汉顺帝|顺帝]]特加[[策问]]，举对以[[阴阳]]闭隔之道（052-004）。[[尚书令]][[左雄]]荐周举为[[尚书]]，后[[举劾]][[左雄]]所荐冯直，不避举主之嫌，时人服其公（052-044）。[[梁商]]大会[[宾客]]，酒阑歌《薤露》，周举叹以为[[哀乐]]失时（052-070）。商[[病笃]]，谓帝曰："臣[[从事中郎]]周举，清高忠正，可重任也。"由是拜[[谏议大夫]]（052-078）。周举以[[清正]]称，为[[东汉]]中世名臣。
