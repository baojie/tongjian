# 周举

周举，[[东汉]]名臣。[[《资治通鉴》]]载其与[[杜乔]]等分行州郡，数进谏言，以正直闻名。

[[陈蕃]]及同郡周举常相谓曰："时月之间不见黄生，则鄙吝之萌复存乎心矣"（050-113）。[[司徒]]掾汝南周举谓[[李郃]]曰："昔瞽瞍常欲杀舜，舜事之逾谨；郑武姜谋杀庄公，庄公誓之黄泉，[[秦始皇]]怨母失行…"（051-022）。[[司徒]]掾周举说[[硃伥]]曰："朝廷在西钟下时，非[[孙程]]等岂立！今忘其大德，录其小过。如道路夭折，帝有杀功臣之讥"（051-037）。
