# 桓范

桓范，[[三国]]魏[[大司农]]。[[曹爽]]兄弟出游，桓范曰："总万机，典禁兵，不宜并出。若有闭城门，谁复内入者？"爽不听（075-028）。及[[司马懿]]起兵，以[[太后]]令召范，范欲应之（075-037）。后与[[曹爽]]、[[何晏]]等同下狱，劾以大逆不道，夷三族（075-040）。

> 司农沛国桓范谓曰："总万机，典禁兵，不宜并出。若有闭城门，谁复内入者？"爽曰："谁敢尔邪！"
> （075-028）
