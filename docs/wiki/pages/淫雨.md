# 淫雨

**淫雨**指持续过久、过量降雨，常导致[[水灾]]泛滥、[[农业]]歉收，为古代常见的[[自然]]灾害之一。

## 淫雨为灾

青、冀之域淫雨漏河，徐、岱之滨海水盆溢，兗、豫蝗蝝滋生（050-111）。慕容儁时淫雨百日，[[饥疫]]并臻（096-026）。[[前秦]]苻生时天降淫雨，七旬乃霁（097-083）。

## 淫雨妨战

[[后梁]][[梁太祖|朱温]]与晋争战，淫雨积旬，粮竭士病（269-045）。[[玄宗|唐玄宗]]时淫雨不已，[[高力士]]谓[[赏罚无章]]，[[阴阳]]失度（217-019）。
