# 陈宝应

陈宝应，晋安人。南朝陈闽州刺史，割据一方。

陈宝应在晋安，与熊昙朗、周迪、留异等共相连结（167-051）。上以陈宝应之父为光禄大夫，子女皆受封爵，命宗正编入属籍。而宝应以留异女为妻，阴与异合（168-120）。及留异为陈军所破，与子脱身奔晋安，依陈宝应（168-131）。宝应终为陈军所破。

> 时熊昙朗在豫章，周迪在临川，留异在东阳，陈宝应在晋安，共相连结。
> （167-051）
