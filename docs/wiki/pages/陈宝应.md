# 陈宝应

陈宝应，[[晋安]]人。[[南朝]]陈闽州[[刺史]]，[[割据]]一方。

陈宝应在[[晋安]]，与[[熊昙朗]]、[[周迪]]、[[留异]]等共相连结（167-051）。上以陈宝应之父为[[光禄大夫]]，子女皆受[[封爵]]，命[[宗正]]编入属籍。而宝应以[[留异]]女为妻，阴与异合（168-120）。及[[留异]]为陈军所破，与子脱身奔[[晋安]]，依陈宝应（168-131）。宝应终为陈军所破。

> 时熊昙朗在豫章，周迪在临川，留异在东阳，陈宝应在晋安，共相连结。
> （167-051）
