# [[显州]]

[[显州]]是[[东魏]]、[[北齐]]、[[隋]]时期设置的州，治所在今[[山西]]孝义一带。[[东魏]]时，[[慕容绍宗]]与[[侯景]]交战，[[显州]]刺史[[张遵业]]为侯景所擒（160-058）。

[[北齐]]时，[[文宣帝]]自[[离石]]道讨[[山胡]]，遣[[斛律金]]从显州道，[[常山王]][[高演]]从[[晋州]]道夹攻，大破之（165-043）。显州在[[北朝]]时是防御[[山胡]]、[[稽胡]]的前沿州郡。
