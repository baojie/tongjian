# [[虞城]]

[[虞城]]是[[隋]][[唐]]置县，位于今[[河南]]虞城，属[[宋州]]（后隶[[汴州]]）。[[唐]]初，[[徐圆朗]]反[[唐]]，攻陷[[楚丘]]，引兵将围虞城（189-064）。[[杞州]]刺史[[李瑰]]遣部将[[崔枢]]、[[张公谨]]率众守虞城，崔枢以质子守城，徐圆朗等惧而不敢进。

此前徐圆朗厚礼[[李彦师]]，使其作书招降虞城。彦师为书与其弟曰："吾奉使无状，为贼所擒，为臣不忠，誓之以死；汝善侍老母，勿以吾为念"（189-063）。虞城遂坚守不下。
