# 上郡

上郡为[[战国]]魏置，[[秦惠文王]]十年取上郡（003-053）。[[秦始皇]]时[[蒙恬]]屯上郡以御[[匈奴]]（007-041）。秦末[[扶苏]]监[[蒙恬]]军于上郡（007-053），后为[[赵高]]、[[胡亥]]矫诏所杀。

[[汉武帝]]时[[李广]]、[[程不识]]分屯上郡各御[[匈奴]]（017-022）。上郡地处[[关中]]正北、[[黄河]]以西，为秦汉时期拱卫[[京师]]的重要屏障，亦为北方游[[牧民]]族南下的必经之地。

## 上郡沿革

魏筑[[长城]]自郑滨洛以北有上郡，与秦接界（002-016）。[[张仪]]说[[魏主|魏王]]，魏因尽入上郡十五县以谢秦（002-109）。秦灭义渠，于[[陇西]]、[[北地]]、上郡筑[[长城]]以拒胡（006-057）。秦灭赵，从[[晋阳|太原]]、上郡归（006-123）。始皇巡北边从上郡入，使[[蒙恬]]发兵三十万北击[[匈奴]]（007-051）。上郡自魏入秦至秦汉，[[始终]]为北疆要塞，历代重兵驻守。
