# 彭宠

**彭宠**，宛人，[[东汉]]初[[渔阳]][[太守]]。

[[王莽]]末，彭宠与[[吴汉]][[亡命]]在[[渔阳]]。乡人韩鸿为更始使，承制拜宠[[偏将军]]、行[[渔阳]][[太守]]事（039-025）。[[刘歆|刘秀]]经营[[河北]]，彭宠发[[渔阳]][[突骑]]助秀（039-037），为[[刘歆|刘秀]]统一[[河北]]的重要力量。

及[[刘歆|刘秀]][[称帝]]，彭宠自负功高，以赏薄[[怨望]]，遂反，自称[[燕王]]。后为[[苍头]]所杀。彭宠之叛为[[东汉]]初年功臣叛乱的典型案例，反映了[[刘歆|刘秀]]对功臣的猜防政策。

## 渔阳叛汉

彭宠发[[渔阳]][[突骑]]助[[刘歆|刘秀]]，[[刘歆|刘秀]]加宠[[大将军]]封[[列侯]]（039-045）。及帝[[称帝]]，宠自负功高，意望甚高，帝接之不能满其望（040-051）。帝遣游击[[将军]][[邓隆]]助[[朱浮]]讨宠，隆军潞南，浮军雍奴，相距百里。帝知其必败，后果如其言（040-064）。[[涿郡]][[张丰]]反，与彭宠[[连兵]]（041-012）。[[耿弇]]请北收[[上谷]]兵定彭宠（041-021）。后宠为[[苍头]]所杀。
