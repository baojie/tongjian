# 古弼

古弼，代郡人。[[北魏]]太武帝将领，安西将军，以忠谨恭勤为太武帝所重。

魏主以古弼等忠谨恭勤，使之给侍[[东宫]]，分典机要（119-074）。命古弼等镇抚阴山以北，弼抚诸部，收其[[贡赋]]，魏之民间马牛羊及氈皮为之价贱（121-067）。魏主使古弼等将兵趣安定，弼伪退以诱夏主，大破之（121-108）。后古弼与[[娥清]]将精骑一万伐燕，克白狼城（123-009, 123-011）。

古弼少忠谨，好读书，初为猎郎，使长安，称旨，迁门下奏事，以敏正著称。太武帝以其忠谨，命辅[[东宫]]。后以直谏忤旨，太武帝怒，弼神色自若，帝叹曰："刚直之臣，国之宝也。"弼历事太武、文成二朝，累迁[[侍中]]、吏部尚书，以清直终。

> 魏主又以典东西部刘絜、门下奏事代人古弼、直郎徒河卢鲁元忠谨恭勤，使之给侍东宫。
> （119-074）
