# 和睦

**和睦**指人际关系和政治局面的和好融洽，是[[儒家]]政治思想中的重要理念。《[[资治通鉴概述|通鉴]]》记载了诸多关于和睦的史事与议论。

## 汉代

[[汉文帝]]以和睦待[[匈奴]]，[[和亲]]修好，使[[边境]][[安宁]]（033-009）。[[汉宣帝]]时，[[匈奴]][[呼韩邪单于]]来朝，宣帝以殊礼待之，汉匈和睦数十年（055-018）。
