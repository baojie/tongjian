# 兼行

**兼行**指日夜不停地急速[[行军]]，为古代军队争取战略主动权的关键手段，常与[[倍道]]连用。

## 兼行救急

[[刘歆|刘秀]]晨夜兼行，蒙犯[[霜雪]]，面皆破裂（039-038）。[[岑彭]][[倍道]]兼行二千馀里，径拔[[武阳]]，所至皆奔散（042-072）。

## 兼行袭敌

[[虞诩]]日夜进道兼行百馀里，令吏士各作两灶，日增倍之，羌不敢逼（049-151）。[[段颎]]晨夜兼行，与羌战于奢延泽，连破之（056-027）。
