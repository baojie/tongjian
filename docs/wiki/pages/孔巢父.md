# 孔巢父

孔巢父，字弱翁，[[冀州]]人。唐[[给事中]]，以辩博著称。

以舒王谟为[[荆襄]]等道[[行营]]都[[元帅]]，右庶子孔巢父为左[[司马]]（228-030）。以[[给事中]]孔巢父为淄青宣慰使（229-032）。[[田悦]][[用兵]]数败，以[[给事中]]孔巢父为魏博宣慰使。巢父性辩博，至魏州，对其众为陈逆顺[[祸福]]，悦及将士皆喜（230-017）。
